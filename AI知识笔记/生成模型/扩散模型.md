# 扩散模型

## 一句话定义

> 扩散模型通过学习从纯噪声逐步去噪生成数据的过程来创建新样本。

## 核心公式

### 前向扩散过程（加噪）

$$
q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t I)
$$

### 反向生成过程（去噪）

$$
p_\theta(x_{t-1} | x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))
$$

### 简化目标

$$
\mathcal{L} = \mathbb{E}_{t, x_0, \epsilon} \left[ \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]
$$

其中 $\epsilon$ 是添加的噪声，$\epsilon_\theta$ 是模型预测的噪声。

## 详细说明

### 1. 扩散模型的核心思想

**两阶段过程：**

```
真实数据 x_0
    ↓ 加噪（T步）
噪声 x_T → 去噪（T步）→ 生成数据
```

**前向过程（训练时）：**
- 逐步添加高斯噪声
- 最终变成纯噪声

**反向过程（生成时）：**
- 学习从噪声恢复数据
- 逐步去噪直到生成样本

### 2. 为什么扩散模型火了？

| 优势 | 说明 |
|------|------|
| 训练稳定 | 不像GAN容易模式崩溃 |
| 生成质量高 | FID分数超越GAN |
| 可控生成 | Classifier-Free Guidance |
| 理论基础好 | 有概率解释 |

### 3. 经典模型

| 模型 | 年份 | 特点 |
|------|------|------|
| DDPM | 2020 | 基础扩散模型 |
| ADM | 2021 | Classifier Diffusion |
| GLIDE | 2022 | Text-to-Image |
| Stable Diffusion | 2022 | 开源，latent diffusion |
| DALL-E 3 | 2023 | GPT-4V字幕，生成质量高 |
| Sora | 2024 | 视频生成 |

### 4. Stable Diffusion 架构

```python
# Latent Diffusion Model 核心
class LatentDiffusion:
    def __init__(self):
        self.vae = VAE()           # 压缩图像到latent空间
        self.unet = UNet()        # 去噪网络
        self.clip = CLIP()        # 文本编码器

    @torch.no_grad()
    def generate(self, prompt, num_steps=50):
        # 文本编码
        text_emb = self.clip.encode(prompt)

        # 初始化噪声
        latents = torch.randn(1, 4, 64, 64)

        # 逐步去噪
        for t in reversed(range(num_steps)):
            noise_pred = self.unet(latents, t, text_emb)
            latents = self.scheduler.step(noise_pred, t, latents)

        # 解码到像素空间
        return self.vae.decode(latents)
```
