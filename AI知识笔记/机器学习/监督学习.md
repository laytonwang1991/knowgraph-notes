# 监督学习

## 一句话定义

> 监督学习是从带有正确答案（标签）的训练数据中学习模型，使其能够对新数据进行预测。

## 核心公式

### 经验风险最小化

$$
\min_{\theta} \frac{1}{N} \sum_{i=1}^{N} L(y_i, f(x_i; \theta))
$$

其中 $L$ 是损失函数，$f$ 是模型，$\theta$ 是参数。

## 详细说明

### 1. 基本流程

```
训练数据 (x, y)
    ↓
学习算法
    ↓
模型 f(x; θ)
    ↓
预测新数据 x' → y'
```

### 2. 主要任务

| 任务类型 | 输出 | 示例 |
|----------|------|------|
| 分类 | 离散标签 | 图像分类、垃圾邮件检测 |
| 回归 | 连续值 | 房价预测、股票价格 |

### 3. 常用算法

**传统机器学习：**
- 线性回归、逻辑回归
- 决策树、随机森林
- SVM（支持向量机）

**深度学习：**
- CNN（图像分类）
- RNN（序列预测）
- Transformer（NLP）

### 4. 代码示例

```python
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import train_test_split

# 划分训练集测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2
)

# 训练模型
model = LogisticRegression()
model.fit(X_train, y_train)

# 预测

```
