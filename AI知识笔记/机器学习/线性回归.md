# 线性回归（Linear Regression）

## 一句话定义

线性回归是一种通过拟合特征的线性组合来预测连续目标变量的监督学习算法，目标是最小化预测值与真实值之间的均方误差。

## 核心公式

**线性模型：**

$$f(x) = w^T x + b = \sum_{i=1}^{n} w_i x_i + b$$

**均方误差损失函数（MSE）：**

$$J(w) = \frac{1}{m} \sum_{i=1}^{m} (f(x^{(i)}) - y^{(i)})^2 = \frac{1}{m} \|Xw - y\|^2$$

**最小二乘法闭式解：**

$$w = (X^T X)^{-1} X^T y$$

**梯度下降更新规则：**

$$w_j := w_j - \alpha \frac{2}{m} \sum_{i=1}^{m} (f(x^{(i)}) - y^{(i)}) x_j^{(i)}$$

**正则化项：**

- L2正则化（Ridge）：$J(w) = \|Xw - y\|^2 + \lambda \|w\|_2^2$
- L1正则化（Lasso）：$J(w) = \|Xw - y\|^2 + \lambda \|w\|_1$

## 详细说明

### 1. 算法原理

- **线性假设**：假设目标变量与特征之间存在线性关系
- **损失函数**：使用均方误差（MSE）衡量预测误差
- **参数学习**：找到使MSE最小化的权重向量 $w$ 和偏置 $b$
- **可解释性**：权重 $w$ 直接反映了各特征对预测结果的贡献程度

### 2. 最小二乘法

**正规方程推导：**

对 MSE 损失求导并令其为零：

$$\frac{\partial J(w)}{\partial w} = 2 X^T (Xw - y) = 0$$

求解得：

$$w = (X^T X)^{-1} X^T y$$

**几何意义**：找到在特征空间中距离所有数据点平方距离和最小的超平面

### 3. 梯度下降法

当特征维度很高或样本量很大时，使用迭代优化：

- **批量梯度下降**：每次使用全部样本计算梯度
- **随机梯度下降（SGD）**：每次使用单个样本更新
- **小批量梯度下降**：每次使用一小批样本

### 4. 正则化

| 类型 | 公式 | 特点 |
|------|------|------|
| **Ridge（L2）** | $\lambda \sum w_i^2$ | 收缩权重，保留所有特征 |
| **Lasso（L1）** | $\lambda \sum |w_i|$ | 产生稀疏解，特征选择 |
| **Elastic Net** | 组合L1和L2 | 结合两者优点 |

### 5. 过拟合处理

- **增加正则化项**：L1/L2正则化约束模型复杂度
- **增加训练数据**：更多样本有助于学到真正的模式
- **特征选择**：移除不相关或冗余特征
- **交叉验证**：选择最佳正则化参数

### 6. 优缺点

**优点：**
- 简单易懂，模型可解释性强
- 训练和预测速度快
- 可处理大规模数据
- 能捕捉特征与目标的线性关系

**缺点：**
- 只能建模线性关系
- 对异常值敏感
- 特征需要预处理（标准化）
- 无法自动进行特征选择（无正则化时）

## 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression, Ridge, Lasso, ElasticNet
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.metrics import mean_squared_error, r2_score
from sklearn.preprocessing import StandardScaler, PolynomialFeatures

# 生成示例数据（带噪声的线性数据）
np.random.seed(42)
X = 2 * np.random.rand(100, 1)
y = 4 + 3 * X + np.random.randn(100, 1)  # y = 4 + 3x + noise

# 划分数据集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 标准线性回归
lin_reg = LinearRegression()
lin_reg.fit(X_train, y_train)

# 预测
y_train_pred = lin_reg.predict(X_train)
y_test_pred = lin_reg.predict(X_test)

# 评估
print(f"训练集 MSE: {mean_squared_error(y_train, y_train_pred):.4f}")
print(f"测试集 MSE: {mean_squared_error(y_test, y_test_pred):.4f}")
print(f"训练集 R²: {r2_score(y_train, y_train_pred):.4f}")
print(f"测试集 R²: {r2_score(y_test, y_test_pred):.4f}")
print(f"权重 w: {lin_reg.coef_[0][0]:.4f}, 偏置 b: {lin_reg.intercept_[0]:.4f}")

# 可视化
plt.scatter(X_test, y_test, color='blue', alpha=0.5, label='实际值')
plt.plot(X_test, y_test_pred, color='red', linewidth=2, label='预测值')
plt.xlabel('X')
plt.ylabel('y')
plt.legend()
plt.title('线性回归拟合')
plt.show()
```

```python
# 从零实现线性回归
class LinearRegressionManual:
    def __init__(self, learning_rate=0.01, n_iterations=1000):
        self.lr = learning_rate
        self.n_iterations = n_iterations
        self.weights = None
        self.bias = None

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0

        # 梯度下降
        for _ in range(self.n_iterations):
            y_pred = np.dot(X, self.weights) + self.bias

            # 计算梯度
            dw = (2 / n_samples) * np.dot(X.T, (y_pred - y))
            db = (2 / n_samples) * np.sum(y_pred - y)

            # 更新参数
            self.weights -= self.lr * dw
            self.bias -= self.lr * db

    def predict(self, X):
        return np.dot(X, self.weights) + self.bias


# 最小二乘法闭式解实现
class LinearRegressionClosedForm:
    def fit(self, X, y):
        # 添加偏置项
        X_b = np.c_[np.ones((X.shape[0], 1)), X]
        # 正规方程: w = (X^T X)^(-1) X^T y
        self.theta = np.linalg.inv(X_b.T @ X_b) @ X_b.T @ y
        self.bias = self.theta[0]
        self.weights = self.theta[1:]

    def predict(self, X):
        return np.dot(X, self.weights) + self.bias
```

```python
# 展示正则化效果
np.random.seed(42)
X = np.random.rand(100, 1)
y = 3 * X + 1.5 + np.random.randn(100, 1) * 0.3

# 原始数据
plt.figure(figsize=(12, 4))

alphas = [0, 0.1, 1, 10]
for i, alpha in enumerate(alphas):
    ridge = Ridge(alpha=alpha)
    ridge.fit(X, y)
    plt.subplot(1, 4, i+1)
    plt.scatter(X, y, alpha=0.5)
    plt.plot(X, ridge.predict(X), color='red', linewidth=2)
    plt.title(f'Ridge (alpha={alpha})\nw={ridge.coef_[0][0]:.2f}')
    plt.ylim(0, 6)

plt.tight_layout()
plt.show()
```

## 应用场景

1. **趋势预测**：
   - 房价趋势预测
   - 销售数据趋势分析
   - 经济增长预测

2. **因果分析**：
   - 广告投入与销售额的关系
   - 药物剂量与疗效关系
   - 环境影响因素分析

3. **风险评估**：
   - 信用评分模型
   - 保险定价
   - 财务风险度量

4. **数据预处理**：
   - 作为基线模型（Baseline）
   - 特征重要性分析
   - 异常值检测（残差分析）

## 相关概念

- **最小二乘法**：通过最小化平方误差来估计参数的方法
- **梯度下降**：迭代优化算法，沿梯度负方向更新参数
- **正则化**：添加约束防止过拟合的技术
- **过拟合**：模型在训练集上表现好但在测试集上表现差
- **MSE损失**：均方误差，常用的回归损失函数
- **Ridge回归**：L2正则化的线性回归
- **Lasso回归**：L1正则化的线性回归，可做特征选择
