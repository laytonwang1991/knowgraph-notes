# 强化学习

## 一句话定义

> 强化学习是智能体通过与环境进行试错交互，以最大化累积奖励为目标学习最优策略的机器学习方法。

## 核心公式

### 累计奖励

$$
G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}
$$

其中 $\gamma \in [0, 1]$ 是折扣因子。

### Q-Learning 更新规则

$$
Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)]
$$

## 详细说明

### 1. 强化学习基本框架

**智能体（Agent）与环境（Environment）交互：**

```
智能体
  ↓ 动作 a
环境 → 状态 s' + 奖励 r
  ↑
  观察 o
```

**MDP（马尔可夫决策过程）：**
- 状态空间 $\mathcal{S}$
- 动作空间 $\mathcal{A}$
- 转移概率 $P(s'|s, a)$
- 奖励函数 $R(s, a, s')$
- 折扣因子 $\gamma$

### 2. 核心概念

| 概念 | 说明 |
|------|------|
| 策略 $\pi$ | 从状态到动作的映射 |
| 价值函数 $V$ | 状态的长期价值 |
| Q函数 $Q$ | 状态-动作对的长期价值 |
| 探索/利用 | 探索新动作 vs 利用已知最优 |

### 3. 经典算法

**无模型方法：**
- Q-Learning：离散动作空间
- SARSA：on-policy学习
- DQN：深度Q网络
- Policy Gradient：直接优化策略

**Actor-Critic：**
- A2C/A3C：异步优势Actor-Critic
- PPO：近端策略优化
- DDPG：深度确定性策略梯度

### 4. 代码示例

```python
import numpy as np

class QLearningAgent:
    def __init__(self, n_states, n_actions, alpha=0.1, gamma=0.95, epsilon=0.1):
        self.q_table = np.zeros((n_states, n_actions))
        self.alpha = alpha      # 学习率
        self.gamma = gamma      # 折扣因子
        self.epsilon = epsilon  # 探索率

    def choose_action(self, state):
        if np.random.random() < self.epsilon:
            return np.random.randint(len(self.q_table[state]))
        return np.argmax(self.q_table[state])

    def update(self, state, action, reward, next_state):
        best_next_q = np.max(self.q_table[next_state])
        td_target = reward + self.gamma * best_next_q
        td_error = td_target - self.q_table[state, action]
        self.q_table[state, action] += self.alpha * td_error
```
