# 聚类

## 一句话定义

> 聚类是将数据集中的样本按照相似性自动分组，使得同组样本相似度高、不同组样本差异大的无监督学习任务。

## 核心公式

### K-means 目标函数

$$
\min_{C} \sum_{i=1}^{k} \sum_{x \in C_i} \| x - \mu_i \|^2
$$

其中 $\mu_i$ 是簇 $C_i$ 的中心（质心）。

### 层次聚类距离度量

| 距离类型 | 公式 |
|----------|------|
| 最近距离 | $d_{min}(C_i, C_j) = \min_{x \in C_i, y \in C_j} \| x - y \|$ |
| 最远距离 | $d_{max}(C_i, C_j) = \max_{x \in C_i, y \in C_j} \| x - y \|$ |
| 平均距离 | $d_{avg}(C_i, C_j) = \frac{1}{|C_i||C_j|} \sum_{x \in C_i} \sum_{y \in C_j} \| x - y \|$ |

### DBSCAN 核心公式

- **核心点**：邻域 $\epsilon$ 内至少包含 $minPts$ 个点
- **密度直达**：点 $p$ 在点 $q$ 的 $\epsilon$ 邻域内，且 $q$ 是核心点
- **密度可达**：存在核心点链 $p \rightarrow q_1 \rightarrow ... \rightarrow q$

## 详细说明

### 1. 主要聚类算法

```
聚类算法
    ├── 划分方法（K-means, K-medoids）
    ├── 层次方法（AGNES, DIANA）
    ├── 密度方法（DBSCAN, OPTICS）
    └── 网格方法（STING, CLIQUE）
```

### 2. K-means 算法流程

```
1. 随机选择K个初始质心
2. 分配：每个样本分配到最近的质心
3. 更新：重新计算各簇质心
4. 重复2-3直到收敛或达到最大迭代
```

### 3. 算法对比

| 算法 | 时间复杂度 | 优点 | 缺点 |
|------|------------|------|------|
| K-means | O(nki) | 简单高效 | 需预设K，对噪声敏感 |
| 层次聚类 | O(n²) | 无需预设K，可得到层次结构 | 计算复杂度高 |
| DBSCAN | O(n²) | 可发现任意形状簇，处理噪声 | 参数敏感 |

### 4. 聚类评估指标

| 指标 | 说明 |
|------|------|
| 轮廓系数 | $[-1, 1]$，越大越好 |
| CH指数 | 越大越好 |
| DB指数 | 越小越好 |
| 类内距离 | 越小越好 |
| 类间距离 | 越大越好 |

### 5. 代码示例

```python
from sklearn.cluster import KMeans, AgglomerativeClustering, DBSCAN
from sklearn.preprocessing import StandardScaler

# 数据标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# K-means聚类
kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
kmeans_labels = kmeans.fit_predict(X_scaled)

# 层次聚类
hier = AgglomerativeClustering(n_clusters=3)
hier_labels = hier.fit_predict(X_scaled)

# DBSCAN
dbscan = DBSCAN(eps=0.5, min_samples=5)
dbscan_labels = dbscan.fit_predict(X_scaled)

# 确定最优K值（肘部法则）
inertias = []
K_range = range(1, 11)
for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_scaled)
    inertias.append(kmeans.inertia_)
```

### 6. 应用场景

- **客户细分**：将客户按行为特征分组
- **图像分割**：将图像像素聚类
- **异常检测**：将正常数据聚类，孤立点为异常
- **文档归类**：将文档按主题聚类
