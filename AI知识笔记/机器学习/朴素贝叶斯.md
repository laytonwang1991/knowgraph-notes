# 朴素贝叶斯（Naive Bayes）

## 一句话定义

朴素贝叶斯是一种基于贝叶斯定理的概率分类算法，它假设特征之间相互条件独立，以此简化后验概率的计算。

## 核心公式

**贝叶斯定理：**

$$P(Y|X) = \frac{P(X|Y) \cdot P(Y)}{P(X)}$$

**朴素贝叶斯分类决策：**

$$\hat{Y} = \arg\max_{y} P(Y=y) \prod_{i=1}^{n} P(X_i | Y=y)$$

**拉普拉斯平滑：**

$$P(X_i = x | Y = y) = \frac{\text{count}(X_i = x, Y = y) + \alpha}{\text{count}(Y = y) + \alpha \cdot |V_i|}$$

## 详细说明

### 1. 算法原理

- **贝叶斯定理**：核心思想是通过先验概率和似然函数计算后验概率
- **条件独立假设**：假设给定类别标签后，所有特征相互条件独立，这是"朴素"一词的来源
- **生成模型**：朴素贝叶斯是生成模型，学习的是特征在类别条件下的分布

### 2. 三种常见变体

| 类型 | 适用场景 | 特征分布 |
|------|----------|----------|
| **多项式朴素贝叶斯** | 文本分类、词频统计 | 多项式分布 |
| **高斯朴素贝叶斯** | 连续特征分类 | 正态分布 |
| **伯努利朴素贝叶斯** | 二值特征分类 | 伯努利分布 |

### 3. 拉普拉斯平滑

- 用于解决零概率问题（某特征值在某类别下从未出现）
- 常用取值为 $\alpha = 1$（也称为拉普拉斯估计）
- 避免因训练数据稀疏导致的概率为0

### 4. 优缺点

**优点：**
- 训练和预测速度快，时间复杂度为 $O(n \cdot d)$
- 对小规模数据效果好
- 天然支持多分类
- 对特征工程不敏感

**缺点：**
- 条件独立假设往往与现实不符
- 对特征分布的假设可能不准确
- 需要解决零概率问题

## 代码示例

```python
import numpy as np
from sklearn.naive_bayes import GaussianNB, MultinomialNB, BernoulliNB
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report

# 高斯朴素贝叶斯示例
X = np.array([[1, 2], [1, 4], [2, 1], [3, 5], [3, 7], [4, 3]])
y = np.array([0, 0, 0, 1, 1, 1])

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 创建高斯朴素贝叶斯分类器
gnb = GaussianNB()

# 训练模型
gnb.fit(X_train, y_train)

# 预测
y_pred = gnb.predict(X_test)

# 评估
print(f"准确率: {accuracy_score(y_test, y_pred):.2f}")
print(f"预测概率:\n{gnb.predict_proba(X_test)}")
```

```python
# 从零实现朴素贝叶斯（高斯）
class NaiveBayes:
    def fit(self, X, y):
        self.classes = np.unique(y)
        self.parameters = {}

        for c in self.classes:
            X_c = X[y == c]
            self.parameters[c] = {
                'mean': X_c.mean(axis=0),
                'var': X_c.var(axis=0) + 1e-9,  # 防止方差为0
                'prior': len(X_c) / len(X)
            }

    def _gaussian_pdf(self, x, mean, var):
        coef = 1 / np.sqrt(2 * np.pi * var)
        exponent = -0.5 * ((x - mean) ** 2 / var)
        return coef * np.exp(exponent)

    def predict(self, X):
        predictions = []
        for x in X:
            posteriors = []
            for c in self.classes:
                prior = self.parameters[c]['prior']
                likelihood = np.prod(
                    self._gaussian_pdf(x,
                                       self.parameters[c]['mean'],
                                       self.parameters[c]['var'])
                )
                posteriors.append(prior * likelihood)
            predictions.append(self.classes[np.argmax(posteriors)])
        return np.array(predictions)
```

## 应用场景

1. **垃圾邮件检测**：文本分类的经典应用，根据邮件内容判断是否为垃圾邮件
2. **文本分类**：新闻主题分类、情感分析、文档分类
3. **推荐系统**：协同过滤中用于估计用户偏好概率
4. **医学诊断**：根据症状推断疾病概率
5. **实时分类系统**：需要快速响应的在线分类任务

## 相关概念

- **贝叶斯定理**：描述条件概率关系的定理，是算法的数学基础
- **条件概率**：在某事件发生的条件下，另一事件发生的概率
- **生成模型**：与判别模型（如逻辑回归、SVM）相对，学习数据的生成过程
- **拉普拉斯平滑**：解决零概率问题的平滑技术
- **先验概率**：根据以往数据分析得到的概率
- **后验概率**：得到某结果后，根据新信息修正的概率
