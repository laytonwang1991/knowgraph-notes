# 无监督学习

## 一句话定义

> 无监督学习是在没有标签指导的情况下，从数据中发现隐藏模式和信息结构的机器学习方法。

## 核心公式

### K-Means 目标函数

$$
\min \sum_{i=1}^{k} \sum_{x \in S_i} \|x - \mu_i\|^2
$$

其中 $\mu_i$ 是簇 $S_i$ 的中心点。

## 详细说明

### 1. 主要任务

| 任务 | 说明 | 算法 |
|------|------|------|
| 聚类 | 将相似数据分组 | K-Means, DBSCAN, 层次聚类 |
| 降维 | 减少特征维度 | PCA, t-SNE, UMAP |
| 异常检测 | 发现异常数据 | Isolation Forest, One-Class SVM |
| 生成 | 学习数据分布 | VAE, GAN |

### 2. 聚类算法

**K-Means：**
```python
from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=3)
labels = kmeans.fit_predict(X)
```

**DBSCAN（密度聚类）：**
```python
from sklearn.cluster import DBSCAN

dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)
```

### 3. 降维技术

**PCA（主成分分析）：**
- 线性降维
- 保留最大方差方向
- 用于数据预处理

**t-SNE：**
- 非线性降维
- 保留局部结构
- 常用于可视化
