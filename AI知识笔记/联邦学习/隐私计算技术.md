# 隐私计算技术

## 一句话定义

隐私计算是指在保护数据隐私的前提下实现数据分析和计算的技术总称，通过密码学、安全硬件、系统安全等手段，实现"数据可用不可见"的隐私保护目标。

## 核心公式

### 安全多方计算基础

#### 秘密分享（Secret Sharing）

将秘密 $s$ 分割为 $n$ 份 $\text{share}_1, ..., \text{share}_n$，满足：

$$s = \text{share}_1 \oplus \text{share}_2 \oplus ... \oplus \text{share}_n$$

恢复时需要至少 $t$ 份shares（(t, n)-阈值方案）。

#### Garbled Circuits

$$f(x, y) = \text{Eval}(\text{Garble}(C), \text{InputEnc}(x), \text{InputEnc}(y))$$

其中 $C$ 为布尔电路，$x, y$ 分别为两方输入。

### 同态加密

#### 加法同态（Paillier）

$$\text{Enc}(m_1) \cdot \text{Enc}(m_2) = \text{Enc}(m_1 + m_2)$$

$$\text{Enc}(m)^k = \text{Enc}(k \cdot m)$$

#### 乘法同态（RSA）

$$\text{Enc}(m_1) \cdot \text{Enc}(m_2) = \text{Enc}(m_1 \cdot m_2)$$

#### Gentry's FHE Scheme

$$\text{Eval}(f, \text{Enc}(x_1), ..., \text{Enc}(x_n)) = \text{Enc}(f(x_1, ..., x_n))$$

支持任意布尔电路或算术电路的同态计算。

### 可信执行环境

#### 远程认证（Remote Attestation）

$$\text{Quote} = \text{Sign}_{\text{IntelKey}}(\text{Report}_{\text{TEE}})$$

客户端验证 TEE 的可信状态。

## 详细说明

### 1. 安全多方计算（Secure Multi-Party Computation, MPC）

#### 定义

MPC 允许 $n$ 方在各自私有输入上联合计算函数 $f(x_1, ..., x_n)$，同时保证每一方只能看到自己的输入和最终输出，无法获取其他方的输入信息。

#### 安全保证

- **隐私性**：任何攻击者（包括合谋攻击）无法获取额外信息
- **正确性**：各方获得正确的计算结果
- **输入独立性**：输入由各方独立提供
- **公平性**：所有各方同时获得结果，或都不获得

#### MPC 协议分类

| 类型 | 代表协议 | 适用场景 | 效率 |
|------|---------|---------|------|
| 基于秘密分享 | SPDZ,诚实多数 | 算术电路 | 高（算术运算） |
| 基于Garbled Circuits | Yao's GC | 布尔电路 | 中等 |
| 混合协议 | ABY, ABY3 | 混合电路 | 高 |
| 不经意传输 | OT, PSI | 安全函数调用 | 取决于具体协议 |

#### 代码示例

```python
# 基于秘密分享的简单 MPC 实现
import numpy as np

class SecretSharingMPC:
    """简单的加法秘密分享 MPC"""

    def __init__(self, n_parties=3, threshold=2):
        self.n_parties = n_parties
        self.threshold = threshold
        self.mod = 2**32  # 模运算

    def share(self, secret):
        """将秘密分割为 n 份"""
        shares = np.random.randint(0, self.mod, self.n_parties - 1)
        shares = list(shares)
        # 最后一份 = 秘密 - 其他份之和
        last_share = (secret - sum(shares)) % self.mod
        shares.append(last_share)
        return shares

    def reconstruct(self, shares):
        """从 shares 重构秘密"""
        return sum(shares) % self.mod

    def add_shares(self, share_lists):
        """加法操作：本地直接相加"""
        result = [sum(shares) % self.mod for shares in zip(*share_lists)]
        return result

    def multiply_scalar(self, scalar, shares):
        """标量乘法：本地直接相乘"""
        return [(scalar * s) % self.mod for s in shares]
```

### 2. 同态加密（Homomorphic Encryption, HE）

#### 定义

同态加密允许在密文上直接进行特定类型的计算，无需解密。解密后的结果与在明文上执行相应计算的结果一致。

#### 发展历程

1. **部分同态加密**：仅支持加法或乘法（RSA, Paillier）
2. ** somewhat HE**：支持有限次数的加法和乘法
3. ** Bootstrappable HE**：可自举，支持任意次数运算（Gentry's scheme）
4. **全同态加密（FHE）**：支持任意次数的加法和乘法运算

#### 主流 FHE 方案

| 方案 | 年份 | 效率 | 特性 |
|------|------|------|------|
| BGV | 2012 | 高 | 批处理效率高 |
| BFV | 2012 | 高 | 类似 BGV |
| CKKS | 2017 | 中 | 支持近似计算，适合机器学习 |
| TFHE | 2017 | 中 | 快速Bootstrapping |

#### 代码示例

```python
# 使用 PySEAL（Microsoft SEAL）进行同态加密
from seal import (
    EncryptionParameters, SchemeType, KeyGenerator,
    Encryptor, Evaluator, Decryptor, IntegerEncoder, BatchEncoder
)

class HomomorphicEncryption:
    """同态加密封装"""

    def __init__(self, poly_modulus_degree=4096, plain_modulus=1024):
        self.params = EncryptionParameters()
        self.params.set_poly_modulus_degree(poly_modulus_degree)
        self.params.set_coeff_modulus([1024, 1024, 1024])
        self.params.set_plain_modulus(plain_modulus)

        self.keygen = KeyGenerator(self.params)
        self.public_key = self.keygen.public_key()
        self.secret_key = self.keygen.secret_key()
        self.relin_keys = self.keygen.relin_keys()

        self.encryptor = Encryptor(self.params, self.public_key)
        self.evaluator = Evaluator(self.params)
        self.decryptor = Decryptor(self.params, self.secret_key)

    def encrypt(self, plaintext):
        """加密"""
        return self.encryptor.encrypt(plaintext)

    def decrypt(self, ciphertext):
        """解密"""
        return self.decryptor.decrypt(ciphertext)

    def add(self, ct1, ct2):
        """密文加法"""
        return self.evaluator.add(ct1, ct2)

    def multiply(self, ct1, ct2):
        """密文乘法"""
        result = self.evaluator.multiply(ct1, ct2)
        # 重线性化（减少密文大小）
        return self.evaluator.relinearize(result, self.relin_keys)

    def add_plain(self, ct, plain):
        """密文与明文加法"""
        return self.evaluator.add_plain(ct, plain)

    def multiply_plain(self, ct, plain):
        """密文与明文乘法"""
        return self.evaluator.multiply_plain(ct, plain)

import torch
import tenseal as ts

class TENSEALEncryptedML:
    """使用 TENSEAL 库的同态加密机器学习"""

    def __init__(self):
        self.context = ts.context(
            ts.SCHEME_TYPE.CKKS,
            poly_modulus_degree=8192,
            coeff_mod_bit_sizes=[60, 40, 40, 60]
        )
        self.context.generate_galois_keys()
        self.context.generate_relin_keys(self.context.secret_key())

    def encrypt_vector(self, vector):
        """加密向量"""
        return ts.ckks_vector(self.context, vector.tolist())

    def encrypted_linear_layer(self, encrypted_input, weight, bias):
        """加密域线性层计算"""
        # y = wx + b
        result = encrypted_input * weight.tolist()
        result = result + bias.tolist()
        return result

    def encrypted_relu(self, encrypted_input):
        """加密域 ReLU（使用多项式近似）"""
        # ReLU(x) ≈ 0.5 + 0.5*x (简化近似)
        # 实际使用更复杂的多项式逼近
        return 0.5 + 0.5 * encrypted_input


# 端到端隐私保护推理示例
def privacy_preserving_inference(model_weights, model_bias, input_data):
    """
    使用同态加密进行隐私保护推理
    - 服务端拥有模型（加密）
    - 客户端拥有输入数据（加密）
    - 推理过程在加密域完成
    """
    he = HomomorphicEncryption()

    # 客户端加密输入
    encrypted_input = he.encrypt_vector(input_data)

    # 加密域推理
    encrypted_output = encrypted_input
    for weight, bias in zip(model_weights, model_bias):
        encrypted_output = he.encrypted_linear_layer(
            encrypted_output, weight, bias
        )

    # 客户端解密结果
    result = he.decrypt(encrypted_output)
    return result
```

### 3. 可信执行环境（Trusted Execution Environment, TEE）

#### 定义

TEE 是处理器内的一个隔离安全区域，与主操作系统（Rich OS）隔离运行，提供硬件级别的安全保证。即使主系统被攻破，TEE 内的代码和数据仍受到保护。

#### 主流 TEE 技术

| 技术 | 厂商 | 应用场景 |
|------|------|---------|
| Intel SGX | Intel | 云服务器、机密计算 |
| ARM TrustZone | ARM | 移动设备、嵌入式 |
| AMD SEV | AMD | 虚拟机加密 |
| RISC-V Keystone | 开源 | 研究定制 |

#### 安全特性

- **隔离执行**：TEE 与 Rich OS 完全隔离
- **内存加密**：SGX 支持 EPC 内存加密
- **远程认证**：向远程验证 TEE 可信状态
- **防物理攻击**：硬件级别的篡改检测

#### 代码示例

```python
class SGXRemoteAttestation:
    """Intel SGX 远程认证流程"""

    def __init__(self, spid, subscription_key):
        self.spd = spid  # Intel SPID
        self.key = subscription_key

    def generate_quote(self, enclave_signing_key, report_data):
        """生成 SGX Quote"""
        # SGX Quote 包含 Enclave 测量值和报告数据
        quote_request = {
            'sig_rl': self._get_sig_rl(),
            'quote': self._create_quote(enclave_signing_key, report_data)
        }
        return quote_request

    def verify_quote(self, quote):
        """验证 Quote 有效性"""
        # 1. 验证签名
        is_signed = self._verify_quote_signature(quote)

        # 2. 验证 Enclave 测量值
        is_measured = self._verify_measurement(quote)

        # 3. 验证 Quote 版本
        is_version_ok = self._check_quote_version(quote)

        return is_signed and is_measured and is_version_ok

    def establish_secure_channel(self, verified_quote):
        """验证通过后建立安全通道"""
        # 使用 Quote 中的公钥建立密钥交换
        session_key = self._ecdh_key_exchange(
            verified_quote['report_body']['pub_key']
        )
        return session_key
```

```python
# 使用 PySyft 实现隐私保护联邦学习
import syft as sy
from syft import federated
from syft.frameworks.torch.dp import pytorch as dp

class PrivacyPreservingFL:
    """隐私保护联邦学习"""

    def __init__(self, epsilon=1.0, delta=1e-5):
        self.hook = sy.TorchHook(torch)
        self.epsilon = epsilon
        self.delta = delta
        self.nodes = []

    def add_worker(self, node_id, data):
        """添加联邦学习节点"""
        worker = self.hook.world.get_worker(node_id)
        worker.data = data
        self.nodes.append(worker)

    def private_train(self, model, federated_data, rounds=10):
        """联邦学习训练循环"""
        loss_history = []

        for round_num in range(rounds):
            # 1. 分发模型到各节点
            model = self._send_model_to_workers(model)

            # 2. 本地训练
            local_updates = []
            for node, node_data in zip(self.nodes, federated_data):
                local_model = self._train_local(
                    model.copy(), node_data
                )
                local_updates.append(local_model)

            # 3. 安全聚合（使用 MPC 或 TEE）
            aggregated_model = self._secure_aggregate(local_updates)

            # 4. 差分隐私（如果启用）
            if self.epsilon > 0:
                aggregated_model = self._apply_dp(aggregated_model)

            model = aggregated_model
            loss_history.append(self._evaluate(model))

        return model, loss_history

    def _send_model_to_workers(self, model):
        """分发模型到各节点"""
        return model.send(self.nodes)

    def _train_local(self, model, data, epochs=5):
        """本地训练"""
        criterion = torch.nn.CrossEntropyLoss()
        optimizer = torch.optim.SGD(model.parameters(), lr=0.01)

        for _ in range(epochs):
            optimizer.zero_grad()
            outputs = model(data['x'])
            loss = criterion(outputs, data['y'])
            loss.backward()
            optimizer.step()

        return model

    def _secure_aggregate(self, local_models):
        """安全聚合（使用加权平均）"""
        # 简单实现：使用秘密分享
        weights = [1.0 / len(local_models)] * len(local_models)

        aggregated_state = {}
        for key in local_models[0].state_dict().keys():
            aggregated_state[key] = sum(
                w * model.state_dict()[key]
                for w, model in zip(weights, local_models)
            )

        aggregated_model = local_models[0]
        aggregated_model.load_state_dict(aggregated_state)
        return aggregated_model
```
