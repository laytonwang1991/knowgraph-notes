# 联邦学习概述

## 一句话定义

联邦学习是一种分布式机器学习范式，多个参与方在保持数据本地化的前提下协作训练模型，实现"数据不动模型动"的隐私保护机器学习。

## 核心公式

### 联邦平均算法（FedAvg）

$$\theta_{t+1} = \sum_{k=1}^{K} \frac{n_k}{n} \theta_{k,t+1}$$

其中：
- $\theta_{t+1}$ 为第 $t+1$ 轮的全局模型参数
- $K$ 为参与方数量
- $n_k$ 为第 $k$ 个参与方的样本数量
- $n = \sum_{k=1}^{K} n_k$ 为总样本数量
- $\theta_{k,t+1}$ 为第 $k$ 个参与方的本地更新

### 目标函数

$$\min_{\theta} \sum_{k=1}^{K} \frac{n_k}{n} F_k(\theta)$$

其中 $F_k(\theta) = \frac{1}{n_k} \sum_{i \in \mathcal{D}_k} f(x_i, y_i; \theta)$ 为第 $k$ 个参与方的本地损失函数。

## 详细说明

### 1. 联邦学习的核心思想

- **数据不动，模型动**：原始数据永不离开本地设备
- **协作学习**：通过聚合本地模型更新构建全局模型
- **隐私保护**：参与方只共享模型参数，不共享原始数据

### 2. 联邦学习的分类

#### 横向联邦学习（Horizontal Federated Learning）

- 适用于各方数据特征相同但样本不同
- 场景：跨银行的反欺诈模型（银行A、B各自拥有不同用户数据）
- 形式：样本水平联邦学习

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  银行A数据   │     │  银行B数据   │     │  银行C数据   │
│ 用户1: 特征  │     │ 用户2: 特征  │     │ 用户3: 特征  │
│ 用户2: 特征  │     │ 用户3: 特征  │     │ 用户4: 特征  │
└──────┬──────┘     └──────┬──────┘     └──────┬──────┘
       │                   │                   │
       ▼                   ▼                   ▼
┌─────────────────────────────────────────────────────┐
│              本地训练 → 模型更新 → 聚合服务器         │
└─────────────────────────────────────────────────────┘
                        │
                        ▼
                 ┌─────────────┐
                 │  全局模型    │
                 └─────────────┘
```

#### 纵向联邦学习（Vertical Federated Learning）

- 适用于各方数据样本相同但特征不同
- 场景：银行与电商联合建模（同一用户不同特征）
- 形式：特征水平联邦学习

#### 联邦迁移学习（Federated Transfer Learning）

- 适用于样本和特征均不重叠的场景
- 通过迁移学习提取共有知识
- 解决跨孤岛数据协作问题

### 3. 联邦学习工作流程

1. **初始化**：中央服务器分发初始模型参数 $\theta_0$
2. **本地训练**：各参与方使用本地数据训练 $E$ 轮
3. **模型上传**：参与方将更新 $\Delta \theta_k$ 上传至服务器
4. **聚合更新**：服务器使用 FedAvg 聚合模型
5. **模型分发**：服务器将聚合后模型分发至各参与方
6. **迭代执行**：重复步骤 2-5 直到收敛

### 4. 隐私保护机制

联邦学习本身并非完全隐私保护，需要结合：

- **差分隐私**：在模型更新中添加噪声
- **安全多方计算**：保护模型聚合过程
- **同态加密**：加密模型参数传输

## 代码示例

### PyTorch 联邦平均实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
from typing import List, Dict

class Federated Averaging:
    """联邦平均算法实现"""

    def __init__(self, model: nn.Module, lr: float = 0.01):
        self.model = model
        self.lr = lr

    def local_train(self, train_data: torch.Tensor,
                    labels: torch.Tensor,
                    epochs: int = 5) -> Dict[str, torch.Tensor]:
        """本地训练"""
        criterion = nn.CrossEntropyLoss()
        optimizer = optim.SGD(self.model.parameters(), lr=self.lr)

        self.model.train()
        for _ in range(epochs):
            optimizer.zero_grad()
            outputs = self.model(train_data)
            loss = criterion(outputs, labels)
            loss.backward()
            optimizer.step()

        # 返回模型更新（而非原始数据）
        return {name: param.data.clone()
                for name, param in self.model.named_parameters()}

    def aggregate(self, client_updates: List[Dict[str, torch.Tensor]],
                  client_weights: List[float]) -> Dict[str, torch.Tensor]:
        """聚合客户端更新"""
        aggregated = {}
        total_weight = sum(client_weights)

        for name in client_updates[0].keys():
            aggregated[name] = sum(
                w * update[name] for w, update in
                zip(client_weights, client_updates)
            ) / total_weight

        return aggregated

    def apply_update(self, state_dict: Dict[str, torch.Tensor]):
        """应用更新到模型"""
        self.model.load_state_dict(state_dict)


# 使用示例
def federated_learning_round(clients_data: List[torch.Tensor],
                              clients_labels: List[torch.Tensor],
                              global_model: nn.Module,
                              client_weights: List[float]):
    """执行一轮联邦学习"""
    fed_avg = Federated Averaging(global_model)

    # 收集各客户端更新
    client_updates = []
    for data, labels in zip(clients_data, clients_labels):
        update = fed_avg.local_train(data, labels)
        client_updates.append(update)

    # 聚合更新
    aggregated = fed_avg.aggregate(client_updates, client_weights)

    # 应用到全局模型
    fed_avg.apply_update(aggregated)

    return global_model
```

## 应用场景

### 1. 医疗健康
- **跨医院疾病预测**：多家医院协作训练诊断模型，患者数据不出院
- **基因数据研究**：在不暴露基因隐私的前提下进行联合分析
- **药物发现**：药企间共享药物分子特征而不暴露商业机密

### 2. 金融领域
- **反欺诈**：银行间共享欺诈模式，提升检测准确率
- **信用评估**：跨机构联合建模，丰富用户画像
- **保险定价**：多保险公司协作优化精算模型

### 3. 移动端应用
- **输入法预测**：手机本地训练语言模型，保护用户打字习惯
- **键盘推荐**：Google Gboard 联邦学习案例
- **语音助手**：本地化语音识别模型训练

### 4. 边缘计算
- **物联网**：边缘设备协作学习，无需上传原始数据
- **自动驾驶**：车内模型个性化训练
- **智能家居**：本地化行为模式学习

### 5. 跨孤岛数据协作
- **数据交易所**：数据不动模型动，实现数据价值流通
- **政府数据开放**：政务数据跨部门协作
- **企业数据中台**：集团内子公司数据协作
