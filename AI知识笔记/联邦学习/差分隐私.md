# 差分隐私

## 一句话定义

差分隐私是一种数学隐私保护框架，通过对查询结果添加精心设计的随机噪声，使得攻击者无法判断数据库中是否包含某个特定个体的数据，从而为隐私提供可证明的量化保证。

## 核心公式

### ε-差分隐私（纯差分隐私）

$$Pr[\mathcal{M}(D) \in S] \leq e^\varepsilon \cdot Pr[\mathcal{M}(D') \in S]$$

其中：
- $\mathcal{M}$ 为隐私机制
- $D$ 和 $D'$ 为仅相差一条记录的两个相邻数据库
- $S$ 为任意输出集合
- $\varepsilon$ 为隐私预算参数

### (ε, δ)-差分隐私（近似差分隐私）

$$Pr[\mathcal{M}(D) \in S] \leq e^\varepsilon \cdot Pr[\mathcal{M}(D') \in S] + \delta$$

其中 $\delta$ 表示允许的隐私保证失败概率。

### 全局敏感度（Global Sensitivity）

$$\Delta f = \max_{D, D'} \| f(D) - f(D') \|_1$$

对于计数查询：$\Delta f = 1$

### 拉普拉斯机制

$$\mathcal{M}(x) = f(x) + \text{Lap}(\Delta f / \varepsilon)$$

其中 $\text{Lap}(b)$ 表示尺度参数为 $b$ 的拉普拉斯分布：
$$p(x) = \frac{1}{2b} e^{-|x|/b}$$

### 高斯机制

$$\mathcal{M}(x) = f(x) + \mathcal{N}(0, \sigma^2)$$

其中 $\sigma$ 满足：
$$\sigma \geq \Delta f \cdot \frac{\sqrt{2 \ln(1.25/\delta)}}{\varepsilon}$$

### 隐私损失随机变量（Privacy Loss Random Variable）

$$\mathcal{L}_D = \ln \frac{Pr[\mathcal{M}(D) = o]}{Pr[\mathcal{M}(D') = o]}$$

###  RDP（Rényi Differential Privacy）

$$D_\alpha(\mathcal{M}(D) \| \mathcal{M}(D')) = \frac{1}{\alpha - 1} \ln \mathbb{E}\left[ e^{(\alpha-1) \mathcal{L}_D} \right] \leq \varepsilon$$

## 详细说明

### 1. 差分隐私的数学基础

#### 相邻数据库（Adjacent Databases）

两个数据库 $D$ 和 $D'$ 被称为相邻，当且仅当它们相差恰好一条记录：

$$|D \Delta D'| = 1$$

这意味着要么 $D'$ 在 $D$ 基础上增加一条记录，要么 $D'$ 在 $D$ 基础上删除一条记录。

#### 隐私保证的核心思想

差分隐私的直觉是：算法的输出分布对于单个数据点的存在或缺失不敏感。因此，攻击者无法从输出中推断出特定个体是否在数据库中。

### 2. 隐私预算 ε 的含义

- **ε 越小**：隐私保护越强，但可用性越低（噪声越大）
- **ε 越大**：可用性越高，但隐私风险越大
- **典型取值**：ε ∈ [0.1, 10]，实际应用中常取 ε ≤ 1
- **ε = 0**：理想情况，输出分布完全相同（不可实现）

### 3. 噪声注入机制

#### 拉普拉斯噪声

适用于数值型查询输出：

```python
import numpy as np

def laplace_mechanism(true_value, sensitivity, epsilon):
    """拉普拉斯机制实现"""
    scale = sensitivity / epsilon
    noise = np.random.laplace(0, scale)
    return true_value + noise

# 示例：计数查询
def noisy_count(query_result, epsilon=1.0):
    sensitivity = 1  # 计数查询敏感度为1
    return laplace_mechanism(query_result, sensitivity, epsilon)
```

#### 高斯噪声

适用于复合查询或 RDP 分析：

```python
def gaussian_mechanism(true_value, sensitivity, epsilon, delta):
    """高斯机制实现"""
    sigma = sensitivity * np.sqrt(2 * np.log(1.25 / delta)) / epsilon
    noise = np.random.normal(0, sigma)
    return true_value + noise
```

#### 指数机制（Exponential Mechanism）

适用于离散输出：

```python
def exponential_mechanism(score_function, epsilon, candidates):
    """
    指数机制实现
    score_function: 评分函数，输出分数越高越好
    """
    sensitivities = []
    for c in candidates:
        max_diff = 0
        for other in candidates:
            diff = abs(score_function(c) - score_function(other))
            max_diff = max(max_diff, diff)
        sensitivities.append(max_diff)

    # 计算概率权重
    weights = np.exp(epsilon * np.array(sensitivities) / 2)
    weights = weights / weights.sum()

    # 采样
    return np.random.choice(candidates, p=weights)
```

### 4. 隐私预算的组合

#### 序列组合定理

对同一数据集连续应用 $k$ 个 $\varepsilon_i$-差分隐私机制，总隐私预算：

$$\varepsilon_{total} = \sum_{i=1}^{k} \varepsilon_i$$

#### 并行组合定理

如果机制作用于数据集的互不相交子集：

$$\varepsilon_{total} = \max(\varepsilon_1, \varepsilon_2, ..., \varepsilon_k)$$

### 5. 用户级 vs 样本级差分隐私

#### 样本级差分隐私（Sample-Level DP）

相邻数据集定义为相差一个样本：

$$D \approx_{sample} D' \iff D' = D \setminus \{x_i\}$$

- 适用于独立同分布数据
- 敏感度分析与单个样本相关

#### 用户级差分隐私（User-Level DP）

相邻数据集定义为相差一个用户的所有数据：

$$D \approx_{user} D' \iff D' = D \setminus \{x_{i,1}, ..., x_{i,n_i}\}$$

- 适用于用户产生多条记录的场景
- 敏感度取决于单个用户的数据量
- 实际应用中更常用，因为用户需要隐私保护

```python
class UserLevelDP:
    """用户级差分隐私"""

    def __init__(self, epsilon, max_user_samples):
        self.epsilon = epsilon
        # 用户级敏感度 = 单个用户贡献的最大范数
        self.sensitivity = max_user_samples

    def add_noise(self, user_gradients):
        """为用户梯度添加噪声"""
        scale = self.sensitivity / self.epsilon
        noisy_gradients = [
            g + np.random.laplace(0, scale, g.shape)
            for g in user_gradients
        ]
        return noisy_gradients
```

## 代码示例

### PyTorch 差分隐私优化器

```python
import torch
import torch.nn as nn
from typing import List, Tuple
import numpy as np

class DPSGD:
    """差分隐私随机梯度下降"""

    def __init__(self, model, optimizer, sigma, max_grad_norm, delta):
        self.model = model
        self.optimizer = optimizer
        self.sigma = sigma  # 噪声标准差
        self.max_grad_norm = max_grad_norm  # 梯度裁剪范数
        self.delta = delta  # 隐私预算的δ参数
        self.epoch = 0
        self.noise_multiplier = sigma / max_grad_norm

    def clip_gradients(self, gradients: List[torch.Tensor]):
        """梯度裁剪"""
        total_norm = torch.sqrt(sum(g.norm() ** 2 for g in gradients))
        clip_coef = self.max_grad_norm / (total_norm + 1e-6)
        clip_coef = min(clip_coef, 1.0)
        return [g * clip_coef for g in gradients]

    def add_noise(self, gradients: List[torch.Tensor]):
        """添加高斯噪声"""
        noisy_gradients = []
        for g in gradients:
            noise = torch.randn_like(g) * self.sigma
            noisy_gradients.append(g + noise)
        return noisy_gradients

    def step(self, batch: Tuple[torch.Tensor, torch.Tensor]):
        """执行一步训练"""
        self.optimizer.zero_grad()

        # 前向传播
        inputs, targets = batch
        outputs = self.model(inputs)
        loss = nn.functional.cross_entropy(outputs, targets)

        # 反向传播
        loss.backward()

        # 获取梯度
        gradients = [p.grad.clone() for p in self.model.parameters()]

        # 梯度裁剪
        gradients = self.clip_gradients(gradients)

        # 添加噪声
        noisy_gradients = self.add_noise(gradients)

        # 更新参数
        for p, g in zip(self.model.parameters(), noisy_gradients):
            p.data = p.data - self.optimizer.param_groups[0]['lr'] * g

        return loss.item()

    def get_privacy_spent(self):
        """计算已消耗的隐私预算"""
        # 使用 RDP 到 (ε, δ)-DP 的转换
        alpha = 32  # Rényi 阶
        epsilon_rdp = self.epoch * (self.noise_multiplier ** 2) * alpha / 2

        # 转换回 (ε, δ)-DP
        epsilon = epsilon_rdp + np.sqrt(2 * alpha * np.log(1.25 / self.delta)) * self.noise_multiplier
        return epsilon, self.delta


# 使用示例
def train_with_dp(model, train_loader, epochs=10, epsilon=1.0, delta=1e-5):
    optimizer = torch.optim.SGD(model.parameters(), lr=0.01, momentum=0.9)
    dp_sgd = DPSGD(
        model=model,
        optimizer=optimizer,
        sigma=1.0,  # 根据隐私预算计算
        max_grad_norm=1.0,
        delta=delta
    )

    for epoch in range(epochs):
        for batch in train_loader:
            loss = dp_sgd.step(batch)

        dp_epsilon, dp_delta = dp_sgd.get_privacy_spent()
        print(f'Epoch {epoch}: Privacy spent (ε={dp_epsilon:.2f}, δ={dp_delta})')
```

### Opacus 库示例（PyTorch 官方差分隐私库）

```python
from opacus import PrivacyEngine
from opacus.utils.module_utils import clip_gradients

def train_with_opacus(model, train_loader, epochs=10, epsilon=1.0, delta=1e-5):
    """使用 Opacus 库训练差分隐私模型"""
    privacy_engine = PrivacyEngine(
        accountant='rdp',  # 使用 Rényi DP 会计师
        secure_mode=False
    )

    optimizer = torch.optim.SGD(model.parameters(), lr=0.01, momentum=0.9)

    model, optimizer, train_loader = privacy_engine.make_private(
        module=model,
        optimizer=optimizer,
        data_loader=train_loader,
        noise_multiplier=1.0,
        max_grad_norm=1.0,
        poisson_sampling=True  # 采样时应用差分隐私
    )

    for epoch in range(epochs):
        for batch in train_loader:
            optimizer.zero_grad()
            outputs = model(batch['input'])
            loss = nn.functional.cross_entropy(outputs, batch['label'])
            loss.backward()
            optimizer.step()

        # 获取隐私预算
        epsilon = privacy_engine.get_epsilon(delta)
        print(f'Epoch {epoch}: ε={epsilon:.2f}, δ={delta}')
```

## 应用场景

### 1. 联邦学习中的差分隐私

#### 本地差分隐私（Local Differential Privacy）

- 每个参与者在本地添加噪声
- 不需要可信第三方
- 噪声叠加导致信号衰减

```
用户A数据 → 添加噪声 → 噪声数据 → 聚合服务器
用户B数据 → 添加噪声 → 噪声数据 ↗
用户C数据 → 添加噪声 → 噪声数据 ↗
```

#### 中央差分隐私（Central Differential Privacy）

- 可信中央服务器聚合数据后添加噪声
- 隐私预算利用更高效
- 需要参与方信任中央服务器
