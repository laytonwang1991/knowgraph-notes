# 推荐系统

## 一句话定义

推荐系统是一种基于用户行为、物品属性和环境上下文信息，通过算法模型预测用户对物品兴趣程度并生成个性化推荐列表的信息过滤系统，核心目标是解决信息过载下的用户内容发现难题。

## 核心公式

### 1. 协同过滤基础

#### 用户-物品评分矩阵

设用户集合为 $U = \{u_1, u_2, ..., u_m\}$，物品集合为 $I = \{i_1, i_2, ..., i_n\}$，则用户-物品评分矩阵为：

$$\mathbf{R}_{m \times n} = \begin{bmatrix} r_{11} & r_{12} & \cdots & r_{1n} \\ r_{21} & r_{22} & \cdots & r_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ r_{m1} & r_{m2} & \cdots & r_{mn} \end{bmatrix}$$

其中 $r_{ui}$ 表示用户 $u$ 对物品 $i$ 的评分，缺失值用 ? 表示。

#### 相似度计算

**余弦相似度：**
$$\text{cosine}(u, v) = \frac{\vec{r}_u \cdot \vec{r}_v}{\|\vec{r}_u\| \cdot \|\vec{r}_v\|} = \frac{\sum_{i \in I_{uv}} r_{ui} \cdot r_{vi}}{\sqrt{\sum_{i \in I_{uv}} r_{ui}^2} \cdot \sqrt{\sum_{i \in I_{uv}} r_{vi}^2}}$$

**皮尔逊相关系数：**
$$\text{pearson}(u, v) = \frac{\sum_{i \in I_{uv}} (r_{ui} - \bar{r}_u)(r_{vi} - \bar{r}_v)}{\sqrt{\sum_{i \in I_{uv}} (r_{ui} - \bar{r}_u)^2} \cdot \sqrt{\sum_{i \in I_{uv}} (r_{vi} - \bar{r}_v)^2}}$$

### 2. 矩阵分解

#### SVD（奇异值分解）

$$\mathbf{R}_{m \times n} \approx \mathbf{U}_{m \times k} \cdot \mathbf{\Sigma}_{k \times k} \cdot \mathbf{V}_{k \times n}^T$$

其中 $k$ 为隐因子维度，预测评分为：

$$\hat{r}_{ui} = \mathbf{U}_u \cdot \mathbf{V}_i^T = \sum_{f=1}^{k} u_{uf} \cdot v_{if}$$

#### FunkSVD（梯度下降求解）

损失函数：
$$\mathcal{L} = \sum_{(u,i) \in \text{obs}} (r_{ui} - \hat{r}_{ui})^2 + \lambda(\|\mathbf{U}\|_F^2 + \|\mathbf{V}\|_F^2)$$

更新公式：
$$u_{uf} \leftarrow u_{uf} + \alpha \cdot (r_{ui} - \hat{r}_{ui}) \cdot v_{if} - \lambda \cdot u_{uf}$$
$$v_{if} \leftarrow v_{if} + \alpha \cdot (r_{ui} - \hat{r}_{ui}) \cdot u_{uf} - \lambda \cdot v_{if}$$

### 3. Embedding在推荐系统中的应用

#### 向量化召回

$$\text{score}(u, i) = \text{cosine}(\mathbf{e}_u, \mathbf{e}_i) = \frac{\mathbf{e}_u \cdot \mathbf{e}_i}{\|\mathbf{e}_u\| \|\mathbf{e}_i\|}$$

其中 $\mathbf{e}_u$ 为用户Embedding，$\mathbf{e}_i$ 为物品Embedding。

#### 双塔模型

用户塔输出：$\mathbf{e}_u = f_u(\text{user_features})$

物品塔输出：$\mathbf{e}_i = f_i(\text{item_features})$

匹配分数：$\text{score}(u, i) = \text{cosine}(\mathbf{e}_u, \mathbf{e}_i)$

## 详细说明

### 1. 推荐系统架构

现代推荐系统通常采用**召回（Recall）-> 粗排（Re-rank）-> 精排（Rank）-> 重排（Rerank）**的多阶段架构：

```
┌─────────────────────────────────────────────────────┐
│                     推荐流程                          │
├─────────────────────────────────────────────────────┤
│ 1. 召回层: 从百万/千万物品中快速召回候选集(百/千级)    │
│    - 协同过滤召回                                    │
│    - 向量相似度召回（Embedding）                      │
│    - 热门/新品/运营规则召回                           │
│                                                     │
│ 2. 粗排层: 对召回候选进行轻量级排序（百级）            │
│    - 简单ML模型                                      │
│    - 特征交叉                                        │
│                                                     │
│ 3. 精排层: 精细化排序（十级）                         │
│    - 深度学习模型（DNN/Wide&Deep/DeepFM）             │
│    - 丰富的用户/物品/上下文特征                        │
│                                                     │
│ 4. 重排层: 最终调整（多样性/商业逻辑/用户体验）        │
│    - MMR（最大边际相关性）                           │
│    - 规则引擎                                        │
│    - 流量分配策略                                    │
└─────────────────────────────────────────────────────┘
```

### 2. 协同过滤（Collaborative Filtering）

#### 基于用户的协同过滤（User-CF）

核心思想：找到与目标用户兴趣相似的用户群，推荐这些相似用户喜欢的物品。

步骤：
1. 计算用户之间的相似度
2. 找到与用户 $u$ 最相似的 $K$ 个用户
3. 根据这 $K$ 个用户的行为，预测用户 $u$ 对物品 $i$ 的评分

$$\hat{r}_{ui} = \bar{r}_u + \frac{\sum_{v \in N_k(u)} w_{uv} \cdot (r_{vi} - \bar{r}_v)}{\sum_{v \in N_k(u)} |w_{uv}|}$$

其中 $N_k(u)$ 为用户 $u$ 的 $K$ 个最相似用户，$w_{uv}$ 为用户 $u$ 和 $v$ 的相似度。

#### 基于物品的协同过滤（Item-CF）

核心思想：用户喜欢与其历史喜欢的物品相似的物品。

$$\hat{r}_{ui} = \frac{\sum_{j \in R_u} w_{ij} \cdot r_{uj}}{\sum_{j \in R_u} |w_{ij}|}$$

其中 $R_u$ 为用户 $u$ 评分过的物品集合，$w_{ij}$ 为物品 $i$ 和 $j$ 的相似度。

### 3. 内容推荐（Content-Based）

基于物品特征和用户偏好进行匹配：

- **用户画像**：从用户历史行为中提取特征（类别偏好、标签偏好等）
- **物品画像**：物品的文本特征、类别、标签、属性等
- **匹配方式**：余弦相似度、逻辑回归、决策树等

### 4. 深度学习推荐

#### Wide&Deep模型

$$\hat{r} = \sigma(\mathbf{w}_{wide}^T [\mathbf{x}, \phi(\mathbf{x})] + \mathbf{w}_{deep}^T a^{(l)} + b)$$

- **Wide部分**：记忆能力，学习浅层特征交叉
- **Deep部分**：泛化能力，学习深度特征表示

#### DeepFM模型

$$\hat{r} = \sigma(\mathbf{w}_0 + \sum_i w_i x_i + \sum_{j} \sum_{k} v_{jk} x_{jk} + \mathbf{w}_{DNN}^T a^{(l)} + b)$$

结合了FM的二阶特征交叉和DNN的高阶特征学习。

### 5. Embedding应用

| 应用场景 | Embedding类型 | 用途 |
|----------|--------------|------|
| 向量召回 | Item Embedding | ANN召回候选集 |
| 用户表征 | User Embedding | 用户兴趣建模 |
| 序列推荐 | Item Sequence Embedding | 捕捉用户兴趣演变 |
| 图推荐 | Graph Embedding | 社交网络、物品图谱 |
| 多模态推荐 | Multimodal Embedding | 图文/视频内容理解 |

## 代码示例

### Python实现基础协同过滤

```python
import numpy as np
from scipy.sparse import csr_matrix
from sklearn.metrics.pairwise import cosine_similarity

class UserBasedCF:
    def __init__(self, n_neighbors=20):
        self.n_neighbors = n_neighbors
        self.user_item_matrix = None
        self.similarity_matrix = None
        self.user_means = None

    def fit(self, user_item_matrix):
        """
        user_item_matrix: numpy array of shape (n_users, n_items)
        """
        self.user_item_matrix = user_item_matrix
        self.user_means = np.nanmean(user_item_matrix, axis=1)

        # 填充缺失值为用户均值
        matrix_filled = user_item_matrix.copy()
        for i in range(len(self.user_means)):
            mask = np.isnan(matrix_filled[i])
            matrix_filled[i, mask] = self.user_means[i]

        # 计算用户相似度
        self.similarity_matrix = cosine_similarity(matrix_filled)

    def predict(self, user_id, item_id):
        """预测用户对物品的评分"""
        scores = self.similarity_matrix[user_id]
        user_ratings = self.user_item_matrix[:, item_id]

        # 获取有评分的用户及其相似度
        rated_users = np.where(~np.isnan(user_ratings))[0]
        if len(rated_users) == 0:
            return self.user_means[user_id]

        # 取最相似的K个用户
        similarities = scores[rated_users]
        top_k_idx = np.argsort(similarities)[-self.n_neighbors:]

        top_k_users = rated_users[top_k_idx]
        top_k_sims = similarities[top_k_idx]

        # 加权平均预测
        numerator = np.sum(top_k_sims * (user_ratings[top_k_users] - self.user_means[top_k_users]))
        denominator = np.sum(np.abs(top_k_sims))

        if denominator == 0:
            return self.user_means[user_id]

        return self.user_means[user_id] + numerator / denominator


# 示例
ratings = np.array([
    [5, 4, np.nan, 2, np.nan],
    [4, np.nan, 5, 3, 3],
    [np.nan, 3, 4, 5, np.nan],
    [2, np.nan, 3, np.nan, 4],
    [np.nan, 4, np.nan, 3, 5],
])

cf = UserBasedCF(n_neighbors=3)
cf.fit(ratings)
print(f"预测用户0对物品4的评分: {cf.predict(0, 4):.2f}")
```

### PyTorch实现Embedding召回

```python
import torch
import torch.nn as nn

class TwoTowerModel(nn.Module):
    """双塔模型：用户塔 + 物品塔"""

    def __init__(self, num_users, num_items, embedding_dim=64, hidden_dim=128):
        super().__init__()

        # 用户塔
        self.user_embedding = nn.Embedding(num_users, embedding_dim)
        self.user_fc = nn.Sequential(
            nn.Linear(embedding_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, embedding_dim)
        )

        # 物品塔
        self.item_embedding = nn.Embedding(num_items, embedding_dim)
        self.item_fc = nn.Sequential(
            nn.Linear(embedding_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, embedding_dim)
        )

    def forward_user(self, user_ids):
        """用户塔前向传播"""
        x = self.user_embedding(user_ids)
        return self.user_fc(x)

    def forward_item(self, item_ids):
        """物品塔前向传播"""
        x = self.item_embedding(item_ids)
        return self.item_fc(x)

    def score(self, user_ids, item_ids):
        """计算用户-物品匹配分数"""
        user_emb = self.forward_user(user_ids)
        item_emb = self.forward_item(item_ids)
        # L2归一化后点积，等价于余弦相似度
        user_emb = torch.nn.functional.normalize(user_emb, dim=-1)
        item_emb = torch.nn.functional.normalize(item_emb, dim=-1)
        return torch.sum(user_emb * item_emb, dim=-1)


# 使用示例
model = TwoTowerModel(num_users=10000, num_items=50000, embedding_dim=64)

# 训练时
user_ids = torch.tensor([123, 456, 789])
item_ids = torch.tensor([1001, 2002, 3003])
scores = model.score(user_ids, item_ids)  # 匹配分数

# Serving时（离线计算所有物品Embedding）
all_item_embeddings = model.forward_item(
    torch.arange(50000)
)  # (50000, 64)

# 在线推理（用户向量查最近邻）
user_emb = model.forward_user(torch.tensor([123]))
# 使用FAISS等库进行ANN检索
```

## 相关概念

- **协同过滤**：基于用户行为数据的推荐方法
- **内容推荐**：基于物品属性的推荐方法
- **矩阵分解**：将高维稀疏矩阵分解为低维稠密矩阵的技术
- **Embedding**：将离散ID映射为稠密向量的技术
- **召回与排序**：推荐系统的两阶段范式
- **多样性推荐**：在准确性和多样性间取得平衡
