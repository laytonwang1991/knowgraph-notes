# 图神经网络进阶

## 一句话定义

图神经网络进阶是在浅层图卷积基础上，通过注意力机制、采样聚合和深层消息传递等技术，实现更强大、更灵活的图结构数据表示学习能力。

## 核心公式

### 1. 图注意力层（GAL）

$$
\alpha_{ij} = \frac{\exp(\text{LeakyReLU}(\mathbf{a}^T[\mathbf{W}\mathbf{h}_i \parallel \mathbf{W}\mathbf{h}_j]))}{\sum_{k \in \mathcal{N}_i} \exp(\text{LeakyReLU}(\mathbf{a}^T[\mathbf{W}\mathbf{h}_i \parallel \mathbf{W}\mathbf{h}_k]))}
$$

其中 $\alpha_{ij}$ 是节点 $i$ 对节点 $j$ 的注意力权重，$\mathbf{W}$ 是权重矩阵，$\mathbf{h}_i$ 是节点 $i$ 的特征向量，$\parallel$ 表示拼接操作。

### 2. GraphSAGE 聚合

$$
\mathbf{h}_i^{(k)} = \sigma(\mathbf{W}^{(k)} \cdot \text{AGG}(\{\mathbf{h}_j^{(k-1)} : j \in \mathcal{N}_i\}))
$$

AGG 可以是：Mean、LSTM、Pool 等聚合函数。

### 3. 消息传递神经网络（MPNN）

$$
\mathbf{m}_i^{(t+1)} = \sum_{j \in \mathcal{N}_i} \mathbf{M}^{(t)}(\mathbf{h}_i^{(t)}, \mathbf{h}_j^{(t)}, \mathbf{e}_{ij})
$$

$$
\mathbf{h}_i^{(t+1)} = \mathbf{U}^{(t)}(\mathbf{h}_i^{(t)}, \mathbf{m}_i^{(t+1)})
$$

## 详细说明

### 1. GAT（注意力图网络）

#### 1.1 核心思想

GAT 将注意力机制引入图卷积，通过学习不同邻居节点的权重，自动聚焦最相关的邻居信息。

#### 1.2 多头注意力机制

通常使用 K 个独立的注意力头，每个头产生一组注意力权重，最终特征为：

$$
\mathbf{h}_i' = \parallel_{k=1}^{K} \sigma\left(\sum_{j \in \mathcal{N}_i} \alpha_{ij}^k \mathbf{W}^k \mathbf{h}_j\right)
$$

#### 1.3 GAT vs GCN

| 特性 | GCN | GAT |
|------|-----|-----|
| 邻居聚合 | 固定权重 $1/\sqrt{d_i d_j}$ | 学习得到的注意力权重 |
| 计算复杂度 | $O(V \cdot E)$ | $O(V \cdot K \cdot E)$ |
| 可解释性 | 低 | 高（注意力权重可解释） |
| 参数量 | 较少 | 较多 |

### 2. GraphSAGE

#### 2.1 采样策略

为处理大图，GraphSAGE 采用邻居采样：

- 1层采样：每节点采样 K1 个邻居
- 2层采样：每节点采样 K2 个邻居
- 总采样数：K1 + K1 × K2

#### 2.2 聚合函数

**Mean 聚合：**

$$
\mathbf{h}_i^{(k)} = \sigma(\mathbf{W} \cdot \text{mean}(\{\mathbf{h}_i^{(k-1)}\} \cup \{\mathbf{h}_j^{(k-1)} : j \in \mathcal{N}_i\}))
$$

**Pool 聚合：**

$$
\mathbf{h}_i^{(k)} = \sigma(\mathbf{W} \cdot \max(\{\mathbf{h}_j^{(k-1)} : j \in \mathcal{N}_i\}))
$$

**LSTM 聚合：**

$$
\mathbf{h}_i^{(k)} = \text{LSTM}(\{\mathbf{h}_j^{(k-1)} : j \in \mathcal{N}_i\})
$$

#### 2.3 归纳学习能力

GraphSAGE 支持 inductive learning，能够泛化到未见过的节点，这是与 Transductive 方法的关键区别。

### 3. 消息传递神经网络（MPNN）

#### 3.1 通用框架

MPNN 将图神经网络的传播分解为消息传递和状态更新两个阶段：

1. **消息阶段**：每个节点计算发送给邻居的消息
2. **更新阶段**：每个节点聚合消息并更新自身状态

#### 3.2 消息函数设计

$$
\mathbf{m}_{ij} = \mathbf{M}(\mathbf{h}_i, \mathbf{h}_j, \mathbf{e}_{ij})
$$

常见消息函数：
- 边特征融合：$\mathbf{m}_{ij} = \mathbf{W}_1 \mathbf{h}_i + \mathbf{W}_2 \mathbf{h}_j + \mathbf{W}_3 \mathbf{e}_{ij}$
- 笛卡尔积：$\mathbf{m}_{ij} = (\mathbf{W}_1 \mathbf{h}_i) \odot (\mathbf{W}_2 \mathbf{h}_j)$

#### 3.3 读取函数

$$
\hat{\mathbf{y}} = \mathbf{R}(\{\mathbf{h}_i^L : i \in V\})
$$

用于图级别的分类/回归任务。

## 代码示例

### PyTorch Geometric 实现 GAT

```python
import torch
import torch.nn.functional as F
from torch_geometric.nn import GATConv

class GAT(torch.nn.Module):
    def __init__(self, in_channels, hidden_channels, out_channels, heads=8):
        super(GAT, self).__init__()
        self.conv1 = GATConv(in_channels, hidden_channels, heads=heads, dropout=0.6)
        self.conv2 = GATConv(hidden_channels * heads, out_channels, heads=1, concat=False, dropout=0.6)

    def forward(self, x, edge_index):
        x = F.dropout(x, p=0.6, training=self.training)
        x = F.elu(self.conv1(x, edge_index))
        x = F.dropout(x, p=0.6, training=self.training)
        x = self.conv2(x, edge_index)
        return x
```

### GraphSAGE 实现

```python
from torch_geometric.nn import SAGEConv

class GraphSAGE(torch.nn.Module):
    def __init__(self, in_channels, hidden_channels, out_channels):
        super(GraphSAGE, self).__init__()
        self.conv1 = SAGEConv(in_channels, hidden_channels)
        self.conv2 = SAGEConv(hidden_channels, out_channels)

    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index).relu()
        x = self.conv2(x, edge_index)
        return x
```

### 自定义消息传递层

```python
from torch_geometric.nn import MessagePassing

class MyMPNN(MessagePassing):
    def __init__(self, in_channels, out_channels):
        super(MyMPNN, self).__init__(aggr='add')
        self.lin = torch.nn.Linear(in_channels, out_channels)

    def forward(self, x, edge_index):
        return self.propagate(edge_index, x=x)

    def message(self, x_j):
        return self.lin(x_j)
```

## 应用场景

### 1. 药物发现

- 分子图表示：原子为节点，化学键为边
- GAT 用于预测分子性质
- MPNN 用于生成新分子结构

### 2. 社交网络分析

- 用户推荐
- 社区检测
- 影响力最大化

### 3. 推荐系统

- 用户-物品交互图建模
- GraphSAGE 用于冷启动问题
- 异构图神经网络处理多模态数据

### 4. 知识图谱

- 知识图谱补全
- 实体关系预测
- 图注意力用于关系路径建模
