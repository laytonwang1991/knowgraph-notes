# 视频理解（Video Understanding）

## 一句话定义

视频理解是让AI模型能够感知、解析和推理视频中时空联合信息的技术，涵盖动作识别、事件检测、时序定位等核心任务。

## 核心公式

### 时空特征表示

$$F_{video} = \mathcal{T}(\mathcal{S}(I_1), \mathcal{S}(I_2), ..., \mathcal{S}(I_T))$$

其中 $\mathcal{S}$ 为空间特征提取器，$\mathcal{T}$ 为时序聚合模块，$I_t$ 为第 $t$ 帧。

### 3D卷积特征

$$y_{t,i,j,k} = \sum_{d,u,v} w_{d,u,v} \cdot x_{t+d, i+u, j+v} \cdot h_{d,u,v} + b$$

### Video Transformer自注意力

$$Attention(Q, K, V) = softmax\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

## 详细说明

### 1. 视频理解的挑战

- **时空联合建模**：同时捕获空间语义与时间动态
- **长程依赖**：视频帧间存在远距离关联
- **计算效率**：视频数据量庞大，处理成本高
- **多尺度问题**：动作可能在不同时间尺度发生

### 2. 核心技术路线

#### (1) 2D CNN + 时序聚合

- 使用预训练2D CNN提取单帧特征
- 通过LSTM/Transformer聚合时序信息
- 代表模型：LRCN、TSN

#### (2) 3D CNN

- 将2D卷积扩展到3D（时间作为第三维）
- 端到端学习时空特征
- 代表模型：I3D、C3D、P3D、R(2+1)D

#### (3) Video Transformer

- 利用自注意力捕获长程依赖
- 近年主流方案
- 代表模型：TimeSformer、ViViT、Video Swin Transformer

#### (4) 多模态融合

- 结合视觉、音频、文本等多种模态
- 提升理解深度与准确性

### 3. 动作识别主流架构

| 架构 | 特点 | 典型模型 |
|------|------|----------|
| 双流网络 | 光流+RGB双分支 | Two-Stream CNN |
| 3D CNN | 时空联合卷积 | I3D, C3D |
| Transformer | 全局注意力建模 | TimeSformer |
| 效率优先 | 分解时空注意力 | MViT, SlowFast |

### 4. 时序建模方法

- **RNN/LSTM**：适合序列建模，但难以捕获长距离依赖
- ** Temporal Pooling**：简单平均/最大池化，信息损失大
- **Temporal Attention**：动态权重聚合关键帧
- **Transformer**：全局自注意力，最新主流方案

## 代码示例

### PyTorch 3D卷积实现

```python
import torch
import torch.nn as nn

class C3D(nn.Module):
    """C3D模型核心实现"""
    def __init__(self, num_classes=400):
        super(C3D, self).__init__()
        self.conv1 = nn.Conv3d(3, 64, kernel_size=3, padding=1)
        self.pool1 = nn.MaxPool3d(kernel_size=(1, 2, 2), stride=(1, 2, 2))

        self.conv2 = nn.Conv3d(64, 128, kernel_size=3, padding=1)
        self.pool2 = nn.MaxPool3d(kernel_size=(2, 2, 2), stride=(2, 2, 2))

        self.conv3a = nn.Conv3d(128, 256, kernel_size=3, padding=1)
        self.conv3b = nn.Conv3d(256, 256, kernel_size=3, padding=1)
        self.pool3 = nn.MaxPool3d(kernel_size=(2, 2, 2), stride=(2, 2, 2))

        self.conv4a = nn.Conv3d(256, 512, kernel_size=3, padding=1)
        self.conv4b = nn.Conv3d(512, 512, kernel_size=3, padding=1)
        self.pool4 = nn.MaxPool3d(kernel_size=(2, 2, 2), stride=(2, 2, 2))

        self.conv5a = nn.Conv3d(512, 512, kernel_size=3, padding=1)
        self.conv5b = nn.Conv3d(512, 512, kernel_size=3, padding=1)
        self.pool5 = nn.MaxPool3d(kernel_size=(2, 2, 2), stride=(2, 2, 2))

        self.fc6 = nn.Linear(8192, 4096)
        self.fc7 = nn.Linear(4096, 4096)
        self.fc8 = nn.Linear(4096, num_classes)

        self.dropout = nn.Dropout(p=0.5)
        self.relu = nn.ReLU()

    def forward(self, x):
        # x: [batch, channels, depth, height, width]
        x = self.pool1(self.relu(self.conv1(x)))
        x = self.pool2(self.relu(self.conv2(x)))

        x = self.relu(self.conv3a(x))
        x = self.pool3(self.relu(self.conv3b(x)))

        x = self.relu(self.conv4a(x))
        x = self.pool4(self.relu(self.conv4b(x)))

        x = self.relu(self.conv5a(x))
        x = self.pool5(self.relu(self.conv5b(x)))

        x = x.view(x.size(0), -1)
        x = self.dropout(self.relu(self.fc6(x)))
        x = self.dropout(self.relu(self.fc7(x)))
        x = self.fc8(x)
        return x
```

### Video Transformer (TimeSformer风格)

```python
import torch
import torch.nn as nn
from einops import rearrange

class SpaceTimeAttention(nn.Module):
    """分离时空注意力"""
    def __init__(self, dim, num_heads=8, qkv_bias=True):
        super().__init__()
        self.num_heads = num_heads
        self.head_dim = dim // num_heads
        self.scale = self.head_dim ** -0.5

        self.qkv = nn.Linear(dim, dim * 3, bias=qkv_bias)
        self.proj = nn.Linear(dim, dim)

    def forward(self, x):
        # x: [B, T, H*W, C]
        B, T, S, C = x.shape

        qkv = self.qkv(x).reshape(B, T, S, 3, self.num_heads, self.head_dim)
        qkv = qkv.permute(3, 0, 4, 1, 2, 5)  # [3, B, heads, T, S, head_dim]
        q, k, v = qkv[0], qkv[1], qkv[2]

        # 时间注意力
        attn_t = (q @ k.transpose(-2, -1)) * self.scale
        attn_t = attn_t.softmax(dim=-1)
        out_t = attn_t @ v  # [B, heads, T, S, head_dim]

        # 空间注意力
        out_t = out_t.transpose(2, 3).reshape(B, T, S, C)
        qkv_s = self.qkv(out_t).reshape(B, T, S, 3, self.num_heads, self.head_dim)
        q_s, k_s, v_s = qkv_s[0], qkv_s[1], qkv_s[2]

        attn_s = (q_s @ k_s.transpose(-2, -1)) * self.scale
        attn_s = attn_s.softmax(dim=-1)
        out_s = attn_s @ v_s

        out_s = out_s.transpose(2, 3).reshape(B, T, S, C)
        return self.proj(out_s)
```

### 视频数据集加载

```python
import torch.utils.data as data
from torchvision.io import read_video

class VideoDataset(data.Dataset):
    def __init__(self, video_paths, transform=None, num_frames=16):
        self.video_paths = video_paths
        self.transform = transform
        self.num_frames = num_frames

    def __getitem__(self, idx):
        path = self.video_paths[idx]
        video, _, _ = read_video(path, pts_unit='sec')

        # 均匀采样帧
        total_frames = len(video)
        indices = torch.linspace(0, total_frames - 1, self.num_frames).long()
        frames = video[indices]  # [T, H, W, C]

        if self.transform:
            frames = self.transform(frames)

        return frames.permute(0, 3, 1, 2).float() / 255.0

    def __len__(self):
        return len(self.video_paths)
```

## 应用场景

### 1. 视频理解核心应用

- **动作识别**：体育动作分类、人体行为检测（如K400、UCF101）
- **视频描述**：生成视频内容的文本描述
- **视频问答**：针对视频内容回答问题
- **异常检测**：监控视频中的异常事件
- **视频摘要**：提取视频关键片段生成摘要

### 2. 行业应用

| 行业 | 应用场景 |
|------|----------|
| 安防监控 | 异常行为检测、人流统计 |
| 自动驾驶 | 道路场景理解、行为预测 |
| 医疗健康 | 手术视频分析、康复动作评估 |
| 内容审核 | 色情/暴力内容检测 |
| 体育分析 | 比赛动作解析、战术分析 |
| 教育 | 课堂行为分析、教学视频检索 |

## 相关概念

- **光流（Optical Flow）**：描述像素运动变化的二维矢量场，是视频理解的重要特征
- **时序建模（Temporal Modeling）**：捕获视频帧间动态关系的技术
- **动作识别（Action Recognition）**：识别视频中人类动作类别的任务
- **时空特征（Spatiotemporal Feature）**：融合空间位置与时间维度的联合特征表示
- **双流网络（Two-Stream Network）**：分别处理空间与光流信息后融合的网络架构
- **3D卷积（3D Convolution）**：在时间维度上扩展的卷积操作，同时提取时空特征

3. **SlowFast Networks for Video Recognition** (2019)
   - 双路径架构分别捕获慢速语义与快速运动

   - 在视频上应用Swin Transformer结构

### 进阶学习路径

1. 基础：CNN/RNN时序模型 → 2D CNN + LSTM
2. 进阶：3D CNN架构 → I3D/ C3D/ R(2+1)D
3. 核心：Video Transformer → TimeSformer / ViViT / MViT
4. 前沿：大规模预训练视频模型 → VideoPaLM, PyVideo-1B
