# 视觉问答（Visual Question Answering, VQA）

## 一句话定义

视觉问答（VQA）是指给定一张图像和一个关于该图像的问题，AI系统需要理解图像内容并生成准确答案的多模态任务。

## 核心公式

### 1. VQA任务形式化
$$\hat{a} = \arg\max_{a \in \mathcal{A}} P(a|I, Q; \theta)$$

其中 $I$ 是图像，$Q$ 是问题，$\mathcal{A}$ 是答案空间，$\theta$ 是模型参数。

### 2. 早期融合模型（Early Fusion）
$$P(a|I, Q) = \text{Softmax}(W_2 \cdot \text{MLP}(W_1 \cdot [\mathbf{v}; \mathbf{q}] + b_1) + b_2)$$

其中 $\mathbf{v} = f_v(I)$ 是图像特征，$\mathbf{q} = f_q(Q)$ 是问题特征，$[\cdot; \cdot]$ 表示拼接。

### 3. 注意力融合（Attention-based Fusion）
$$\alpha_i = \text{Attention}(Q, v_i) = \frac{\exp(\mathbf{q}^T \mathbf{W}_a \mathbf{v}_i)}{\sum_j \exp(\mathbf{q}^T \mathbf{W}_a \mathbf{v}_j)}$$
$$\mathbf{v}_{\text{att}} = \sum_i \alpha_i \mathbf{v}_i$$

### 4. 双线性注意力（Bilinear Attention）
$$\mathbf{F} = \mathbf{V}^T \mathbf{W}_a \mathbf{Q}, \quad \mathbf{V}_{\text{att}} = \mathbf{V} \cdot \text{Softmax}(\mathbf{F})$$

### 5. 大型多模态模型（LMM）范式
$$P(a|I, Q) = \text{LM}(a | \text{VisionEncoder}(I), \text{LLM}(Q))$$

其中LM是大语言模型，负责根据视觉特征和文本生成答案。

## 详细说明

### VQA发展历程

- **2014-2016**：CNN+LSTM时代 - 使用预训练CNN提取图像特征，RNN编码问题
- **2017-2019**：注意力机制崛起 - SAN, MCAN, DFFN, 引入视觉注意力
- **2020-2021**：Transformer时代 - VilBERT, LXMERT, UNITER
- **2022-2024**：大型多模态模型 - GPT-4V, Gemini, LLaVA, Qwen-VL

### VQA数据集

| 数据集 | 问题类型 | 图像数量 | 特点 |
|--------|---------|---------|------|
| VQA v1 | 开放域 | 204K | 通用问答 |

### VQA问题类型

1. **开放域问题**
   - "这张图片里有什么？"
   - "描述这个场景"

2. **属性识别**
   - "这个物体是什么颜色？"
   - "有多少个XX？"

3. **空间关系**
   - "XX在YY的左边还是右边？"
   - "什么在前景/背景？"

4. **动作识别**
   - "人们在做什么？"
   - "这个人在吃什么？"

5. **知识推理**
   - "这是什么历史事件？"
   - "这个建筑是什么风格的？"

6. **文本阅读（OCR）**
   - "图片中的文字是什么？"
   - "表格里的数据是多少？"

### 模型架构演进

#### 早期：CNN+LSTM+融合
```
图像 → ResNet → 视觉特征 → 融合 → MLP → 答案
问题 → LSTM → 文本特征 ↗
```

#### 中期：双流+注意力
```
图像 → 视觉编码器 → 多头注意力 ↗
                                    → 跨模态注意力 → 答案
问题 → 文本编码器 → 多头注意力 ↗
```

#### 当前：大型多模态模型
```
图像 → Vision Encoder → 视觉Token → LLM → 文本答案
问题 → Tokenizer → 文本Token → ↗
```

### 关键技术

1. **视觉特征提取**
   - 早期：ResNet, VGG
   - 当前：CLIP Vision, SigLIP, DINOv2

2. **跨模态对齐**
   - 对比学习：CLIP, ALIGN
   - 融合学习：Flamingo, BLIP-2

3. **大型多模态连接器**
   - MLP投影器
   - Q-Former（BLIP-2）
   - Projector+适配器（LLaVA）

4. **上下文学习**
   - Few-shot示例
   - Chain-of-Thought推理

## 代码示例

### 基础VQA模型（PyTorch）

```python
import torch
import torch.nn as nn
from transformers import BertModel, BertTokenizer

class SimpleVQA(nn.Module):
    """简化版VQA模型：早期融合"""

    def __init__(self, num_answers, vocab_size=30522):
        super().__init__()
        # 图像编码器
        self.image_encoder = nn.Sequential(
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(),
            nn.AdaptiveAvgPool2d((7, 7))
        )

        # 文本编码器
        self.text_encoder = BertModel.from_pretrained('bert-base-uncased')
        self.text_proj = nn.Linear(768, 512)

        # 融合层
        self.fusion = nn.Sequential(
            nn.Linear(1024, 1024),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(1024, 512),
            nn.ReLU()
        )

        # 答案分类器
        self.classifier = nn.Linear(512, num_answers)

    def forward(self, image_features, input_ids, attention_mask):
        # 图像编码
        v = self.image_encoder(image_features)  # [B, 512, 7, 7]
        v = v.flatten(2).transpose(1, 2)  # [B, 49, 512]
        v = v.mean(dim=1)  # [B, 512]

        # 文本编码
        q = self.text_encoder(input_ids=input_ids, attention_mask=attention_mask)
        q = q.last_hidden_state[:, 0]  # [B, 768]
        q = self.text_proj(q)  # [B, 512]

        # 融合
        fused = torch.cat([v, q], dim=1)  # [B, 1024]
        fused = self.fusion(fused)

        # 分类
        logits = self.classifier(fused)

        return logits
```

### 使用Hugging Face进行VQA

```python
from transformers import pipeline
from PIL import Image
import requests

# 加载VQA pipeline
vqa_pipeline = pipeline(
    "visual-question-answering",
    model="Salesforce/blip-vqa-base",
    device="cuda"

image = Image.open("example.jpg")

### 使用LLaVA进行复杂VQA

```python
from llava.model.builder import load_pretrained_model
from llava.mm_utils import process_images, tokenizer_image_token
from llava.constants import IMAGE_TOKEN, DEFAULT_IMAGE_TOKEN
import torch

# 加载LLaVA模型
model_path = "liuhaotian/llava-v1.5-7b"
model, tokenizer, image_processor, context_len = load_pretrained_model(
    model_path=model_path,
    model_base=None,
    model_name="llava-v1.5-7b",
    device="cuda"
)

def ask_about_image(image_path: str, question: str):
    """使用LLaVA回答关于图像的问题"""

    # 加载和处理图像
    image = Image.open(image_path).convert('RGB')
    image_tensor = process_images([image], image_processor)[0]

    # 构建提示
    prompt = f"{DEFAULT_IMAGE_TOKEN}\nQuestion: {question}\nAnswer:"

    # tokenize
    input_ids = tokenizer_image_token(
        prompt, tokenizer, IMAGE_TOKEN, return_tensors='pt'
    ).unsqueeze(0).cuda()

    # 生成答案
    with torch.no_grad():
        output_ids = model.generate(
            input_ids,
            images=image_tensor.unsqueeze(0).half().cuda(),
            max_new_tokens=100,
            do_sample=False
        )

    # 解码
    answer = tokenizer.decode(output_ids[0], skip_special_tokens=True).strip()

    return answer

# 使用示例
answer = ask_about_image(
    "photo.jpg",
    "Describe what you see in this image and explain what might happen next."
)
print(f"答案: {answer}")
```

### GPT-4V API调用

```python
from openai import OpenAI
from PIL import Image
import base64
import io

client = OpenAI(api_key="your-api-key")

def ask_gpt4v(image_path: str, question: str) -> str:
    """使用GPT-4V进行视觉问答"""

    # 读取并编码图像
    with open(image_path, "rb") as img_file:
        image_data = base64.b64encode(img_file.read()).decode('utf-8')

    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {
                "role": "user",
                "content": [
                    {
                        "type": "image_url",
                        "image_url": {
                            "url": f"data:image/jpeg;base64,{image_data}"
                        }
                    },
                    {
                        "type": "text",
                        "text": question
                    }
                ]
            }
        ],
        max_tokens=500
    )

    return response.choices[0].message.content

# 使用示例
answer = ask_gpt4v("scene.jpg", "What unusual activity can you spot in this image?")
print(answer)
```

### 多轮对话VQA

```python
class MultiTurnVQA:
    """多轮对话式视觉问答"""

    def __init__(self, model_name="Salesforce/blip-vqa-base"):
        self.pipeline = pipeline("visual-question-answering", model=model_name)
        self.history = {}  # 按图像路径存储历史

    def ask(self, image_path: str, question: str, session_id: str = "default"):
        # 加载图像
        image = Image.open(image_path)

        # 生成答案
        answer = self.pipeline(image, question)["answer"]

        # 更新历史
        if session_id not in self.history:
            self.history[session_id] = []
        self.history[session_id].append({
            "question": question,
            "answer": answer
        })

        return answer

    def get_history(self, session_id: str = "default"):
        return self.history.get(session_id, [])

    def clear_history(self, session_id: str = "default"):
        if session_id in self.history:
            del self.history[session_id]
