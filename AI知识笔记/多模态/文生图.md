# 文生图（Text-to-Image）

## 一句话定义

文生图（Text-to-Image）是指通过文本描述生成对应图像的深度学习技术，是多模态AI领域的核心任务之一。

## 核心公式

### 1. 扩散模型前向过程
$$q(\mathbf{x}_{1:T}|\mathbf{x}_0) = \prod_{t=1}^{T} q(\mathbf{x}_t|\mathbf{x}_{t-1}), \quad q(\mathbf{x}_t|\mathbf{x}_{t-1}) = \mathcal{N}(\mathbf{x}_t; \sqrt{1-\beta_t}\mathbf{x}_{t-1}, \beta_t\mathbf{I})$$

### 2. 扩散模型反向过程（去噪）
$$p_\theta(\mathbf{x}_{0:T}) = p(\mathbf{x}_T)\prod_{t=1}^{T} p_\theta(\mathbf{x}_{t-1}|\mathbf{x}_t), \quad p_\theta(\mathbf{x}_{t-1}|\mathbf{x}_t) = \mathcal{N}(\mathbf{x}_{t-1}; \mu_\theta(\mathbf{x}_t, t), \Sigma_\theta(\mathbf{x}_t, t))$$

### 3. 条件生成（Classifier-Free Guidance）
$$\hat{\epsilon}_\theta(\mathbf{x}_t, t, c) = \epsilon_\theta(\mathbf{x}_t, t) + w \cdot (\epsilon_\theta(\mathbf{x}_t, t, c) - \epsilon_\theta(\mathbf{x}_t, t))$$

其中 $w$ 为引导强度，$c$ 为文本条件。

### 4. UNet噪声预测
$$\epsilon_\theta(\mathbf{x}_t, t, c) = \text{UNet}(\mathbf{x}_t, t, \text{CLIP}(c))$$

### 关键技术组件

1. **文本编码器**：通常使用CLIP text encoder或大型语言模型（如T5、PaLM）
2. **生成模型**：主要采用扩散模型（Diffusion Model）或GAN
3. **图像解码器**：将潜在表示解码为像素图像
4. **条件机制**：将文本信息融入生成过程

### 训练目标

扩散模型的训练目标是最小化噪声预测损失：

$$\mathcal{L} = \mathbb{E}_{t, \mathbf{x}_0, \epsilon, c}\left[\|\epsilon - \epsilon_\theta(\sqrt{\bar{\alpha}_t}\mathbf{x}_0 + \sqrt{1-\bar{\alpha}_t}\epsilon, t, c)\|^2\right]$$

## 代码示例

### 使用Hugging Face Diffusers生成图像

```python
from diffusers import StableDiffusionPipeline, DPMSolverMultistepScheduler
import torch

# 加载模型
pipe = StableDiffusionPipeline.from_pretrained(
    "stabilityai/stable-diffusion-2-1",
    torch_dtype=torch.float16,
    safety_checker=None
)

# 优化调度器
pipe.scheduler = DPMSolverMultistepScheduler.from_config(
    pipe.scheduler.config,
    use_karras_sigmas=True
)

# 移动到GPU
pipe = pipe.to("cuda")

# 生成图像
prompt = "A majestic tiger in a bamboo forest, photorealistic, 8k"
image = pipe(
    prompt,
    num_inference_steps=25,
    guidance_scale=7.5,
    height=512,
    width=512
).images[0]

# 保存图像
image.save("tiger.png")
```

### 自定义扩散模型采样

```python
import torch
from diffusers import DDPMScheduler, UNet2DConditionModel
from transformers import CLIPTextModel, CLIPTokenizer

def generate_with_custom_guidance(
    pipe,
    prompt,
    num_inference_steps=50,
    guidance_scale=7.5,
    negative_prompt=""
):
    # 编码文本
    text_input = pipe.tokenizer(
        prompt,
        padding="max_length",
        max_length=pipe.tokenizer.model_max_length,
        truncation=True,
        return_tensors="pt"
    )

    with torch.no_grad():
        text_embeddings = pipe.text_encoder(
            text_input.input_ids.to(pipe.device)
        ).last_hidden_state

    # 负向提示词编码
    if negative_prompt:
        uncond_input = pipe.tokenizer(
            [negative_prompt],
            padding="max_length",
            max_length=pipe.tokenizer.model_max_length,
            return_tensors="pt"
        )
        with torch.no_grad():
            uncond_embeddings = pipe.text_encoder(
                uncond_input.input_ids.to(pipe.device)
            ).last_hidden_state
    else:
        uncond_embeddings = None

    # 初始化潜在变量
    latents = torch.randn(
        (1, pipe.unet.in_channels, 64, 64),
        device=pipe.device,
        dtype=text_embeddings.dtype
    )

    # DDPM采样
    pipe.scheduler.set_timesteps(num_inference_steps)

    for t in pipe.scheduler.timesteps:
        # 扩展潜在变量以同时预测
        latent_model_input = torch.cat([latents] * 2)

        with torch.no_grad():
            if uncond_embeddings is not None:
                text_embeddings_combined = torch.cat([uncond_embeddings, text_embeddings])
            else:
                text_embeddings_combined = torch.cat([text_embeddings, text_embeddings])

        # 预测噪声
        noise_pred = pipe.unet(
            latent_model_input,
            t,
            encoder_hidden_states=text_embeddings_combined
        ).sample

        # 执行去噪步骤
        latents = pipe.scheduler.step(noise_pred, t, latents).prev_sample

    return latents
```

### 使用ComfyUI工作流逻辑（Python模拟）

```python
class TextToImagePipeline:
    """简化版文生图流程"""

    def __init__(self, model_path):
        self.clip = CLIPModel.load(model_path)
        self.unet = UNetModel.load(model_path)
        self.vae = VAEModel.load(model_path)

    def __call__(self, prompt, cfg_scale=7.0, steps=20):
        # 1. 文本编码
        text_tokens = self.clip.encode_text(prompt)

        # 2. 初始化噪声
        latents = torch.randn(1, 4, 64, 64)

        # 3. 迭代去噪
        for step in range(steps):
            t = 1.0 - step / steps  # 时间步从1到0

            # 预测噪声（有条件 + 无条件）
            noise_cond = self.unet(latents, t, text_tokens)
            noise_uncond = self.unet(latents, t, None)

            # CFG合并
            noise = noise_uncond + cfg_scale * (noise_cond - noise_uncond)

            # 一步去噪
            latents = self.denoise_step(latents, noise, t)

        # 4. VAE解码
        image = self.vae.decode(latents)

        return image
```

## 相关概念

### 核心概念
- **扩散模型（Diffusion Model）**：通过逐步加噪和去噪学习数据分布的生成模型
- **CLIP（Contrastive Language-Image Pre-training）**：连接文本和图像的多模态模型
- **潜在扩散模型（Latent Diffusion Model）**：在压缩的潜在空间中进行扩散，更高效
- **CFG（Classifier-Free Guidance）**：无分类器的条件引导技术

### 相关技术
- **ControlNet**：通过额外条件控制生成过程
- **LoRA（Low-Rank Adaptation）**：轻量级微调技术
- **DreamBooth**：个性化主题微调
- **IP-Adapter**：图像提示注入

### 对比学习
- **文生图 vs 图生图**：前者完全从文本生成，后者基于现有图像改造
- **GAN vs 扩散模型**：GAN训练不稳定但采样快，扩散模型训练稳定但采样慢
- **闭源 vs 开源**：DALL-E/Midjourney闭源，Stable Diffusion开源可本地部署
