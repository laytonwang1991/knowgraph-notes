# 3D视觉（3D Computer Vision）

## 一句话定义

3D视觉是让AI系统能够感知、重建和理解三维空间结构的技术，通过从2D图像或多传感器数据中恢复场景的几何、外观与语义信息。

## 核心公式

### 针孔相机模型

$$s \begin{bmatrix} u \\ v \\ 1 \end{bmatrix} = K \begin{bmatrix} R & t \end{bmatrix} \begin{bmatrix} X \\ Y \\ Z \\ 1 \end{bmatrix}$$

其中 $K$ 为内参矩阵，$[R|t]$ 为外参矩阵。

### NeRF隐式表示

$$F_\theta: (\mathbf{x}, \mathbf{d}) \rightarrow (\mathbf{c}, \sigma)$$

输入3D位置 $\mathbf{x}=(x,y,z)$ 和视角方向 $\mathbf{d}=(\theta,\phi)$，输出颜色 $\mathbf{c}$ 和密度 $\sigma$。

### 神经辐射场的体渲染

$$C(\mathbf{r}) = \sum_{i=1}^{N} T_i \cdot \alpha_i \cdot \mathbf{c}_i, \quad T_i = \prod_{j=1}^{i-1}(1 - \alpha_j)$$

其中 $\alpha_i = 1 - e^{-\sigma_i \delta_i}$，$\delta_i$ 为采样间隔。

### 深度估计（视差与深度关系）

$$depth = \frac{f \cdot B}{dispersion}$$

其中 $f$ 为焦距，$B$ 为基线距离。

## 详细说明

### 1. 3D视觉的任务类型

- **3D重建**：从多视角恢复场景几何结构
- **深度估计**：预测每个像素对应的深度值
- **表面重建**：恢复物体或场景的连续表面
- **3D目标检测**：定位和识别三维空间中的目标
- **场景理解**：理解三维空间的语义布局

### 2. 3D表示方法对比

| 表示方法 | 优点 | 缺点 | 典型应用 |
|----------|------|------|----------|
| **深度图** | 存储紧凑、2D兼容 | 只适用单一视角 | 单目深度估计 |
| **体素** | 结构化、易处理 | 内存立方增长 $O(n^3)$ | 3D语义分割 |
| **点云** | 简单灵活 | 无拓扑结构 | 3D目标检测 |
| **网格** | 紧凑、拓扑清晰 | 拓扑学习困难 | 3D生成 |
| **隐式表示** | 分辨率无关 | 渲染计算量大 | NeRF |
| **多平面图像(MSI)** | 高效推理 | 限定视角范围 | Immersive视频 |

### 3. NeRF技术体系

#### 3.1 经典NeRF

- 使用MLP将3D位置和视角映射到颜色和密度
- 通过体渲染合成新视角图像
- 缺点：训练慢、无法建模大型场景、无法处理动态物体

#### 3.2 NeRF改进方向

| 改进方向 | 代表工作 | 核心贡献 |
|----------|----------|----------|
| 训练效率 | Instant NGP | 使用Hash编码+GPU并行 |
| 大规模场景 | MegaNeRF / Block-NeRF | 空间分块+动态加载 |
| 动态场景 | DyanmicNeRF / Nerfies | 添加时间维度 |
| 编辑能力 | EditNeRF | 条件生成+语义编辑 |
| 几何先验 | Depth-supervised NeRF | 深度监督加速收敛 |

#### 3.3 体渲染实现

```python
def render_volume_density_density(volume_density, ray_dir, ray_origin, near, far, num_samples=128):
    """
    体渲染公式: C = sum(T_i * alpha_i * c_i)
    """
    t_vals = torch.linspace(near, far, num_samples)
    points = ray_origin + ray_dir.unsqueeze(0) * t_vals.unsqueeze(-1)

    densities, colors = evaluate_network(points, ray_dir)

    delta = t_vals[1:] - t_vals[:-1]
    alpha = 1.0 - torch.exp(-densities * delta)

    T = torch.cumprod(1.0 - alpha + 1e-10, dim=-1)
    T = torch.cat([torch.ones_like(T[..., :1]), T[..., :-1]], dim=-1)

    weights = T * alpha
    rgb = torch.sum(weights.unsqueeze(-1) * colors, dim=-2)

    return rgb, weights
```

### 4. 深度估计方法

#### 4.1 单目深度估计

$$depth_{pred} = \mathcal{F}_\theta(Image)$$

- **监督学习**：需要真实深度标签
- **自监督**：利用时序/双目约束
- **有符号距离函数(SDF)**：预测SDF而非直接深度

#### 4.2 双目/多目深度估计

- **立体匹配**：寻找左右图像对应点
- **多视角融合**：从多个视角综合深度信息

### 5. 点云处理

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class PointNet(nn.Module):
    """PointNet: 点云特征提取的经典架构"""
    def __init__(self, global_feat=True, feature_transform=True):
        super(PointNet, self).__init__()
        self.global_feat = global_feat
        self.feature_transform = feature_transform

        self.conv1 = nn.Conv1d(3, 64, 1)
        self.conv2 = nn.Conv1d(64, 128, 1)
        self.conv3 = nn.Conv1d(128, 1024, 1)

        self.bn1 = nn.BatchNorm1d(64)
        self.bn2 = nn.BatchNorm1d(128)
        self.bn3 = nn.BatchNorm1d(1024)

        if feature_transform:
            self.conv4 = nn.Conv1d(1024, 512, 1)
            self.conv5 = nn.Conv1d(512, 256, 1)
            self.bn4 = nn.BatchNorm1d(512)
            self.bn5 = nn.BatchNorm1d(256)

    def feature_transform_net(self, x):
        n = x.size(2)
        net = torch.matmul(x, x.transpose(2, 1))
        net = net / n
        y = torch.abs(net)
        return y

    def forward(self, x):
        # x: [B, 3, N] - N个点
        x = F.relu(self.bn1(self.conv1(x)))
        x = F.relu(self.bn2(self.conv2(x)))
        x = F.relu(self.bn3(self.conv3(x)))
        x = torch.max(x, 2, keepdim=True)[0]  # 全局特征
        x = x.view(-1, 1024)

        if self.feature_transform:
            trans_feat = self.feature_transform_net(x)
            x = trans_feat

        return x
```

```python
class VoxelNet(nn.Module):
    """3D目标检测的体素编码器"""
    def __init__(self, in_channels=4, voxel_size=[0.4, 0.4, 0.4],
                 point_cloud_range=[0, -40, -3, 70.4, 40, 1]):
        super().__init__()
        self.voxel_size = voxel_size
        self.point_cloud_range = point_cloud_range

        self.conv3d_layers = nn.Sequential(
            nn.Conv3d(in_channels, 16, 3, padding=1),
            nn.BatchNorm3d(16),
            nn.ReLU(),
            nn.Conv3d(16, 32, 3, padding=1),
            nn.BatchNorm3d(32),
            nn.ReLU(),
            nn.Conv3d(32, 64, 3, padding=1),
            nn.BatchNorm3d(64),
            nn.ReLU(),
        )
        self.middle_layer = nn.Sequential(
            nn.Conv3d(64, 64, 3, padding=1),
            nn.BatchNorm3d(64),
            nn.ReLU(),
        )
        self.out_channel = 64

    def forward(self, voxels, num_points, coordinates):
        # voxels: [M, max_points, C]
        # coordinates: [M, 4] (batch_idx, z, y, x)
        batch_size = int(coordinates[:, 0].max()) + 1

        voxel_feature = voxels.mean(dim=1)  # [M, C]
        spatial_feature = torch.zeros(
            batch_size, self.out_channel,
            *self.vol_shape, device=voxels.device
        )

        return spatial_feature
```

## 相关概念

- **NeRF（神经辐射场）**：使用神经网络隐式表示3D场景的方法，通过体渲染合成新视角
- **SLAM（即时定位与地图构建）**：机器人在未知环境中同时进行定位与地图构建
- **多视角几何（Multi-View Geometry）**：从多张2D图像恢复3D结构的几何原理
- **SDF（有符号距离函数）**：用连续函数表示物体表面的方法
- **点云配准（Point Cloud Registration）**：将不同视角的点云对齐到统一坐标系
- **BA（束调整）**：联合优化相机参数与3D点的全局优化算法
