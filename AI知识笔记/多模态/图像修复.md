# 图像修复（Image Inpainting）

## 一句话定义

图像修复是利用AI技术自动推断并补全图像中缺失、损坏或被遮挡区域的技术，使修复后的图像在视觉上连贯自然、符合上下文语义。

## 核心公式

### 传统方法：基于偏微分方程（PDE）

$$I_{completed}(x,y) = I_{known}(x,y) + \nabla^2 I(x,y) \cdot \delta t$$

基于热传导方程，边缘信息向缺失区域扩散。

### Context Encoder损失函数

$$\mathcal{L} = \mathcal{L}_{reconstruction} + \lambda_1 \mathcal{L}_{adversarial} + \lambda_2 \mathcal{L}_{perceptual}$$

其中重建损失保证像素级准确性，对抗损失保证语义一致性，感知损失提升视觉质量。

### 扩散模型修复

$$p_\theta(x_{0:M}|y, m) = p(x_M) \prod_{t=1}^{M} p_\theta(x_{t-1}|x_t, y, m)$$

其中 $y$ 为条件图像，$m$ 为掩码，$M$ 为去噪步数。

### LaMa感知显著性掩码

$$m_{salient} = \mathcal{S}(I_{input}) = \sigma(W_{salient} \cdot I_{input} + b_{salient})$$

通过感知显著性图引导修复过程，优先保持显著区域完整性。

## 详细说明

### 1. 图像修复的核心挑战

- **语义一致性**：补全区域需与周围上下文语义匹配
- **纹理细节**：自然纹理与结构的保持
- **边缘平滑**：修复区域与原图边缘的无缝过渡
- **大规模缺失**：当缺失区域较大时的合理推断
- **结构连贯性**：线条、边缘等几何结构的自然延伸

### 2. 技术发展路线

#### (1) 传统方法（2010年前）

| 方法 | 原理 | 局限性 |
|------|------|--------|
| Biharmonic插值 | 基于边界样条的数学插值 | 仅适合小面积缺失 |
| BSCB模型 | 偏微分方程驱动扩散 | 模糊、无法处理语义 |
| Exemplar方法 | 从图像其他区域复制纹理块 | 结构断裂、重复纹理 |

#### (2) 深度学习早期（2016-2019）

**Context Encoder (CVPR 2016)**

- 首次端到端深度学习方法
- 编码器-解码器架构 + GAN
- 通道级全连接层捕获全局上下文

**LaMa (ICCV 2019)**

- 引入感知显著性掩码
- Fast Fourier Convolutions全局感受野
- 在大面积缺失上效果显著

**MAT (AAAI 2021)**

- Multi-modal Attention Transformer
- 更好地建模长程依赖
- 纹理细节更清晰

#### (3) 扩散模型时代（2022至今）

- **LDM (Latent Diffusion Models)**：在潜空间进行扩散，适合高分辨率
- **Stable Diffusion Inpainting**：开源方案，生态成熟
- **DALLE-3 / Imagen**：大模型加持，语义理解更强
- **ControlNet + Inpainting**：结合边缘/深度等条件控制

### 3. 主流架构对比

| 架构 | 年份 | 核心创新 | 适用场景 |
|------|------|----------|----------|
| Context Encoder | 2016 | 端到端+GAN | 中等缺失 |
| GntI2I | 2019 | 无监督学习 | 域迁移 |
| LaMa | 2019 | FFT卷积+显著掩码 | 大面积缺失 |
| MAT | 2021 | Transformer注意力 | 高质量细节 |
| SD Inpainting | 2022+ | 潜空间扩散 | 通用场景 |
| PowerPaint | 2023 | 条件控制 | 精细控制 |

### 4. 代码实现

#### 4.1 Context Encoder核心结构

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class ContextEncoder(nn.Module):
    """
    Context Encoder: 编码器-解码器结构 + 全局上下文感知
    """
    def __init__(self, in_channels=3, latent_dim=4000):
        super().__init__()

        # 编码器
        self.encoder = nn.Sequential(
            nn.Conv2d(in_channels, 64, 4, stride=2, padding=1),  # 64x64 -> 32x32
            nn.LeakyReLU(0.2),
            nn.Conv2d(64, 128, 4, stride=2, padding=1),  # 32x32 -> 16x16
            nn.BatchNorm2d(128),
            nn.LeakyReLU(0.2),
            nn.Conv2d(128, 256, 4, stride=2, padding=1),  # 16x16 -> 8x8
            nn.BatchNorm2d(256),
            nn.LeakyReLU(0.2),
            nn.Conv2d(256, 512, 4, stride=2, padding=1),  # 8x8 -> 4x4
            nn.BatchNorm2d(512),
            nn.LeakyReLU(0.2),
        )

        # 通道级全连接层（捕获全局上下文）
        self.fc = nn.Linear(512 * 4 * 4, latent_dim)

        # 解码器
        self.decoder_fc = nn.Linear(latent_dim, 512 * 4 * 4)

        self.decoder = nn.Sequential(
            nn.ConvTranspose2d(512, 256, 4, stride=2, padding=1),  # 4x4 -> 8x8
            nn.BatchNorm2d(256),
            nn.ReLU(),
            nn.ConvTranspose2d(256, 128, 4, stride=2, padding=1),  # 8x8 -> 16x16
            nn.BatchNorm2d(128),
            nn.ReLU(),
            nn.ConvTranspose2d(128, 64, 4, stride=2, padding=1),  # 16x16 -> 32x32
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.ConvTranspose2d(64, 3, 4, stride=2, padding=1),  # 32x32 -> 64x64
            nn.Tanh(),
        )

    def forward(self, x, mask):
        """
        x: 输入图像 [B, 3, H, W]
        mask: 缺失掩码 [B, 1, H, W], 1表示缺失区域
        """
        # 编码
        h = self.encoder(x)
        h = h.view(h.size(0), -1)
        h = self.fc(h)

        # 全局感知重构
        h = self.decoder_fc(h)
        h = h.view(h.size(0), 512, 4, 4)

        # 解码
        reconstructed = self.decoder(h)

        # 与原图融合
        output = x * (1 - mask) + reconstructed * mask
        return output
```

#### 4.3 LaMa风格的FFT卷积

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class FFTConv(nn.Module):
    """
    Fast Fourier Convolution: 全局感受野的频域卷积
    来自 LaMa (ICCV 2019)
    """
    def __init__(self, in_channels, out_channels, modes=32):
        super().__init__()
        self.in_channels = in_channels
        self.out_channels = out_channels
        self.modes = modes  # 频域模式数

        self.conv1x1 = nn.Conv2d(in_channels, out_channels, 1)
        self.fft_conv = nn.Conv2d(
            in_channels, 2 * out_channels, 1,
            groups=in_channels  # depthwise
        )
        self.norm = nn.LayerNorm([out_channels, None, None])

    def forward(self, x):
        # x: [B, C, H, W]
        B, C, H, W = x.shape

        # 1x1卷积进行通道调整
        x_1x1 = self.conv1x1(x)

        # FFT卷积分支
        x_fft = torch.fft.rfft2(x, dim=(2, 3), norm='ortho')
        x_fft_real = x_fft.real[:, :self.modes, :, :]
        x_fft_imag = x_fft.imag[:, :self.modes, :, :]

        fft_feat = torch.cat([x_fft_real, x_fft_imag], dim=1)
        fft_feat = self.fft_conv(fft_feat)

        fft_real, fft_imag = fft_feat.chunk(2, dim=1)
        x_fft_combine = torch.complex(fft_real, fft_imag)

        x_fft_irfft = torch.fft.irfft2(x_fft_combine, s=(H, W), dim=(2, 3), norm='ortho')

        # 残差连接
        out = x_1x1 + x_fft_irfft
        out = self.norm(out)

        return F.relu(out)
```

## 应用场景

### 1. 核心应用

- **照片修复**：去除照片中的杂物、水印、日期标注
- **老照片修复**：补全破损、褪色老照片的缺失部分
- **物体移除**：删除图像中不需要的物体（如游客、车辆）
- **面部修复**：修复遮挡面部的物品（如眼镜、口罩）
- **AI创作**：根据文字描述生成并替换图像区域

### 2. 行业应用

| 行业 | 应用场景 |
|------|----------|
| 摄影/后期 | 商业修图、婚纱摄影、证件照处理 |
| 影视制作 | 特效去除、场景扩展、穿帮镜头修复 |
| 游戏/元宇宙 | 虚拟场景构建、物体编辑 |
| 医学影像 | 医学图像修复、组织重建 |
| 文档处理 | 修复扫描文档中的污渍、折痕 |
| 监控安防 | 修复监控盲区、遮挡区域 |

## 相关概念

- **掩码（Mask）**：标记图像中需要修复区域的二值图像
- **Context Encoder**：一种编码器-解码器架构，通过GAN损失进行图像修复
- **扩散模型（Diffusion Model）**：通过逐步加噪去噪生成图像的生成模型
- **GAN（生成对抗网络）**：由生成器和判别器对抗训练的生成模型
- **感知损失（Perceptual Loss）**：基于预训练VGG网络特征计算的损失函数
- **FFT卷积（快速傅里叶卷积）**：在频域进行的高效全局卷积操作
