# 向量检索技术

## 一句话定义

**向量检索（Vector Retrieval）** 是从大规模向量数据库中高效找出与查询向量最相似的K个向量的技术，通过近似最近邻（ANN）算法在精度和速度之间取得平衡。

## 核心公式

### 1. 精确最近邻 vs 近似最近邻

$$\text{ANN} = \{ v_i \in D \mid \text{sim}(q, v_i) \text{ 在Top-K中} \}$$

目标：找到近似最优解，而非穷举精确解。

### 2. HNSW 分层可导航小世界图

$$O(\log \bar{M} \times \log N)$$

其中 $\bar{M}$ 是每层平均连接数，$N$ 是向量总数。查询复杂度为对数级别。

### 3. IVF 倒排索引

$$O(n_q + k \cdot \sqrt{N / n_c})}$$

其中 $n_q$ 是查询数，$k$ 是每个查询探针数，$n_c$ 是聚类数。

### 4. PQ 乘积量化

$$\text{Memory} = n \times \frac{D}{m} \times 1 \text{ byte}$$

原始存储需要 $n \times D \times 4$ bytes，PQ量化后可大幅降低。

## 详细说明

### 向量检索的核心问题

- **维度灾难**：高维空间中距离度量失效
- **大规模数据**：十亿级向量的高效检索
- **精度vs速度**：ANN在可接受精度损失下换取速度提升

### 主流算法分类

#### 1. **基于图的算法**

**HNSW (Hierarchical Navigable Small World)**

- 分层结构：多层图，上层稀疏、下层密集
- 贪心搜索：从顶层入口点逐层向下搜索
- 优点：查询速度快、精度高
- 缺点：内存占用较大、构建时间长

| 参数 | 说明 | 推荐值 |
|-----|------|-------|
| M | 每节点连接数 | 16-64 |
| efConstruction | 构建时动态列表大小 | 100-400 |
| efSearch | 查询时搜索范围 | 50-1000 |

#### 2. **基于聚类的算法**

**IVF (Inverted File Index)**

- 先聚类后索引
- 查询时只搜索最近的几个聚类
- 可与其他方法组合（IVF+PQ）

**Flat**: 不进行任何优化的暴力搜索（基准线）

#### 3. **基于量化的算法**

**PQ (Product Quantization)**

- 将高维向量分割为多个子空间
- 每个子空间独立进行K-means聚类
- 用聚类中心ID代替原始向量
- 压缩比可达10-50倍

**SQ (Scalar Quantization)**

- 标量量化（FP32 -> INT8）
- 简单但压缩比较低

#### 4. **混合算法**

**HNSW + PQ**: 高速查询+高压缩比
**IVF + PQ**: 聚类过滤+量化压缩
**SCANN**: Google的专利算法，结合树结构和量化

### 索引类型对比

| 索引类型 | 查询速度 | 内存占用 | 构建速度 | 精度 | 适用规模 |
|---------|---------|---------|---------|------|---------|
| Flat | 慢 | 高 | 无 | 100% | <100万 |
| IVF | 中 | 中 | 中 | 95-99% | 100万-1亿 |
| HNSW | 快 | 高 | 慢 | 95-99% | 100万-10亿 |
| PQ | 快 | 低 | 中 | 80-95% | 任意规模 |
| HNSW+PQ | 很快 | 低 | 慢 | 90-98% | >10亿 |

## 代码示例

### Python + FAISS 基础使用

```python
import numpy as np
import faiss

# 生成随机测试数据
np.random.seed(42)
dimension = 128
num_vectors = 10000
num_queries = 100

# 准备数据库向量和查询向量
database_vectors = np.random.random((num_vectors, dimension)).astype('float32')
query_vectors = np.random.random((num_queries, dimension)).astype('float32')

# 归一化（用于余弦相似度）
faiss.normalize_L2(database_vectors)
faiss.normalize_L2(query_vectors)

print(f"数据库: {database_vectors.shape}")
print(f"查询: {query_vectors.shape}")
```

### HNSW 索引

```python
# 创建HNSW索引
d = 128  # 向量维度
M = 32   # 每节点连接数
efSearch = 64  # 搜索参数
efConstruction = 200  # 构建参数

index = faiss.IndexHNSWFlat(d, M)
index.hnsw.efSearch = efSearch
index.hnsw.efConstruction = efConstruction

# 添加向量
index.add(database_vectors)
print(f"HNSW索引向量数: {index.ntotal}")

# 搜索
k = 5  # 返回最近5个
distances, indices = index.search(query_vectors, k)
print(f"搜索结果形状: 距离={distances.shape}, 索引={indices.shape}")
print(f"示例距离: {distances[0]}")
print(f"示例索引: {indices[0]}")
```

### IVF + PQ 索引

```python
# 创建IVF-PQ复合索引
nlist = 100  # 聚类中心数
m = 16       # PQ子空间数

# IVF-PQ索引
quantizer = faiss.IndexFlatIP(d)  # 使用内积作为度量
index = faiss.IndexIVFPQ(quantizer, d, nlist, m, 8)
# 8 = 每个子向量用8bits表示 (256个聚类中心)

index.train(database_vectors[:1000])  # 需要训练数据
index.add(database_vectors)
print(f"IVF-PQ索引向量数: {index.ntotal}")

# 设置搜索范围
index.nprobe = 10  # 搜索的聚类数
distances, indices = index.search(query_vectors, k=5)
```

### IVF-HNSW 索引

```python
# IVF-HNSW结合了两者的优点
quantizer = faiss.IndexHNSWFlat(d, 32)
index = faiss.IndexIVF(quantizer, d, nlist)
index.hnsw.efSearch = 64
index.hnsw.efConstruction = 200

index.train(database_vectors)
index.add(database_vectors)

# 搜索
distances, indices = index.search(query_vectors, k=5)
```

### 完整的向量检索流程

```python
import numpy as np
import faiss

class VectorSearchEngine:
    def __init__(self, dimension, index_type='HNSW'):
        self.dimension = dimension
        self.index_type = index_type
        self.index = None
        self._build_index()

    def _build_index(self):
        if self.index_type == 'HNSW':
            self.index = faiss.IndexHNSWFlat(self.dimension, 32)
        elif self.index_type == 'IVF':
            quantizer = faiss.IndexFlatIP(self.dimension)
            self.index = faiss.IndexIVFPQ(quantizer, self.dimension, 100, 16, 8)
        else:
            self.index = faiss.IndexFlatIP(self.dimension)

    def add_vectors(self, vectors):
        """添加向量到索引"""
        vectors = np.array(vectors).astype('float32')
        faiss.normalize_L2(vectors)
        self.index.add(vectors)

    def search(self, query_vectors, top_k=10):
        """搜索最相似的向量"""
        query_vectors = np.array(query_vectors).astype('float32')
        faiss.normalize_L2(query_vectors)
        distances, indices = self.index.search(query_vectors, top_k)
        return distances, indices

    def save(self, path):
        """保存索引"""
        faiss.write_index(self.index, path)

    @classmethod
    def load(cls, path, dimension, index_type='HNSW'):
        """加载索引"""
        engine = cls(dimension, index_type)
        engine.index = faiss.read_index(path)
        return engine

# 使用示例
dimension = 768
engine = VectorSearchEngine(dimension, 'HNSW')

# 添加10000个嵌入向量
embeddings = np.random.randn(10000, dimension).astype('float32')
engine.add_vectors(embeddings)

# 查询
query = np.random.randn(1, dimension).astype('float32')
distances, indices = engine.search(query, top_k=5)
print(f"最相似的5个向量索引: {indices[0]}")
print(f"对应的距离: {distances[0]}")
```

### 混合搜索实现

```python
import numpy as np
import faiss

class HybridSearchEngine:
    """支持向量检索和关键词过滤的混合搜索"""

    def __init__(self, dimension):
        self.dimension = dimension
        self.index = faiss.IndexHNSWFlat(dimension, 32)
        self.metadata = []  # 存储元数据用于过滤

    def add_vectors(self, vectors, metadata_list):
        vectors = np.array(vectors).astype('float32')
        faiss.normalize_L2(vectors)
        self.index.add(vectors)
        self.metadata.extend(metadata_list)

    def search_with_filter(self, query, top_k=10, filter_func=None):
        """
        带过滤条件的搜索
        filter_func: 接收metadata返回True/False
        """
        query = np.array(query).astype('float32')
        faiss.normalize_L2(query)

        # 搜索更多候选结果以应对过滤
        search_k = top_k * 10
        distances, indices = self.index.search(query, search_k)

        # 应用过滤器
        if filter_func:
            filtered_results = []
            for i, (dist, idx) in enumerate(zip(distances[0], indices[0])):
                if idx >= 0 and filter_func(self.metadata[idx]):
                    filtered_results.append((dist, idx))
                    if len(filtered_results) >= top_k:
                        break

            if filtered_results:
                distances = np.array([[d for d, _ in filtered_results]])
                indices = np.array([[idx for _, idx in filtered_results]])
            else:
                distances = np.array([[]])
                indices = np.array([[]])

        return distances, indices

# 使用示例
engine = HybridSearchEngine(128)

# 添加带元数据的向量
for i in range(1000):
    vector = np.random.randn(128).astype('float32')
    metadata = {
        'id': f'item_{i}',
        'category': ['electronics', 'clothing', 'food'][i % 3],
        'price': (i % 100) + 10
    }
    engine.add_vectors([vector], [metadata])

# 搜索，只返回电子产品
query = np.random.randn(128).astype('float32')
distances, indices = engine.search_with_filter(
    query,
    top_k=5,
    filter_func=lambda m: m['category'] == 'electronics'
)
print(f"电子产品搜索结果索引: {indices[0]}")
```
