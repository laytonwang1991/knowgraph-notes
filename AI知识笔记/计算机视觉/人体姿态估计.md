# 人体姿态估计

## 一句话定义

人体姿态估计（Human Pose Estimation, HPE）是通过深度学习模型预测人体各个关键点（如头部、肩膀、肘部、手腕、髋部、膝盖、脚踝等）在图像或视频中的二维或三维空间坐标的技术。

## 核心公式

### 1. 关键点热力图（Heatmap）回归

对于每个关键点 $k$，模型预测一个热力图 $H_k \in \mathbb{R}^{H \times W}$：

$$H_k(x, y) = \exp\left(-\frac{(x - x_k)^2 + (y - y_k)^2}{2\sigma^2}\right)$$

其中 $(x_k, y_k)$ 为第 $k$ 个关键点的真实坐标，$\sigma$ 为高斯核标准差。

### 2. OKS（Object Keypoint Similarity）

$$OKS = \frac{\sum_k \exp\left(-\frac{d_k^2}{2s_k^2 \sigma_k^2}\right)}{\sum_k \delta(k_i \neq 0)}$$

其中 $d_k$ 为预测点与真实点的欧氏距离，$s$ 为目标面积，$\sigma_k$ 为第 $k$ 个关键点的衰减常数。

### 3. PCK（Percentage of Correct Keypoints）

$$PCK = \frac{1}{N} \sum_{i=1}^{N} \delta\left(\frac{d_{ik}}{d_{b_i}} \leq \tau\right)$$

其中 $d_{ik}$ 为第 $i$ 个样本第 $k$ 个关键点的误差，$d_{b_i}$ 为躯干直径，$\tau$ 为阈值（通常取 0.5）。

### 4. 3D 姿态lifting（从2D到3D）

$$p_{3D} = \arg\min_{p_{3D}} \sum_{i} \|f_{lift}(p_{2D}^{(i)}) - p_{3D}^{(i)}\|^2 + \lambda R(p_{3D})$$

其中 $f_{lift}$ 为 lifting 网络，$R$ 为正则化项。

## 详细说明

### 1. 方法分类：Top-down vs Bottom-up

#### 1.1 Top-down（自上而下）

```
检测器定位所有人 → 对每个人单独估计姿态
```

- **优点**：精度高，每个人的关键点独立估计
- **缺点**：速度随人数线性增长，遮挡处理困难
- **代表模型**：HRNet, CPM (Convolutional Pose Machines)

#### 1.2 Bottom-up（自下而上）

```
检测所有关键点 → 将关键点分组到各个人
```

- **优点**：速度与人数无关，适合密集场景
- **缺点**：关键点分组困难，遮挡时精度下降
- **代表模型**：OpenPose, Associative Embedding, HigherHRNet

### 2. 主要模型架构

#### 2.1 OpenPose

- **核心思想**：使用 Part Affinity Fields（PAFs）学习关节点之间的关联向量场
- **网络结构**：VGG-19 特征提取 + 两分支多阶段预测（PAF + Heatmap）
- **关键点数量**：COCO 17点 / MPII 25点 / 脸部 68点
- **优势**：开源最早，PAF 机制影响深远

#### 2.2 HRNet（High-Resolution Network）

- **核心思想**：在整个网络中保持高分辨率表征，多尺度并行融合
- **网络结构**：并行多分辨率子网，通过反复多尺度融合增强特征
- **优势**：关键点定位精度高，至今仍是许多下游任务的主干网络
- **性能**：COCO val mAP = 76.1%（HRNet-W48）

#### 2.3 Hourglass Network（沙漏网络）

- **核心思想**：编码器-解码器结构，配对池化和上采样捕获多尺度信息
- **创新**：残余模块 + 多层堆叠沙漏结构
- **应用**：是许多人体姿态估计模型的基石

#### 2.4 3D 姿态估计

- **思路1（直接回归）**：直接从图像回归 3D 坐标（VoNet, VideoPose3D）
- **思路2（2D → 3D Lifting）**：先用 2D 检测器，再用 lifting 网络恢复 3D（Simple Baseline 3D）
- **主流方法**：时序平滑 + 骨骼长度的几何约束

### 3. COCO Keypoint 数据集格式

COCO 定义的 17 个关键点：

| ID | 名称 | 说明 |
|----|------|------|
| 0 | nose | 鼻子 |
| 1 | left_eye | 左眼 |
| 2 | right_eye | 右眼 |
| 3 | left_ear | 左耳 |
| 4 | right_ear | 右耳 |
| 5 | left_shoulder | 左肩 |
| 6 | right_shoulder | 右肩 |
| 7 | left_elbow | 左肘 |
| 8 | right_elbow | 右肘 |
| 9 | left_wrist | 左手腕 |
| 10 | right_wrist | 右手腕 |
| 11 | left_hip | 左髋 |
| 12 | right_hip | 右髋 |
| 13 | left_knee | 左膝 |
| 14 | right_knee | 右膝 |
| 15 | left_ankle | 左脚踝 |
| 16 | right_ankle | 右脚踝 |

### 4. 评估指标

- **OKS（Object Keypoint Similarity）**：COCO 官方指标，综合关键点位置和目标大小
- **AP / AR**：COCO 风格的平均精度 / 召回率
- **PCK**：关键点正确率（以躯干大小归一化）
- **AUC**：PCK 曲线下面积

### 5. 实时人体姿态估计优化

- **模型轻量化**：MobileNetV3 backbone, TensorRT INT8 量化
- **热力图解码优化**：无偏解码方法（UDP），避免量化误差
- **剪枝与蒸馏**：用大模型监督小模型
- **视频级时序平滑**：卡尔曼滤波或 One Euro 滤波器

## 代码示例

### 使用 OpenCV DNN 模块 + 预训练模型进行姿态估计

```python
import cv2
import numpy as np

# OpenPose Caffe 模型路径（需下载）
protoFile = "pose/coco/pose_deploy_linevec.prototxt"
weightsFile = "pose/coco/pose_iter_440000.caffemodel"

net = cv2.dnn.readNetFromCaffe(protoFile, weightsFile)

# 读取图像
frame = cv2.imread("person.jpg")
frameWidth = frame.shape[1]
frameHeight = frame.shape[0]

inpBlob = cv2.dnn.blobFromImage(frame, 1.0 / 255, (256, 256), (0, 0, 0), swapRB=False, crop=False)
net.setInput(inpBlob)

output = net.forward()
H, W = output.shape[2], output.shape[3]

# 提取关键点
points = []
for i in range(17):  # COCO 17 keypoints
    probMap = output[0, i, :, :]
    _, prob, _, point = cv2.minMaxLoc(probMap)
    x = (frameWidth * point[0]) / W
    y = (frameHeight * point[1]) / H
    if prob > 0.1:
        points.append((int(x), int(y)))
    else:
        points.append(None)

# 绘制骨架
骨架连接 = [(0,1),(1,2),(2,3),(3,4),(1,5),(5,6),(6,7),(1,11),(11,12),(0,14),(14,16)]
for (i, j) in 骨架连接:
    if points[i] and points[j]:
        cv2.line(frame, points[i], points[j], (0, 255, 0), 2)
        cv2.circle(frame, points[i], 3, (0, 0, 255), -1)

cv2.imshow("Pose Estimation", frame)
cv2.waitKey(0)
```

### 使用 MMPose (HRNet) 进行姿态估计

```python
from mmpose.apis import init_model, inference_topdown
from mmpose.registry import VISUALIZERS
from mmpose.structures import merge_data_samples
import cv2

# 初始化模型（HRNet）
config_file = "configs/body/2d_kpt_sview_rgb_img/topdown_heatmap/coco/hrnet_w32_coco_256x192.py"
checkpoint_file = "hrnet_w32_coco_256x192-85aa8c6b_20200712.pth"
model = init_model(config_file, checkpoint_file, device='cuda:0')

# 推理
results = inference_topdown(model, "person.jpg")
result = merge_data_samples(results[0])

# 可视化
visualizer = VISUALIZERS.build(model.cfg.visualizer)
visualizer.add_datasample('result', cv2.imread("person.jpg"), result, draw_heatmap=True, show=True)
```

### 使用 MediaPipe 进行实时姿态估计

```python
import cv2
import mediapipe as mp

mp_pose = mp.solutions.pose
pose = mp_pose.Pose(static_image_mode=False, model_complexity=2, enable_segmentation=True)

cap = cv2.VideoCapture(0)
while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        break

    rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    results = pose.process(rgb_frame)

    if results.pose_landmarks:
        mp.solutions.drawing_utils.draw_landmarks(
            frame, results.pose_landmarks, mp_pose.POSE_CONNECTIONS)

    cv2.imshow("MediaPipe Pose", frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```

## 相关概念

- **热力图回归（Heatmap Regression）**：通过预测每个关键点的高斯热力图来定位关键点
- **直接坐标回归（Direct Regression）**：直接回归关键点的 (x, y) 坐标
- **Part Affinity Fields（PAF）**：OpenPose 提出的关节点关联向量场，用于 Bottom-up 分组
- **关联嵌入（Associative Embedding）**：通过在每个关键点附加嵌入向量来学习分组
- **Hourglass Network**：编码器-解码器沙漏结构，用于捕获多尺度信息
- **Lifting Network**：从 2D 关键点提升到 3D 关键点的网络
- **时序平滑（Temporal Smoothing）**：在视频中利用时序信息平滑关键点轨迹
