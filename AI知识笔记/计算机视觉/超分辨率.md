# 超分辨率

## 一句话定义

超分辨率（Super Resolution）是通过深度学习模型将低分辨率图像提升为高分辨率图像，同时尽可能恢复真实纹理细节的技术。

## 核心公式

### PSNR（峰值信噪比）

$$
\text{PSNR} = 10 \cdot \log_{10}\left( \frac{\text{MAX}_I^2}{\text{MSE}} \right)
$$

其中 $\text{MAX}_I$ 是像素最大值（通常为255），MSE 是均方误差。

### SSIM（结构相似性指数）

$$
\text{SSIM}(x, y) = \frac{(2\mu_x\mu_y + c_1)(2\sigma_{xy} + c_2)}{(\mu_x^2 + \mu_y^2 + c_1)(\sigma_x^2 + \sigma_y^2 + c_2)}
$$

### L1 损失函数

$$
\mathcal{L}_{L1} = \frac{1}{N} \sum_{i=1}^{N} |I_{HR}^i - \hat{I}_{HR}^i|_1
$$

## 详细说明

### 1. 技术发展脉络

- **SRCNN（2014）**：首次将 CNN 应用于超分辨率，使用 3 层卷积网络，端到端学习 LR 到 HR 的映射
- **FSRCNN（2016）**：改用转置卷积上采样，在特征空间进行上采样，减少计算量
- **ESPCN（2016）**：提出亚像素卷积层（Sub-pixel Convolution），高效实现空间上采样
- **EDSR（2017）**：移除批归一化层，增加残差块数量，显著提升性能
- **RCAN（2018）**：引入通道注意力机制（Channel Attention），自适应调整特征权重
- **ESRGAN（2018）**：将 GAN 引入超分辨率，生成更真实的纹理
- **EDiff-SR（2023）**：基于扩散模型的条件超分辨率，兼顾生成质量与保真度

### 2. 上采样方法对比

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 双线性插值 | 线性加权 | 快速 | 模糊 |
| 转置卷积 | 可学习上采样 | 端到端 | 棋盘效应 |
| 亚像素卷积 | 通道重组 | 无棋盘效应 | 感受野受限 |
| ISR | 逐级上采样 | 稳定 | 慢 |

### 3. 损失函数选择

- **像素损失**：L1/L2 损失，保证整体结构
- **感知损失**：VGG 特征损失，增强视觉真实感
- **对抗损失**：GAN 损失，生成更细纹理
- **风格损失**：Gram 矩阵损失，控制纹理风格

## 代码示例

### PyTorch 实现 SRCNN

```python
import torch
import torch.nn as nn

class SRCNN(nn.Module):
    def __init__(self, num_channels=3):
        super(SRCNN, self).__init__()
        # 特征提取
        self.conv1 = nn.Conv2d(num_channels, 64, kernel_size=9, padding=4)
        # 非线性映射
        self.conv2 = nn.Conv2d(64, 32, kernel_size=1, padding=0)
        # 重建
        self.conv3 = nn.Conv2d(32, num_channels, kernel_size=5, padding=2)
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.relu(self.conv1(x))
        x = self.relu(self.conv2(x))
        x = self.conv3(x)
        return x

# 使用示例
model = SRCNN()
lr_image = torch.randn(1, 3, 64, 64)  # 低分辨率输入
hr_image = model(lr_image)  # 高分辨率输出
print(f"输出尺寸: {hr_image.shape}")  # [1, 3, 64, 64]
```

### ESRGAN 推理

```python
import torch
from basicsr.archs.rrdbnet_arch import RRDBNet

model = RRDBNet(num_in_ch=3, num_out_ch=3, num_feat=64, num_block=23)

```

## 应用场景

1. **医学影像**：CT、MRI 图像增强，辅助诊断
2. **卫星遥感**：航拍图像分辨率提升，降低拍摄成本
3. **视频监控**：老旧监控画质增强，人脸/车牌识别
4. **游戏渲染**：将低分辨率纹理实时超分，节省显存
5. **老照片修复**：家庭老照片、视频画质修复
6. **通信传输**：传输低分辨率图像，接收端超分恢复
