# 图像风格迁移

## 一句话定义

图像风格迁移（Neural Style Transfer）是通过深度神经网络将内容图像的结构与风格图像的纹理分离，再将两者高效重组，生成兼具内容结构与风格纹理的新图像。

## 核心公式

### Gatys 风格损失（Gram 矩阵）

$$
G_{ij}^l = \sum_k F_{ik}^l \cdot F_{jk}^l
$$

其中 $F_{ik}^l$ 是第 $l$ 层第 $i$ 个通道第 $k$ 个位置的特征值。

### AdaIN（自适应实例归一化）

$$
\text{AdaIN}(x, y) = \sigma(y) \cdot \frac{x - \mu(x)}{\sigma(x)} + \mu(y)
$$

其中 $x$ 是内容特征，$y$ 是风格特征，$\mu$ 和 $\sigma$ 分别是均值和标准差。

### WCT（ whitening & coloring transform）

$$
\text{WCT}: \hat{x} = E_y D_y^{1/2} E_x^T x + \mu_y
$$

将内容图像特征 $x$ 的协方差转换为风格图像 $y$ 的协方差。

### 总损失函数

$$
\mathcal{L}_{total} = \alpha \cdot \mathcal{L}_{content} + \beta \cdot \mathcal{L}_{style}
$$

## 详细说明

### 1. 技术发展脉络

- **Gatys (2015)**：首次提出基于 Gram 矩阵的神经风格迁移，速度慢但效果震撼
- **Johnson et al. (2016)**：提出感知损失，训练前馈网络实现实时风格迁移
- **Dumoulin et al. (2016)**：提出条件实例归一化（CIN），单网络多风格
- **Huang et al. (2017)**：AdaIN 实现任意风格迁移，无需重新训练
- **Li et al. (2017)**：WCT 通过协方差变换实现风格迁移
- **GANimation (2018)**：基于 GAN 的人脸表情风格迁移
- **StyleGAN (2019)**：高质量人脸风格控制
- **CLIPStye (2021)**：文本驱动的风格迁移

### 2. 方法分类

| 类型 | 代表方法 | 特点 |
|------|----------|------|
| 慢速（非迭代） | Gatys | 每次迭代优化，效果好但慢 |
| 快速（前馈） | Johnson | 单次前馈，实时，风格固定 |
| 任意风格 | AdaIN/WCT | 无需预训练，任意风格 |
| 多风格 | CIN | 单网络多风格 |
| 文本驱动 | CLIPStye | 文本描述控制风格 |

### 3. 风格表示方法

- **Gram 矩阵**：捕获纹理统计信息
- **均值/方差**：简化的风格统计
- **直方图**：更丰富的分布信息
- **协方差**：二阶统计量

## 代码示例

### PyTorch 实现 Gatys 风格迁移

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import models, transforms

class StyleTransfer:
    def __init__(self, content_img, style_img, device='cuda'):
        self.device = device
        self.content_img = content_img.to(device)
        self.style_img = style_img.to(device)
        # 固定内容图像为可求导变量
        self.target = content_img.clone().requires_grad_(True).to(device)
        self.optimizer = optim.LBFGS([self.target])

        # 加载 VGG19 特征提取器
        self.vgg = models.vgg19(pretrained=True).features.to(device).eval()
        for param in self.vgg.parameters():
            param.requires_grad_(False)

    def get_features(self, img):
        """提取 VGG 各层特征"""
        layers = {
            '0': 'conv1_1', '5': 'conv2_1', '10': 'conv3_1',
            '19': 'conv4_1', '21': 'conv4_2', '28': 'conv5_1'
        }
        features = {}
        x = img
        for name, layer in self.vgg._modules.items():
            x = layer(x)
            if name in layers:
                features[layers[name]] = x
        return features

    def gram_matrix(self, x):
        """计算 Gram 矩阵"""
        b, c, h, w = x.size()
        features = x.view(b, c, h * w)
        gram = torch.bmm(features, features.transpose(1, 2))
        return gram / (c * h * w)

    def train(self, content_weight=1, style_weight=1e5, steps=300):
        """训练"""
        content_features = self.get_features(self.content_img)
        style_features = self.get_features(self.style_img)
        style_grams = {k: self.gram_matrix(v) for k, v in style_features.items()}

        def closure():
            self.optimizer.zero_grad()
            target_features = self.get_features(self.target)

            # 内容损失
            content_loss = content_weight * torch.mean(
                (target_features['conv4_2'] - content_features['conv4_2']) ** 2
            )

            # 风格损失
            style_loss = 0
            for layer, style_gram in style_grams.items():
                target_gram = self.gram_matrix(target_features[layer])
                style_loss += style_weight * torch.mean(
                    (target_gram - style_gram) ** 2
                )

            total_loss = content_loss + style_loss
            total_loss.backward()
            return total_loss

        for i in range(steps):
            self.optimizer.step(closure)
            if i % 50 == 0:
                print(f"Step {i}: loss = {closure().item():.4f}")

        return self.target.detach()
```

### AdaIN 实现任意风格迁移

```python
import torch
import torch.nn.functional as F

def adaptive_instance_norm(content, style):
    """
    AdaIN: 将内容特征的统计量替换为风格特征的统计量
    content: [B, C, H, W] 内容特征
    style: [B, C, H, W] 风格特征
    """
    assert content.size()[:2] == style.size()[:2]

    # 计算均值和标准差
    content_mean = content.mean(dim=[2, 3], keepdim=True)
    content_std = content.std(dim=[2, 3], keepdim=True) + 1e-6
    style_mean = style.mean(dim=[2, 3], keepdim=True)
    style_std = style.std(dim=[2, 3], keepdim=True) + 1e-6

    # 归一化内容特征，然后用风格统计量重新缩放
    normalized = (content - content_mean) / content_std
    return normalized * style_std + style_mean

# 使用示例
content_feat = torch.randn(1, 256, 64, 64)
style_feat = torch.randn(1, 256, 64, 64)
styled_feat = adaptive_instance_norm(content_feat, style_feat)
print(f"输出特征形状: {styled_feat.shape}")  # [1, 256, 64, 64]
```

## 应用场景

1. **艺术创作**：将照片转换为梵高、毕加索等大师风格
2. **图像滤镜**：手机相机实时艺术滤镜
3. **电影/游戏**：风格化图像生成，节省手绘时间
4. **人像美化**：人脸表情/风格迁移（GANimation）
5. **服装试穿**：服装风格迁移到用户照片
6. **室内设计**：将设计草图风格化为实景效果图

## 相关概念

| 概念 | 说明 |
|------|------|
| 感知损失 | 基于 VGG 特征的损失，比像素损失更符合人类感知 |
| Gram 矩阵 | 风格统计表示，捕捉通道间相关性 |
| 实例归一化 | 每个样本独立归一化，适合风格迁移 |
| 风格量化 | 将风格表示为可量化的向量/矩阵 |
