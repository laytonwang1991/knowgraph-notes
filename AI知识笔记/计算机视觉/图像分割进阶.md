# 图像分割进阶

## 一句话定义

图像分割进阶涵盖从语义分割到实例分割、全景分割，再到统一视觉分割模型（SAM）的技术演进，以及向三维医学影像和点云分割扩展的前沿方向，核心在于如何统一处理"Things"（可计数目标）和"Stuff"（背景区域）。

## 核心公式

### 1. 全景分割 Quality Fat Loss

$$L_{QF} = \L_{mask} + \lambda \cdot \L_{cls}$$

其中 $\L_{mask}$ 为掩码预测损失，$\L_{cls}$ 为分类损失，$\lambda$ 为权重平衡因子。

### 2. SAM（Segment Anything Model）提示编码

$$\phi_{mask} = \text{Attention}(Q, K, V) \cdot M$$

其中 $Q = \text{Decoder}(p)$, $K, V = \text{Encoder}(image)$, $p$ 为提示（点/框/文本），$M$ 为预测掩码。

### 3. PerSAM 语义相似度匹配

$$S_{i,j} = \frac{\exp(-\|f_i - f_j\|^2 / 2\sigma^2)}{\sum_{k} \exp(-\|f_k - f_j\|^2 / 2\sigma^2)}$$

其中 $f_i$ 为图像_patch_特征，$f_j$ 为参考掩码区域特征，$\sigma$ 为温度参数。

### 4. 3D 点云分割体素化

$$\text{Voxel}(p) = \left( \left\lfloor \frac{p_x - O_x}{v_x} \right\rfloor, \left\lfloor \frac{p_y - O_y}{v_y} \right\rfloor, \left\lfloor \frac{p_z - O_z}{v_z} \right\rfloor \right)$$

其中 $O$ 为体素网格原点，$v$ 为体素分辨率，$p$ 为三维点坐标。

## 详细说明

### 1. 分割任务分类体系

```
图像分割
├── 语义分割（Semantic Segmentation）→ 像素级类别标签
│   ├── FCN, DeepLab系列, UNet, SegFormer
│   └── 开放词汇语义分割（OVS）→ CLIP语义
│
├── 实例分割（Instance Segmentation）→ 区分同类不同个体
│   ├── Mask R-CNN, Cascade Mask R-CNN, HTC
│   └── SOLOv2（动态分割 kernel）
│
├── 全景分割（Panoptic Segmentation）→ 统一 Things + Stuff
│   ├── Mask2Former, Panoptic FPN, Swin-L
│   └── 先验驱动：Panoptic SegFormer
│
└── 交互式分割（Interactive Segmentation）→ 用户引导
    ├── SAM, PerSAM, Semantic-SAM
    └── Grounded SAM（检测+分割）
```

### 2. Mask2Former：统一分割的Transformer架构

#### 2.1 核心创新

- **Meta Architecture**：统一处理语义/实例/全景分割
- **Masked Attention**：在预测掩码区域内做注意力，捕获局部信息
- **多尺度可变形注意力**：Deformable DETR 风格的跨尺度特征融合

#### 2.2 架构图

```
Image → Backbone(Swin/ResNet) → Pixel Decoder
                                      ↓
                              Global Object Queries (N个)
                                      ↓
                              Transformer Decoder
                              (Masked Cross-Attention)
                                      ↓
                              预测：class + mask + quality
```

#### 2.3 性能对比

| 模型 | Backbone | PQ (Panoptic) | mAP (Instance) | mIoU (Semantic) |
|------|----------|----------------|----------------|-----------------|
| Mask2Former | Swin-L | 57.8 | 49.1 | - |
| Mask2Former | ResNet50 | 53.2 | 45.1 | - |
| Panoptic FPN | ResNet101 | 48.3 | 41.2 | 42.8 |
| Seamless | Swin-L | 63.5 | 53.4 | - |

### 3. SAM（Segment Anything Model）

#### 3.1 核心组件

| 组件 | 说明 |
|------|------|
| Image Encoder | MAE 预训练 ViT-H (1056 patches) |
| Prompt Encoder | 点/框/文本 → sparse embeddings |
| Mask Decoder | 两层交叉注意力 + 动态 MLP |

#### 3.2 SAM 的 prompt 模式

- **点提示（Point）**：前景点/背景点指定分割目标
- **框提示（Box）**：通过边界框指定目标
- **文本提示（Text）**：CLIP 编码的自然语言描述
- **全图提示（Everything）**：无需 prompt，全图分割

#### 3.3 SAM 的问题与改进

| 问题 | 改进方案 | 代表模型 |
|------|----------|----------|
| 边缘粗糙 | 级联后处理 | SAM-PTUR, EdgeSAM |
| 小目标差 | 高分辨率版本 | SAM-HQ, HQ-Former |
| 语义弱 | 与CLIP结合 | Grounded SAM, Semantic-SAM |
| 计算重 | 轻量化 | MobileSAM, EfficientSAM |
| 缺乏实例区分 | 嵌入向量匹配 | PerSAM, SA-Former |

### 4. PerSAM：个性化语义分割

#### 4.1 核心思想

PerSAM 利用用户提供的参考掩码（1-3 张图），通过特征相似度匹配实现"一次训练，随处分割"的个性化语义分割。

#### 4.2 分割流程

```
1. 输入：参考图像 + 参考掩码 + 目标图像
2. 提取：CLIP 图像编码器 → 提取参考掩码区域的特征
3. 匹配：计算目标图像各 patch 与参考特征的余弦相似度
4. 分割：通过相似度热力图生成二值掩码
5. 优化：可选联合优化 1-3 张参考图进行微调
```

### 5. 3D 分割

#### 5.1 医学影像3D分割

- **3D U-Net**：将 2D U-Net 扩展到三维体素
- **nnU-Net**：自适应的 3D 分割框架（医学影像 SOTA）
- **Swin UNETR**：Transformer 在 3D 医学影像的应用

#### 5.2 点云分割

| 方法 | 代表模型 | 核心思想 |
|------|----------|----------|
| 逐点MLP | PointNet, PointNet++ | 逐点特征提取 + 对称函数聚合 |
| 图卷积 | DGCNN, GACNet | 图结构上的消息传递 |
| 体素化 | PointPillars, VoxelNet | 将点云体素化后用3D CNN |
| Transformer | PointTransformer, PTv2 | 全局注意力在点云上 |
| 2D投影 | SqueezeSeg, RangeNet | 深度图/前视图 + 2D分割 |

### 6. 开放词汇分割（Open-Vocabulary Segmentation）

#### 6.1 核心问题

传统分割只支持预定义的类别，开放词汇分割通过预训练视觉-语言模型（CLIP, SAM）实现任意类别的分割。

#### 6.2 代表方法

| 方法 | 视觉编码器 | 语言编码器 | 特点 |
|------|------------|------------|------|
| CLIPSeg | ViT-L | CLIP Text | 零样本图像分割 |
| Lseg | ResNet50 | CLIP Text | 语义+开放词汇 |
| OVSeg | Mask2Former | CLIP Text | Mask-level 开放词汇 |
| Grounded SAM | SAM | BLIP-2 | 检测+分割联合 |

## 代码示例

### 使用 Grounded SAM 进行开放词汇检测+分割

```python
from grounded_sam import GroundedSAM
import cv2

# 初始化
gsam = GroundedSAM(
    sam_version="vit_h",
    ckpt_dir="./checkpoints",
    device="cuda"
)

# 加载图像
image = cv2.imread("street.jpg")

# 文本提示：检测所有车辆和行人
detections = gsam.detect_and_segment(
    image,
    text_prompt="car, pedestrian, bicycle, traffic light",
    box_threshold=0.3,
    nms_threshold=0.5,
)

# 遍历所有检测结果
for det in detections:
    class_name = det["class_name"]
    box = det["box"]
    mask = det["mask"]
    score = det["confidence"]

    # 在图像上绘制
    color = gsam.color_palette[class_name]
    image = gsam.draw_mask(image, mask, color)
    image = gsam.draw_box(image, box, class_name, score)

cv2.imwrite("grounded_sam_result.jpg", image)
```

### 使用 SAM 进行交互式分割

```python
from segment_anything import SamPredictor, sam_model_registry
import cv2
import numpy as np

# 加载 SAM 模型
sam = sam_model_registry["vit_h"](checkpoint="sam_vit_h.pth")
predictor = SamPredictor(sam)

# 读取图像
image = cv2.imread("object.jpg")
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
predictor.set_image(image_rgb)

# 点提示：点击目标中心（正向）和背景（负向）
input_point = np.array([[150, 200], [50, 50]])  # 正向点，负向点
input_label = np.array([1, 0])  # 1=前景，0=背景

# 预测掩码
masks, scores, logits = predictor.predict(
    point_coords=input_point,
    point_labels=input_label,
    multimask_output=True
)

# 选择最佳掩码
best_mask = masks[np.argmax(scores)]

# 可视化
image[best_mask] = image[best_mask] * 0.5 + np.array([0, 255, 0]) * 0.5
cv2.imwrite("sam_result.jpg", image)
```

### 使用 Mask2Former 进行全景分割

```python
from detectron2.config import get_cfg
from detectron2.projects.deeplab import add_deeplab_config
from detectron2.engine.defaults import DefaultPredictor
from detectron2.data import MetadataCatalog
import cv2

# 配置
cfg = get_cfg()
add_deeplab_config(cfg)
cfg.merge_from_file("configs/coco/panoptic-segmentation/maskformer2_R101_bs16_90k.yaml")
cfg.MODEL.WEIGHTS = "model_final_2c908d.pkl"
cfg.MODEL.MASK_FORMER.TEST.PANOPTIC.FUSE_CONF = 0.9

predictor = DefaultPredictor(cfg)

# 推理
im = cv2.imread("cityscape.jpg")
outputs = predictor(im)

# 解析全景分割结果
panoptic_seg = outputs["panoptic_seg"][0].to("cpu")
segments_info = outputs["panoptic_seg"][1]

# 重建全分辨率全景图
result = panoptic_seg.cpu().numpy()
print(f"检测到 {len(segments_info)} 个分割实例")
```

### 3D U-Net 医学影像分割

```python
import torch
import torch.nn as nn
from monai.networks.nets import UNet

# 3D U-Net for Medical Imaging
model = UNet(
    spatial_dims=3,
    in_channels=1,      # CT/MRI 单通道
    out_channels=2,     # 背景 + 器官
    channels=(32, 64, 128, 256),
    strides=(2, 2, 2),
    num_res_units=2,
).cuda()

# 输入： (B, 1, D, H, W) - 3D体素
volume = torch.randn(1, 1, 128, 128, 128).cuda()

# 推理
with torch.no_grad():
    output = model(volume)
    prediction = torch.argmax(output, dim=1)

print(f"输出形状: {prediction.shape}")  # (1, 128, 128, 128)
```

## 相关概念

- **Things vs Stuff**：Things 是可数的独立目标（人、车），Stuff 是不可数的背景区域（天空、道路）
- **Panoptic Segmentation**：全景分割，统一实例分割和语义分割
- **Masked Attention**：在掩码区域内进行注意力计算，避免无关区域干扰
- **Deformable Attention**：可变形注意力，只在参考点附近采样，降低计算量
- **开放词汇（Open Vocabulary）**：利用预训练视觉-语言模型支持任意类别
- **零样本分割（Zero-shot Segmentation）**：无需训练即可分割新类别
- **NeRF / 3D Gaussian Splatting**：新型 3D 场景表征，可从中提取分割结果
