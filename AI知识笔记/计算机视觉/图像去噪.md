# 图像去噪

## 一句话定义

图像去噪（Image Denoising）是通过深度学习模型从噪声观测中恢复干净图像的过程，核心目标是分离噪声与信号，同时最大程度保留图像细节。

## 核心公式

### 加性高斯白噪声（AWGN）模型

$$
y = x + n, \quad n \sim \mathcal{N}(0, \sigma^2)
$$

其中 $y$ 是观测到的噪声图像，$x$ 是干净图像，$n$ 是均值为0、方差为 $\sigma^2$ 的高斯噪声。

### MSE 损失函数

$$
\mathcal{L}_{MSE} = \frac{1}{N} \sum_{i=1}^{N} |\hat{x}_i - x_i|_2^2
$$

### 感知损失（Perceptual Loss）

$$
\mathcal{L}_{perceptual} = \frac{1}{C_j H_j W_j} \|\phi_j(\hat{x}) - \phi_j(x)\|_2^2
$$

其中 $\phi_j$ 是预训练网络（如 VGG）第 $j$ 层的特征提取函数。

### 扩散模型前向过程

$$
q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t} x_{t-1}, \beta_t I)
$$

### 扩散模型反向去噪（简化为单步）

$$
\hat{x}_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)
$$

## 详细说明

### 1. 技术发展脉络

- **BM3D（2007）**：传统方法巅峰，利用非局部相似块进行去噪
- **DnCNN（2017）**：将残差学习与批归一化结合，深度卷积网络去噪
- **FFDNet（2018）**：可处理空间变化的噪声和噪声水平图
- **NBNet（2021）**：通过注意力机制选择性地重建信号子空间
- **SwinIR（2021）**：基于 Transformer 的图像重建，刷新多项去噪基准
- **DiffMB（2022）**：基于扩散模型的盲去噪，无需知道噪声水平
- **Restormer（2022）**：高效 Transformer 用于多尺度图像重建
- **DeepCache（2023）**：加速扩散模型去噪的实用方案

### 2. 噪声类型

| 类型 | 特点 | 典型场景 |
|------|------|----------|
| 高斯噪声 | 加性、服从正态分布 | 传感器噪声、低照度 |
| 泊松噪声 | 信号依赖、乘性 | 医学影像（荧光显微镜） |
| 椒盐噪声 | 随机黑白像素 | 传输错误 |
| 斑点噪声 | 乘性、相干 | SAR 雷达、超声 |
| 复杂噪声 | 混合多种噪声 | 真实场景 |

### 3. 网络架构演进

```
MLP → CNN → ResNet → U-Net → Attention → Transformer → Diffusion
```

- **U-Net**：编码器-解码器 + 跳跃连接，适合低层次视觉任务
- **残差学习**：学习噪声残差而非直接重建，加速收敛
- **注意力机制**：非局部注意力捕获长程依赖
- **Transformer**：自注意力捕获全局上下文信息

### 4. 评价指标

| 指标 | 说明 | 优缺点 |
|------|------|--------|
| PSNR | 峰值信噪比 | 常用，但与人眼感知不一致 |
| SSIM | 结构相似性 | 考虑结构信息 |
| LPIPS | 学习感知补丁相似性 | 更符合人眼感知 |
| NIQE | 自然图像质量评价 | 无参考质量评价 |

## 代码示例

### PyTorch 实现 DnCNN

```python
import torch
import torch.nn as nn

class DnCNN(nn.Module):
    def __init__(self, channels=3, num_layers=20, features=64):
        super(DnCNN, self).__init__()
        layers = []

        # 第一层：特征提取
        layers.append(nn.Conv2d(channels, features, kernel_size=3, padding=1))
        layers.append(nn.ReLU(inplace=True))

        # 中间层：残差块
        for _ in range(num_layers - 2):
            layers.append(nn.Conv2d(features, features, kernel_size=3, padding=1))
            layers.append(nn.BatchNorm2d(features))
            layers.append(nn.ReLU(inplace=True))

        # 最后一层：重建
        layers.append(nn.Conv2d(features, channels, kernel_size=3, padding=1))

        self.network = nn.Sequential(*layers)

    def forward(self, x):
        # 残差学习：预测噪声
        noise = self.network(x)
        return x - noise  # 干净图像 = 噪声图像 - 预测噪声

# 使用示例
model = DnCNN(channels=3, num_layers=20)
noisy_img = torch.randn(1, 3, 256, 256)
clean_img = model(noisy_img)
print(f"输入形状: {noisy_img.shape}, 输出形状: {clean_img.shape}")
```

### NBNet 去噪核心模块

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class NBNetDecoder(nn.Module):
    """NBNet 的信号空间选择解码器"""
    def __init__(self, in_channels, reduction=16):
        super(NBNetDecoder, self).__init__()
        self.signal_proj = nn.Conv2d(in_channels, in_channels // reduction, 1)
        self.noise_proj = nn.Conv2d(in_channels, in_channels // reduction, 1)
        self.reconstruct = nn.Conv2d(in_channels // reduction, in_channels, 1)

    def forward(self, feat, residual):
        """
        feat: 特征图
        residual: 残差（噪声估计）
        """
        # 信号空间选择
        signal = self.signal_proj(feat)
        noise = self.noise_proj(residual)

        # 计算注意力权重
        # 高的 signal - noise 表示更可能是信号主导区域
        attention = torch.sigmoid(signal - noise)
        # 选择性重建
        selected = signal * attention
        reconstructed = self.reconstruct(selected)
        return reconstructed + residual
```

### 扩散模型单步去噪（概念代码）

```python
def denoise_step(x_t, t, model, alphas_cumprod, beta_t):
    """
    扩散模型单步去噪
    x_t: 当前带噪声图像
    t: 当前时间步
    model: 去噪网络，预测噪声 epsilon
    """
    alpha_t = alphas_cumprod[t]
    beta_t = beta_t[t]

    # 模型预测噪声
    epsilon_pred = model(x_t, t)

    # 计算均值参数
    sqrt_alpha_t = torch.sqrt(alpha_t)
    sqrt_one_minus_alpha = torch.sqrt(1 - alpha_t)

    # 去噪公式
    mean = (x_t - beta_t / sqrt_one_minus_alpha * epsilon_pred) / sqrt_alpha_t

    # 添加少量噪声（采样）
    if t > 0:
        noise = torch.randn_like(x_t)
        x_t_minus_1 = mean + torch.sqrt(beta_t) * noise
    else:
        x_t_minus_1 = mean

    return x_t_minus_1

# 完整去噪循环
def denoise_diffusion(x_noisy, model, alphas_cumprod, beta_t, num_steps=100):
    x = x_noisy
    for t in reversed(range(num_steps)):
        x = denoise_step(x, t, model, alphas_cumprod, beta_t)
    return x
```

## 应用场景

1. **医学影像**：CT、MRI、X光图像去噪，降低辐射剂量
2. **天文图像**：深空天体图像去噪，弱信号增强
3. **低照度摄影**：手机/监控夜间图像去噪
4. **计算摄影**：多帧去噪、HDR 去噪
5. **旧照片修复**：去除老照片噪点、划痕
6. **自动驾驶**：车载相机去噪，提升感知精度
7. **显微成像**：荧光显微镜图像降噪

## 相关概念

| 概念 | 说明 |
|------|------|
| 残差学习 | 学习噪声残差而非直接重建，加速收敛 |
| 盲去噪 | 未知噪声水平的去噪 |
| 乘性噪声 | 噪声与信号相乘，如斑点噪声 |
| 自监督去噪 | 无需干净图像配对训练 |
| 扩散模型 | 通过迭代去噪生成高质量图像 |
| score-based model | 基于分数的生成模型 |
