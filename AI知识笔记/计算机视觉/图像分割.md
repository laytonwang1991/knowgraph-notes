# 图像分割

## 一句话定义

图像分割是将整张图像按照语义或实例级别划分为若干有意义区域的任务，每个区域对应一个物体或前景/背景类别。

## 核心公式

### 1. 语义分割的像素级分类

给定输入图像 $I \in \mathbb{R}^{H \times W \times 3}$，输出分割掩码 $M \in \mathbb{Z}^{H \times W}$，每个像素 $(i,j)$ 的类别预测为：

$$
\hat{y}_{i,j} = \arg\max_{c \in \{0,1,...,K\}} P_c(I_{i,j})
$$

其中 $P_c$ 是类别 $c$ 的概率，$K$ 为类别总数。

### 2. IoU (Intersection over Union)

$$
\text{IoU} = \frac{|A \cap B|}{|A \cup B|}
$$

### 3. Dice Loss

$$
\mathcal{L}_{\text{Dice}} = 1 - \frac{2|X \cap Y|}{|X| + |Y|}
$$

### 4. FCN 的转置卷积（上采样）

$$
y_{i,j} = \sum_{a,b} x_{i+s \cdot a, j+s \cdot b} \cdot w_{a,b}
$$

其中 $s$ 为步长（stride），$w$ 为卷积核权重。

## 详细说明

### 1. 图像分割的三种类型

#### 语义分割 (Semantic Segmentation)

- 为图像中每个像素分配类别标签
- 不区分同类物体的不同实例
- 场景：自动驾驶道路分割、医学影像组织分割

#### 实例分割 (Instance Segmentation)

- 在语义分割基础上，进一步区分同类物体的不同实例
- 每个实例有独立 ID
- 场景：拥挤场景行人计数、产品缺陷检测

#### 全景分割 (Panoptic Segmentation)

- 统一语义分割和实例分割
- Thing 类（可数物体）用实例分割
- Stuff 类（不可数背景）用语义分割
- 公式化：全景质量 $PQ = \frac{\sum_{(g,p)\in TP} IoU(g,p)}{|TP| + 0.5|FP| + 0.5|FN|}$

### 2. 经典网络架构

#### FCN (Fully Convolutional Network)

- 首个全卷积网络，用于端到端语义分割
- 将 VGG/AlexNet 的全连接层替换为 $1\times1$ 卷积
- 通过转置卷积实现上采样，恢复空间分辨率
- 跳跃连接（Skip Connection）融合浅层细节和深层语义

#### U-Net

- 编码器-解码器对称结构
- 编码器：下采样提取特征（Contracting Path）
- 解码器：上采样恢复分辨率（Expanding Path）
- 跳跃连接：同分辨率的特征拼接
- 广泛应用于医学图像分割

```
编码器: 572x572 -> 批归一化 -> ReLU -> 3x3卷积 -> 64通道 -> 2x2最大池化 -> 128通道 -> ...
解码器: 上采样 -> 2x2卷积 -> 特征拼接 -> 3x3卷积 -> ... -> 1x1卷积 -> 分割掩码
```

#### DeepLab 系列

| 版本 | 核心创新 | 空洞卷积膨胀率 |
|------|----------|----------------|
| DeepLabv1 | 空洞卷积 + CRF | 6, 12, 18, 24 |
| DeepLabv2 | ASPP (多尺度空洞卷积池化) | 6, 12, 18 |
| DeepLabv3 | 改进ASPP + 全局上下文 | 6, 12, 18 |
| DeepLabv3+ | 编码器-解码器 + Xception骨干 | 6, 12, 18 |

ASPP (Atrous Spatial Pyramid Pooling):
$$
Y_c(k) = \sum_{i} x[k + r_i \cdot a] \cdot w_c[i]
$$
其中 $r_i$ 是第 $i$ 个空洞卷积的膨胀率。

#### Mask R-CNN

- 两阶段实例分割框架
- 基础：Faster R-CNN (RPN + ROI Align)
- 新增 Mask 分支：预测每个 ROI 的二进制分割掩码
- ROI Align 替代 ROI Pooling，避免量化误差
- 支持同时检测、分类、分割

```
输入图像 -> CNN骨干网络 -> RPN -> ROI Align -> 类别分类 + 边界框回归 + 掩码预测
```

### 3. 评估指标

- **Pixel Accuracy**：正确分类像素占总像素比例
- **Mean IoU**：各类别 IoU 的平均值
- **FWIoU** (Frequency Weighted IoU)：按类别频率加权的 IoU
- **AP / mAP**：按实例计算的精度均值
- **PQ** (Panoptic Quality)：全景分割质量指标

## 代码示例

### 使用 PyTorch 和 Torchvision 进行语义分割

```python
import torch
import torchvision
from torchvision.models.segmentation import fcn_resnet50, FCN_ResNet50_Weights
from torchvision.transforms import functional as F
from PIL import Image
import numpy as np

# 加载预训练 FCN 模型
model = fcn_resnet50(weights=FCN_ResNet50_Weights.DEFAULT)
model.eval()

# 预处理输入图像
image = Image.open("input.jpg").convert("RGB")
input_tensor = F.pil_to_tensor(image).float() / 255.0
input_batch = input_tensor.unsqueeze(0)

# 推理
with torch.no_grad():
    output = model(input_batch)["out"]
    predictions = output.argmax(1)

# 转换为可视化的彩色掩码
mask = predictions.squeeze().numpy()
print(f"分割类别数: {len(np.unique(mask))}")
print(f"图像尺寸: {mask.shape}")
```

### U-Net 实现（简化版）

```python
import torch
import torch.nn as nn

class DoubleConv(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(in_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True)
        )
    def forward(self, x):
        return self.conv(x)

class UNet(nn.Module):
    def __init__(self, in_channels=3, out_channels=1, features=[64, 128, 256, 512]):
        super().__init__()
        self.downs = nn.ModuleList()
        self.ups = nn.ModuleList()
        self.pool = nn.MaxPool2d(2)

        # 编码器（下采样）
        for feature in features:
            self.downs.append(DoubleConv(in_channels, feature))
            in_channels = feature

        # 瓶颈层
        self.bottleneck = DoubleConv(features[-1], features[-1] * 2)

        # 解码器（上采样）
        for feature in reversed(features):
            self.ups.append(
                nn.ConvTranspose2d(feature * 2, feature, kernel_size=2, stride=2)
            )
            self.ups.append(DoubleConv(feature * 2, feature))

        # 输出层
        self.final_conv = nn.Conv2d(features[0], out_channels, kernel_size=1)

    def forward(self, x):
        skip_connections = []
        for down in self.downs:
            x = down(x)
            skip_connections.append(x)
            x = self.pool(x)

        x = self.bottleneck(x)
        skip_connections = skip_connections[::-1]

        for idx in range(0, len(self.ups), 2):
            x = self.ups[idx](x)
            skip = skip_connections[idx // 2]
            x = torch.cat([skip, x], dim=1)
            x = self.ups[idx + 1](x)

        return self.final_conv(x)

# 使用示例
model = UNet(in_channels=3, out_channels=1)
x = torch.randn(1, 3, 572, 572)
output = model(x)
print(f"输出尺寸: {output.shape}")  # [1, 1, 388, 388] (因Valid卷积会缩小)
```

### Mask R-CNN 使用示例

```python
from torchvision.models.detection import maskrcnn_resnet50_fpn, MaskRCNN_ResNet50_FPN_Weights
from torchvision.utils import draw_segmentation_masks
import torchvision.transforms as T
import torch

# 加载预训练 Mask R-CNN
model = maskrcnn_resnet50_fpn(weights=MaskRCNN_ResNet50_FPN_Weights.DEFAULT)
model.eval()

# 预处理
transform = T.Compose([T.ToTensor()])
image = Image.open("people.jpg").convert("RGB")
image_tensor = transform(image)

# 推理
with torch.no_grad():
    predictions = model([image_tensor])

# 查看预测结果
pred = predictions[0]
print(f"检测到实例数: {len(pred['boxes'])}")
print(f"类别: {pred['labels']}")
print(f"置信度: {pred['scores']}")
print(f"掩码数量: {len(pred['masks'])}")

# 可视化（保留置信度 > 0.5 的实例）
keep = pred['scores'] > 0.5
result_image = draw_segmentation_masks(
    (image_tensor * 255).byte(),
    pred['masks'][keep].squeeze(1) > 0.5,
    alpha=0.6
)
```

## 应用场景

| 场景 | 任务类型 | 典型模型 | 说明 |
|------|----------|----------|------|
| 自动驾驶 | 语义分割 | DeepLabv3+, U-Net | 道路、行人、车辆、建筑分割 |
| 医学影像 | 语义分割 | U-Net, nnU-Net | 肿瘤、器官、病灶区域检测 |
| 卫星遥感 | 语义分割 | DeepLabv3, FCN | 土地利用、建筑物、植被分类 |
| 人像处理 | 实例分割 | Mask R-CNN | 人物与背景分离，抠图 |
| 工业质检 | 实例分割 | YOLACT++, BlendMask | 产品表面缺陷检测与定位 |
| 全景感知 | 全景分割 | Panoptic FPN, UPSNet | 统一理解场景中所有物体和背景 |

## 相关概念

- **语义分割**：为每个像素分配类别，不区分实例
- **实例分割**：在检测基础上预测每个实例的掩码
- **全景分割**：统一语义和实例分割的全面场景理解
- **目标检测**：提供边界框，粒度较粗
- **图像分类**：图像级标签，无法定位
- **转置卷积**：上采样操作，恢复空间分辨率
- **空洞卷积**：增大感受野，不损失分辨率
- **ROI Align**：精确对齐 ROI 特征，避免量化误差
