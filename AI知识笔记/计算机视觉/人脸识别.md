# 人脸识别

## 一句话定义

人脸识别是通过提取和分析面部特征来验证或识别个体身份的一种生物特征识别技术。

## 核心公式

**FaceNet 嵌入向量 (Triplet Loss)**：

$$L = \sum_{i}^{N} \left[ \|f(x_i^a) - f(x_i^p)\|_2^2 - \|f(x_i^a) - f(x_i^n)\|_2^2 + \alpha \right]_{+}$$

其中 $x^a$ 是锚样本，$x^p$ 是正样本，$x^n$ 是负样本，$\alpha$ 是间隔边界。

**余弦相似度**（人脸比对）：

$$cos(\theta) = \frac{A \cdot B}{\|A\| \|B\|} = \frac{\sum_{i=1}^{n} A_i B_i}{\sqrt{\sum_{i=1}^{n} A_i^2} \sqrt{\sum_{i=1}^{n} B_i^2}}$$

## 详细说明

### 1. 识别流程

完整的人脸识别系统包含四个阶段：
1. **人脸检测**：定位图像中所有人脸区域
2. **人脸对齐**：检测关键点并进行几何标准化
3. **特征提取**：提取判别性面部特征向量
4. **特征匹配**：与数据库比对验证身份

### 2. 关键技术演进

**传统方法**：
- Eigenfaces：基于PCA降维
- Fisherfaces：LDA判别分析
- LBP + SVM：局部二值模式特征

**深度学习时代**：
- DeepFace (2014)：9层CNN，FaceBook提出
- FaceNet (2015)：三元组损失，直接学习嵌入
- ArcFace (2018)：加性角度间隔损失，性能优异
- InsightFace (2018)：开源实现，MiFace论文

### 3. 损失函数

| 损失函数 | 年份 | 特点 |
|---------|------|------|
| Contrastive Loss | 2006 | 成对样本距离约束 |
| Triplet Loss | 2015 | 三元组难例挖掘 |
| Center Loss | 2016 | 类内紧凑性 |
| SphereFace | 2017 | 乘性角度间隔 |
| CosFace | 2018 | 加性余弦间隔 |
| ArcFace | 2018 | 加性角度间隔 |

### 4. 评估指标与应用场景

**评估指标**：
- **FAR (False Acceptance Rate)**：假阳性率
- **FRR (False Rejection Rate)**：假阴性率
- **Verification Rate**：验证准确率
- **TAR@FAR=0.1%**：特定FAR下TAR

**应用场景**：
- 人脸支付与金融认证
- 安防监控与门禁系统
- 身份验证与手机解锁
- 失踪人口搜寻
- 社交媒体标签建议
