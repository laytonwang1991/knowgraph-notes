# 图像生成

## 一句话定义

图像生成是利用深度神经网络从随机噪声或条件信息（如文本、类别标签）中学习数据分布，并生成高质量、多样化图像的技术，是生成式人工智能的核心组成部分。

## 核心公式

### 1. GAN 的价值函数（零和博弈）

$$
\min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{data}(x)}[\log D(x)] + \mathbb{E}_{z \sim p_z(z)}[\log(1 - D(G(z)))]
$$

其中 $D$ 为判别器，$G$ 为生成器，$z$ 为随机噪声，$x$ 为真实数据。

### 2. VAE 的变分下界 (ELBO)

$$
\mathcal{L}(\theta, \phi; x) = \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)] - D_{KL}(q_\phi(z|x) \| p(z))
$$

第一项为重构损失，第二项为潜在空间正则化（KL 散度）。

### 3. 扩散模型前向过程（加噪）

$$
q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t I)
$$

简化为闭式：

$$
q(x_t | x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t} x_0, (1 - \bar{\alpha}_t)I)
$$

其中 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{i=1}^t \alpha_i$。

### 4. 扩散模型反向过程（去噪）

$$
p_\theta(x_{t-1} | x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \sigma_t^2 I)
$$

预测均值：

$$
\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)
$$

其中 $\epsilon_\theta$ 为网络预测的噪声。

### 5. DDPM 整体目标

$$
\mathcal{L} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(x_t, t) \|^2 \right]
$$

## 详细说明

### 1. GAN (生成对抗网络)

#### 核心架构

- **生成器 G**：从潜在空间采样，生成逼真图像
- **判别器 D**：区分真实图像与生成图像
- 两者对抗训练，纳什均衡时 $p_G = p_{data}$

#### 训练技巧

| 技巧 | 说明 | 作用 |
|------|------|------|
| 标签平滑 | 真实标签用 0.9 代替 1.0 | 防止判别器过拟合 |
| 谱归一化 | 判别器权重除以 spectral norm | 约束判别器 Lipschitz 常数 |
| 特征匹配 | 匹配中间层特征统计量 | 稳定训练 |
| 小批量歧视 | 判别器看到小批量统计量 | 增加生成多样性 |
| 条件GAN | 标签/文本作为额外输入 | 实现条件生成 |

#### 经典 GAN 变体

| 模型 | 年份 | 核心贡献 |
|------|------|----------|
| DCGAN | 2015 | 深度卷积GAN，稳定训练 |
| WGAN | 2017 | Wasserstein距离，解决模式崩溃 |
| WGAN-GP | 2017 | 梯度惩罚正则化 |
| Progressive GAN | 2017 | 渐进式增长分辨率 |
| BigGAN | 2018 | 大规模类别条件生成 |
| StyleGAN | 2018 | 风格化隐空间控制 |
| StyleGAN2 | 2020 | 权重解耦，伪影消除 |
| StyleGAN3 | 2021 | 连续性改进，适合视频 |

#### 隐空间与特征解耦

StyleGAN 的隐空间映射：

$$
w = f(z) = MLP(z)
$$

然后通过自适应实例归一化（AdaIN）注入到生成器：

$$
\text{AdaIN}(x, y) = y_s \cdot \frac{x - \mu(x)}{\sigma(x)} + y_b
$$

其中 $y_s, y_b$ 分别为从 $w$ 学习的缩放和偏置参数。

### 2. VAE (变分自编码器)

#### 核心思想

- 学习数据的潜在表示 $z \sim q(z|x)$
- 通过解码器重构：$x' = p(x|z)$
- KL 散度约束潜在分布接近先验 $p(z) = \mathcal{N}(0, I)$

#### 架构特点

```
编码器: x -> q(z|x) = N(μ(x), σ(x)) -> 重参数化 z
解码器: z -> p(x|z) -> x'
```

重参数化技巧：$z = \mu + \sigma \odot \epsilon$，其中 $\epsilon \sim \mathcal{N}(0, I)$

#### 优缺点

- 优点：训练稳定，有潜在空间，可插值
- 缺点：生成图像模糊，潜在空间可能不够解耦

### 3. 扩散模型 (Diffusion Model)

#### 前向过程（加噪）

逐步向图像添加高斯噪声，$T$ 步后接近各向同性高斯噪声：

$$
x_0 \xrightarrow{q} x_1 \xrightarrow{q} x_2 \rightarrow \cdots \rightarrow x_T \approx \mathcal{N}(0, I)
$$

#### 反向过程（去噪）

学习反向条件概率 $p_\theta(x_{t-1}|x_t)$，逐步从噪声恢复图像：

$$
x_T \xrightarrow{p_\theta} x_{T-1} \xrightarrow{p_\theta} \cdots \xrightarrow{p_\theta} x_0
$$

#### 条件扩散模型

Classifier-Free Guidance (CFG)：

$$
\tilde{\epsilon}_\theta(x_t, t, c) = (1 + w) \epsilon_\theta(x_t, t, c) - w \epsilon_\theta(x_t, t, \emptyset)
$$

其中 $w$ 为引导强度，$c$ 为条件（如文本），$\emptyset$ 为无条件输入。

#### 主流扩散模型

| 模型 | 机构 | 特点 |
|------|------|------|
| DDPM | OpenAI | 理论基础，2020 |
| DDIM | Tsinghua | 加速采样，50步即可 |
| Stable Diffusion | Stability AI | Latent扩散，开源 |
| DALL-E 2 | OpenAI | CLIP引导，文本生成 |
| Imagen | Google | 超分辨率，文字渲染强 |
| SDXL | Stability AI | 高分辨率，1024px |
| SD 3 | Stability AI | Transformer架构 |
| FLUX | Black Forest Labs | 卓越文字渲染 |

#### Latent Diffusion Model (Stable Diffusion)

在隐空间而非像素空间进行扩散：

1. 编码器：$z = \text{VAE}(x)$
2. 隐空间扩散：学习 $p(z_T|z_0)$ 的反向过程
3. 条件注入：Cross-Attention 将文本条件注入 U-Net
4. 解码器：$\hat{x} = \text{VAE}^{-1}(\hat{z}_0)$

$$
\mathcal{L} = \mathbb{E}_{z_0, c, \epsilon, t} \left[ \| \epsilon - \epsilon_\theta(z_t, t, c) \|^2 \right]
$$

### 4. 图像生成评估指标

| 指标 | 全称 | 说明 |
|------|------|------|
| Inception Score (IS) | - | 基于 Inception 网络的类别预测熵 |
| FID | Fréchet Inception Distance | 真实与生成特征的 Fréchet 距离，越低越好 |
| LPIPS | Learned Perceptual Image Patch Similarity | 感知相似度 |
| Precision/Recall | - | 精确率/召回率（多样性 vs 质量） |
| CLIP Score | - | 文本-图像匹配度 |

$$
\text{FID} = \| \mu_r - \mu_g \|^2 + \text{Tr}(\Sigma_r + \Sigma_g - 2\sqrt{\Sigma_r \Sigma_g})
$$

### GAN 训练（PyTorch 简化版）

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

# 定义生成器和判别器
class Generator(nn.Module):
    def __init__(self, latent_dim=100, img_shape=(1, 28, 28)):
        super().__init__()
        self.img_shape = img_shape
        self.model = nn.Sequential(
            nn.Linear(latent_dim, 128),
            nn.LeakyReLU(0.2),
            nn.Linear(128, 256),
            nn.BatchNorm1d(256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 512),
            nn.BatchNorm1d(512),
            nn.LeakyReLU(0.2),
            nn.Linear(512, 1024),
            nn.BatchNorm1d(1024),
            nn.LeakyReLU(0.2),
            nn.Linear(1024, int(torch.prod(torch.tensor(img_shape)))),
            nn.Tanh()
        )

    def forward(self, z):
        img = self.model(z)
        return img.view(img.size(0), *self.img_shape)


class Discriminator(nn.Module):
    def __init__(self, img_shape=(1, 28, 28)):
        super().__init__()
        self.model = nn.Sequential(
            nn.Linear(int(torch.prod(torch.tensor(img_shape))), 512),
            nn.LeakyReLU(0.2),
            nn.Linear(512, 256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 128),
            nn.LeakyReLU(0.2),
            nn.Linear(128, 1),
            nn.Sigmoid()
        )

    def forward(self, img):
        return self.model(img.view(img.size(0), -1))


# 训练循环
latent_dim = 100
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
G = Generator(latent_dim).to(device)
D = Discriminator().to(device)

criterion = nn.BCELoss()
optimizer_G = optim.Adam(G.parameters(), lr=0.0002, betas=(0.5, 0.999))
optimizer_D = optim.Adam(D.parameters(), lr=0.0002, betas=(0.5, 0.999))

# 训练步骤
for epoch in range(200):
    for batch_idx, (real_imgs, _) in enumerate(dataloader):
        real_imgs = real_imgs.to(device)
        batch_size = real_imgs.size(0)

        # 训练判别器
        D.zero_grad()
        label_real = torch.ones(batch_size, 1).to(device)
        label_fake = torch.zeros(batch_size, 1).to(device)

        output_real = D(real_imgs)
        loss_D_real = criterion(output_real, label_real)

        z = torch.randn(batch_size, latent_dim).to(device)
        fake_imgs = G(z)
        output_fake = D(fake_imgs.detach())
        loss_D_fake = criterion(output_fake, label_fake)

        loss_D = (loss_D_real + loss_D_fake) / 2
        loss_D.backward()
        optimizer_D.step()

        # 训练生成器
        G.zero_grad()
        output = D(fake_imgs)
        loss_G = criterion(output, label_real)
        loss_G.backward()
        optimizer_G.step()

    print(f"Epoch {epoch}: Loss_D={loss_D:.4f}, Loss_G={loss_G:.4f}")
```

### VAE 实现（PyTorch）

```python
class VAE(nn.Module):
    def __init__(self, latent_dim=20):
        super().__init__()
        self.latent_dim = latent_dim

        # 编码器
        self.encoder = nn.Sequential(
            nn.Linear(784, 400),
            nn.ReLU(),
            nn.Linear(400, 200),
            nn.ReLU()
        )
        self.fc_mu = nn.Linear(200, latent_dim)
        self.fc_logvar = nn.Linear(200, latent_dim)

        # 解码器
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 200),
            nn.ReLU(),
            nn.Linear(200, 400),
            nn.ReLU(),
            nn.Linear(400, 784),
            nn.Sigmoid()
        )

    def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

    def forward(self, x):
        x = x.view(x.size(0), -1)
        h = self.encoder(x)
        mu = self.fc_mu(h)
        logvar = self.fc_logvar(h)
        z = self.reparameterize(mu, logvar)
        x_recon = self.decoder(z)
        return x_recon, mu, logvar

    def loss_function(self, x_recon, x, mu, logvar):
        recon_loss = nn.functional.binary_cross_entropy(x_recon, x.view(x.size(0), -1), reduction='sum')
        kl_loss = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp())
        return recon_loss + kl_loss
```

### 简化 Diffusion 前向与反向过程

```python
import torch
import torch.nn.functional as F

def forward_diffusion(x0, t, betas):
    """前向加噪过程（闭式计算）"""
    alphas = 1 - betas
    alpha_bar = torch.cumprod(alphas, dim=0)

    # 添加噪声
    noise = torch.randn_like(x0)
    sqrt_alpha_bar = torch.sqrt(alpha_bar[t])
    sqrt_one_minus_alpha_bar = torch.sqrt(1 - alpha_bar[t])

    # 根据 t 索引获取对应 timestep 的系数
    sqrt_alpha_bar_t = sqrt_alpha_bar[t].view(-1, 1, 1, 1)
    sqrt_one_minus_t = sqrt_one_minus_alpha_bar[t].view(-1, 1, 1, 1)

    x_t = sqrt_alpha_bar_t * x0 + sqrt_one_minus_t * noise
    return x_t, noise

class SimpleDiffusionUNet(nn.Module):
    """简化的 UNet 用于去噪预测"""
    def __init__(self, in_channels=3, out_channels=3, time_emb_dim=100):
        super().__init__()
        self.time_mlp = nn.Sequential(
            nn.Linear(1, time_emb_dim),
            nn.SiLU(),
            nn.Linear(time_emb_dim, time_emb_dim)
        )
        # 编码器
        self.enc1 = nn.Conv2d(in_channels, 64, 3, padding=1)
        self.enc2 = nn.Conv2d(64, 128, 3, padding=1)
        # 解码器
        self.dec2 = nn.Conv2d(128 + time_emb_dim, 64, 3, padding=1)
        self.dec1 = nn.Conv2d(64 + time_emb_dim, out_channels, 3, padding=1)

    def forward(self, x_t, t):
        t_emb = self.time_mlp(t.float().unsqueeze(1))
        h1 = F.relu(self.enc1(x_t))
        h2 = F.relu(self.enc2(F.max_pool2d(h1, 2)))
        # 上采样并融合时间嵌入
        h2_up = F.interpolate(h2, scale_factor=2, mode='bilinear')
        h2_cat = torch.cat([h2_up, t_emb.unsqueeze(2).unsqueeze(3).expand(-1, -1, h2_up.shape[2], h2_up.shape[3])], dim=1)
        h1_out = F.relu(self.dec2(h2_cat))
        h_out = F.interpolate(h1_out, scale_factor=2, mode='bilinear')
        return self.dec1(torch.cat([h_out, t_emb.unsqueeze(2).unsqueeze(3).expand(-1, -1, h_out.shape[2], h_out.shape[3])], dim=1))

def ddpm_sampling(model, x_T, betas, device, num_timesteps=1000):
    """DDPM 采样过程"""
    model.eval()
    alphas = 1 - betas
    alphas_cumprod = torch.cumprod(alphas, dim=0)

    x_t = x_T
    for t in reversed(range(num_timesteps)):
        t_batch = torch.tensor([t] * x_T.size(0)).to(device)
        with torch.no_grad():
            epsilon_pred = model(x_t, t_batch)

        alpha_t = alphas[t]
        alpha_bar_t = alphas_cumprod[t]
        sigma_t = torch.sqrt(betas[t])

        if t > 0:
            noise = torch.randn_like(x_t)
        else:
            noise = 0

        mean = (1 / torch.sqrt(alpha_t)) * (x_t - (betas[t] / torch.sqrt(1 - alpha_bar_t)) * epsilon_pred)
        x_t = mean + sigma_t * noise

    return x_t
```

## 相关概念

- **生成对抗网络 (GAN)**：零和博弈的对抗训练框架
- **变分自编码器 (VAE)**：基于变分推断的生成模型
- **扩散模型 (Diffusion Model)**：渐进式加噪-去噪的生成范式
- **Latent Diffusion**：在隐空间进行扩散的高效生成方法
- **CLIP**：对比语言-图像预训练模型，用于条件生成
- **隐空间插值**：在 VAE/GAN 隐空间中线性插值生成过渡图像
- **Classifier-Free Guidance**：无需额外分类器的条件引导技术
- **ControlNet**：通过条件控制网络精准控制生成结构
