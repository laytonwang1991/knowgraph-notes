# 语音识别 (Automatic Speech Recognition, ASR)

## 一句话定义

语音识别是将人类语音转换为文本的技术，使机器能够"听懂"人类的口头语言。

## 核心公式/技术要点

### 声学模型核心架构
```
输入: 音频特征 (MFCC/FBANK/Filter Bank)
      ↓
声学特征提取 → 声学模型 (CTC/Attention) → 音素/字符序列
      ↓
语言模型 → 解码输出最终文本
```

### CTC (Connectionist Temporal Classification) 损失
$$L_{CTC} = -\sum_{(X,Y)} \log P(Y|X)$$

### Attention-based 端到端模型
$$H = \text{Encoder}(X)$$
$$C_i = \text{Attention}(H, h_{i-1})$$
$$y_i = \text{Decoder}(y_{i-1}, C_i)$$

## 详细说明

### 1. 语音识别发展历程

| 阶段 | 代表技术 | 特点 |
|------|----------|------|
| 早期 | GMM-HMM | 统计模型，需手动设计特征 |
| 深度学习时代 | DNN-HMM | 深度神经网络替代GMM |
| 端到端时代 | CTC, Attention | 直接音频到文本 |

### 2. 主流模型架构

**WaveNet (DeepMind, 2016)**
- 使用空洞卷积 (Dilated Convolution)
- 自回归生成模型
- 原始音频波形输入

**Deep Speech (Baidu, 2014)**
- 端到端 RNN 架构
- CTC 损失函数
- 超过人类识别率（特定数据集）

**Conformer (Google, 2020)**
- 结合 Transformer 和 CNN
- 局部+全局特征捕获
- 当前主流骨干网络

### 3. 关键组件

#### 声学特征提取
- **MFCC**: Mel-frequency Cepstral Coefficients
- **FBANK**: Filter Bank 特征
- **SpecAugment**: 数据增强

#### 解码策略
- Greedy Decoding: 选择最高概率token
- Beam Search: 保留多条候选路径
- Language Model Integration: 加入语言模型约束

### 4. 评估指标

- **WER (Word Error Rate)**: $WER = \frac{S + D + I}{N}$
  - S: Substitution (替换)
  - D: Deletion (删除)
  - I: Insertion (插入)
  - N: 总词数
- **CER (Character Error Rate)**: 字符级别错误率
