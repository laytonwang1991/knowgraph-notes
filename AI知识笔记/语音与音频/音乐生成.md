# 音乐生成

## 一句话定义

AI音乐生成是利用深度学习模型自动创作音乐的技术，能够根据文本描述、参考音频或符号输入生成具有特定风格、旋律和节奏的音乐作品。

## 详细说明

### 1. 主要技术范式

#### 1.1 音频到音频生成

直接生成音频波形，无需中间符号表示：

- **MusicLM**（Google）：大规模音乐生成模型，支持文本条件
- **Stable Audio**（Stability AI）：基于扩散模型的高质量音频生成
- **Riffusion**：基于Stable Diffusion的音频生成，支持参考旋律

#### 1.2 符号音乐生成

生成MIDI、ABC notation等符号表示：

- **符号驱动模型**：生成乐谱而非波形
- **优势**：可编辑、可解释、文件小
- **局限**：需要额外合成器转为声音

### 2. MusicLM 详解

Google的MusicLM是当前最著名的音乐生成系统：

**核心能力**：
- 文本到音乐（Text-to-Music）
- 风格保持（Style Conditioning）
- 旋律续写（Melody Continuation）

**架构特点**：
- MuLAN + SoundStream 联合训练
- 24kHz 高保真输出
- 支持长达数分钟的连贯生成

### 3. Stable Audio

Stability AI的扩散模型方案：

```
文本输入 → CLIP文本编码器 → 扩散过程 → 音频解码器 → 波形输出
```

**优势**：
- 推理速度快
- 音质优异
- 开源可商用

### 4. Riffusion

Riffusion创新性地将音频转换为梅尔频谱图，利用Stable Diffusion进行生成：

```python
# Riffusion 风格生成伪代码
from riffusion import RiffusionModel

model = RiffusionModel.load("riffusion/riffusion-model-v1")

# 基于参考音频和文本描述生成
result = model.generate(
    prompt="jazz piano solo, relaxed mood",
    audio_seed=load_audio("reference_melody.wav"),
    num_inference_steps=50,
    guidance_scale=7.5
)

save_audio(result, "jazz_piano.wav")
```

### 5. 符号音乐生成模型

| 模型 | 输入 | 输出 | 特点 |
|------|------|------|------|
| Transformer-MM | 符号序列 | MIDI | 大规模预训练 |
| PopMAG | 流行音乐 | MIDI | 多轨道生成 |
| ABCNet | ABC记谱法 | 音频 | 轻量级 |
| MusicVAE | 潜在空间 | MIDI | 插值编辑 |

## 代码示例

```python
# MusicLM API 调用示例
import musiclm

model = musiclm.load_model("google/musiclm-base")

# 文本条件生成
music = model.generate(
    text="calm piano music with soft strings in the background",
    duration=30,  # 秒
    temperature=0.9
)

# 保存生成结果
musiclm.save(music, "generated_music.wav")
```

```python
# Stable Audio 生成示例
from stability_sdk import audio

client = audio.Client()

generated = client.generate(
    prompt="upbeat electronic dance music with strong bass",
    negative_prompt="low quality, distorted audio",
    duration_seconds=15,
    steps=100,
    seed=42
)

with open("edm_track.wav", "wb") as f:
    f.write(generated.audio_content)
```

## 应用场景

| 场景 | 说明 |
|------|------|
| 游戏背景音乐 | 根据游戏场景动态生成适配音乐 |
| 视频配乐 | 自动为短视频创作者生成背景音乐 |
| 音乐制作辅助 | 提供旋律或和声灵感 |
| 音乐治疗 | 生成特定情绪氛围的音频 |
| 教育音乐 | 生成不同风格的学习背景音乐 |
| 广告音乐 | 快速生成符合品牌调性的配乐 |

## 相关概念

- **梅尔频谱图（Mel Spectrogram）**：音频的时频表示
- **扩散模型（Diffusion Model）**：通过逐步去噪生成数据的模型
- **CLIP对比学习**：连接文本和图像/音频的跨模态模型
- **MIDI协议**：数字音乐接口标准
- **节奏感（Rhythm）**：音乐的时序结构
- **和声进行（Chord Progression）**：和弦的序列组织
