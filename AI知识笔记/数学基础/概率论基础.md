# 概率论基础

## 一句话定义

概率论是研究随机现象数量规律性的数学分支，通过概率分布描述不确定性，是机器学习中进行统计推断和决策的核心工具。

---

## 核心公式

### 贝叶斯定理
$$
P(\theta | \mathbf{X}) = \frac{P(\mathbf{X} | \theta) P(\theta)}{P(\mathbf{X})}
$$

其中 $P(\theta)$ 是先验概率，$P(\mathbf{X}|\theta)$ 是似然，$P(\theta|\mathbf{X})$ 是后验概率。

### 期望与方差
$$
\mathbb{E}[X] = \sum_{i} x_i P(x_i) = \int x p(x) dx
$$
$$
\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2
$$

### 最大似然估计 (MLE)
$$
\hat{\theta}_{MLE} = \arg\max_{\theta} \prod_{i=1}^{n} P(x_i | \theta)
$$
通常使用对数形式：
$$
\hat{\theta}_{MLE} = \arg\max_{\theta} \sum_{i=1}^{n} \log P(x_i | \theta)
$$

### 链式法则
$$
P(x_1, x_2, ..., x_n) = P(x_1) P(x_2|x_1) P(x_3|x_1,x_2) ... P(x_n|x_1,...,x_{n-1})
$$

### KL散度
$$
D_{KL}(P || Q) = \sum_{x} P(x) \log \frac{P(x)}{Q(x)}
$$

## 详细说明

### 1. 概率分布

#### 离散分布
- **伯努利分布**：单次二值实验，$P(X=1) = p, P(X=0) = 1-p$
- **二项分布**：$n$次独立伯努利实验，$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$
- **泊松分布**：稀疏事件计数，$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- **多项分布**：多类别的二项分布推广

#### 连续分布
- **正态/高斯分布**：$p(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
- **指数分布**：$p(x) = \lambda e^{-\lambda x}, x \geq 0$
- **拉普拉斯分布**：$p(x) = \frac{1}{2b} e^{-\frac{|x-\mu|}{b}}$
- **Beta分布**：$[0,1]$区间的概率分布先验
- **Dirichlet分布**：多项分布的共轭先验

### 2. 统计量

- **期望**：随机变量的加权平均值
- **方差**：随机变量离散程度的度量
- **协方差**：两个随机变量共同变化的趋势
- **相关系数**：协方差的归一化版本，$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$

### 3. 贝叶斯定理

- **核心思想**：根据新证据更新信念
- **先验概率**：观察数据前的信念
- **似然函数**：数据在给定参数下的概率
- **后验概率**：观察数据后的更新信念
- **共轭先验**：先验与后验属于同一分布族

### 4. 估计方法

- **最大似然估计 (MLE)**：找到使观测数据概率最大的参数
- **最大后验估计 (MAP)**：加入先验的最大化后验概率
- **贝叶斯估计**：对参数进行积分，输出完整的后验分布

### 5. 不确定性类型

- **认知不确定性 (Epistemic)**：由于知识不足导致，可通过更多数据减少
- **偶然不确定性 (Aleatoric)**：数据本身的随机性，无法消除

## 代码示例

```python
import numpy as np
from scipy import stats

# 高斯分布
mu, sigma = 0, 1
x = np.linspace(-5, 5, 100)
pdf = stats.norm.pdf(x, mu, sigma)

# 采样
samples = np.random.normal(mu, sigma, 1000)

# 期望与方差计算
mean = np.mean(samples)
variance = np.var(samples)

# 贝叶斯更新示例 (Beta-Bernoulli)
alpha, beta = 1, 1  # Beta先验参数
data = [1, 0, 1, 1, 0]  # 观测数据

for obs in data:
    if obs == 1:
        alpha += 1
    else:
        beta += 1

# 后验 Beta(alpha, beta)
posterior_mean = alpha / (alpha + beta)
print(f"后验均值: {posterior_mean:.3f}")

# MLE 示例
from scipy.optimize import minimize

def neg_log_likelihood(theta, data):
    # 假设数据来自泊松分布
    lambda_est = theta[0]
    return -np.sum(stats.poisson.logpmf(data, lambda_est))

result = minimize(neg_log_likelihood, [1.0], args=(data,), method='L-BFGS-B')
print(f"ML估计的λ: {result.x[0]:.3f}")
```
