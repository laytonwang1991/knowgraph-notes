# 线性代数基础

## 一句话定义

线性代数是研究向量空间和线性映射的数学分支，是机器学习中描述、变换和压缩数据的核心工具。

---

## 核心公式

### 矩阵乘法
$$
\mathbf{C} = \mathbf{A} \times \mathbf{B}, \quad c_{ij} = \sum_{k=1}^{n} a_{ik} b_{kj}
$$

### 特征值分解
$$
\mathbf{A} = \mathbf{Q} \boldsymbol{\Lambda} \mathbf{Q}^{-1}
$$
其中 $\mathbf{Q}$ 是特征向量矩阵，$\boldsymbol{\Lambda}$ 是特征值对角矩阵。

### 奇异值分解 (SVD)
$$
\mathbf{A} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^{\top}
$$
其中 $\mathbf{U}$ 和 $\mathbf{V}$ 是正交矩阵，$\boldsymbol{\Sigma}$ 是奇异值对角矩阵。

### 行列式
$$
\det(\mathbf{A}) = \sum_{\sigma \in S_n} \text{sgn}(\sigma) \prod_{i=1}^{n} a_{i,\sigma(i)}
$$

---

## 详细说明

### 1. 向量与向量空间

- **向量**：有序的数值列表，表示方向和大小
  - 列向量：$\mathbf{v} = [v_1, v_2, ..., v_n]^{\top}$
  - 行向量：$\mathbf{v}^{\top} = [v_1, v_2, ..., v_n]$

- **向量空间**：满足加法和数乘封闭性的集合
  - 基向量：线性无关且能生成整个空间的向量组
  - 维数：基向量的个数

### 2. 矩阵运算

- **加法**：同维度矩阵对应元素相加
- **乘法**：行×列的点积，第$i$行与第$j$列交叉相乘求和
- **转置**：行列互换，$\mathbf{A}^{\top}[i,j] = \mathbf{A}[j,i]$
- **逆矩阵**：$\mathbf{A}^{-1}$，满足 $\mathbf{A}\mathbf{A}^{-1} = \mathbf{I}$
- **行列式**：矩阵"体积"的标量度量，$\det(\mathbf{A}) = 0$ 时矩阵奇异

### 3. 特征值与特征向量

- **定义**：$\mathbf{A}\mathbf{v} = \lambda \mathbf{v}$
- **几何意义**：特征向量指示变换的方向，特征值表示伸缩倍数
- **性质**：
  - 特征值之和等于矩阵迹（trace）
  - 特征值之积等于矩阵行列式

### 4. 奇异值分解 (SVD)

- 将任意矩阵分解为三个特殊矩阵的乘积
- **应用**：降维、去噪、推荐系统、图像压缩
- 截断SVD保留前$k$个奇异值实现有损压缩

### 5. 范数

- **L1范数**：$\|\mathbf{x}\|_1 = \sum |x_i|$（稀疏性度量）
- **L2范数**：$\|\mathbf{x}\|_2 = \sqrt{\sum x_i^2}$（欧氏距离）
- **Frobenius范数**：$\|\mathbf{A}\|_F = \sqrt{\sum_{i,j} a_{ij}^2}$

## 代码示例

```python
import numpy as np

# 矩阵运算
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = np.dot(A, B)  # 矩阵乘法

# 特征值分解
eigenvalues, eigenvectors = np.linalg.eig(A)

# SVD 分解
U, S, Vt = np.linalg.svd(A)

# 矩阵求逆
A_inv = np.linalg.inv(A)

# 范数计算
x = np.array([3, 4])
l2_norm = np.linalg.norm(x, 2)
l1_norm = np.linalg.norm(x, 1)

print(f"矩阵乘法结果:\n{C}")
print(f"特征值: {eigenvalues}")
print(f"SVD 奇异值: {S}")
print(f"L2范数: {l2_norm:.2f}")
```
