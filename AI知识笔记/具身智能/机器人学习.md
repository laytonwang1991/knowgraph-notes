# 机器人学习（Robot Learning）

## 一句话定义

机器人学习是使用机器学习方法让机器人获得感知、决策和控制能力，实现从简单操作到复杂任务自主执行的技术学科。

## 详细说明

### 1. 核心概念

机器人学习是具身智能的核心子领域，它将机器学习技术应用于机器人系统，涵盖以下几个关键方面：

- **技能获取**：让机器人学会执行特定任务（如抓取、装配、导航）
- **策略学习**：学习从状态到动作的映射策略
- **环境适应**：使机器人能够适应新环境和新任务
- **自主决策**：在复杂场景中进行实时决策

### 2. 机器人学习框架

### 3. 主要学习方法

#### 3.1 模仿学习（Imitation Learning）

通过观察专家演示来学习策略。

**行为克隆（Behavior Cloning）**：
```python
class BehaviorCloning:
    """行为克隆：直接从专家轨迹学习"""

    def __init__(self, state_dim, action_dim):
        self.policy = NeuralNetwork(state_dim, action_dim)

    def train(self, expert_trajectories):
        """
        Args:
            expert_trajectories: 专家轨迹列表
            每个轨迹: [(state, action), ...]
        """
        states = []
        actions = []

        for traj in expert_trajectories:
            for state, action in traj:
                states.append(state)
                actions.append(action)

        # 监督学习：minimize ||policy(s) - a||²
        self.policy.fit(np.array(states), np.array(actions))

    def predict(self, state):
        return self.policy.forward(state)
```

**DAgger（Dataset Aggregation）**：
```python
def dagger(env, expert_policy, learner_policy, num_iterations=10):
    """
    DAgger: 迭代式模仿学习

    1. 使用专家生成初始数据
    2. 训练学习器策略
    3. 运行学习器，询问专家更正
    4. 聚合数据，重复
    """
    dataset = []

    for iteration in range(num_iterations):
        # 运行当前策略收集轨迹
        episodes = collect_episodes(env, learner_policy)

        # 对于每个状态，查询专家获取正确动作
        for state, _ in episodes:
            expert_action = expert_policy(state)
            dataset.append((state, expert_action))

        # 用聚合数据重新训练
        learner_policy.train(dataset)

    return learner_policy
```

#### 3.2 强化学习控制（Reinforcement Learning Control）

通过与环境交互学习最优控制策略。

**策略梯度方法**：
```python
class PolicyGradient:
    """策略梯度算法的简化实现"""

    def __init__(self, state_dim, action_dim, learning_rate=0.001):
        self.policy = NeuralNetwork(state_dim, action_dim)
        self.learning_rate = learning_rate

    def select_action(self, state):
        """根据当前策略选择动作"""
        action_probs = self.policy.forward(state)
        # 简化为确定性策略
        return np.argmax(action_probs)

    def update(self, trajectory, rewards):
        """
        基于轨迹更新策略

        Args:
            trajectory: 状态-动作序列
            rewards: 累积回报
        """
        states, actions = zip(*trajectory)

        # 计算梯度：∇J = E[∇log π(a|s) * R]
        for state, action in zip(states, actions):
            action_dist = self.policy.forward(state)
            # 简化的梯度更新
            gradient = np.zeros_like(self.policy.weights)
            gradient[action] = state * rewards
            self.policy.weights += self.learning_rate * gradient

    def compute_returns(self, rewards, gamma=0.99):
        """计算折扣累积回报"""
        returns = []
        running_return = 0

        for r in reversed(rewards):
            running_return = r + gamma * running_return
            returns.insert(0, running_return)

        return np.array(returns)
```

**Soft Actor-Critic (SAC) 示例**：
```python
class SoftActorCritic:
    """SAC算法的核心组件"""

    def __init__(self, state_dim, action_dim):
        self.actor = Actor(state_dim, action_dim)  # 策略网络
        self.critic1 = Critic(state_dim, action_dim)  # Q网络1
        self.critic2 = Critic(state_dim, action_dim)  # Q网络2
        self.target_critic1 = Critic(state_dim, action_dim)
        self.target_critic2 = Critic(state_dim, action_dim)
        self.log_alpha = np.log(0.1)  # 熵温度参数

    def update(self, replay_buffer, batch_size=256, gamma=0.99, tau=0.005):
        """从经验回放更新网络"""
        batch = replay_buffer.sample(batch_size)
        states, actions, rewards, next_states, dones = batch

        # 1. 更新critic：最小化Bellman误差
        with torch.no_grad():
            next_actions, log_pi = self.actor.sample(next_states)
            target_q = torch.min(
                self.target_critic1(next_states, next_actions),
                self.target_critic2(next_states, next_actions)
            ) - self.log_alpha.exp() * log_pi
            target_q = rewards + gamma * (1 - dones) * target_q

        q1_loss = F.mse_loss(self.critic1(states, actions), target_q)
        q2_loss = F.mse_loss(self.critic2(states, actions), target_q)

        # 2. 更新actor：最大化Q值 + 熵
        new_actions, log_pi = self.actor.sample(states)
        q1_new = self.critic1(states, new_actions)
        actor_loss = (self.log_alpha.exp() * log_pi - q1_new).mean()

        # 3. 自动调整熵温度
        alpha_loss = -(self.log_alpha * (log_pi + self.target_entropy).detach()).mean()

        # 4. 软更新目标网络
        for target, source in [(self.target_critic1, self.critic1),
                                (self.target_critic2, self.critic2)]:
            for tp, sp in zip(target.parameters(), source.parameters()):
                tp.data.copy_(tau * sp.data + (1 - tau) * tp.data)
```

#### 3.3 视觉伺服（Visual Servoing）

基于视觉反馈的机器人控制技术。

**图像雅可比矩阵方法**：
```python
class VisualServoController:
    """基于图像的视觉伺服控制器"""

    def __init__(self, lambda_v=0.1):
        self.lambda_v = lambda_v  # 收敛速率
        self.epsilon = 1e-8  # 防止除零

    def compute_image_jacobian(self, feature, z, f=500):
        """
        计算图像雅可比矩阵

        Args:
            feature: 图像特征点位置 (u, v)
            z: 特征点深度
            f: 相机焦距

        Returns:
            J: 2x6 图像雅可比矩阵
        """
        u, v = feature
        J = np.array([
            [-f/z, 0, u/z, u*v/f, -(f + u**2/f), v],
            [0, -f/z, v/z, f + v**2/f, -u*v/f, -u]
        ])
        return J

    def compute_error(self, current_features, desired_features):
        """计算特征误差"""
        return (current_features - desired_features).flatten()

    def compute_velocity(self, current_features, desired_features, z_estimates):
        """
        计算末端执行器速度

        Args:
            current_features: 当前图像特征
            desired_features: 期望图像特征
            z_estimates: 特征点深度估计

        Returns:
            v_cam: 相机速度 (6,)
        """
        error = self.compute_error(current_features, desired_features)

        # 堆叠雅可比矩阵
        J_stack = []
        for feat, z in zip(current_features, z_estimates):
            J_stack.append(self.compute_image_jacobian(feat, z))

        J = np.vstack(J_stack)

        # 伪逆控制 law: v = λ * J^+ * e
        J_pseudo_inv = J.T @ np.linalg.inv(J @ J.T + self.epsilon * np.eye(len(error)))
        v_cam = self.lambda_v * J_pseudo_inv @ error

        return v_cam

    def control_loop(self, robot, camera, desired_features, max_iterations=100):
        """视觉伺服控制循环"""
        for iteration in range(max_iterations):
            # 1. 获取当前图像特征
            current_features = camera.detect_features()

            # 2. 估计深度
            z_estimates = self.estimate_depth(current_features)

            # 3. 计算控制速度
            v_cam = self.compute_velocity(current_features, desired_features, z_estimates)

            # 4. 转换为关节速度并执行
            v_task = self.camera_to_task(v_cam)
            joint_velocities = robot.cartesian_to_joint_velocity(v_task)
            robot.execute_velocity(joint_velocities)

            # 5. 检查收敛
            error_norm = np.linalg.norm(
                self.compute_error(current_features, desired_features)
            )
            if error_norm < 0.01:
                print(f"Converged at iteration {iteration}")
                return True

        return False
```

#### 3.4 运动规划（Motion Planning）

生成从起始状态到目标状态的无碰撞路径。

**RRT（快速随机树）算法**：
```python
class RRTPlanner:
    """RRT路径规划算法"""

    def __init__(self, workspace, obstacle_list, max_iterations=10000,
                 step_size=0.1, goal_bias=0.1):
        self.workspace = workspace
        self.obstacle_list = obstacle_list
        self.max_iterations = max_iterations
        self.step_size = step_size
        self.goal_bias = goal_bias
        self.tree = {}

    def sample_configuration(self):
        """采样配置空间"""
        if np.random.random() < self.goal_bias:
            return self.workspace['goal']
        else:
            return np.array([
                np.random.uniform(*self.workspace['x']),
                np.random.uniform(*self.workspace['y']),
                np.random.uniform(*self.workspace['z'])
            ])

    def distance(self, q1, q2):
        """配置空间距离"""
        return np.linalg.norm(q1 - q2)

    def steer(self, q_nearest, q_sample):
        """从最近节点向采样点 Steering"""
        direction = q_sample - q_nearest
        dist = self.distance(q_nearest, q_sample)

        if dist < self.step_size:
            return q_sample
        else:
            return q_nearest + (direction / dist) * self.step_size

    def is_collision_free(self, q):
        """检查配置是否无碰撞"""
        for obstacle in self.obstacle_list:
            if self.distance(q, obstacle['position']) < obstacle['radius']:
                return False
        return True

    def find_nearest(self, q):
        """找到树中最近的节点"""
        min_dist = float('inf')
        nearest = None

        for node_q in self.tree.keys():
            dist = self.distance(node_q, q)
            if dist < min_dist:
                min_dist = dist
                nearest = node_q

        return nearest

    def plan(self, start, goal):
        """
        RRT路径规划

        Returns:
            path: 从起点到终点的路径
        """
        self.tree = {tuple(start): None}

        for _ in range(self.max_iterations):
            # 1. 采样
            q_sample = self.sample_configuration()

            # 2. 找最近节点
            q_nearest = self.find_nearest(q_sample)

            # 3. Steering
            q_new = self.steer(q_nearest, q_sample)

            # 4. 碰撞检测
            if self.is_collision_free(q_new):
                self.tree[tuple(q_new)] = q_nearest

                # 5. 检查是否到达目标
                if self.distance(q_new, goal) < self.step_size:
                    self.tree[tuple(goal)] = q_new
                    return self.extract_path(goal)

        return None  # 规划失败

    def extract_path(self, goal):
        """提取从起点到终点的路径"""
        path = [goal]
        current = tuple(goal)

        while self.tree[current] is not None:
            current = self.tree[current]
            path.append(np.array(current))

        path.reverse()
        return path
```

**PRM（概率路线图）算法**：
```python
class PRMPlanner:
    """概率路线图（PRM）算法"""

    def __init__(self, num_samples=1000, k_nearest=10):
        self.num_samples = num_samples
        self.k_nearest = k_nearest
        self.roadmap = {}  # {(x,y): neighbors}

    def sample_free_configurations(self, workspace, obstacles):
        """采样无碰撞配置"""
        samples = []

        for _ in range(self.num_samples):
            q = np.array([
                np.random.uniform(*workspace['x']),
                np.random.uniform(*workspace['y']),
                np.random.uniform(*workspace['z'])
            ])

            if not self.check_collision(q, obstacles):
                samples.append(q)

        return samples

    def check_collision(self, q, obstacles):
        """碰撞检测"""
        for obs in obstacles:
            if np.linalg.norm(q - obs['position']) < obs['radius']:
                return True
        return False

    def build_roadmap(self, samples, obstacles):
        """构建概率路线图"""
        # 添加所有样本到路线图
        for sample in samples:
            self.roadmap[tuple(sample)] = []

        # 连接相邻样本
        for i, q1 in enumerate(samples):
            distances = []
            for j, q2 in enumerate(samples):
                if i != j:
                    distances.append((self.distance(q1, q2), j))

            # 取k个最近邻
            distances.sort(key=lambda x: x[0])
            k_nearest = distances[:self.k_nearest]

            for dist, j in k_nearest:
                q2 = samples[j]
                if not self.check_segment_collision(q1, q2, obstacles):
                    self.roadmap[tuple(q1)].append(tuple(q2))
                    self.roadmap[tuple(q2)].append(tuple(q1))

    def distance(self, q1, q2):
        return np.linalg.norm(q1 - q2)

    def query(self, start, goal, obstacles):
        """查询起点到终点的路径"""
        # 如果起点终点不在路线图中，添加它们
        self.add_to_roadmap(start, obstacles)
        self.add_to_roadmap(goal, obstacles)

        # BFS 搜索
        from collections import deque
        queue = deque([start])
        visited = {tuple(start): None}

        while queue:
            current = queue.popleft()

            if tuple(current) == tuple(goal):
                return self.extract_path(visited, goal)

            for neighbor in self.roadmap.get(tuple(current), []):
                if tuple(neighbor) not in visited:
                    visited[neighbor] = current
                    queue.append(np.array(neighbor))

        return None  # 无路径
```

## 应用场景

### 1. 工业制造

- **装配线自动化**：汽车、电子产品组装
- **柔性制造**：多品种小批量生产
- **质量检测**：视觉引导的缺陷检测

### 2. 服务机器人

- **餐饮服务**：送餐、烹饪辅助
- **酒店服务**：行李搬运、房间服务
- **家庭服务**：扫地、洗衣、收纳

### 3. 医疗手术

- **腹腔镜手术**：微创手术精确控制
- **骨科手术**：关节置换、骨折固定
- **介入治疗**：血管内操作

### 4. 农业自动化

- **果实采摘**：草莓、苹果等软果实采摘
- **作物监测**：苗圃管理、病虫害检测
- **精准农业**：播种、施肥、灌溉

### 5. 探索与救援

- **行星探索**：火星探测器、月球车
- **深海作业**：水下地形测绘、资源勘探
- **灾害救援**：地震、火灾现场的搜救
