# 长短期记忆网络

## 一句话定义

> LSTM通过引入门控机制和细胞状态，能够选择性记住或遗忘信息，有效解决了传统RNN难以学习长距离依赖的问题。

## 核心公式

### LSTM 门控机制

**遗忘门：** 决定丢弃什么信息
$$
f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)
$$

**输入门：** 决定更新什么信息
$$
i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)
$$

**输出门：** 决定输出什么信息
$$
o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)
$$

### 细胞状态更新

$$
C_t = f_t * C_{t-1} + i_t * \tilde{C}_t
$$

## 详细说明

### 1. LSTM 的核心思想

传统 RNN 的问题：梯度消失，难以学习长期依赖。

LSTM 的解决方案：
- **细胞状态**：一条信息传输的"高速公路"
- **门控机制**：控制信息的流入和流出
- **遗忘门**：决定丢弃什么旧信息
- **输入门**：决定添加什么新信息

### 2. LSTM 结构

```
细胞状态 C_t ←────── + ←────── * ←────── *
                ↑       ↑       ↑
           遗忘门   输入门   新候选
                ↑       ↑
              σ ←───┐   σ ←───┐
                ↑       ↑
输入 x_t → [h_{t-1}] ←─ h_t
```

### 3. 代码实现

```python
class LSTMCell(nn.Module):
    def __init__(self, input_size, hidden_size):
        super().__init__()
        self.hidden_size = hidden_size

        # 四个门：遗忘门、输入门、细胞候选、输出门
        self.W_ih = nn.Linear(input_size, 4 * hidden_size)
        self.W_hh = nn.Linear(hidden_size, 4 * hidden_size)

    def forward(self, x, state):
        h, c = state

        # 计算四个门
        gates = self.W_ih(x) + self.W_hh(h)
        i, f, g, o = gates.chunk(4, dim=-1)

        # 激活
        i = torch.sigmoid(i)
        f = torch.sigmoid(f)
        g = torch.tanh(g)
        o = torch.sigmoid(o)

        # 更新细胞状态和隐藏状态
        c_next = f * c + i * g
        h_next = o * torch.tanh(c_next)

        return h_next, c_next
```
