# 注意力机制

## 一句话定义

> 注意力机制是一种让模型能够"关注"输入序列中与当前任务最相关部分的技术。

## 核心公式

### Scaled Dot-Product Attention

$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

参数说明：
- $Q$（Query）：查询向量，代表当前位置想要关注的内容
- $K$（Key）：键向量，代表每个位置的特征
- $V$（Value）：值向量，代表每个位置的实际信息
- $d_k$：键向量的维度，用于缩放防止梯度消失

## 详细说明

### 1. 为什么需要注意力机制？

传统 RNN 的问题：
- 顺序处理，无法并行
- 长序列梯度消失/爆炸
- 难以捕捉长距离依赖

注意力机制解决了以上问题：
- 可以并行计算
- 直接建模任意距离的依赖
- 让模型自动学习关注什么

### 2. 工作原理

```
输入序列 → 计算Q、K、V → 计算注意力权重 → 加权求和 → 输出
```

步骤：
1. **线性变换**：输入通过三个权重矩阵得到 Q、K、V
2. **计算得分**：$score = QK^T$
3. **缩放**：$\frac{score}{\sqrt{d_k}}$
4. **softmax**：得到注意力权重
5. **加权求和**：权重 × V 得到输出

### 3. 多头注意力（Multi-Head Attention）

```python
class MultiHeadAttention:
    def __init__(self, d_model, num_heads):
        self.num_heads = num_heads
        self.d_k = d_model // num_heads
        self.W_q = Linear(d_model, d_model)
        self.W_k = Linear(d_model, d_model)
        self.W_v = Linear(d_model, d_model)
        self.W_o = Linear(d_model, d_model)

    def split_heads(self, x, batch_size):
        x = x.reshape(batch_size, -1, self.num_heads, self.d_k)
        return x.transpose(1, 2)

    def forward(self, query, key, value, mask=None):
        batch_size = query.size(0)

        # 线性变换并分头
        Q = self.split_heads(self.W_q(query), batch_size)
        K = self.split_heads(self.W_k(key), batch_size)
        V = self.split_heads(self.W_v(value), batch_size)

        # 缩放点积注意力
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        attention_weights = F.softmax(scores, dim=-1)
        output = torch.matmul(attention_weights, V)

        # 合并多头并输出
        output = output.transpose(1, 2).contiguous().view(batch_size, -1, self.num_heads * self.d_k)
        return self.W_o(output)
```

### 4. 应用场景

- 机器翻译
- 文本生成
- 图像描述
- 语音识别
