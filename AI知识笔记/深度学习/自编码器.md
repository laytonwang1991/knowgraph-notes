# 自编码器

## 一句话定义

> 自编码器是一种无监督学习的神经网络，通过编码器将输入数据压缩到低维潜在空间，再由解码器重建原始输入数据，学习数据的有效表征。

## 核心公式

### 基本自编码器结构

$$L = \|x - D(E(x))\|^2$$

其中 $E$ 是编码器，$D$ 是解码器，$x$ 是原始输入，$L$ 是重建损失。

### 变分自编码器 (VAE)

$$L = \|x - D(z)\|^2 + KL(q(z|x) \| p(z))$$

第一项是重建损失，第二项是 KL 散度正则化项，确保潜在空间的结构化。

## 详细说明

### 1. 基本结构

- **编码器 (Encoder)**：将高维输入 $x$ 映射到低维潜在空间 $z = f(x)$
- **解码器 (Decoder)**：从潜在表征 $z$ 重建输入 $\hat{x} = g(z)$
- **潜在空间 (Latent Space)**：压缩后的紧凑表示，捕捉数据的核心特征

### 2. 核心类型

#### 欠完备自编码器 (Undercomplete AE)

- 潜在空间维度小于输入维度
- 强制学习有用的数据表征
- 可用于降维和特征提取

#### 降噪自编码器 (Denoising AE)

- 输入被破坏（添加噪声），目标是重建原始干净数据
- 学习更鲁棒的表征
- 公式：$L = \|x - D(E(\tilde{x}))\|^2$，其中 $\tilde{x}$ 是噪声输入

#### 变分自编码器 (VAE)

- 潜在空间被建模为概率分布
- 使用重参数化技巧：$z = \mu + \sigma \cdot \epsilon$
- 生成模型的基础组件

#### 稀疏自编码器 (Sparse AE)

- 在损失函数中加入稀疏惩罚项
- 公式：$L = \|x - D(E(x))\|^2 + \lambda \sum_j |h_j|$
- 鼓励学习分散的表征

### 3. 训练机制

- 无需标签，使用重构误差作为监督信号
- 端到端训练，编码器和解码器同时优化
- 早停防止过拟合（记忆原始数据）

## 代码实现

### PyTorch 简单自编码器

```python
import torch
import torch.nn as nn

class Autoencoder(nn.Module):
    def __init__(self, input_dim, latent_dim):
        super().__init__()
        # 编码器
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, 256),
            nn.ReLU(),
            nn.Linear(256, latent_dim),
            nn.ReLU()
        )
        # 解码器
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.ReLU(),
            nn.Linear(256, input_dim),
            nn.Sigmoid()
        )

    def forward(self, x):
        z = self.encoder(x)
        x_recon = self.decoder(z)
        return x_recon, z

# 训练循环
model = Autoencoder(input_dim=784, latent_dim=32)
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
```

### 变分自编码器实现

```python
class VAE(nn.Module):
    def __init__(self, input_dim, latent_dim):
        super().__init__()
        self.fc1 = nn.Linear(input_dim, 256)
        self.fc_mu = nn.Linear(256, latent_dim)
        self.fc_logvar = nn.Linear(256, latent_dim)
        self.fc_decode = nn.Linear(latent_dim, 256)
        self.fc_out = nn.Linear(256, input_dim)

    def encode(self, x):
        h = torch.relu(self.fc1(x))
        mu = self.fc_mu(h)
        logvar = self.fc_logvar(h)
        return mu, logvar

    def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

    def decode(self, z):
        h = torch.relu(self.fc_decode(z))
        return torch.sigmoid(self.fc_out(h))

    def forward(self, x):
        mu, logvar = self.encode(x)
        z = self.reparameterize(mu, logvar)
        x_recon = self.decode(z)
        return x_recon, mu, logvar

def vae_loss(x_recon, x, mu, logvar):
    recon_loss = nn.functional.mse_loss(x_recon, x, reduction='sum')
    kl_loss = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp())
    return recon_loss + kl_loss
```

## 应用场景

### 1. 数据降维与可视化

- 将高维数据（如图像）降到 2-3 维进行可视化
- 比 PCA 更强大的非线性降维

### 2. 特征提取

- 学习到的潜在表征可用于下游任务
- 作为其他神经网络的预训练权重

### 3. 图像去噪

- 降噪自编码器可以去除图像噪声
- 医学图像处理、老照片修复

### 4. 异常检测

- 正常数据的重建误差小，异常数据误差大
- 工业缺陷检测、金融欺诈检测

### 5. 图像生成

- VAE 可以从潜在空间采样生成新图像
- 图像插值、风格迁移

### 6. 推荐系统

- 学习用户和物品的潜在表征
- 协同过滤的神经扩展

## 相关概念

- **变分自编码器 (VAE)**：自编码器的概率化扩展，支持生成建模
- **生成对抗网络 (GAN)**：另一种生成模型，使用对抗训练
- **对比学习 (Contrastive Learning)**：自监督学习方法，学习区分性表征
- **去噪扩散模型 (DDPM)**：基于去噪的生成模型
- **神经网络基础**：了解全连接层、激活函数等基础组件
