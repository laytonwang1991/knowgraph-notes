# 循环神经网络

## 一句话定义

> RNN是一种具有内部状态（记忆）的神经网络，能够处理任意长度的序列数据。

## 核心公式

### RNN 前向传播

$$
h_t = \tanh(W_{xh}x_t + W_{hh}h_{t-1} + b_h)
$$
$$
y_t = W_{hy}h_t + b_y
$$

其中 $h_t$ 是时刻 $t$ 的隐藏状态，$h_{t-1}$ 是上一时刻的状态。

## 详细说明

### 1. 为什么需要 RNN？

传统神经网络的局限：
- 输入输出长度固定
- 无法处理序列数据
- 无法共享不同位置的信息

RNN 的优势：
- 处理变长序列
- 记忆之前的信息
- 参数共享

### 2. RNN 的问题

**梯度消失/爆炸**

- 长时间序列的反向传播中，梯度会指数级衰减或爆炸
- 难以学习长距离依赖

**解决方案：**
- LSTM（长短期记忆网络）
- GRU（门控循环单元）
- Gradient Clipping（梯度裁剪）

### 3. LSTM 结构

```python
class LSTMCell(nn.Module):
    def __init__(self, input_size, hidden_size):
        super().__init__()
        self.hidden_size = hidden_size
        # 遗忘门
        self.W_f = nn.Linear(input_size + hidden_size, hidden_size)
        # 输入门
        self.W_i = nn.Linear(input_size + hidden_size, hidden_size)
        # 输出门
        self.W_o = nn.Linear(input_size + hidden_size, hidden_size)
        # 候选记忆
        self.W_c = nn.Linear(input_size + hidden_size, hidden_size)

    def forward(self, x, h, c):
        # 遗忘门：决定忘记多少过去的信息
        f = torch.sigmoid(self.W_f(torch.cat([x, h], dim=-1)))
        # 输入门：决定更新多少新信息
        i = torch.sigmoid(self.W_i(torch.cat([x, h], dim=-1)))
        # 候选记忆
        c_tilde = torch.tanh(self.W_c(torch.cat([x, h], dim=-1)))
        # 更新细胞状态
        c_new = f * c + i * c_tilde
        # 输出门
        o = torch.sigmoid(self.W_o(torch.cat([x, h], dim=-1)))
        # 新的隐藏状态
        h_new = o * torch.tanh(c_new)
        return h_new, c_new
```

### 4. 应用场景

| 任务 | 应用 |
|------|------|
| NLP | 文本生成、机器翻译 |
| 语音 | 语音识别、语音合成 |
| 时序 | 股票预测、天气预测 |

## 优缺点

| 优点 | 缺点 |
|------|------|
| 处理变长序列 | 训练慢（无法并行） |
| 记忆能力 | 梯度消失问题 |
| 参数共享 | 难以捕捉长距离依赖 |
