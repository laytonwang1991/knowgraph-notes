# 损失函数

## 一句话定义

损失函数是衡量神经网络预测值与真实值之间差异的函数，网络的训练过程就是最小化这个差异的过程。

## 核心公式

### 交叉熵损失 (Cross-Entropy Loss)

**二分类交叉熵**:
$$
L = -[y \log(\hat{y}) + (1-y) \log(1-\hat{y})]
$$

**多分类交叉熵 (Softmax Loss)**:
$$
L = -\sum_{c=1}^{C} y_c \log(\hat{y}_c)
$$
其中 $\hat{y}_c = \frac{e^{z_c}}{\sum_{k=1}^{C} e^{z_k}}$ 为 softmax 函数

### 均方误差 (MSE / L2 Loss)
$$
L_{\text{MSE}} = \frac{1}{n} \sum_{i=1}^{n}(y_i - \hat{y}_i)^2
$$

### Huber Loss (Smooth L1 Loss)
$$
L_{\text{Huber}}(y, \hat{y}) =
\begin{cases}
\frac{1}{2}(y - \hat{y})^2 & \text{if } |y - \hat{y}| \leq \delta \\
\delta |y - \hat{y}| - \frac{1}{2}\delta^2 & \text{otherwise}
\end{cases}
$$

### Focal Loss
$$
L_{\text{Focal}} = -\alpha_t (1 - \hat{y}_t)^\gamma \log(\hat{y}_t)
$$
其中 $\gamma$ 为聚焦参数，通常取 2

### 对比损失 (Contrastive Loss)
$$
L = \frac{1}{2N} \sum_{i=1}^{N} y_i d_i^2 + (1-y_i) \max(\margin - d_i, 0)^2
$$
其中 $d_i = \|a_i - b_i\|_2$ 为样本对距离

## 详细说明

### 1. 交叉熵损失 (Cross-Entropy Loss)

- **类型**: 分类任务首选
- **优点**:
  - 梯度表达优良，避免梯度消失
  - 概率意义上的最优损失函数
  - 与 softmax 函数配合使用
- **缺点**:
  - 对噪声标签敏感
  - 仅关注正确类别的概率

### 2. 均方误差 (MSE)

- **类型**: 回归任务首选
- **优点**:
  - 易于理解和实现
  - 处处可导
  - 对大误差惩罚更大
- **缺点**:
  - 对异常值（outliers）敏感
  - 在分类任务中收敛慢

### 3. Huber Loss

- **类型**: 回归任务，对异常值更鲁棒
- **优点**:
  - 结合了 L1 和 L2 损失的优点
  - 对异常值不敏感
  - 梯度平滑
- **缺点**:
  - 需要手动设置 $\delta$ 参数

### 4. Focal Loss

- **由 Lin et al. 在 RetinaNet 中提出 (2017)**
- **用途**: 处理类别不平衡的目标检测问题
- **优点**:
  - 降低易分类样本的权重
  - 聚焦于难分类样本
- **缺点**: 引入额外超参数 $\alpha$ 和 $\gamma$

### 5. 对比损失 (Contrastive Loss)

- **用途**: 度量学习、对比学习
- **优点**:
  - 使相似样本距离更近
  - 不相似样本距离更远
- **缺点**: 需要成对训练数据

## 代码示例

```python
import numpy as np
import torch
import torch.nn as nn
import torch.nn.functional as F

# ============ PyTorch 实现 ============

# 1. 交叉熵损失 (多分类)
criterion = nn.CrossEntropyLoss()
logits = torch.randn(32, 10)  # batch_size=32, 10个类别
labels = torch.randint(0, 10, (32,))
loss = criterion(logits, labels)

# 2. 二分类交叉熵
bce_loss = nn.BCEWithLogitsLoss()
logits = torch.randn(32, 1)
labels = torch.randn(32, 1)
loss = bce_loss(logits, labels)

# 3. MSE 损失
mse_loss = nn.MSELoss()
predictions = torch.randn(32, 1)
targets = torch.randn(32, 1)
loss = mse_loss(predictions, targets)

# 4. Huber Loss
huber_loss = nn.HuberLoss(delta=1.0)
loss = huber_loss(predictions, targets)

# 5. Focal Loss
class FocalLoss(nn.Module):
    def __init__(self, alpha=0.25, gamma=2.0):
        super().__init__()
        self.alpha = alpha
        self.gamma = gamma

    def forward(self, inputs, targets):
        bce_loss = F.binary_cross_entropy_with_logits(
            inputs, targets, reduction='none'
        )
        pt = torch.exp(-bce_loss)
        focal_loss = self.alpha * (1 - pt) ** self.gamma * bce_loss
        return focal_loss.mean()

# 6. Contrastive Loss
class ContrastiveLoss(nn.Module):
    def __init__(self, margin=1.0):
        super().__init__()
        self.margin = margin

    def forward(self, output1, output2, label):
        euclidean_distance = F.pairwise_distance(output1, output2)
        loss = 0.5 * (label * euclidean_distance.pow(2) +
                      (1 - label) * F.relu(self.margin - euclidean_distance).pow(2))
        return loss.mean()
```

```python
# ============ NumPy 实现（简单示例）============

def cross_entropy_loss(y_pred, y_true):
    """交叉熵损失"""
    eps = 1e-15
    y_pred = np.clip(y_pred, eps, 1 - eps)
    return -np.sum(y_true * np.log(y_pred)) / len(y_true)

def mse_loss(y_pred, y_true):
    """均方误差"""
    return np.mean((y_pred - y_true) ** 2)

def huber_loss(y_pred, y_true, delta=1.0):
    """Huber Loss"""
    diff = np.abs(y_pred - y_true)
    loss = np.where(diff <= delta,
                    0.5 * diff ** 2,
                    delta * diff - 0.5 * delta ** 2)
    return np.mean(loss)
```

## 应用场景

| 损失函数 | 适用场景 |
|---------|---------|
| 交叉熵 (CE) | 图像分类、文本分类、语音识别 |
| BCE | 二分类、多标签分类 |
| MSE | 房价预测、年龄估计、回归任务 |
| Huber Loss | 姿态估计、目标检测（回归框） |
| Focal Loss | 类别不平衡的目标检测、细粒度分类 |
| Contrastive Loss | 人脸识别、度量学习、对比学习 (SimCLR, MoCo) |
| Triplet Loss | 人脸识别、行人重识别 |

## 相关概念

- **激活函数**: 输出层激活函数决定输出分布，与损失函数配合
- **优化器**: 基于损失函数梯度更新模型参数
- **神经网络基础**: 前向传播计算损失，反向传播最小化损失
