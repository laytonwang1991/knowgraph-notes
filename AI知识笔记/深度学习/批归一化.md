# 批归一化（Batch Normalization）

## 一句话定义

批归一化通过对每一批次的输入数据进行均值和方差的规范化，将每层神经网络的输入分布稳定到标准正态分布，从而加速深度网络的训练并缓解梯度消失问题。

## 核心公式

**规范化过程：**

$$
\hat{x}^{(k)} = \frac{x^{(k)} - \mu_B^{(k)}}{\sqrt{\sigma_B^{(k)2} + \epsilon}}
$$

**仿射变换（可学习参数）：**

$$
y^{(k)} = \gamma^{(k)} \hat{x}^{(k)} + \beta^{(k)}
$$

其中：
- $\mu_B$：批次均值
- $\sigma_B^2$：批次方差
- $\gamma$：缩放参数（scale）
- $\beta$：平移参数（shift）
- $\epsilon$：防止除零的小常数（通常为 $10^{-5}$）

## 详细说明

### 1. Internal Covariate Shift 问题

- **定义**：深度神经网络中，由于前面层的参数不断更新，导致后续层的输入分布在训练过程中不断变化
- **影响**：
  - 后续层需要不断适应这种变化，降低了训练效率
  - 深层网络底层参数的微小变化会被逐层放大
  - 导致梯度变得不稳定，引发梯度消失或梯度爆炸

### 2. 批归一化的数学原理

**训练阶段：**
- 对每个mini-batch计算均值和方差
- 使用移动平均来记录全局统计量供推理时使用

**推理阶段：**
- 使用训练阶段累积的移动均值和移动方差
- 不再计算批次统计量

### 3. 可学习参数 γ 和 β

- **γ（scale）**：恢复模型的表达能力，避免简单归一化限制模型
- **β（shift）**：恢复模型的表达能力，允许学习到恒等映射
- 当 $\gamma = \sigma_B$, $\beta = \mu_B$ 时，输出即为原始输入

### 4. 推理阶段的处理

$$
\mu_{final} = E(\mu_B)
$$

$$
\sigma_{final}^2 = \frac{1}{m} \sum_{i=1}^{m} \sigma_B^{(i)2}
$$

推理时使用固定的全局均值和方差，而不是当前输入的统计量。

## 代码示例

### PyTorch 实现

```python
import torch
import torch.nn as nn

class BatchNorm1d(nn.Module):
    """一维批归一化（适用于全连接层输出）"""
    def __init__(self, num_features, eps=1e-5, momentum=0.1):
        super().__init__()
        self.num_features = num_features
        self.eps = eps
        self.momentum = momentum
        # 可学习参数
        self.gamma = nn.Parameter(torch.ones(num_features))
        self.beta = nn.Parameter(torch.zeros(num_features))
        # 推理时使用的移动统计量
        self.register_buffer('running_mean', torch.zeros(num_features))
        self.register_buffer('running_var', torch.ones(num_features))

    def forward(self, x):
        if self.training:
            # 计算批次均值和方差
            batch_mean = x.mean(dim=0)
            batch_var = x.var(dim=0, unbiased=False)
            # 更新移动统计量
            self.running_mean = (1 - self.momentum) * self.running_mean + self.momentum * batch_mean
            self.running_var = (1 - self.momentum) * self.running_var + self.momentum * batch_var
        else:
            batch_mean = self.running_mean
            batch_var = self.running_var

        # 归一化
        x_norm = (x - batch_mean) / torch.sqrt(batch_var + self.eps)
        # 仿射变换
        return self.gamma * x_norm + self.beta
```

### 使用示例

```python
import torch.nn.functional as F

class MLPWithBN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.bn1 = nn.BatchNorm1d(hidden_size)
        self.fc2 = nn.Linear(hidden_size, hidden_size)
        self.bn2 = nn.BatchNorm1d(hidden_size)
        self.fc3 = nn.Linear(hidden_size, output_size)

    def forward(self, x):
        x = self.fc1(x)
        x = self.bn1(x)
        x = F.relu(x)
        x = self.fc2(x)
        x = self.bn2(x)
        x = F.relu(x)
        x = self.fc3(x)
        return x
```

### 自定义实现对比

```python
def batch_norm_forward(x, gamma, beta, eps=1e-5):
    """批归一化前向传播"""
    N, D = x.shape

    # 计算批次均值
    mu = x.mean(axis=0)
    # 计算批次方差
    var = x.var(axis=0, unbiased=False)
    # 归一化
    x_norm = (x - mu) / np.sqrt(var + eps)
    # 仿射变换
    out = gamma * x_norm + beta

    cache = (x, x_norm, mu, var, gamma, beta, eps)
    return out, cache

def batch_norm_backward(dout, cache):
    """批归一化反向传播"""
    x, x_norm, mu, var, gamma, beta, eps = cache
    N, D = x.shape

    dgamma = (dout * x_norm).sum(axis=0)
    dbeta = dout.sum(axis=0)

    dx_norm = dout * gamma
    dvar = (dx_norm * (x - mu)).sum(axis=0) * (-0.5) * (var + eps)**(-1.5)
    dmu = (dx_norm * (-1) / np.sqrt(var + eps)).sum(axis=0) + dvar * (-2/N) * (x - mu).sum(axis=0)
    dx = dx_norm / np.sqrt(var + eps) + dvar * 2 * (x - mu) / N + dmu / N

    return dx, dgamma, dbeta
```

## 应用场景

1. **计算机视觉**
   - 图像分类（ResNet、VGG 等网络的标准组件）
   - 目标检测（YOLO、SSD 等）
   - 语义分割（FCN、UNet 等）

2. **自然语言处理**
   - 早期NLP模型（如TextCNN）
   - 部分Transformer变体

3. **训练加速**
   - 允许使用更大的学习率
   - 减少对Dropout的依赖
   - 加速收敛速度

4. **不适用的场景**
   - Batch Size 很小的时候（统计量不稳定）
   - RNN 等序列模型（序列长度可变）
   - 动态图谱模型

## 相关概念

| 概念 | 区别 |
|------|------|
| **层归一化** | 对单个样本的所有特征归一化，不依赖batch |
| **实例归一化** | 对每个样本的每个通道独立归一化 |
| **组归一化** | 将通道分组，组内归一化 |
| **权重归一化** | 对权重参数进行归一化，而非激活值 |

### 归一化方法对比

| 方法 | 归一化维度 | Batch依赖 | RNN适用 |
|------|-----------|----------|---------|
| Batch Norm | Batch × Feature | 是 | 否 |
| Layer Norm | Feature | 否 | 是 |
| Instance Norm | Batch × Channel | 否 | 是 |
| Group Norm | Batch × (Channel/G) | 否 | 是 |

## 延伸阅读

1. **原始论文**：Ioffe & Szegedy, "Batch Normalization: Accelerating Deep Network Training by Reducing Internal Covariate Shift", 2015

2. **深入理解**：
   - Batch Renormalization（限制训练和推理分布差异）
   - Synchronized Batch Norm（分布式训练中的批归一化）
   - Evolving Norm（自适应归一化）

3. **实践技巧**：
   - 注意区分训练模式和推理模式的batch统计量处理
   - 卷积层的BatchNorm通常跟在Conv之后、激活函数之前
   - 不建议与Dropout同时使用（效果可能变差）
