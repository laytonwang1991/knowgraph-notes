# 激活函数

## 一句话定义

激活函数是神经网络中引入非线性特性的数学函数，它对神经元加权求和后的结果进行变换，决定该神经元是否被"激活"输出。

## 核心公式

### Sigmoid
$$
\sigma(x) = \frac{1}{1 + e^{-x}}
$$

### Tanh
$$
\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
$$

### ReLU (Rectified Linear Unit)
$$
\text{ReLU}(x) = \max(0, x)
$$

### Leaky ReLU
$$
\text{LeakyReLU}(x) = \max(\alpha x, x) \quad \alpha \approx 0.01
$$

### Swish
$$
\text{Swish}(x) = x \cdot \sigma(\beta x) \quad \beta \in \mathbb{R}
$$

### GELU (Gaussian Error Linear Unit)
$$
\text{GELU}(x) = x \cdot \Phi(x) = x \cdot \frac{1}{2}\left[1 + \text{erf}\left(\frac{x}{\sqrt{2}}\right)\right]
$$

## 详细说明

### 1. Sigmoid 函数

- **输出范围**: (0, 1)，适合二分类概率输出
- **优点**: 平滑可导，输出可以被解释为概率
- **缺点**:
  - 梯度消失问题（导数最大值仅 0.25）
  - 输出不是零中心（zero-centered）
  - 计算开销较大（涉及指数运算）

### 2. Tanh 函数

- **输出范围**: (-1, 1)，零中心化
- **优点**: 收敛速度通常比 Sigmoid 快
- **缺点**: 同样存在梯度消失问题

### 3. ReLU 函数

- **输出范围**: [0, +∞)
- **优点**:
  - 计算效率高（仅需比较运算）
  - 缓解梯度消失问题
  - 稀疏激活特性
- **缺点**:
  - "dying ReLU"问题：负值区域梯度为零
  - 输出不是零中心

### 4. Leaky ReLU

- **改进**: 负值区域给予极小斜率 $\alpha$
- **优点**: 解决了 "dying ReLU" 问题
- **缺点**: $\alpha$ 需要人工设定

### 5. Swish

- **由 Google Brain 提出 (2017)**
- **优点**:
  - 自门控（self-gated）机制
  - 在深层网络中表现优于 ReLU
- **缺点**: 计算量稍大

### 6. GELU

- **由 BERT 等 Transformer 模型广泛采用**
- **近似计算**: $\text{GELU}(x) \approx 0.5x\left(1 + \tanh\left(\sqrt{\frac{2}{\pi}}(x + 0.044715x^3)\right)\right)$
- **优点**:
  - 平滑非凸
  - 在 NLP 任务中表现优异

## 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def tanh(x):
    return np.tanh(x)

def relu(x):
    return np.maximum(0, x)

def leaky_relu(x, alpha=0.01):
    return np.where(x > 0, x, alpha * x)

def swish(x, beta=1.0):
    return x * sigmoid(beta * x)

def gelu(x):
    # 近似实现
    return 0.5 * x * (1 + np.tanh(np.sqrt(2/np.pi) * (x + 0.044715 * x**3)))

# 绘制各激活函数对比图
x = np.linspace(-5, 5, 1000)
funcs = {
    'Sigmoid': sigmoid(x),
    'Tanh': tanh(x),
    'ReLU': relu(x),
    'Leaky ReLU': leaky_relu(x),
    'Swish': swish(x),
    'GELU': gelu(x)
}
```

## 应用场景

| 激活函数 | 适用场景 |
|---------|---------|
| Sigmoid | 二分类输出层、概率输出 |
| Tanh | 循环神经网络 (RNN/LSTM) 的隐藏层 |
| ReLU | 卷积神经网络 (CNN) 隐藏层（默认选择） |
| Leaky ReLU | 避免神经元死亡 |
| Swish | 深层网络、超参数搜索 |
| GELU | Transformer架构、NLP任务、BERT等 |

## 相关概念

- **损失函数**: 衡量预测值与真实值的差异，是网络训练的优化目标
- **优化器**: 基于梯度更新网络参数，配合激活函数实现反向传播
- **神经网络基础**: 理解神经元、层、前向传播等基本概念
