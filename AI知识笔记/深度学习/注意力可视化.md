# 注意力可视化

## 一句话定义

注意力可视化是通过图形化方式展示Transformer模型中注意力权重分布，帮助理解模型如何在输入序列中分配计算资源，学习token之间的依赖关系和语义关联。

## 详细说明

### 1. 注意力热力图（Attention Heatmap）

注意力热力图是最直观的注意力可视化方式，用颜色深浅表示token之间的注意力强度。

**可视化原理**：
- 行表示查询（Query）token
- 列表示键（Key）token
- 颜色深浅表示注意力权重的大小
- 权重范围通常为0到1，颜色从浅到深

**解读方法**：
- 对角线高亮：表示token关注自身（自注意力特征）
- 结构化模式：显示学习到的语法/语义关系
- 远距离依赖：捕捉长程依赖关系
- 噪声模式：可能表示学习不充分

**应用场景**：
- 分析句子中词语间的依赖关系
- 验证模型是否学习到了有意义的语法结构
- 发现特定任务相关的注意力模式

### 2. 头部分析（Head Analysis）

Transformer使用多头注意力，不同头学习不同的注意力模式。

**头类型分类**：
- **句法注意力头**：关注语法关系（如主语-动词、修饰语）
- **语义注意力头**：关注语义相关性
- **位置注意力头**：主要关注相邻token
- **长距离注意力头**：关注远距离token
- **对角注意力头**：关注自身和附近位置

**分析维度**：
- **头特异性**：一个头对特定关系的专注程度
- **头冗余性**：不同头之间的相似性
- **头的功能互补性**：不同头如何协作

### 3. Token重要性（Token Importance）

Token重要性分析识别输入序列中对输出贡献最大的token。

**方法**：
- **注意力加权求和**：用注意力权重加权词嵌入
- **梯度重要性**：计算输出对输入的梯度
- **扰动分析**：逐个遮挡token观察输出变化
- **Layer-wise Relevance Propagation (LRP)**：反向传播相关性分数

**应用**：
- 解释模型对特定输入的预测
- 识别关键决策因素
- 验证模型是否关注了正确的特征

### 4. 显著图（Saliency Map）

显著图通过计算输入特征对输出的影响程度，生成强调重要区域的图像。

**图像领域的显著图**：
- **梯度显著图**：直接使用反向传播梯度
- **积分梯度**：考虑输入路径的积分梯度
- **Grad-CAM/Grad-CAM++**：针对CNN设计，也可用于ViT

**文本领域的显著图**：
- **输入梯度**：词语级别的重要性分数
- **注意力流**：结合注意力权重的分析方法
- **LIME**：局部近似解释

## 代码示例

### 注意力热力图可视化

```python
import torch
import torch.nn as nn
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
from transformers import AutoModel, AutoTokenizer

class AttentionVisualizer:
    def __init__(self, model_name='bert-base-chinese'):
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model = AutoModel.from_pretrained(model_name, output_attentions=True)
        self.model.eval()

        self.attention_weights = None

    def get_attention(self, text):
        """获取注意力权重"""
        inputs = self.tokenizer(text, return_tensors='pt')

        with torch.no_grad():
            outputs = self.model(**inputs)

        self.attention_weights = outputs.attentions
        self.tokens = self.tokenizer.convert_ids_to_tokens(inputs['input_ids'][0])

        return self.attention_weights

    def plot_attention_heatmap(self, layer_idx=0, head_idx=0,
                               save_path='attention_heatmap.png'):
        """绘制单个注意力头热力图"""
        if self.attention_weights is None:
            raise ValueError("请先调用 get_attention 获取注意力权重")

        # 获取指定层和头的注意力权重
        attention = self.attention_weights[layer_idx][0, head_idx].cpu().numpy()

        plt.figure(figsize=(12, 10))
        sns.heatmap(attention,
                    xticklabels=self.tokens,
                    yticklabels=self.tokens,
                    cmap='Blues',
                    annot=False,
                    fmt='.2f',
                    square=True)
        plt.title(f'Layer {layer_idx + 1}, Head {head_idx + 1}')
        plt.xlabel('Key Tokens')
        plt.ylabel('Query Tokens')
        plt.tight_layout()
        plt.savefig(save_path, dpi=150, bbox_inches='tight')
        plt.show()

    def plot_multi_head_attention(self, layer_idx=0,
                                  save_path='multi_head_attention.png'):
        """绘制某一层的所有注意力头"""
        if self.attention_weights is None:
            raise ValueError("请先调用 get_attention 获取注意力权重")

        num_heads = self.attention_weights[0].shape[1]
        attention = self.attention_weights[layer_idx][0].cpu().numpy()

        fig, axes = plt.subplots(2, num_heads // 2, figsize=(16, 8))
        axes = axes.flatten()

        for head in range(num_heads):
            sns.heatmap(attention[head],
                        xticklabels=self.tokens,
                        yticklabels=self.tokens,
                        cmap='Blues',
                        ax=axes[head],
                        cbar=False)
            axes[head].set_title(f'Head {head + 1}')
            axes[head].tick_params(axis='x', rotation=45)
            axes[head].tick_params(axis='y', rotation=0)

        plt.tight_layout()
        plt.savefig(save_path, dpi=150)
        plt.show()

    def plot_average_attention(self, layer_idx=None,
                              save_path='average_attention.png'):
        """绘制平均注意力权重"""
        if self.attention_weights is None:
            raise ValueError("请先调用 get_attention 获取注意力权重")

        if layer_idx is not None:
            # 单层平均
            attention = self.attention_weights[layer_idx][0].mean(dim=0).cpu().numpy()
        else:
            # 所有层平均
            attention_list = [att[0].mean(dim=0).cpu().numpy()
                             for att in self.attention_weights]
            attention = np.mean(attention_list, axis=0)

        plt.figure(figsize=(12, 10))
        sns.heatmap(attention,
                    xticklabels=self.tokens,
                    yticklabels=self.tokens,
                    cmap='Blues',
                    annot=False,
                    fmt='.2f')
        plt.title('Average Attention Weights')
        plt.xlabel('Key Tokens')
        plt.ylabel('Query Tokens')
        plt.tight_layout()
        plt.savefig(save_path, dpi=150, bbox_inches='tight')
        plt.show()

# 使用示例
visualizer = AttentionVisualizer('bert-base-chinese')
text = "今天天气很好，我和朋友去公园散步。"
visualizer.get_attention(text)
visualizer.plot_attention_heatmap(layer_idx=5, head_idx=3)
visualizer.plot_multi_head_attention(layer_idx=5)
```

### Token重要性分析

```python
import torch
import torch.nn.functional as F
import matplotlib.pyplot as plt
import numpy as np
from transformers import AutoModel, AutoTokenizer

class TokenImportanceAnalyzer:
    def __init__(self, model_name='bert-base-chinese'):
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model = AutoModel.from_pretrained(model_name)
        self.model.eval()

    def compute_gradient_importance(self, text, target_class=None):
        """基于梯度计算token重要性"""
        inputs = self.tokenizer(text, return_tensors='pt')
        input_ids = inputs['input_ids']

        input_ids.requires_grad = True

        outputs = self.model(input_ids)
        logits = outputs.last_hidden_state  # (batch, seq_len, hidden)

        if target_class is None:
            target_class = logits.argmax(dim=-1).item()

        # 使用[CLS] token的表示进行分类
        cls_output = logits[0, 0]

        # 计算对特定类别的梯度
        model_output = cls_output.sum()
        model_output.backward()

        # 梯度即为重要性分数
        importance = input_ids.grad.abs().sum(dim=-1)[0].cpu().numpy()
        tokens = self.tokenizer.convert_ids_to_tokens(input_ids[0])

        return tokens, importance

    def compute_attention_importance(self, text):
        """基于注意力权重计算token重要性"""
        inputs = self.tokenizer(text, return_tensors='pt')
        input_ids = inputs['input_ids']

        with torch.no_grad():
            outputs = self.model(input_ids, output_attentions=True)
            attentions = outputs.attentions

        # 平均所有层和所有头的注意力权重
        attention_weights = torch.stack(attentions, dim=0)  # (num_layers, batch, num_heads, seq_len, seq_len)
        avg_attention = attention_weights.mean(dim=(0, 1, 2))[0]  # (seq_len, seq_len)

        # 对[CLS]的注意力作为其他token的重要性
        importance = avg_attention[:, 0].cpu().numpy()

        tokens = self.tokenizer.convert_ids_to_tokens(input_ids[0])

        return tokens, importance

    def compute_perturbation_importance(self, text, target_class=None,
                                       num_perturbations=100):
        """基于扰动计算token重要性"""
        inputs = self.tokenizer(text, return_tensors='pt')
        input_ids = inputs['input_ids']
        tokens = self.tokenizer.convert_ids_to_tokens(input_ids[0])

        # 获取原始预测
        with torch.no_grad():
            outputs = self.model(input_ids)
            logits = outputs.last_hidden_state[0, 0]

            if target_class is None:
                target_class = logits.argmax(dim=-1).item()

            original_score = logits[target_class].item()

        # 对每个token进行扰动
        importance = np.zeros(len(tokens))

        for i in range(1, len(tokens)):  # 跳过[CLS]
            perturbed_scores = []

            for _ in range(num_perturbations):
                perturbed_ids = input_ids.clone()
                # 用[MASK]替换该token
                perturbed_ids[0, i] = self.tokenizer.mask_token_id

                with torch.no_grad():
                    outputs = self.model(perturbed_ids)
                    logits = outputs.last_hidden_state[0, 0]
                    perturbed_scores.append(logits[target_class].item())

            # 分数下降越大，token越重要
            importance[i] = original_score - np.mean(perturbed_scores)

        return tokens, importance

    def plot_token_importance(self, tokens, importance,
                             title='Token Importance',
                             save_path='token_importance.png'):
        """绘制token重要性条形图"""
        # 过滤特殊token用于显示
        display_tokens = []
        display_importance = []

        for token, imp in zip(tokens, importance):
            if token not in ['[CLS]', '[SEP]', '[PAD]', '[UNK]']:
                display_tokens.append(token)
                display_importance.append(imp)

        # 排序
        sorted_pairs = sorted(zip(display_importance, display_tokens), reverse=True)
        sorted_importance, sorted_tokens = zip(*sorted_pairs)

        plt.figure(figsize=(12, max(6, len(sorted_tokens) * 0.3)))
        colors = plt.cm.Reds(np.linspace(0.3, 0.9, len(sorted_tokens)))
        plt.barh(range(len(sorted_tokens)), sorted_importance, color=colors)
        plt.yticks(range(len(sorted_tokens)), sorted_tokens)
        plt.xlabel('Importance Score')
        plt.title(title)
        plt.tight_layout()
        plt.savefig(save_path, dpi=150)
        plt.show()

# 使用示例
analyzer = TokenImportanceAnalyzer('bert-base-chinese')
text = "这部电影真的太好看了，强烈推荐！"

tokens, grad_importance = analyzer.compute_gradient_importance(text)
tokens, att_importance = analyzer.compute_attention_importance(text)
tokens, pert_importance = analyzer.compute_perturbation_importance(text)

analyzer.plot_token_importance(tokens, grad_importance, 'Gradient-based Importance')
analyzer.plot_token_importance(tokens, att_importance, 'Attention-based Importance')
```

### Vision Transformer (ViT) 显著图

```python
import torch
import torch.nn.functional as F
import matplotlib.pyplot as plt
import numpy as np
from PIL import Image
from transformers import ViTForImageClassification, ViTImageProcessor

class ViTSaliencyVisualizer:
    def __init__(self, model_name='google/vit-base-patch16-224'):
        self.processor = ViTImageProcessor.from_pretrained(model_name)
        self.model = ViTForImageClassification.from_pretrained(model_name)
        self.model.eval()

    def compute_saliency(self, image_path, target_class=None):
        """计算图像的显著图"""
        # 加载并预处理图像
        image = Image.open(image_path).convert('RGB')
        inputs = self.processor(images=image, return_tensors='pt')

        image_tensor = inputs['pixel_values'].requires_grad_(True)

        # 前向传播
        outputs = self.model(image_tensor)
        logits = outputs.logits

        if target_class is None:
            target_class = logits.argmax(dim=-1).item()

        # 反向传播获取梯度
        model_output = logits[0, target_class]
        model_output.backward()

        # 获取梯度作为显著图
        gradient = image_tensor.grad[0]  # (3, 224, 224)

        # 取梯度的绝对值并归一化
        saliency = gradient.abs().max(dim=0)[0].cpu().numpy()
        saliency = (saliency - saliency.min()) / (saliency.max() - saliency.min() + 1e-8)

        return saliency, image

    def compute_gradcam(self, image_path, target_layer=None, target_class=None):
        """计算Grad-CAM显著图"""
        image = Image.open(image_path).convert('RGB')
        inputs = self.processor(images=image, return_tensors='pt')

        image_tensor = inputs['pixel_values']
        image_tensor.requires_grad_(True)

        # 获取目标层（最后一层Transformer块）
        if target_layer is None:
            target_layer = self.model.vit.encoder.layer[-1].attention.attention

        gradients = None
        activations = None

        def forward_hook(module, input, output):
            nonlocal activations
            activations = output[0].detach()

        def backward_hook(module, grad_input, grad_output):
            nonlocal gradients
            gradients = grad_output[0].detach()

        forward_handle = target_layer.register_forward_hook(forward_hook)
        backward_handle = target_layer.register_full_backward_hook(backward_hook)

        # 前向传播
        outputs = self.model(image_tensor)
        logits = outputs.logits

        if target_class is None:
            target_class = logits.argmax(dim=-1).item()

        # 反向传播
        self.model.zero_grad()
        logits[0, target_class].backward()

        forward_handle.remove()
        backward_handle.remove()

        # 计算Grad-CAM
        gradients = gradients[0]  # (seq_len, hidden)
        activations = activations[0]  # (seq_len, hidden)

        # 全局平均池化梯度得到权重
        weights = gradients.mean(dim=0)  # (hidden,)

        # 加权求和
        cam = torch.matmul(activations, weights)
        cam = F.relu(cam)  # (seq_len,)

        # 将CAMreshape为14x14（假设patch_size=16, 224/16=14）
        cam = cam[1:]  # 移除[CLS]token
        cam = cam.view(14, 14).cpu().numpy()
        cam = (cam - cam.min()) / (cam.max() - cam.min() + 1e-8)

        # 调整大小到原始图像尺寸
        import cv2
        cam = cv2.resize(cam, (224, 224))

        return cam, image

    def plot_saliency(self, saliency, image,
                     title='Saliency Map',
                     save_path='saliency.png'):
        """绘制显著图"""
        fig, axes = plt.subplots(1, 3, figsize=(15, 5))

        # 原图
        axes[0].imshow(image)
        axes[0].set_title('Original Image')
        axes[0].axis('off')

        # 显著图
        axes[1].imshow(saliency, cmap='hot')
        axes[1].set_title(title)
        axes[1].axis('off')

        # 叠加图
        image_resized = np.array(image.resize((224, 224))) / 255.0
        overlay = image_resized * 0.5 + plt.cm.hot(saliency)[..., :3] * 0.5
        axes[2].imshow(overlay)
        axes[2].set_title('Overlay')
        axes[2].axis('off')

        plt.tight_layout()
        plt.savefig(save_path, dpi=150)
        plt.show()

# 使用示例
visualizer = ViTSaliencyVisualizer()
saliency, image = visualizer.compute_saliency('example.jpg')
visualizer.plot_saliency(saliency, image)
```

### 1. 模型可解释性
- 理解Transformer的决策过程

- 向非技术人员解释AI模型行为

### 2. 模型调试
- 发现注意力机制的异常模式
- 诊断训练过程中的问题
- 验证模型架构的合理性

### 3. 任务分析
- 分析特定任务所需的注意力模式
- 理解不同任务对注意力机制的利用方式
- 发现任务相关的专业知识

### 4. 模型优化
- 识别冗余的注意力头进行剪枝
- 分析头的功能专门化
- 指导模型架构改进

## 相关概念

| 概念 | 说明 |
|------|------|
| 自注意力 | 计算序列内部token之间依赖关系的机制 |
| 多头注意力 | 并行的多个注意力机制，捕捉不同类型的关系 |
| Query/Key/Value | 注意力计算的三个向量 |
| 显著图 | 显示输入区域对输出影响程度的可视化 |
| Grad-CAM | 基于梯度的类激活映射方法 |
| LRP | 基于反向传播的相关性传播方法 |
| 头特异性 | 特定注意力头对特定关系模式的专注程度 |
