# 神经网络可视化

## 一句话定义

神经网络可视化是通过各种技术手段将神经网络的内部运作机制、特征学习和决策过程以直观的形式展示出来，帮助理解深度模型究竟"看"到了什么。

## 详细说明

### 1. 特征图可视化（Feature Map Visualization）

特征图是卷积层对输入数据进行特征提取后产生的二维/三维数组，每张特征图代表网络对某种特定模式的响应。

**低层特征图**：
- 主要捕捉边缘、纹理、颜色等基本视觉元素
- 形状简单，模式直观易懂
- 类似于传统计算机视觉中的Gabor滤波器

**高层特征图**：
- 捕捉更抽象的概念，如物体部件、纹理模式
- 响应模式更加复杂和专业化
- 神经元激活更加稀疏和选择性

### 2. 激活图可视化（Activation Visualization）

激活图展示了神经网络在处理特定输入时各层神经元的响应强度。

**可视化方法**：
- **直接绘制**：将各层激活值作为图像绘制出来
- **最大激活**：找到使特定神经元激活最大的输入图像
- **闭包可视化**：通过优化输入最大化目标神经元激活

### 3. 热力图可视化（Heatmap Visualization）

热力图通过颜色编码显示网络中不同区域的重要程度。

**主流技术**：
- **Grad-CAM**：使用梯度信息生成类别判别热力图
- **CAM（Class Activation Mapping）**：基于全局平均池化的热力图
- **LIME**：局部可解释模型无关解释
- **SHAP**：基于博弈论的特征贡献分析

### 4. t-SNE降维可视化

t-SNE是一种非线性降维技术，将高维特征空间映射到二维/三维空间进行可视化。

**应用场景**：
- 观察不同类别在特征空间中的分布
- 发现潜在的聚类结构
- 分析网络学习到的特征表示质量

## 代码示例

### PyTorch特征图可视化

```python
import torch
import torch.nn as nn
import matplotlib.pyplot as plt
import numpy as np

class FeatureExtractor:
    def __init__(self, model, target_layer):
        self.model = model
        self.target_layer = target_layer
        self.features = None
        self.gradients = None

        # 注册钩子获取特征和梯度
        target_layer.register_forward_hook(self.save_features)
        target_layer.register_full_backward_hook(self.save_gradients)

    def save_features(self, module, input, output):
        self.features = output.detach()

    def save_gradients(self, module, grad_input, grad_output):
        self.gradients = grad_output[0].detach()

    def get_feature_maps(self, input_tensor):
        self.model.eval()
        with torch.no_grad():
            _ = self.model(input_tensor)
        return self.features

def visualize_feature_maps(feature_maps, num_cols=8, figsize=(16, 16)):
    """可视化特征图"""
    batch_size, channels, h, w = feature_maps.shape
    num_cols = min(num_cols, channels)
    num_rows = (channels + num_cols - 1) // num_cols

    fig, axes = plt.subplots(num_rows, num_cols, figsize=figsize)
    axes = axes.flatten() if channels > 1 else [axes]

    for i in range(channels):
        feature_map = feature_maps[0, i].cpu().numpy()
        axes[i].imshow(feature_map, cmap='viridis')
        axes[i].set_title(f'Channel {i}')
        axes[i].axis('off')

    # 隐藏多余的子图
    for i in range(channels, len(axes)):
        axes[i].axis('off')

    plt.tight_layout()
    plt.savefig('feature_maps.png', dpi=150)
    plt.show()

# 使用示例
model = torch.hub.load('pytorch/vision', 'resnet18', pretrained=True)
extractor = FeatureExtractor(model, model.layer4)

# 加载示例图像
from torchvision import transforms
from PIL import Image

transform = transforms.Compose([
    transforms.Resize(224),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

img = Image.open('example.jpg')
input_tensor = transform(img).unsqueeze(0)

feature_maps = extractor.get_feature_maps(input_tensor)
visualize_feature_maps(feature_maps)
```

### Grad-CAM热力图实现

```python
import torch
import torch.nn.functional as F
import cv2
import numpy as np

class GradCAM:
    def __init__(self, model, target_layer):
        self.model = model
        self.target_layer = target_layer
        self.gradients = None
        self.activations = None

        target_layer.register_full_backward_hook(self.save_gradient)
        target_layer.register_forward_hook(self.save_activation)

    def save_activation(self, module, input, output):
        self.activations = output.detach()

    def save_gradient(self, module, grad_input, grad_output):
        self.gradients = grad_output[0].detach()

    def generate_cam(self, input_tensor, target_class=None):
        self.model.eval()
        self.model.zero_grad()

        # 前向传播
        output = self.model(input_tensor)

        if target_class is None:
            target_class = output.argmax(dim=1).item()

        # 反向传播
        one_hot = torch.zeros_like(output)
        one_hot[0][target_class] = 1
        output.backward(gradient=one_hot, retain_graph=True)

        # 计算权重
        gradients = self.gradients[0]  # (C, H, W)
        activations = self.activations[0]  # (C, H, W)

        weights = gradients.mean(dim=(1, 2))  # (C,)

        # 加权组合
        cam = torch.zeros(activations.shape[1:], dtype=torch.float32)
        for i, w in enumerate(weights):
            cam += w * activations[i]

        # ReLU和归一化
        cam = F.relu(cam)
        cam = cam.cpu().numpy()
        cam = cv2.resize(cam, (224, 224))
        cam = (cam - cam.min()) / (cam.max() - cam.min() + 1e-8)

        return cam

def overlay_cam_on_image(img, cam, alpha=0.5):
    """将CAM热力图叠加到原图"""
    heatmap = cv2.applyColorMap(np.uint8(255 * cam), cv2.COLORMAP_JET)
    heatmap = cv2.cvtColor(heatmap, cv2.COLOR_BGR2RGB)
    heatmap = np.float32(heatmap) / 255

    if np.max(img) > 1:
        img = np.float32(img) / 255

    overlay = heatmap * alpha + img * (1 - alpha)
    overlay = overlay / np.max(overlay)

    return np.uint8(255 * overlay)

# 使用示例
img_tensor = transform(img).unsqueeze(0).requires_grad_(True)
gradcam = GradCAM(model, model.layer4)
cam = gradcam.generate_cam(img_tensor)

original_img = np.array(img.resize((224, 224)))
result = overlay_cam_on_image(original_img, cam)

plt.figure(figsize=(10, 10))
plt.imshow(result)
plt.axis('off')
plt.savefig('gradcam_result.png', dpi=150, bbox_inches='tight')
```

### t-SNE特征可视化

```python
from sklearn.manifold import TSNE
import seaborn as sns
import pandas as pd

def extract_features(model, dataloader, device):
    """提取特征并收集标签"""
    model.eval()
    features = []
    labels = []

    with torch.no_grad():
        for images, targets in dataloader:
            images = images.to(device)
            features.append(model(images))
            labels.extend(targets.numpy())

    return torch.cat(features).cpu().numpy(), np.array(labels)

def visualize_tsne(features, labels, title="t-SNE Visualization"):
    """t-SNE降维可视化"""
    tsne = TSNE(n_components=2, random_state=42, perplexity=30, n_iter=1000)
    features_2d = tsne.fit_transform(features)

    df = pd.DataFrame({
        'x': features_2d[:, 0],
        'y': features_2d[:, 1],
        'label': labels
    })

    plt.figure(figsize=(12, 10))
    sns.scatterplot(data=df, x='x', y='y', hue='label', palette='tab20', s=50)
    plt.title(title)
    plt.legend(bbox_to_anchor=(1.05, 1), loc='upper left')
    plt.tight_layout()
    plt.savefig('tsne_visualization.png', dpi=150)
    plt.show()

# 使用示例
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = torch.hub.load('pytorch/vision', 'resnet18', pretrained=True)
model.fc = nn.Identity()  # 移除分类头，获取特征
model = model.to(device)

# 提取特征
features, labels = extract_features(model, test_loader, device)
visualize_tsne(features, labels, "ResNet18 Feature Distribution")
```

## 应用场景

### 1. 模型调试与性能分析
- 识别网络是否学习了有意义的特征
- 发现过拟合或欠拟合问题
- 检测数据集质量问题（如标签错误）

### 2. 模型可解释性
- 向非技术人员解释模型决策
- 满足模型审计和合规要求
- 建立用户对AI系统的信任

### 3. 模型优化
- 识别冗余的卷积核或通道
- 指导网络架构设计
- 分析模型精简化的影响

### 4. 迁移学习分析
- 选择合适的预训练模型
- 理解预训练模型学到的特征
- 诊断微调过程中的问题

## 相关概念

| 概念 | 说明 |
|------|------|
| 特征图 | 卷积层输出的二维激活阵列 |
| 激活函数 | 引入非线性的函数，如ReLU |
| 热力图 | 用颜色编码重要性的可视化方法 |
| 降维 | 将高维数据映射到低维空间的技术 |
| 类激活图 | 显示分类决策重要区域的热力图 |
| 显著图 | 显示输入像素对输出影响程度的图像 |
