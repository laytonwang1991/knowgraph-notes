# 层归一化（Layer Normalization）

## 一句话定义

层归一化通过对单个样本的所有神经元输出进行均值和方差的规范化，使模型对输入尺度变化具有不变性，广泛应用于 Transformer 等序列模型中。

## 核心公式

**层归一化定义：**

$$
\mu^{(i)} = \frac{1}{H} \sum_{j=1}^{H} x_j^{(i)}
$$

$$
\sigma^{(i)} = \sqrt{\frac{1}{H} \sum_{j=1}^{H} (x_j^{(i)} - \mu^{(i)})^2}
$$

$$
\hat{x}_j^{(i)} = \frac{x_j^{(i)} - \mu^{(i)}}{\sigma^{(i)} + \epsilon}
$$

$$
y^{(i)} = \gamma \odot \hat{x}^{(i)} + \beta
$$

其中：
- $H$：隐藏层神经元数量
- $x^{(i)}$：第 $i$ 个样本的隐藏层输出向量
- $\gamma, \beta$：可学习的缩放和平移参数（Element-wise）
- $\odot$：元素级乘法

## 详细说明

### 1. 与 Batch Norm 的核心区别

| 特性 | Batch Norm | Layer Norm |
|------|-----------|-----------|
| 归一化维度 | 跨 Batch | 跨 Hidden Units |
| Batch 依赖 | 是 | 否 |
| 统计量来源 | 当前 mini-batch | 当前样本自身 |
| RNN 适用性 | 不适用（序列长度可变） | 完全适用 |
| 推理依赖训练Batch | 是（移动平均） | 否（直接计算） |

### 2. Layer Norm 的几何解释

层归一化可以看作是将样本映射到以 $\gamma$ 和 $\beta$ 定义的球面上：

- 归一化使向量具有单位方差
- $\gamma$ 和 $\beta$ 控制球面半径和位置
- 增强模型对输入尺度变化的鲁棒性

### 3. RMS Norm（Root Mean Square Norm）

RMS Norm 是 Layer Norm 的简化版本，只计算 RMS 统计量：

$$
RMS(x) = \sqrt{\frac{1}{H} \sum_{j=1}^{H} x_j^2}
$$

$$
\hat{x}_j = \frac{x_j}{RMS(x)}
$$

**优势**：
- 减少 40% 的计算量
- 去掉了均值计算（更少的内存访问）
- 在许多任务上效果与 Layer Norm 相当

### 4. Pre-Norm vs Post-Norm

**Pre-Norm（Transformer-LN）：**

```
x = x + SubLayer(LayerNorm(x))
```

- 在残差连接之前应用 Layer Norm
- 训练更稳定，但理论表达能力可能较弱
- 被大多数现代 Transformer 采用

**Post-Norm（标准ResNet风格）：**

```
x = LayerNorm(x + SubLayer(x))
```

- 在残差连接之后应用 Layer Norm
- 训练初期梯度范数更大
- 传统BERT、原始Transformer使用

### 5. 可学习参数的作用

与 Batch Norm 类似，Layer Norm 引入 $\gamma$（缩放）和 $\beta$（平移）：

- 恢复模型的表达能力
- 当 $\gamma = \sigma$, $\beta = \mu$ 时，还原原始输入
- 每个维度有独立的 scale 和 shift

## 代码示例

### PyTorch 实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class LayerNorm(nn.Module):
    """层归一化实现"""
    def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
        super().__init__()
        if isinstance(normalized_shape, int):
            normalized_shape = (normalized_shape,)
        self.normalized_shape = tuple(normalized_shape)
        self.eps = eps
        self.elementwise_affine = elementwise_affine

        if elementwise_affine:
            self.gamma = nn.Parameter(torch.ones(normalized_shape))
            self.beta = nn.Parameter(torch.zeros(normalized_shape))
        else:
            self.register_parameter('gamma', None)
            self.register_parameter('beta', None)

    def forward(self, x):
        # x shape: (batch, ..., feature)
        # 计算沿特征维度的均值和标准差
        mean = x.mean(dim=-1, keepdim=True)
        var = x.var(dim=-1, keepdim=True, unbiased=False)
        x_norm = (x - mean) / torch.sqrt(var + self.eps)

        if self.elementwise_affine:
            return self.gamma * x_norm + self.beta
        return x_norm
```

### RMS Norm 实现

```python
class RMSNorm(nn.Module):
    """RMS Norm 实现"""
    def __init__(self, normalized_shape, eps=1e-5):
        super().__init__()
        if isinstance(normalized_shape, int):
            normalized_shape = (normalized_shape,)
        self.normalized_shape = tuple(normalized_shape)
        self.eps = eps
        self.scale = nn.Parameter(torch.ones(normalized_shape))

    def forward(self, x):
        # 计算 RMS
        rms = torch.sqrt(torch.mean(x ** 2, dim=-1, keepdim=True) + self.eps)
        x_norm = x / rms
        return self.scale * x_norm
```

### Transformer 中的应用

```python
class TransformerLayer(nn.Module):
    """Pre-Norm Transformer 层"""
    def __init__(self, d_model, nhead, dim_feedforward):
        super().__init__()
        self.self_attn = nn.MultiheadAttention(d_model, nhead)
        self.linear1 = nn.Linear(d_model, dim_feedforward)
        self.linear2 = nn.Linear(dim_feedforward, d_model)

        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(0.1)

    def forward(self, x, mask=None):
        # Pre-Norm: 先归一化，再残差
        x = x + self.dropout(self.self_attn(self.norm1(x), self.norm1(x), self.norm1(x))[0])
        x = x + self.dropout(self.linear2(F.gelu(self.linear1(self.norm2(x)))))
        return x

class PostNormTransformerLayer(nn.Module):
    """Post-Norm Transformer 层"""
    def __init__(self, d_model, nhead, dim_feedforward):
        super().__init__()
        self.self_attn = nn.MultiheadAttention(d_model, nhead)
        self.linear1 = nn.Linear(d_model, dim_feedforward)
        self.linear2 = nn.Linear(dim_feedforward, d_model)

        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(0.1)

    def forward(self, x, mask=None):
        # Post-Norm: 先残差，再归一化
        attn_out = self.self_attn(x, x, x, attn_mask=mask)[0]
        x = self.norm1(x + self.dropout(attn_out))

        ff_out = self.linear2(F.gelu(self.linear1(x)))
        x = self.norm2(x + self.dropout(ff_out))
        return x
```

### BERT 中的应用

```python
class BERTLayer(nn.Module):
    """BERT（Post-Norm）实现简化"""
    def __init__(self, d_model, nhead, dim_feedforward):
        super().__init__()
        self.attention = nn.MultiheadAttention(d_model, nhead)
        self.norm1 = nn.LayerNorm(d_model)

        self.ffn = nn.Sequential(
            nn.Linear(d_model, dim_feedforward),
            nn.GELU(),
            nn.Linear(dim_feedforward, d_model)
        )
        self.norm2 = nn.LayerNorm(d_model)

    def forward(self, x, mask=None):
        # Post-Norm 风格
        attn_out = self.attention(x, x, x, attn_mask=mask)[0]
        x = self.norm1(x + attn_out)

        ffn_out = self.ffn(x)
        x = self.norm2(x + ffn_out)
        return x
```

## 应用场景

### 1. Transformer 架构
- **GPT、BERT、T5 等语言模型**：几乎所有现代大语言模型
- **ViT（Vision Transformer）**：图像分类
- **Stable Diffusion**：去噪网络中
- **Whisper**：语音识别

### 2. RNN/LSTM 变体
- 层归一化替代 Batch Norm
- 处理变长序列

### 3. 扩散模型
- UNet 架构中的条件归一化
- 文本嵌入调制

### 4. 归一化方法选择指南

```
RNN/LSTM              -> Layer Norm
Small Batch + CNN     -> Group Norm / Layer Norm
Large Batch + CNN     -> Batch Norm
Transformer           -> Layer Norm / RMS Norm
```

## 相关概念对比

### 四种主流归一化方法对比

| 方法 | 计算维度 | 公式 |
|------|---------|------|
| **Batch Norm** | $(B, C, H, W)$ 跨 B | $\frac{x - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}$ |
| **Layer Norm** | $(B, C, H, W)$ 跨 $CHW$ | $\frac{x - \mu_L}{\sqrt{\sigma_L^2 + \epsilon}}$ |
| **Instance Norm** | $(B, C, H, W)$ 跨 $HW$ | $\frac{x - \mu_I}{\sqrt{\sigma_I^2 + \epsilon}}$ |
| **Group Norm** | $(B, C, H, W)$ 跨 $G\cdot HW$ | $\frac{x - \mu_G}{\sqrt{\sigma_G^2 + \epsilon}}$ |

### Layer Norm 的变体

1. **RMS Norm**：只保留RMS统计量，计算更高效
2. **Deep Norm**：DeepNorm，用于稳定超深层Transformer训练
3. **AdaLN**：自适应Layer Norm，引入条件调制
