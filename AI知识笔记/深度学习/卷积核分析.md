# 卷积核分析

## 一句话定义

卷积核分析是通过研究和可视化卷积神经网络中滤波器的权重及其对输入的响应，理解CNN如何从原始像素中逐步提取层次化特征的技术。

## 详细说明

### 1. 第一层卷积核分析

第一层卷积核直接与输入图像交互，是理解网络学习起点的重要窗口。

**特点**：
- 滤波器大小通常为3x3、5x5或7x7
- 直接处理原始RGB像素值
- 学到的模式类似Gabor滤波器：边缘检测、颜色条纹

**常见模式**：
- **边缘检测器**：检测不同方向的线条
- **颜色/纹理检测器**：对特定颜色组合有响应
- **高反差检测器**：对亮度变化剧烈的区域敏感

**可视化价值**：
- 第一层是最容易解释的
- 可以验证网络是否学习到了合理的低级特征
- 为分析更深层提供基准

### 2. 深层卷积核模式

随着网络层数加深，卷积核学到的模式越来越抽象和复杂。

**中层模式**：
- 纹理模式：条纹、斑点、网格
- 局部形状：角点、弧线、局部轮廓
- 颜色组合：特定的颜色空间分布

**高层模式**：
- 物体部件：眼睛、轮子、叶片
- 纹理模式：羽毛、皮毛、砖墙
- 高度抽象的概念模式

**层级递进**：
```
输入图像 → 像素值
    ↓
Layer1: 边缘、颜色斑点
    ↓
Layer2: 纹理、角点、局部形状
    ↓
Layer3: 物体部件、纹理模式
    ↓
Layer4+: 完整物体、复杂概念
```

### 3. 滤波器响应分析

滤波器响应分析研究不同输入图像激活特定滤波器的方式。

**分析方法**：
- **空间响应图**：显示滤波器在图像各位置的激活强度
- **响应统计**：分析激活值的分布（均值、方差、稀疏度）
- **选择性分析**：评估滤波器对特定模式的选择性

**响应模式类型**：
- **均匀响应**：滤波器对整张图像均匀响应（通常是背景相关）
- **局部响应**：滤波器仅在图像特定区域激活（特征相关）
- **选择性激活**：滤波器对特定类别物体响应强烈

### 4. 卷积核可视化技术

**直接权重可视化**：
- 将卷积核权重直接绘制为图像
- 适用于第一层，因为可以直接对应到RGB通道
- 深层卷积核难以直接解释

**最大激活图像**：
- 通过优化寻找使滤波器激活最大的输入图像
- 显示滤波器实际"喜欢"什么样的模式
- 可使用梯度上升或生成对抗方法

**反卷积/转置卷积**：
- 将特征图反向投影回输入空间
- 显示触发特定激活的输入模式
- 需要保存中间激活和梯度

## 代码示例

### 卷积核权重可视化

```python
import torch
import torch.nn as nn
import matplotlib.pyplot as plt
import numpy as np

def visualize_first_layer_kernels(model, save_path='first_layer_kernels.png'):
    """可视化第一层卷积核权重"""
    # 获取第一层卷积权重
    if hasattr(model, 'conv1'):
        weights = model.conv1.weight.data
    elif hasattr(model, 'features'):
        weights = model.features[0].weight.data
    else:
        raise ValueError("无法找到第一层卷积")

    num_kernels = weights.shape[0]
    num_cols = min(8, num_kernels)
    num_rows = (num_kernels + num_cols - 1) // num_cols

    fig, axes = plt.subplots(num_rows, num_cols, figsize=(2*num_cols, 2*num_rows))
    axes = axes.flatten() if num_kernels > 1 else [axes]

    for i in range(num_kernels):
        kernel = weights[i]  # (C, H, W) - C通常是3或1

        if kernel.shape[0] == 3:  # RGB图像
            # 转换为 (H, W, C) 并归一化
            kernel_img = kernel.permute(1, 2, 0).cpu().numpy()
            kernel_img = (kernel_img - kernel_img.min()) / (kernel_img.max() - kernel_img.min() + 1e-8)
        else:
            kernel_img = kernel[0].cpu().numpy()
            kernel_img = (kernel_img - kernel_img.min()) / (kernel_img.max() - kernel_img.min() + 1e-8)

        axes[i].imshow(kernel_img)
        axes[i].set_title(f'Kernel {i}')
        axes[i].axis('off')

    # 隐藏多余的子图
    for i in range(num_kernels, len(axes)):
        axes[i].axis('off')

    plt.tight_layout()
    plt.savefig(save_path, dpi=150, bbox_inches='tight')
    plt.show()

# 使用示例
import torchvision.models as models

model = models.vgg16(pretrained=True)
visualize_first_layer_kernels(model)
```

### 最大激活图像生成（Gradient Ascent）

```python
import torch
import torch.nn as nn
import torch.optim as optim
import matplotlib.pyplot as plt
from torchvision import transforms

class MaxActivationGenerator:
    def __init__(self, model, layer, channel_index):
        self.model = model
        self.layer = layer
        self.channel_index = channel_index
        self.activation = None

        # 注册钩子获取目标层的输出
        layer.register_forward_hook(self.get_activation)

    def get_activation(self, module, input, output):
        self.activation = output

    def generate(self, input_size=(1, 3, 224, 224),
                 num_steps=50,
                 learning_rate=5.0,
                 weight_decay=1e-4,
                 blur_radius=2):
        """通过梯度上升生成最大激活图像"""
        # 初始化随机输入
        img = torch.randn(input_size) * 0.1
        img.requires_grad = True

        optimizer = optim.LFP([img], lr=learning_rate, weight_decay=weight_decay)

        for step in range(num_steps):
            optimizer.zero_grad()

            # 前向传播
            self.model(img)

            # 获取目标通道的激活均值作为损失
            loss = -self.activation[0, self.channel_index].mean()

            # 反向传播
            loss.backward()

            # 更新
            optimizer.step()

            # 周期性模糊以生成更平滑的模式
            if step % 10 == 0 and blur_radius > 0:
                img.data = self.apply_blur(img.data, blur_radius)

            if step % 20 == 0:
                print(f"Step {step}, Loss: {-loss.item():.4f}")

        return img.detach()

    @staticmethod
    def apply_blur(tensor, radius):
        """简单的平均模糊"""
        kernel_size = 2 * radius + 1
        kernel = torch.ones(1, 1, kernel_size, kernel_size) / (kernel_size ** 2)
        return torch.nn.functional.conv2d(
            tensor, kernel, padding=radius, groups=tensor.shape[1]
        )

def visualize_max_activations(model, layer, num_channels=16, input_size=(1, 3, 224, 224)):
    """可视化多个通道的最大激活图像"""
    num_cols = 4
    num_rows = (num_channels + num_cols - 1) // num_cols

    fig, axes = plt.subplots(num_rows, num_cols, figsize=(num_cols * 3, num_rows * 3))
    axes = axes.flatten() if num_channels > 1 else [axes]

    transform = transforms.Compose([
        transforms.Normalize(mean=[-0.485/0.229, -0.456/0.224, -0.406/0.225],
                           std=[1/0.229, 1/0.224, 1/0.225])
    ])

    for i in range(num_channels):
        print(f"Generating channel {i}...")
        generator = MaxActivationGenerator(model, layer, i)
        img = generator.generate(input_size)

        # 反归一化用于显示
        img_display = transform(img[0]).permute(1, 2, 0).cpu().numpy()
        img_display = np.clip(img_display, 0, 1)

        axes[i].imshow(img_display)
        axes[i].set_title(f'Channel {i}')
        axes[i].axis('off')

    # 隐藏多余的子图
    for i in range(num_channels, len(axes)):
        axes[i].axis('off')

    plt.tight_layout()
    plt.savefig('max_activations.png', dpi=150)
    plt.show()

# 使用示例
model = models.vgg16(pretrained=True).features[:10]
target_layer = model[7]  # 选择一个中间层

visualize_max_activations(model, target_layer, num_channels=12)
```

### 滤波器响应分析

```python
import torch
import torch.nn.functional as F
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import entropy

class FilterResponseAnalyzer:
    def __init__(self, model, layer):
        self.model = model
        self.layer = layer
        self.activations = {}

        layer.register_forward_hook(self.save_activation)

    def save_activation(self, module, input, output):
        self.activations['output'] = output.detach()

    def compute_response_statistics(self, dataloader):
        """计算滤波器响应统计"""
        self.model.eval()

        stats = {
            'mean': [],
            'std': [],
            'sparsity': [],
            'max': [],
            'entropy': []
        }

        with torch.no_grad():
            for images, _ in dataloader:
                self.model(images)

                activation = self.activations['output'][0]  # (C, H, W)

                for c in range(activation.shape[0]):
                    channel_act = activation[c].flatten()

                    # 计算统计量
                    stats['mean'].append(channel_act.mean().item())
                    stats['std'].append(channel_act.std().item())
                    stats['max'].append(channel_act.max().item())

                    # 稀疏度：激活值接近0的比例
                    sparsity = (channel_act.abs() < 0.1).float().mean().item()
                    stats['sparsity'].append(sparsity)

                    # 熵：激活值分布的复杂性
                    hist = torch.histc(channel_act, bins=10)
                    hist = hist / hist.sum()
                    ent = entropy(hist.cpu().numpy() + 1e-10)
                    stats['entropy'].append(ent)

        return {k: np.array(v) for k, v in stats.items()}

    def analyze_filter_selectivity(self, images, labels, num_top=5):
        """分析滤波器对类别的选择性"""
        self.model.eval()

        filter_class_response = {i: [] for i in range(100)}

        with torch.no_grad():
            for img, label in zip(images, labels):
                img = img.unsqueeze(0)
                self.model(img)

                activation = self.activations['output'][0]  # (C, H, W)
                mean_activation = activation.mean(dim=(1, 2))  # (C,)

                for c in range(len(mean_activation)):
                    filter_class_response[c].append((mean_activation[c].item(), label.item()))

        # 找出每个滤波器响应最强的类别
        filter_top_classes = {}
        for c, responses in filter_class_response.items():
            if responses:
                sorted_responses = sorted(responses, reverse=True)[:num_top]
                filter_top_classes[c] = [(label, score) for score, label in sorted_responses]

        return filter_top_classes

def plot_filter_statistics(stats, save_path='filter_stats.png'):
    """绘制滤波器统计信息"""
    fig, axes = plt.subplots(2, 2, figsize=(14, 10))

    axes[0, 0].hist(stats['mean'], bins=50, edgecolor='black', alpha=0.7)
    axes[0, 0].set_title('Mean Activation Distribution')
    axes[0, 0].set_xlabel('Mean Value')
    axes[0, 0].set_ylabel('Count')

    axes[0, 1].hist(stats['std'], bins=50, edgecolor='black', alpha=0.7, color='orange')
    axes[0, 1].set_title('Std Activation Distribution')
    axes[0, 1].set_xlabel('Std Value')
    axes[0, 1].set_ylabel('Count')

    axes[1, 0].hist(stats['sparsity'], bins=50, edgecolor='black', alpha=0.7, color='green')
    axes[1, 0].set_title('Sparsity Distribution (ratio of near-zero activations)')
    axes[1, 0].set_xlabel('Sparsity')
    axes[1, 0].set_ylabel('Count')

    axes[1, 1].hist(stats['entropy'], bins=50, edgecolor='black', alpha=0.7, color='red')
    axes[1, 1].set_title('Activation Entropy Distribution')
    axes[1, 1].set_xlabel('Entropy')
    axes[1, 1].set_ylabel('Count')

    plt.tight_layout()
    plt.savefig(save_path, dpi=150)
    plt.show()

# 使用示例
from torchvision import datasets, transforms
from torch.utils.data import DataLoader

transform = transforms.Compose([
    transforms.Resize(224),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

dataset = datasets.CIFAR10(root='./data', train=False, download=True, transform=transform)
dataloader = DataLoader(dataset, batch_size=32, shuffle=False)

model = torch.hub.load('pytorch/vision', 'resnet18', pretrained=True)
analyzer = FilterResponseAnalyzer(model, model.layer4)

stats = analyzer.compute_response_statistics(dataloader)
plot_filter_statistics(stats)
```

## 应用场景

### 1. 网络架构设计
- 验证网络是否学习到了合理的滤波器
- 发现无效或冗余的滤波器
- 指导网络架构的改进方向

### 2. 模型压缩
- 识别低贡献滤波器进行剪枝
- 分析滤波器冗余度
- 优化模型的计算效率

### 3. 迁移学习分析
- 选择适合目标任务的基础网络
- 理解预训练模型的特征质量
- 诊断微调过程中的特征迁移问题

### 4. 调试和错误分析
- 发现训练问题（如梯度消失、特征崩溃）
- 识别数据集标注问题
- 分析模型在特定输入上的失败原因

## 相关概念

| 概念 | 说明 |
|------|------|
| 卷积核/滤波器 | 执行卷积操作的权重矩阵 |
| 特征图 | 卷积操作的输出结果 |
| 感受野 | 滤波器能"看到"的输入区域大小 |
| 权重共享 | 同一滤波器参数在整个输入上复用 |
| 反卷积 | 将特征图反向映射到输入空间的技术 |
| 通道 | 特征图的深度维度 |
| 稀疏性 | 激活值中零或近零值的比例 |
