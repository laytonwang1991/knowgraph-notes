# 神经网络基础

## 一句话定义

> 神经网络是由大量神经元通过权重连接组成的并行计算模型，能够从数据中学习特征表示。

## 核心公式

### 神经元激活

$$
y = f(\sum_{i=1}^{n} w_i x_i + b)
$$

其中：
- $x_i$：输入特征
- $w_i$：权重
- $b$：偏置
- $f$：激活函数

### 常用激活函数

| 激活函数 | 公式 | 特点 |
|----------|------|------|
| Sigmoid | $\sigma(x) = \frac{1}{1+e^{-x}}$ | 输出0-1，易饱和 |
| ReLU | $f(x) = \max(0, x)$ | 计算快，梯度不衰减 |
| Tanh | $\tanh(x)$ | 输出-1到1，零中心化 |

## 详细说明

### 1. 基本原理

神经网络的核心思想是**通过层次化的表示学习来解决复杂问题**。

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 输出层
```

每个神经元接收上一层的输出，进行加权求和 + 激活函数，产生当前层的输出。

### 2. 关键机制

**前向传播（Forward Propagation）**
```python
# 简化的前向传播
def forward(x, weights, bias):
    z = np.dot(x, weights) + bias
    a = sigmoid(z)
    return a
```

**反向传播（Backpropagation）**
- 计算损失函数对每个参数的梯度
- 使用梯度下降更新参数
- 链式法则高效计算导数

### 3. 代码实现

```python
import numpy as np

class SimpleNeuralNetwork:
    def __init__(self, input_size, hidden_size, output_size):
        # Xavier 初始化
        self.W1 = np.random.randn(input_size, hidden_size) * np.sqrt(2.0 / input_size)
        self.b1 = np.zeros((1, hidden_size))
        self.W2 = np.random.randn(hidden_size, output_size) * np.sqrt(2.0 / hidden_size)
        self.b2 = zeros((1, output_size))

    def sigmoid(self, x):
        return 1 / (1 + np.exp(-x))

    def forward(self, X):
        self.z1 = np.dot(X, self.W1) + self.b1
        self.a1 = self.sigmoid(self.z1)
        self.z2 = np.dot(self.a1, self.W2) + self.b2
        return self.z2
```

### 4. 应用场景

- 图像分类（CNN）
- 自然语言处理（RNN/Transformer）
- 语音识别
- 推荐系统

## 优缺点

| 优点 | 缺点 |
|------|------|
| 能自动学习特征 | 训练需要大量数据 |
| 拟合任意复杂函数 | 可解释性差 |
| 泛化能力强 | 训练耗时，需要调参 |
