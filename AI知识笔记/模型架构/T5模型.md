# T5模型

## 一句话定义

> T5（Text-to-Text Transfer Transformer）是将所有NLP任务统一建模为"输入文本转换为输出文本"的框架，基于此范式训练的模型可以在任意任务上进行微调。

## 核心公式

### Text-to-Text统一框架

$$P(output | input; \theta)$$

无论任务类型如何，模型接收文本输入并生成文本输出。

### Span Corruption目标

$$L = -\sum_{i \in \text{masked}} \log P(x_i | x_{\setminus i}; \theta)$$

用连续span替换原始token，训练模型重建被破坏的文本片段。

### Span Corruption采样策略

对于长度为N的序列，以概率p随机选择起始位置，以概率q选择span长度：

$$\text{span_length} \sim \text{Geometric}(q=0.2)$$

典型配置：平均mask掉3个连续token。

## 详细说明

### 1. Text-to-Text设计理念

**核心思想：**
- 任务统一：翻译、分类、问答、摘要等均表示为文本到文本的转换
- 格式一致：使用任务前缀（如"translate English to German:"）区分不同任务
- 损失函数：标准的语言模型交叉熵损失

**任务前缀示例：**
```
translate English to German: I love machine learning.
summarize: Long document text here...
cola: The sentence is grammatically correct.
stsb: Sentence 1 and sentence 2 similarity rating:
```

### 2. Span Corruption预训练

**与BERT的区别：**
- BERT使用随机单个token掩码
- T5使用连续span掩码，用特殊符号[S]标记

**训练目标：**
- 预测被mask掉的连续token序列
- 输出自然流畅的文本片段

### 3. 模型规模对比

| 模型 | 层数 | d_model | 参数量 | 用途 |
|------|------|---------|--------|------|
| T5-Small | 6 | 512 | 60M | 轻量级实验 |
| T5-Base | 12 | 768 | 220M | 基准对比 |
| T5-Large | 24 | 1024 | 770M | 高性能任务 |
| T5-3B | 24 | 1024 | 3B | 大规模预训练 |
| T5-11B | 24 | 1024 | 11B | 最大规模 |

### 4. T5.1.1改进

- 移除了语言建模头（LM head）的相对位置偏置
- 使用Mesh TensorFlow实现高效分布式训练
- 预训练数据：C4数据集（750GB英文语料）

## 代码示例

### 使用Hugging Face加载T5

```python
from transformers import T5Tokenizer, T5ForConditionalGeneration

tokenizer = T5Tokenizer.from_pretrained('t5-base')
model = T5ForConditionalGeneration.from_pretrained('t5-base')

# 翻译任务
input_text = "translate English to German: I love machine learning."
inputs = tokenizer(input_text, return_tensors="pt")

outputs = model.generate(inputs.input_ids, max_length=50)
print(tokenizer.decode(outputs[0]))
```

### 文本摘要任务

```python
from transformers import T5Tokenizer, T5ForConditionalGeneration

model = T5ForConditionalGeneration.from_pretrained('t5-base')
tokenizer = T5Tokenizer.from_pretrained('t5-base')

# 摘要任务
article = "summarize: The quick brown fox jumps over the lazy dog. " * 10
inputs = tokenizer(article, return_tensors="pt", max_length=512, truncation=True)

summary_ids = model.generate(inputs.input_ids, num_beams=4, max_length=150, early_stopping=True)
summary = tokenizer.decode(summary_ids[0], skip_special_tokens=True)
print(summary)
```

### 问答任务

```python
from transformers import T5Tokenizer, T5ForConditionalGeneration

model = T5ForConditionalGeneration.from_pretrained('t5-base')
tokenizer = T5Tokenizer.from_pretrained('t5-base')

# 问答任务
question = "answer: What is the capital of France? Paris is the capital of France."
inputs = tokenizer(question, return_tensors="pt")

outputs = model.generate(inputs.input_ids, max_length=50)
print(tokenizer.decode(outputs[0], skip_special_tokens=True))
```

## 应用场景

| 任务类型 | 示例 | 效果 |
|----------|------|------|
| 机器翻译 | en→de, en→fr | 高质量翻译 |
| 文本摘要 | 长文→短文 | 抽取/生成结合 |
| 问答系统 | 问题+上下文→答案 | 开放域问答 |
| 文本分类 | 文本→类别 | FLAN微调后优秀 |
| 对话生成 | 对话历史→回复 | 生成连贯性良好 |
