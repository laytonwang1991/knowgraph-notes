# 剪枝技术

## 一句话定义

剪枝技术通过识别并移除神经网络中贡献度低（即"不重要"）的权重或结构单元，在保持模型精度的同时实现模型压缩和推理加速。

## 核心公式

### 1. 幅度剪枝（Magnitude Pruning）

$$
w_{pruned} = \begin{cases}
0 & \text{if } |w| < \tau \\
w & \text{otherwise}
\end{cases}
$$

- $\tau$：剪枝阈值（threshold）
- 通常以权重的某个百分位数作为阈值（如剪除50%的最小绝对值权重）

### 2. 梯度剪枝（Gradient Pruning）

$$
\Delta w_{pruned} = \begin{cases}
0 & \text{if } |\Delta w| < \tau_g \\
\Delta w & \text{otherwise}
\end{cases}
$$

- $\Delta w = -\eta \cdot \frac{\partial L}{\partial w}$：梯度更新量
- 只保留梯度幅度大于阈值的更新

### 3. 稀疏度（Sparsity）

$$
s = \frac{\text{num\_pruned\_weights}}{\text{total\_weights}}
$$

- 结构化剪枝中常以目标稀疏度（如50%、75%）作为剪枝比例

### 4. 彩票假说（Lottery Ticket Hypothesis）

设原始权重为 $w_0$，最佳迭代权重为 $w^*$，则获胜彩票满足：

$$
w^* \odot \mathbf{1}_{|w_0| > \tau} = w^*
$$

即存在一个初始子网络（子 lottery），经过相同训练可达到原始网络同等精度。

## 详细说明

### 1. 非结构化剪枝（Unstructured Pruning）

- 剪除单个权重元素，将目标权重设为0
- **优点**：细粒度控制，可达到极高稀疏度（90%+）
- **缺点**：稀疏矩阵存储不规则，实际加速困难，需要稀疏矩阵库支持
- 典型方法：幅度剪枝、随机剪枝

### 2. 结构化剪枝（Structured Pruning）

- 以结构为单位剪枝（channel、filter、block等）
- **优点**：可直接利用硬件的规则计算获得加速
- **缺点**：粒度较粗，精度损失较大
- 典型方法：Channel Pruning、Filter Pruning、Head Pruning

| 维度 | 说明 |
|------|------|
| Channel | 移除整个输入/输出通道 |
| Filter | 移除整个卷积核 |
| Layer | 移除整个网络层 |

### 3. 幅度剪枝（Magnitude Pruning）

- 最简单直观的剪枝方法
- 按权重绝对值大小排序，剪除最小的部分
- 通常结合渐进式剪枝（Iterative Magnitude Pruning）：
  1. 训练 → 剪枝 → 再训练 → 剪枝... 循环迭代

### 4. 梯度剪枝（Gradient Pruning）

- 基于梯度幅度而非权重幅度
- 只更新重要权重，减少通信开销（分布式训练场景）
- 适合模型并行和数据并行场景

### 5. 彩票假说（Lottery Ticket Hypothesis）

- 核心观点：随机初始化的神经网络中存在一个"中奖彩票"子网络
- 该子网络单独训练可达到原网络相同精度
- 识别方法：训练后反推初始权重mask中保留下来的权重
- 实验发现：仅保留3%~7%的初始权重即可训练成功

### 6. 剪枝调度策略

| 策略 | 描述 |
|------|------|
| One-shot | 一次性剪枝到目标稀疏度 |
| Iterative | 逐步剪枝，每步剪除少量 |
| Automated | 基于优化器自动确定剪枝时机 |

## 代码示例

```python
import torch
import torch.nn as nn
import copy

def magnitude_pruning(model, pruning_ratio=0.5):
    """
    幅度剪枝：将权重绝对值最小的部分设为0

    Args:
        model: 待剪枝的模型
        pruning_ratio: 剪枝比例 [0, 1]
    """
    for name, param in model.named_parameters():
        if 'weight' not in name:
            continue

        # 获取权重矩阵
        weight = param.data

        # 计算阈值（权重绝对值的分位数）
        threshold = torch.quantile(torch.abs(weight), pruning_ratio)

        # 创建mask：绝对值小于阈值的设为0
        mask = torch.abs(weight) > threshold
        param.data = weight * mask.float()

        # 将mask注册到param，用于反向传播时自动应用
        param.mask = mask
        print(f"Pruned {name}: {(~mask).sum().item()}/{mask.numel()} weights set to zero")


def iterative_pruning(model, dataloader, criterion, optimizer, epochs=10,
                       pruning_ratio=0.2, pruning_freq=2):
    """
    迭代剪枝：每个pruning_freq个epoch后进行一次剪枝

    Args:
        model: 待剪枝模型
        dataloader: 训练数据
        criterion: 损失函数
        optimizer: 优化器
        epochs: 训练轮数
        pruning_ratio: 每轮剪枝比例
        pruning_freq: 剪枝频率（每隔几个epoch剪枝一次）
    """
    model.train()
    current_pruning_ratio = 0.0

    for epoch in range(epochs):
        # 正常训练
        total_loss = 0
        for batch in dataloader:
            inputs, targets = batch
            optimizer.zero_grad()
            outputs = model(inputs)
            loss = criterion(outputs, targets)
            loss.backward()
            optimizer.step()
            total_loss += loss.item()

        # 定期剪枝
        if (epoch + 1) % pruning_freq == 0:
            # 累积剪枝比例计算
            # 例如：每次剪枝20%，3次后约为1-(0.8^3)=49%
            current_pruning_ratio = 1 - (1 - current_pruning_ratio) * (1 - pruning_ratio)
            magnitude_pruning(model, current_pruning_ratio)
            print(f"Epoch {epoch+1}: Cumulative pruning ratio = {current_pruning_ratio:.2%}")

    return model


class LSTMSparse(nn.Module):
    """LSTM结构化剪枝示例"""
    def __init__(self, input_size, hidden_size, num_layers):
        super().__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers)

    def prune_lstm(self, pruning_ratio=0.3):
        """剪枝LSTM的权重矩阵"""
        for name, param in self.lstm.named_parameters():
            if 'weight_ih' in name:  # input-to-hidden权重
                # 沿hidden维度剪枝
                weight = param.data
                weight_norm = torch.norm(weight, dim=0)  # 每列的L2范数
                threshold = torch.quantile(weight_norm, pruning_ratio)
                mask = weight_norm > threshold
                param.data = weight * mask.float().unsqueeze(0)
