# ML模型评估指标

## 一句话定义

模型评估指标是量化机器学习模型预测能力的标准度量体系，通过对比模型预测值与真实标签，提供模型性能的可比较数值依据，是模型选择、超参调优和线上评估的核心工具。

## 核心公式

### 混淆矩阵基础

|  | 预测正例 (Pred=1) | 预测负例 (Pred=0) |
|--|-------------------|------------------|
| **实际正例 (True=1)** | TP (真阳性) | FN (假阴性) |
| **实际负例 (True=0)** | FP (假阳性) | TN (真阴性) |

### 基础比率指标

**准确率（Accuracy）** — 整体正确率：
$$
\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN} = \frac{\text{正确预测数}}{\text{总样本数}}
$$

**精确率（Precision）** — 预测为正的样本中真正为正的比例：
$$
\text{Precision} = \frac{TP}{TP + FP} = \frac{\text{TP}}{\text{预测正例总数}}
$$

**召回率（Recall / Sensitivity）** — 实际为正的样本中被正确预测的比例：
$$
\text{Recall} = \frac{TP}{TP + FN} = \frac{\text{TP}}{\text{实际正例总数}}
$$

**特异度（Specificity）** — 实际为负的样本中被正确预测的比例：
$$
\text{Specificity} = \frac{TN}{TN + FP}
$$

### F1 分数

精确率和召回率的调和平均，避免极端值影响：
$$
\text{F1} = \frac{2 \times \text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}} = \frac{2 \times TP}{2 \times TP + FP + FN}
$$

更一般的 $F_\beta$ 分数（$\beta$ 控制召回率权重）：
$$
F_\beta = (1 + \beta^2) \times \frac{\text{Precision} \times \text{Recall}}{(\beta^2 \times \text{Precision}) + \text{Recall}}
$$

当 $\beta = 1$ 时为 F1，$\beta = 2$ 时更重视召回率（$F_2$），$\beta = 0.5$ 时更重视精确率（$F_{0.5}$）。

### ROC 与 AUC

**假阳性率（False Positive Rate）**：
$$
\text{FPR} = \frac{FP}{FP + TN}
$$

**真阳性率（True Positive Rate，即召回率）**：
$$
\text{TPR} = \frac{TP}{TP + FN}
$$

**ROC 曲线**：以 FPR 为 x 轴、TPR 为 y 轴，绘制分类阈值从 0 到 1 变化时的点对构成的曲线。理想分类器 ROC 曲线经过左上角 (0,1)，随机分类器沿对角线 (0,0)-(1,1)。

**AUC（Area Under Curve）**：ROC 曲线下的面积，取值 [0, 1]，等价于"随机抽取一个正样本和一个负样本，正样本得分高于负样本的概率"：
$$
\text{AUC} = P(S_{\text{pos}} > S_{\text{neg}})
$$

### PR 曲线与 AP

**PR 曲线**：以召回率为 x 轴、精确率为 y 轴，适用于类别不平衡场景。

**Average Precision (AP)**：
$$
\text{AP} = \sum_{n} (R_n - R_{n-1}) \times P_n
$$
即 PR 曲线下面积。

### 回归指标

**MAE（Mean Absolute Error）**：
$$
\text{MAE} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|
$$

**MSE（Mean Squared Error）**：
$$
\text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

**RMSE（Root MSE）**：$\text{RMSE} = \sqrt{\text{MSE}}$

**MAPE（Mean Absolute Percentage Error）**：
$$
\text{MAPE} = \frac{100\%}{n} \sum_{i=1}^{n} \left| \frac{y_i - \hat{y}_i}{y_i} \right|
$$

## 详细说明

### 1. 混淆矩阵的延伸指标

**Matthews 相关系数（MCC）**，取值 [-1, +1]，平衡准确率，在类别不平衡时比 F1 更可靠：
$$
\text{MCC} = \frac{TP \times TN - FP \times FN}{\sqrt{(TP+FP)(TP+FN)(TN+FP)(TN+FN)}}
$$

**balanced Accuracy**，各类别的平均准确率：
$$
\text{Balanced Accuracy} = \frac{1}{2}\left(\frac{TP}{TP+FN} + \frac{TN}{TN+FP}\right)
$$

### 2. 阈值选择策略

模型默认阈值 0.5 不一定是最优选择，阈值选择方法：

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| **Youden's J** | $J = \max(\text{TPR} - \text{FPR})$ | 找最优敏感度-特异度平衡点 |
| **F1 最优** | 最大化 $F_1$ 分数 | 平衡精确率和召回率 |
| **成本敏感** | 最小化 $C_{\text{FN}} \times FN + C_{\text{FP}} \times FP$ | 业务成本不对称 |
| **目标检测 mAP@IoU** | IoU > threshold 时计算 AP | 目标检测 |

**Youden's J 公式**：
$$
t^* = \arg\max_t [\text{TPR}(t) - \text{FPR}(t)]
$$

### 3. 类别不平衡的处理

当正负样本比例严重失衡（如 1:1000）时，准确率失去意义：

- **使用 AUC/PR**：AUC 对类别不平衡不敏感，PR 曲线直接反映少数类性能
- **SMOTE 过采样**：合成少数类样本
- **阈值下移**：将阈值从 0.5 降至 0.1-0.3，优先提高召回率
- **加权损失函数**：在 loss 中给少数类更高权重

### 4. 多分类扩展

对于 $K$ 分类问题，使用**宏平均（Macro）**和**微平均（Micro）**：

- **Macro F1**：各类分别计算 F1 后取算术平均
  $$
  \text{Macro F1} = \frac{1}{K} \sum_{k=1}^{K} F1_k
  $$

- **Micro F1**：将所有 TP/FP/FN 合并后统一计算 F1
  $$
  \text{Micro F1} = \frac{2 \times \sum TP_k}{2 \times \sum TP_k + \sum FP_k + \sum FN_k}
  $$

## 应用场景

1. **二分类模型评估**：欺诈检测、疾病诊断、垃圾邮件过滤
2. **多分类模型评估**：图像分类、文本分类、语音识别
3. **排序模型评估**：推荐系统、搜索排序、CTR 预估
4. **目标检测**：mAP（mean Average Precision）评估检测框精度
5. **回归模型评估**：价格预测、销量预测、时间序列预测

## 相关概念

- **混淆矩阵**：展示分类模型预测结果与真实标签对应关系的矩阵
- **交叉验证（K-Fold CV）**：将数据划分为 K 份，轮流使用 K-1 份训练、1 份验证，获得更稳定的指标估计
- **留一法（LOO CV）**：K = 样本数，适用于小数据集
- **Bootstrap**：通过有放回抽样估计指标置信区间
- **早停（Early Stopping）**：监控验证集指标，指标不再提升时停止训练
- **阈值搜索**：在验证集上搜索最优分类阈值

## 代码示例

```python
from sklearn.metrics import (
    accuracy_score, precision_score, recall_score,
    f1_score, roc_auc_score, roc_curve, precision_recall_curve,
    confusion_matrix, classification_report
)
import numpy as np

y_true = np.array([1, 0, 1, 1, 0, 1, 0, 0, 1, 0])
y_score = np.array([0.9, 0.1, 0.8, 0.7, 0.2, 0.85, 0.3, 0.4, 0.6, 0.15])
y_pred = (np.array(y_score) > 0.5).astype(int)

# 基础指标
print(f"Accuracy:  {accuracy_score(y_true, y_pred):.4f}")
print(f"Precision: {precision_score(y_true, y_pred):.4f}")
print(f"Recall:    {recall_score(y_true, y_pred):.4f}")
print(f"F1:        {f1_score(y_true, y_pred):.4f}")
print(f"AUC:       {roc_auc_score(y_true, y_score):.4f}")

# 混淆矩阵
print(confusion_matrix(y_true, y_pred))
# [[TN, FP], [FN, TP]]

# 最优阈值 (Youden's J)
fpr, tpr, thresholds = roc_curve(y_true, y_score)
j_scores = tpr - fpr
optimal_idx = np.argmax(j_scores)
optimal_threshold = thresholds[optimal_idx]
print(f"Optimal threshold (Youden's J): {optimal_threshold:.4f}")

# PR 曲线下面积 (AP)
precision, recall, _ = precision_recall_curve(y_true, y_score)
ap = np.sum((recall[1:] - recall[:-1]) * precision[1:])
print(f"Average Precision (AP): {ap:.4f}")
```
