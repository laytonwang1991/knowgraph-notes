# 数据清洗

## 一句话定义

数据清洗是通过检测和修正原始数据中的错误、不一致和重复记录，提升数据质量，为机器学习模型提供可靠输入的过程。

## 详细说明

### 1. 缺失值处理

缺失值是现实数据集最常见的问题之一，处理方式包括：

- **删除法**：直接删除含有缺失值的行或列，适用于缺失比例较小且随机分布的情况
- **均值/中位数填充**：对数值型特征用均值或中位数填充，对类别型特征用众数填充
- **插值填充**：使用前后有效值进行线性或多项式插值
- **模型预测填充**：用其他特征训练模型（如KNN、随机森林）预测缺失值
- **哑变量标记**：在填充的同时添加标记变量，记录该值原本缺失

```python
import pandas as pd
import numpy as np
from sklearn.impute import SimpleImputer, KNNImputer

# 删除缺失值过多的列
df = df.dropna(thresh=len(df) * 0.5, axis=1)

# 均值填充数值型特征
num_imputer = SimpleImputer(strategy='mean')
df['age'] = num_imputer.fit_transform(df[['age']])

# 众数填充类别型特征
cat_imputer = SimpleImputer(strategy='most_frequent')
df['city'] = cat_imputer.fit_transform(df[['city']])

# KNN插补（考虑特征间的相似性）
knn_imputer = KNNImputer(n_neighbors=5)
df_filled = knn_imputer.fit_transform(df)

# 标记缺失值
df['age_missing'] = df['age'].isna().astype(int)
```

### 2. 异常值检测

异常值会显著影响模型训练效果，需要谨慎处理：

- **Z-score方法**：Z > 3 或 Z < -3 的点视为异常
- **IQR方法**：小于 Q1 - 1.5*IQR 或大于 Q3 + 1.5*IQR 的点为异常
- **Isolation Forest**：基于树的异常检测算法，适用于多维数据
- **DBSCAN**：基于密度的聚类方法，将低密度区域的点标记为异常
- **箱线图可视化**：直观发现分布末端的数据点

```python
from scipy import stats
from sklearn.ensemble import IsolationForest
import numpy as np

# Z-score 方法
z_scores = np.abs(stats.zscore(df['value']))
df_clean = df[(z_scores < 3)]

# IQR 方法
Q1 = df['value'].quantile(0.25)
Q3 = df['value'].quantile(0.75)
IQR = Q3 - Q1
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR
df_clean = df[(df['value'] >= lower) & (df['value'] <= upper)]

# Isolation Forest
iso_forest = IsolationForest(contamination=0.05, random_state=42)
outliers = iso_forest.fit_predict(df[['feature1', 'feature2', 'feature3']])
df_clean = df[outliers == 1]
```

### 3. 数据平衡

类别不平衡会严重影响模型对少数类的识别能力：

- **过采样（Oversampling）**：对少数类进行重复采样，如SMOTE合成新样本
- **欠采样（Undersampling）**：减少多数类样本，如RandomUnderSampler
- **SMOTE**：在少数类样本之间进行插值，合成新样本
- **ADASYN**：自适应生成合成样本，生成密度与学习难度成正比
- **类别权重调整**：在模型训练时给少数类更高权重

```python
from imblearn.over_sampling import SMOTE, ADASYN
from imblearn.under_sampling import RandomUnderSampler
from imblearn.combine import SMOTETomek
from sklearn.utils import class_weight

# SMOTE 过采样
smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X_train, y_train)

# SMOTE + Tomek Links 组合
smote_tomek = SMOTETomek(random_state=42)
X_resampled, y_resampled = smote_tomek.fit_resample(X_train, y_train)

# 类别权重
weights = class_weight.compute_class_weight('balanced', classes=np.unique(y_train), y=y_train)
class_weights = dict(zip(np.unique(y_train), weights))
# 在模型中使用: model.fit(X, y, class_weight=class_weights)
```

### 4. 去重方法

重复数据会导致模型过拟合和评估偏差：

- **完全相同去重**：基于所有特征或主键进行完全匹配去重
- **模糊去重**：基于相似度计算（如编辑距离、Jaccard相似度）识别近似重复
- **基于规则去重**：结合业务逻辑，如同一手机号+姓名为重复
- **_merge标记**：在合并时标记重复来源，便于后续分析

```python
import pandas as pd
from rapidfuzz import fuzz

# 完全相同去重
df_deduped = df.drop_duplicates()

# 基于主键去重
df_deduped = df.drop_duplicates(subset=['user_id', 'phone'], keep='first')

# 模糊去重 - 编辑距离
def find_similar_pairs(df, threshold=90):
    similar_pairs = []
    for i in range(len(df)):
        for j in range(i+1, len(df)):
            ratio = fuzz.ratio(df.iloc[i]['name'], df.iloc[j]['name'])
            if ratio > threshold:
                similar_pairs.append((i, j, ratio))
    return similar_pairs

# merge 标记重复来源
df_merged = df1.merge(df2, on='user_id', how='outer', indicator=True)
df_merged['_merge'].value_counts()  # 查看左右表匹配情况
```

## 应用场景

- **推荐系统**：清洗用户行为数据，去除机器人刷量数据，提升推荐质量
- **风控模型**：处理征信数据中的缺失值和异常值，提高风控准确性
- **NLP任务**：去除文本中的HTML标签、特殊字符、乱码等噪声
- **图像识别**：清洗标注错误的图像数据，提升模型鲁棒性
- **数据仓库**：ETL流程中的数据清洗环节，保证数据一致性

## 相关概念

- 特征工程：数据清洗是特征工程的重要组成部分
- 数据标注：清洗后的高质量数据是标注质量的前提
- 数据验证：清洗后的数据需要通过验证确保质量
- ETL流程：数据清洗是大数据处理管道的核心环节
