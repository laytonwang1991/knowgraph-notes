# 知识蒸馏

## 一句话定义

知识蒸馏是将大型高性能模型（Teacher）中蕴含的暗知识（soft labels / 暗知识）迁移到小型高效模型（Student）的模型压缩技术。

## 核心公式

**蒸馏损失函数：**

$$
L = \alpha \cdot L_{soft} + (1-\alpha) \cdot L_{hard}
$$

其中：

$$
L_{soft} = -\sum_{i} p_i^T \cdot \log p_i^S = -\sum_{i} \frac{e^{z_i^T/T}}{\sum_j e^{z_j^T/T}} \cdot \log \frac{e^{z_i^S/T}}{\sum_j e^{z_j^S/T}}
$$

$$
L_{hard} = -\sum_{i} y_i \cdot \log p_i^S
$$

- $T$：温度参数（temperature），控制软化的程度
- $z^T$：Teacher模型的logits
- $z^S$：Student模型的logits
- $p^T$：Teacher的软概率分布（soft targets）
- $p^S$：Student的软概率分布
- $\alpha$：软硬标签的平衡权重

## 详细说明

### 1. Teacher-Student 架构

- **Teacher模型**：参数量大、精度高，通常是训练好的成熟模型
- **Student模型**：参数量小、推理快，是最终要部署的轻量模型
- 两者结构可以相似（如ResNet→MobileNet），也可以完全不同

### 2. 温度参数（Temperature）

- $T=1$ 时等同于普通softmax
- $T>1$ 时分布更平滑， logits 被"软化"
- $T \to \infty$ 时趋近均匀分布
- $T \to 0$ 时趋近argmax（硬标签）
- 通常 $T \in [2, 10]$

### 3. 软标签（Soft Labels）

- Teacher输出的概率分布包含类别间的相似关系
- 例如：猫的图片被预测为"猫 0.7、狗 0.2、虎 0.05"
- 这种暗知识（dark knowledge）比硬标签更丰富

### 4. 对比蒸馏（Contrastive Distillation）

- 引入对比学习思想，拉近相似样本、推开不相似样本
- 损失函数引入对比项：
$$
L_{contrastive} = -\log \frac{\exp(sim(z^T, z^S)/\tau)}{\sum \exp(sim(z^T, z_j^S)/\tau)}
$$
- $\tau$ 为对比温度参数

### 5. 蒸馏类型

| 类型 | 描述 |
|------|------|
| 离线蒸馏 | Teacher先训练好，再蒸馏Student |
| 在线蒸馏 | Teacher和Student同时训练 |
| 自蒸馏 | Student模型自己蒸馏自己 |

## 代码示例

```python
import torch
import torch.nn.functional as F

def distillation_loss(student_logits, teacher_logits, labels, T=4.0, alpha=0.7):
    """
    知识蒸馏损失函数

    Args:
        student_logits: Student模型的输出logits [batch_size, num_classes]
        teacher_logits: Teacher模型的输出logits [batch_size, num_classes]
        labels: 真实标签 [batch_size]
        T: 温度参数
        alpha: 软标签权重
    """
    # 软损失：KL散度（使用温度缩放）
    soft_student = F.log_softmax(student_logits / T, dim=-1)
    soft_teacher = F.softmax(teacher_logits / T, dim=-1)
    L_soft = F.kl_div(soft_student, soft_teacher, reduction='batchmean') * (T * T)

    # 硬损失：交叉熵
    L_hard = F.cross_entropy(student_logits, labels)

    # 组合损失
    return alpha * L_soft + (1 - alpha) * L_hard


class DistillationTrainer:
    def __init__(self, teacher, student, optimizer, T=4.0, alpha=0.7):
        self.teacher = teacher
        self.student = student
        self.optimizer = optimizer
        self.T = T
        self.alpha = alpha

    def train_step(self, batch):
        x, y = batch
        self.optimizer.zero_grad()

        # Teacher前向（不更新梯度）
        with torch.no_grad():
            teacher_logits = self.teacher(x)

        # Student前向
        student_logits = self.student(x)

        # 计算蒸馏损失
        loss = distillation_loss(student_logits, teacher_logits, y, self.T, self.alpha)

        loss.backward()
        self.optimizer.step()
        return loss.item()
```

## 应用场景

1. **移动端部署**：将BERT蒸馏为DistilBERT，在手机上运行NLP模型
2. **实时推理**：自动驾驶中的目标检测模型压缩
3. **边缘计算**：IoT设备上的模型轻量化
4. **模型集成**：将多个模型的ensemble知识压缩到单一模型
5. **跨语言迁移**：大语言模型到小语言模型的知识迁移

## 相关概念

- **模型压缩**：包含蒸馏、剪枝、量化等技术的总称
- **剪枝**：通过移除权重实现压缩，与蒸馏互补
- **量化**：将浮点数权重转换为低精度表示
- **迁移学习**：知识从一个域迁移到另一个域
- **对比学习**：通过对比样本学习表示
