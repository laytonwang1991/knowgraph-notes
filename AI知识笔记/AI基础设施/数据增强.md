# 数据增强

## 一句话定义

数据增强是通过对原始训练数据施加各种可学习的变换，以低成本扩充数据集规模，提升模型泛化能力和对对抗样本鲁棒性的技术。

## 详细说明

### 1. 图像翻转与旋转

基础几何变换，是图像数据增强最常用的方法：

- **水平翻转**：最常用的增强方法，适用于大多数物体识别任务
- **垂直翻转**：需根据任务判断是否适用（如人脸不适合垂直翻转）
- **随机旋转**：在一定角度范围内随机旋转图像
- **随机裁剪**：从原图中随机裁剪一块区域进行训练
- **随机缩放**：在一定比例范围内缩放图像

```python
import torchvision.transforms as T
from PIL import Image

# torchvision 图像增强
train_transform = T.Compose([
    T.RandomHorizontalFlip(p=0.5),           # 水平翻转，概率0.5
    T.RandomVerticalFlip(p=0.3),             # 垂直翻转，概率0.3
    T.RandomRotation(degrees=15),             # 随机旋转，±15度
    T.RandomResizedCrop(224, scale=(0.8, 1.0)),  # 随机裁剪并缩放到224x224
    T.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2),  # 颜色抖动
    T.ToTensor(),
    T.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

# Albumentations 库（更高效的图像增强）
import albumentations as A

train_transform = A.Compose([
    A.HorizontalFlip(p=0.5),
    A.ShiftScaleRotate(shift_limit=0.1, scale_limit=0.1, rotate_limit=15, p=0.5),
    A.RandomBrightnessContrast(p=0.3),
    A.GaussNoise(var_limit=(10.0, 50.0), p=0.3),
    A.CoarseDropout(max_holes=8, max_height=32, max_width=32, p=0.3),
])
```

### 2. Mixup

通过混合两张图像及其标签创建软标签样本：

- **原理**：将两张图像按比例混合，同时混合对应的标签
- **优点**：显著提升模型泛化能力，减少过拟合
- **变体**：CutMix（将一张图的区块粘贴到另一张图上）
- **参数**：混合系数alpha通常取0.2-0.4

```python
import torch
import torch.nn.functional as F

def mixup_data(x, y, alpha=0.2):
    """Mixup 数据增强"""
    if alpha > 0:
        lam = np.random.beta(alpha, alpha)
    else:
        lam = 1

    batch_size = x.size()[0]
    index = torch.randperm(batch_size).to(x.device)

    mixed_x = lam * x + (1 - lam) * x[index, :]
    y_a, y_b = y, y[index]
    return mixed_x, y_a, y_b, lam

def mixup_criterion(criterion, pred, y_a, y_b, lam):
    """Mixup 损失计算"""
    return lam * criterion(pred, y_a) + (1 - lam) * criterion(pred, y_b)

# 训练循环中使用
for data, target in dataloader:
    data, target = data.to(device), target.to(device)
    data, target_a, target_b, lam = mixup_data(data, target, alpha=0.2)

    optimizer.zero_grad()
    output = model(data)
    loss = mixup_criterion(criterion, output, target_a, target_b, lam)
    loss.backward()
    optimizer.step()
```

### 3. Cutmix

将一张图像的矩形区域粘贴到另一张图像上：

- **原理**：在图像中随机选择矩形区域，用另一张图像的对应区域替换
- **优点**：强迫模型关注局部特征而非整体纹理
- **特点**：保留像素级信息的同时提供多样性

```python
def cutmix_data(x, y, alpha=1.0):
    """Cutmix 数据增强"""
    lam = np.random.beta(alpha, alpha)
    batch_size = x.size()[0]
    index = torch.randperm(batch_size).to(x.device)

    # 生成随机矩形框
    W, H = x.size()[2], x.size()[3]
    cut_rat = np.sqrt(1. - lam)
    cut_w = int(W * cut_rat)
    cut_h = int(H * cut_rat)
    cx = np.random.randint(W)
    cy = np.random.randint(H)

    bbx1 = np.clip(cx - cut_w // 2, 0, W)
    bby1 = np.clip(cy - cut_h // 2, 0, H)
    bbx2 = np.clip(cx + cut_w // 2, 0, W)
    bby2 = np.clip(cy + cut_h // 2, 0, H)

    x[:, :, bbx1:bbx2, bby1:bby2] = x[index, :, bbx1:bbx2, bby1:bby2]
    lam = 1 - ((bbx2 - bbx1) * (bby2 - bby1) / (W * H))
    y_a, y_b = y, y[index]
    return x, y_a, y_b, lam

import timm

# timm 库内置了 MixUp 和 CutMix 支持
```

### 4. 文本增强技术

针对自然语言处理任务的数据增强：

- **同义词替换**：随机替换词语为其同义词
- **随机插入**：在文本中随机位置插入词语的同义词
- **随机交换**：随机交换文本中两个词语的位置
- **回译**：将文本翻译成其他语言再翻译回来
- **EDA**：简单数据增强，包含以上四种操作

```python
import random
from googletrans import Translator

# 同义词替换
def synonym_replacement(sentence, n=2):
    words = sentence.split()
    new_words = words.copy()
```

## 应用场景

- **计算机视觉**：图像分类、目标检测、语义分割等任务
- **自然语言处理**：文本分类、情感分析、问答系统
- **语音识别**：对音频添加噪声、变调、变速等变换
- **医学影像**：弥补医学数据标注困难、数据量少的问题
- **小样本学习**：在数据稀缺的场景下提升模型泛化能力

## 相关概念

- 迁移学习：数据增强常与预训练模型配合使用
- 正则化：数据增强本质上是一种数据层面的正则化
- 对抗训练：结合对抗样本进行训练，提升鲁棒性
- AutoAugment：自动搜索最优增强策略
