# 训练数据选择

## 一句话定义

训练数据选择是通过规则、模型或算法从海量原始数据中筛选出高质量、高信息量的样本，用于模型训练，以提升模型性能、降低训练成本和缩短迭代周期。

## 详细说明

### 1. 数据过滤

基于规则或统计方法过滤低质量数据：

- **规则过滤**：基于业务规则过滤，如删除敏感词、过滤无效内容
- **统计过滤**：基于数据分布统计，过滤分布异常的数据点
- **质量分数过滤**：基于预训练模型对数据进行打分，过滤低分样本
- **多样性采样**：在过滤时保持数据的多样性分布

```python
import pandas as pd
import numpy as np

# 规则过滤
def rule_based_filter(df):
    # 过滤长度异常的数据
    df = df[(df['text'].str.len() >= 10) & (df['text'].str.len() <= 500)]

    # 过滤包含敏感词的样本
    sensitive_words = ['敏感词1', '敏感词2']
    pattern = '|'.join(sensitive_words)
    df = df[~df['text'].str.contains(pattern, na=False)]

    # 过滤重复内容
    df = df.drop_duplicates(subset=['text'])

    return df

# 统计过滤 - 基于Z-score
def statistical_filter(df, column, z_threshold=3):
    z_scores = np.abs(stats.zscore(df[column]))
    return df[z_scores < z_threshold]

# 质量分数过滤
def quality_filter(df, score_column='quality_score', threshold=0.6):
    return df[df[score_column] >= threshold]

# 使用预训练模型过滤
from transformers import pipeline

quality_classifier = pipeline("text-classification", model="textattack/distilbert-MRPC")

def model_based_filter(df, batch_size=32):
    scores = []
    for i in range(0, len(df), batch_size):
        batch = df['text'].iloc[i:i+batch_size].tolist()
        results = quality_classifier(batch)
        scores.extend([r['score'] for r in results])
    df['model_score'] = scores
    return df[df['model_score'] >= 0.7]
```

### 2. 数据去毒

清除训练数据中的后门触发器、对抗样本和污染数据：

- **后门检测**：识别可能被植入后门的数据样本
- **对抗样本过滤**：过滤精心构造的对抗输入
- **数据溯源**：追踪数据来源，过滤不可信来源的数据
- **激活分析**：分析模型激活模式，识别异常数据的影响

```python
# 基于激活值的后门检测
def activation_analysis(model, dataloader, device):
    """通过分析神经元激活值检测潜在后门样本"""
    activations = []
    labels = []

    def hook_fn(module, input, output):
        activations.append(output.detach())

    handle = model.layer[-1].register_forward_hook(hook_fn)

    with torch.no_grad():
        for batch in dataloader:
            inputs = batch['input'].to(device)
            outputs = model(inputs)
            labels.extend(batch['label'].cpu().numpy())

    handle.remove()

    activations = torch.cat(activations)
    # 检测激活异常样本
    anomaly_scores = torch.var(activations, dim=0)
    return anomaly_scores

# 基于SIF的去毒方法
from sklearn.decomposition import PCA

def spectral_signature(activations, labels):
    """使用谱签名检测后门样本"""
    pca = PCA(n_components=2)
    features_2d = pca.fit_transform(activations)

    # 计算每个类别的平均表示
    class_means = {}
    for feat, label in zip(features_2d, labels):
        if label not in class_means:
            class_means[label] = []
        class_means[label].append(feat)

    # 识别偏离类中心的样本
    outliers = []
    for i, (feat, label) in enumerate(zip(features_2d, labels)):
        class_center = np.mean(class_means[label], axis=0)
        distance = np.linalg.norm(feat - class_center)
        if distance > 2 * np.std([np.linalg.norm(f - class_center) for f in class_means[label]]):
            outliers.append(i)

    return outliers
```

### 3. 质量评分

多维度评估训练数据质量：

- **流畅度评分**：评估文本是否通顺自然
- **信息量评分**：评估数据的信息密度和多样性
- **一致性评分**：评估标签与内容是否一致
- **难度评分**：评估样本的学习难度（用于课程学习）

```python
# 使用语言模型评估流畅度
from transformers import GPT2LMHeadModel, GPT2Tokenizer

def fluency_score(text, model, tokenizer):
    """使用GPT-2计算文本流畅度（困惑度）"""
    inputs = tokenizer(text, return_tensors='pt', truncation=True, max_length=512)
    with torch.no_grad():
        outputs = model(**inputs, labels=inputs['input_ids'])
        loss = outputs.loss
        # 困惑度
        perplexity = torch.exp(loss).item()
    return 1 / perplexity if perplexity > 0 else 0

# 标签一致性评分
from sklearn.metrics import cohen_kappa_score

def label_consistency(df, n_splits=3):
    """通过交叉验证评估标签一致性"""
    from sklearn.model_selection import KFold

    kf = KFold(n_splits=n_splits)
    kappa_scores = []

    for train_idx, val_idx in kf.split(df):
        train_data = df.iloc[train_idx]
        val_data = df.iloc[val_idx]

        # 训练简单分类器
        clf = train_classifier(train_data)
        predictions = clf.predict(val_data)
        kappa = cohen_kappa_score(val_data['label'], predictions)
        kappa_scores.append(kappa)

    return np.mean(kappa_scores)
```

### 4. 数据合成

通过生成模型合成高质量训练数据：

- **LLM合成**：使用大语言模型生成符合分布的新数据
- **扩散模型合成**：使用扩散模型生成图像数据
- **对抗合成**：通过对抗训练框架生成难例样本
- **插值合成**：在已有样本之间插值生成新样本

```python
# 使用GPT生成合成数据
from transformers import GPT2LMHeadModel, GPT2Tokenizer
import json

def synthetic_data_generation(prompt, n_samples=100, max_length=100):
    """使用GPT-2生成多样化合成文本"""
    model = GPT2LMHeadModel.from_pretrained('gpt2')
    tokenizer = GPT2Tokenizer.from_pretrained('gpt2')

    synthetic_data = []
    for _ in range(n_samples):
        input_ids = tokenizer.encode(prompt, return_tensors='pt')
        output = model.generate(
            input_ids,
            max_length=max_length,
            num_return_sequences=1,
            temperature=0.8,
            do_sample=True,
            top_k=50
        )
        text = tokenizer.decode(output[0], skip_special_tokens=True)
        synthetic_data.append(text)

    return synthetic_data

# 使用扩散模型合成图像
import torch
from diffusers import StableDiffusionPipeline

def synthetic_image(prompt, n_images=10):
    """使用Stable Diffusion生成合成图像"""
    pipe = StableDiffusionPipeline.from_pretrained(
        "runwayml/stable-diffusion-v1-5",
        torch_dtype=torch.float16
    )
    pipe = pipe.to("cuda")

    images = pipe(prompt, num_images_per_prompt=n_images).images
    return images

# 使用SMOTE合成表格数据
from imblearn.over_sampling import SMOTE, ADASYN

def synthetic_tabular(X, y, method='smote'):
    """使用SMOTE或ADASYN合成表格数据"""
    if method == 'smote':
        sampler = SMOTE(random_state=42)
    else:
        sampler = ADASYN(random_state=42)

    X_resampled, y_resampled = sampler.fit_resample(X, y)
    return X_resampled, y_resampled
```

## 应用场景

- **大语言模型训练**：从海量网页数据中筛选高质量文本
- **自动驾驶**：从行车记录中筛选关键场景的训练数据
- **推荐系统**：过滤用户刷量和机器人行为数据
- **内容审核**：筛选高质量标注数据进行审核模型训练
- **医学AI**：从医学影像中筛选高质量标注数据

## 相关概念

- 主动学习：优先标注高不确定性样本，减少标注成本
- 课程学习：按难度顺序学习样本，由易到难
- 迁移学习：利用源域筛选策略迁移到目标域
- 数据管理：训练数据的选择和管理是MLOps的重要组成部分
