# 权重初始化

## 一句话定义

权重初始化是为神经网络各层权重赋予初始值的过程，合理的初始化策略能有效防止梯度消失/爆炸，加速模型收敛。

## 详细说明

### 为什么权重初始化很重要

神经网络的训练过程本质是优化权重以最小化损失函数。初始权重选择不当会导致：

- **梯度消失**：深层网络反向传播时，梯度逐层指数级衰减
- **梯度爆炸**：梯度逐层指数级增大，导致训练不稳定
- **对称性破坏**：相同初始化的神经元学到相同特征
- **收敛速度慢**：不合理的初始化需要更多迭代才能收敛

### 经典初始化方法

#### 1. Xavier/Glorot初始化

由 Xavier Glorot 和 Yoshua Bengio 于 2010 年提出，适用于 Sigmoid 和 Tanh 激活函数。

**核心思想**：保持前向传播和反向传播时方差一致

```python
# 正态分布形式
W ~ N(0, sqrt(2 / (fan_in + fan_out)))

# 均匀分布形式
W ~ U(-sqrt(6 / (fan_in + fan_out)), sqrt(6 / (fan_in + fan_out)))
```

其中 `fan_in` 是输入神经元数，`fan_out` 是输出神经元数。

#### 2. Kaiming/He初始化

由何恺明等人于 2015 年提出，专门为 ReLU 系列激活函数设计。

**核心思想**：考虑 ReLU 对信号的非线性截断效应

```python
# 正态分布形式
W ~ N(0, sqrt(2 / fan_in))

# 均匀分布形式
W ~ U(-sqrt(6 / fan_in), sqrt(6 / fan_in))
```

**MSRA初始化**：Kaiming 初始化的特殊形式，用于 PReLU 等变体。

#### 3. Normal初始化（正态分布）

直接使用正态分布采样权重。

```python
W ~ N(mean, std)
```

- **问题**：标准差选择困难，易导致梯度问题
- **适用场景**：小规模实验或特殊需求

#### 4. 零初始化的问题

将所有权重初始化为零是**严重错误**：

- 所有神经元学到相同特征（对称性）
- 反向传播时梯度相同，无法学习多样化特征
- **偏置可以初始化为零**：对对称性无影响

### 初始化方法对比

| 方法 | 适用激活函数 | 方差计算 | 特点 |
|------|-------------|---------|------|
| Xavier | Sigmoid, Tanh | 2 / (fan_in + fan_out) | 平衡前向/反向方差 |
| Kaiming | ReLU, Leaky ReLU | 2 / fan_in | 考虑ReLU截断效应 |
| Normal | 通用 | 自定义 | 灵活但需调参 |

### 实践建议

1. **ReLU 网络**：优先使用 Kaiming 初始化
2. **Tanh/Sigmoid 网络**：使用 Xavier 初始化
3. **ResNet**：使用 Kaiming + 零偏置
4. **Transformer**：使用 N(0, 0.02) 或 Xavier
5. **避免过大初始值**：导致梯度爆炸
6. **避免过小初始值**：导致梯度消失

## 代码示例

### PyTorch 官方接口

```python
import torch.nn as nn

# Xavier 均匀分布
nn.init.xavier_uniform_(layer.weight)

# Xavier 正态分布
nn.init.xavier_normal_(layer.weight)

# Kaiming 均匀分布（推荐用于ReLU）
nn.init.kaiming_uniform_(layer.weight, mode='fan_in', nonlinearity='relu')

# Kaiming 正态分布
nn.init.kaiming_normal_(layer.weight, mode='fan_in', nonlinearity='relu')
```

### 完整示例

```python
import torch
import torch.nn as nn
import math

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 128)
        self.fc3 = nn.Linear(128, 10)
        self.relu = nn.ReLU()

        # 自定义初始化
        self._init_weights()

    def _init_weights(self):
        for m in self.modules():
            if isinstance(m, nn.Linear):
                # 使用Kaiming初始化（适合ReLU）
                nn.init.kaiming_normal_(m.weight, mode='fan_in', nonlinearity='relu')
                if m.bias is not None:
                    nn.init.zeros_(m.bias)

    def forward(self, x):
        x = x.view(x.size(0), -1)
        x = self.relu(self.fc1(x))
        x = self.relu(self.fc2(x))
        x = self.fc3(x)
        return x
```

### 验证初始化效果

```python
def check_gradients(model, inputs, labels):
    """检查梯度是否正常（无消失/爆炸）"""
    criterion = nn.CrossEntropyLoss()
    outputs = model(inputs)
    loss = criterion(outputs, labels)
    loss.backward()

    grad_norms = {}
    for name, param in model.named_parameters():
        if param.grad is not None:
            grad_norms[name] = param.grad.norm().item()

    return grad_norms

# 正常初始化时，各层梯度范数应该在相近范围内
```

## 应用场景

| 场景 | 推荐初始化 | 原因 |
|------|----------|------|
| ImageNet分类（CNN） | Kaiming (ReLU) | 适合ReLU激活函数 |
| NLP任务（Transformer） | N(0, 0.02) / Xavier | 标准实践 |
| GANs | 谨慎初始化 | 避免训练不稳定 |
| 简单MLP | Xavier / Kaiming | 通用选择 |
| 预训练模型微调 | 保持原初始化 | 不需要重新初始化 |

## 相关概念

- **Batch Normalization**：减少对初始化的依赖，使训练更稳定
- **残差连接**（ResNet）：缓解梯度消失，允许更自由的初始化
- **梯度消失/爆炸**：与初始化密切相关的训练问题
- **LSUV**：层序单位方差初始化，一种数据驱动的初始化方法
