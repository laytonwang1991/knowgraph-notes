# 迁移学习

## 一句话定义

迁移学习是一种利用已在一个任务（源域）上学到的知识来提升另一个相关任务（目标域）性能的技术，旨在减少目标任务所需的训练数据和计算资源。

## 核心公式

### 1. 域适应损失函数

给定源域分布 $P_s(X, Y)$ 和目标域分布 $P_t(X, Y)$，域适应希望学习一个特征映射 $f$，使得：

$$\min_f \mathcal{L}_t(f) + \lambda \cdot \mathcal{D}(P_s(f(X_s)), P_t(f(X_t)))$$

其中 $\mathcal{L}_t$ 是目标域任务的损失，$\mathcal{D}$ 是域间距离度量，$\lambda$ 是平衡参数。

### 2. MMD (Maximum Mean Discrepancy)

$$MMD^2(X_s, X_t) = \left\| \frac{1}{n_s} \sum_{i=1}^{n_s} \phi(x_i^s) - \frac{1}{n_t} \sum_{j=1}^{n_t} \phi(x_j^t) \right\|^2$$

### 3. Fine-tuning损失

$$L_{total} = \alpha \cdot L_{task}(y, \hat{y}) + \beta \cdot L_{regularization}(\theta)$$

其中 $\alpha$ 通常设为较小值（如0.1-0.5）以微调预训练权重。

### 4. 多任务学习联合损失

$$\mathcal{L}_{MTL} = \sum_{k=1}^{K} w_k \cdot \mathcal{L}_k(\theta; D_k)$$

其中 $w_k$ 是任务 $k$ 的权重，$K$ 是任务数量。

## 详细说明

### 一、迁移学习核心概念

#### 1. 源域与目标域
- **源域 (Source Domain)**: 已有丰富标注数据的任务或领域
- **目标域 (Target Domain)**: 标注数据稀少、需要迁移知识的任务或领域
- **域偏移 (Domain Shift)**: 源域和目标域数据分布的差异

#### 2. 迁移学习的基本假设
- 源域和目标域相关但分布不同
- 特征空间存在可迁移的共性
- 任务之间共享底层知识表示

### 二、迁移学习的主要方法

#### 1. 特征迁移 (Feature Transfer)
- 将源域学到的特征表示应用于目标域
- 适用于特征语义通用的场景
- 典型方法：深度特征提取

#### 2. 模型迁移 (Model Transfer)
- 迁移模型参数或模型结构
- 常见形式：预训练模型微调
- 适用场景：源域和目标域任务相似

#### 3. 关系迁移 (Relation Transfer)
- 迁移数据之间的关系知识
- 适用于结构化数据的迁移

### 三、Fine-tuning策略

#### 1. 完全微调
- 解冻所有层，在目标数据上训练所有参数
- 适用于目标域数据量较大的情况
- 风险：可能丢失预训练知识

#### 2. 冻结微调 (Frozen Fine-tuning)
- 冻结预训练模型的底层参数，只训练顶层
- 底层通常学习通用特征（如边缘、纹理）
- 顶层学习任务特定特征

#### 3. 逐层解冻 (Layer-wise Unfreezing)
- 从顶层到底层逐层解冻进行训练
- 策略：先训练顶层，再逐步解冻底层
- 优点：避免灾难性遗忘

#### 4. Adapter方法
- 在Transformer层之间插入小型适配模块
- 只训练适配器参数，保持原模型不变
- 高效参数利用，适用于少数据场景

### 四、域适应技术

#### 1. 判别域适应
- **DAN (Deep Adaptation Network)**: 多核MMD匹配层均值
- **JAN (Joint Adaptation Network)**: 联合适配网络
- **CORAL (Correlation Alignment)**: 协方差对齐

#### 2. 对抗域适应
- **DANN (Domain-Adversarial Neural Network)**: 引入域判别器和梯度反转层
- **ADDA (Adversarial Discriminative Domain Adaptation)**: 对抗双向适应

#### 3. 生成域适应
- 使用GAN进行域转换
- CycleGAN在图像风格迁移中的应用

### 五、多任务学习

#### 1. 硬参数共享
- 所有任务共享底层隐藏层
- 顶层任务特定层独立
- 优点：强正则化效果，减少过拟合

#### 2. 软参数共享
- 每个任务有独立模型参数
- 通过正则化鼓励参数相似
- 灵活性更高

#### 3. 任务权重学习
- **GradNorm**: 梯度归一化自动调整任务权重
- **Uncertainty Weighting**: 基于任务不确定性的加权
- **Dynamic Weight Average**: 动态权重平均

### 六、领域专家模型

#### 1. 专家混合模型
- 多个专家网络处理不同子领域
- 门控网络决定专家激活权重
- $y = \sum_{i=1}^{K} G(x)_i \cdot E_i(x)$

#### 2. LoRA (Low-Rank Adaptation)
- 在原始权重旁添加低秩更新矩阵
- $\Delta W = A \cdot B$（低秩分解）
- 大幅减少可训练参数量

#### 3. Prefix Tuning / Prompt Tuning
- 学习任务特定的连续前缀向量
- 不改变模型主体参数

## 代码示例

### Python - PyTorch模型微调

```python
import torch
import torch.nn as nn
from torchvision import models, transforms
from torch.optim import AdamW

class FineTuner:
    def __init__(self, model_name='resnet50', num_classes=10, pretrained=True):
        # 加载预训练模型
        if model_name == 'resnet50':
            self.model = models.resnet50(pretrained=pretrained)
            num_features = self.model.fc.in_features
            # 替换分类头
            self.model.fc = nn.Linear(num_features, num_classes)
        else:
            raise ValueError(f"Unsupported model: {model_name}")

    def freeze_layers(self, num_layers=5):
        """冻结前num_layers个层"""
        modules = list(self.model.children())
        for i, module in enumerate(modules[:-1][:num_layers]):
            for param in module.parameters():
                param.requires_grad = False

    def unfreeze_all(self):
        """解冻所有层"""
        for param in self.model.parameters():
            param.requires_grad = True

    def train_epoch(self, train_loader, criterion, optimizer, device):
        self.model.train()
        total_loss = 0
        for batch_idx, (data, target) in enumerate(train_loader):
            data, target = data.to(device), target.to(device)
            optimizer.zero_grad()
            output = self.model(data)
            loss = criterion(output, target)
            loss.backward()
            optimizer.step()
            total_loss += loss.item()
        return total_loss / len(train_loader)

# 使用示例
fine_tuner = FineTuner(model_name='resnet50', num_classes=10)
fine_tuner.freeze_layers(num_layers=3)  # 冻结前3个模块

# 设置优化器（只优化未冻结参数）
optimizer = AdamW(filter(lambda p: p.requires_grad, fine_tuner.model.parameters()), lr=1e-4)
```

### Python - 使用Hugging Face Transformers微调

```python
from transformers import AutoModelForSequenceClassification, AutoTokenizer, TrainingArguments, Trainer
from datasets import load_dataset

# 加载预训练模型和分词器
model_name = "bert-base-uncased"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForSequenceClassification.from_pretrained(model_name, num_labels=2)

# 加载数据集
dataset = load_dataset("glue", "sst2")
dataset = dataset.map(lambda examples: tokenizer(examples['sentence'], padding=True, truncation=True), batched=True)

# 设置训练参数
training_args = TrainingArguments(
    output_dir="./results",
    num_train_epochs=3,
    per_device_train_batch_size=16,
    learning_rate=2e-5,
    warmup_ratio=0.1,
    weight_decay=0.01,
    evaluation_strategy="epoch",
    save_strategy="epoch",
    load_best_model_at_end=True,
    metric_for_best_model="accuracy",
)

# 创建Trainer
trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=dataset['train'],
    eval_dataset=dataset['validation'],
    compute_metrics=lambda p: {"accuracy": (p.predictions[0].argmax(-1) == p.label_ids).mean()},
)

# 开始微调
trainer.train()
```

### Python - LoRA实现

```python
import torch
import torch.nn as nn
import math

class LoRALinear(nn.Module):
    def __init__(self, original_layer, rank=4, alpha=1.0):
        super().__init__()
        self.original = original_layer
        self.rank = rank
        self.alpha = alpha

        # 获取原始层维度
        in_features = original_layer.in_features
        out_features = original_layer.out_features

        # 冻结原始权重
        self.original.weight.requires_grad = False
        if hasattr(self.original, 'bias') and self.original.bias is not None:
            self.original.bias.requires_grad = False

        # 初始化LoRA矩阵
        self.lora_A = nn.Parameter(torch.zeros(in_features, rank))
        self.lora_B = nn.Parameter(torch.zeros(rank, out_features))
        self.scaling = alpha / rank

        # 初始化A矩阵（使用随机正交初始化）
        nn.init.normal_(self.lora_A, mean=0, std=0.02)

    def forward(self, x):
        # 原始前向传播
        original_output = self.original(x)
        # LoRA更新
        lora_update = (x @ self.lora_A @ self.lora_B) * self.scaling
        return original_output + lora_update

# 应用LoRA到模型
def apply_lora_to_model(model, rank=4, alpha=1.0, target_modules=None):
    """将LoRA应用到模型的指定层"""
    for name, module in model.named_children():
        if isinstance(module, nn.Linear):
            if target_modules is None or name in target_modules:
                setattr(model, name, LoRALinear(module, rank, alpha))
        else:
            apply_lora_to_model(module, rank, alpha, target_modules)
    return model
```

## 应用场景

### 1. 计算机视觉
- **ImageNet预训练模型迁移**: 将ResNet、VGG等模型迁移到医疗影像诊断
- **风格迁移**: 将艺术风格迁移到照片
- **目标检测**: 使用COCO预训练模型检测新类别

### 2. 自然语言处理
- **BERT/ GPT系列微调**: 通用语言模型适配特定任务
- **情感分析**: 通用预训练模型微调为产品评论分析
- **命名实体识别**: 跨领域NER迁移

### 3. 语音处理
- **ASR模型迁移**: 通用语音识别迁移到特定方言
- **语音合成**: 将声音特征迁移到新说话人

### 4. 跨语言迁移
- **多语言模型**: 如mBERT、XLM-R跨语言迁移
- **低资源语言**: 从高资源语言迁移到低资源语言

### 5. 个性化推荐
- **冷启动问题**: 利用相关用户/物品知识迁移
- **跨平台迁移**: 将一个平台知识迁移到新平台

## 相关概念

| 概念 | 说明 |
|------|------|
| **元学习 (Meta-Learning)** | 学习如何学习，快速适应新任务 |
| **持续学习 (Continual Learning)** | 在连续任务中避免灾难性遗忘 |
| **零样本学习 (Zero-shot Learning)** | 完全未见过类别的识别 |
| **少样本学习 (Few-shot Learning)** | 仅用极少样本进行学习 |
| **知识蒸馏 (Knowledge Distillation)** | 将大模型知识迁移到小模型 |
| **多任务学习 (Multi-task Learning)** | 共享表示学习多个相关任务 |
