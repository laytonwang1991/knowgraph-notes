# 命名实体识别 (Named Entity Recognition, NER)

## 一句话定义

NER是序列标注任务，通过识别文本中的人名、地名、机构名等实体并为其分配预定义类别标签来抽取结构化信息。

## 核心公式

### BIO 标注模式

```
B-X: 实体的开始 (Begin)
I-X: 实体的中间/延续 (Inside)
O: 非实体 (Outside)

示例: "张三 [B-PER] 住在 [O] 北京 [B-LOC]"
```

### CRF 层输出

```
P(y|x) = (1/Z(x)) × exp(∑ λk × fk(x,y))
其中:
  x = 输入序列
  y = 标签序列
  fk = 特征函数
  λk = 权重参数
  Z(x) = 归一化常数
```

### Transformer-based NER

```
Entity = argmax ∏ P(wi|context) × P(label|context)
```

## 详细说明

### 1. NER 任务定义

**输入**: 文本序列 $T = (t_1, t_2, ..., t_n)$
**输出**: 实体序列 $E = (e_1, e_2, ..., e_m)$，每个实体包含类型和边界

### 2. 实体类型体系

| 实体类型 | 说明 | 示例 |
|----------|------|------|
| PER | 人物 | 张三、John Smith |
| LOC | 地点 | 北京、华盛顿 |
| ORG | 组织 | 清华大学、Google |
| TIME | 时间 | 2024年3月、下午3点 |
| MONEY | 货币 | 100美元、50元 |

### 3. 技术演进

**早期方法 (统计机器学习)**
- HMM: 隐马尔可夫模型
- CRF: 条件随机场（主流）
- 特征工程驱动

**深度学习时代**
- BiLSTM-CRF: 双向LSTM + CRF层
- BERT-BiLSTM-CRF: 预训练语言模型增强
- 端到端Transformer: BERT-CRF, BERT-MRC

**最新进展**
- Multi-task Learning: 联合学习实体和关系
- Fine-grained NER: 细分实体类型
- Document-level NER: 跨句推理

### 4. 模型架构

```
输入层: 字/词嵌入
     ↓
编码层: BiLSTM / BERT / RoBERTa
     ↓
解码层: CRF / 线性层 + Softmax
     ↓
输出: 实体序列 (PER/LOC/ORG...)
```

### 5. 评估指标

```
Precision = 正确识别的实体数 / 识别出的实体总数
Recall = 正确识别的实体数 / 实际实体总数
F1 = 2 × P × R / (P + R)

严格匹配: 边界+类型完全正确
松弛匹配: 边界或类型正确即可
```
