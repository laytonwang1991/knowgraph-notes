# 文本生成 (Text Generation)

## 一句话定义

文本生成是让机器根据给定条件（ prompt、关键词、结构化数据等）自动产生符合语言规范和语义逻辑的自然语言文本。

## 核心公式

### 语言模型概率

```
P(w_1, w_2, ..., w_n) = ∏ P(w_i|w_1, ..., w_{i-1})
                      = ∏ P(w_i|context_{<i})
```

### 自回归生成

```
for t = 1 to max_length:
    P(w_t|w_{<t}) = LM(w_{<t})
    w_t ~ P(w_t|w_{<t})  # 采样
```

### 温度采样 (Temperature Sampling)

```
P'(w_i) = softmax(log P(w_i) / T)
       = exp(log P(w_i)/T) / Σ exp(log P(w_j)/T)

T > 1: 更随机、更分散
T < 1: 更确定、更尖锐
T → 0: 近似贪婪解码
```

### Top-K 采样

```
选择概率最高的K个词
在新分布中重新归一化
P'(w) = P(w) if w in Top-K else 0
```

### Top-P (Nucleus) 采样

```
选择概率累加达到P的最小词集合
重新归一化该集合内的概率
```

## 详细说明

### 1. 生成范式

| 范式 | 说明 | 示例 |
|------|------|------|
| 条件生成 | 给定输入条件 | 机器翻译、摘要 |
| 开放域生成 | 无特定输入 | 故事写作 |
| 对话生成 | 多轮交互 | 聊天机器人 |
| 受控生成 | 特定属性控制 | 情感/风格控制 |

### 2. 技术演进

**早期统计方法**
- N-gram语言模型
- 基于模板的生成
- 噪声通道模型

**神经语言模型**
- RNN语言模型 (LSTM/GRU)
- Seq2Seq + Attention
- 复制机制 (Copy Mechanism)

**预训练语言模型**
- GPT系列: 单向Transformer解码器
- BERT系列: 双向上下文编码器
- T5/BART: Encoder-Decoder统一架构

**大语言模型时代**
- GPT-3/4: 175B+ 参数，涌现能力
- LLaMA: 开源可商用
- ChatGPT/Claude: 对话优化

### 3. 生成策略

**贪婪搜索 (Greedy)**
```
w_t = argmax P(w|w_{<t})
```
- 简单但容易陷入重复

**束搜索 (Beam Search)**
```
beam_size = K
维护K条最优路径
```
- 质量较好，计算量适度

**随机采样**
```
从概率分布采样
多样性高，可能不连贯
```

### 4. 生成质量评估

| 指标 | 说明 | 适用范围 |
|------|------|----------|
| BLEU | N-gram重叠度 | 机器翻译 |
| ROUGE | 召回率导向 | 摘要 |
| METEOR | 多指标综合 | 通用 |
| Perplexity | 语言模型困惑度 | 语言质量 |
| BERTScore | 语义相似度 | 语义评估 |
| 人工评估 | 流畅性/相关性/安全性 | 终极标准 |

### 5. 关键挑战

**重复问题**
- 解码端重复: 使用n-gram惩罚
- 语义重复: 语义规划或层次解码

**长度控制**
- 长度归一化
- 目标长度惩罚
- 特殊终止符

**安全性和对齐**
- 有害内容过滤
- RLHF (人类反馈强化学习)
- Constitutional AI

### 6. 应用场景

| 场景 | 代表应用 | 模型 |
|------|----------|------|
| 内容创作 | 小说、诗歌 | GPT系列 |
| 对话系统 | 客服、个人助理 | ChatGPT |
| 代码生成 | Copilot | CodeX |
| 数据到文本 | 财报摘要 | T5 |
| 文本改写 |  paraphrase | BART |
