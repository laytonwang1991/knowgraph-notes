# 大语言模型

## 一句话定义

> 大语言模型是基于大规模文本数据训练的超大规模深度学习模型，能够理解上下文并生成连贯的人类语言。

## 核心概念

### 模型规模

| 规模 | 参数量 | 代表模型 |
|------|--------|----------|
| 小型 | < 1B | GPT-2 Small |
| 中型 | 1B - 10B | LLaMA 7B |
| 大型 | 10B - 100B | GPT-3 175B |
| 超大 | > 100B | GPT-4, Claude |

### Scaling Law

$$
L(N) = L_{\infty} + \left(\frac{N_0}{N}\right)^{\alpha}
$$

模型性能（损失 $L$）与模型参数 $N$ 的关系：
- 更多参数 + 更多数据 → 更好的性能
- 存在可预测的缩放规律

## 详细说明

### 1. 训练流程

**预训练（Pre-training）：**
```
海量互联网文本 → Token化 → 自回归语言建模
                ↓
         通用语言能力
```

**指令微调（Instruction Tuning）：**
```
预训练模型 + 指令数据 → 指令遵循能力
```

**人类反馈强化学习（RLHF）：**
```
指令模型 + 人类反馈 → 与人类偏好对齐
```

### 2. 关键技术

| 技术 | 说明 |
|------|------|
| Transformer | 基础架构 |
| Attention | 核心机制 |
| Tokenization | BPE/WordPiece |
| Flash Attention | 高效注意力计算 |
| LoRA | 低秩适配器微调 |

### 3. 能力涌现

大规模模型展现出小模型没有的能力：
- 思维链（Chain-of-Thought）
- 零样本任务迁移
- 代码生成
- 复杂推理

### 4. 代表模型

| 模型 | 公司 | 特点 |
|------|------|------|
| GPT-4 | OpenAI | 多模态，最强推理 |
| Claude 3 | Anthropic | 安全对齐，有帮助 |
| Gemini | Google | 原生多模态 |
| LLaMA | Meta | 开源可商用 |
| ChatGLM | 智谱 | 中文优化 |

## 应用场景

- 智能对话助手
- 代码生成
- 文本摘要/翻译
- 知识问答
- 创意写作
