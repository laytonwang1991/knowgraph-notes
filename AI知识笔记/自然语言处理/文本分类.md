# 文本分类 (Text Classification)

## 一句话定义

文本分类是使用算法将文本映射到预定义类别的监督学习任务，是信息组织和内容理解的基础。

## 核心公式

### 文本表示 (TF-IDF)

```
TF(t,d) = t在文档d中出现次数
IDF(t) = log(N / df(t))
TF-IDF(t,d) = TF(t,d) × IDF(t)
```

### Softmax 分类器

```
P(c|x) = exp(Wc·x + bc) / Σ exp(Wk·x + bk)
预测类别: argmax_c P(c|x)
```

### 交叉熵损失

```
L = -Σ yi × log(P(yi|xi))
其中:
  yi = 真实标签(one-hot)
  P(yi|xi) = 预测概率
```

## 详细说明

### 1. 任务类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 二分类 | 两选一 | 垃圾邮件检测 |
| 多分类 | 单标签多选 | 新闻主题分类 |
| 多标签分类 | 多标签共存 | 标签推荐 |
| 层次分类 | 类别有层级 | 电商商品分类 |

### 2. 技术发展历程

**传统机器学习方法**
- Naive Bayes: 基于贝叶斯定理，朴素独立性假设
- SVM: 支持向量机，高维稀疏特征效果好
- LR: 逻辑回归，可解释性强

**词袋模型时代**
- Bag-of-Words: 词频统计
- TF-IDF: 词频-逆文档频率加权
- N-gram: 捕获局部词序信息

**深度学习方法**
- TextCNN: 卷积神经网络捕获n-gram特征
- LSTM/GRU: 捕获长距离依赖
- Attention: 关注关键信息

**预训练模型时代**
- BERT: 双向上下文建模
- RoBERTa: 优化的BERT变体
- DeBERTa: 分离位置编码

### 3. 模型架构

```
文本 → 分词/向量化 → BERT编码 → [CLS]向量 → Linear → 类别概率
```

### 4. 关键技巧

**数据增强**
- 回译: 翻译成其他语言再翻译回来
- 同义词替换: 保持语义替换词
- 数据合成: 规则生成训练样本

**类别不平衡处理**
- 重采样: 过采样少数类/欠采样多数类
- 类别权重: 给少数类更高权重
- 阈值调整: 降低多数类阈值

### 5. 评估指标

```
准确率 (Accuracy) = 正确预测数 / 总数
精确率 (Precision) = TP / (TP + FP)
召回率 (Recall) = TP / (TP + FN)
F1 = 2 × P × R / (P + R)

Macro-F1: 每个类F1的算术平均
Micro-F1: 汇总TP/FP/FN后计算
```
