# 对话系统

## 一句话定义

对话系统是使人能够通过自然语言与机器进行交互的系统，涵盖任务完成和信息咨询两大类型。

## 核心公式

意图识别（分类问题）：

$$
P(\text{intent} | \text{utterance}) = \text{Softmax}(W \cdot \text{Enc}(\text{utterance}) + b)
$$

槽位填充（序列标注问题）：

$$
P(\text{slots} | \text{utterance}) = \text{CRF}(\text{Enc}(\text{utterance}))
$$

对话状态跟踪：

$$
B_t = \text{DST}(B_{t-1}, a_{t-1}, o_t)
$$

对话策略：

$$
a_t = \pi(B_t)
$$

## 详细说明

### 1. 对话系统类型

**任务型对话系统（Task-Oriented Dialogue）**

- 目标：帮助用户完成特定任务（订餐、订票、查询天气等）
- 核心组件：意图识别、槽位填充、对话管理、回复生成
- 特点：需要准确的结构化信息提取

**闲聊系统（Chitchat / Open-domain Dialogue）**

- 目标：提供情感陪伴和自由交流
- 技术：基于检索或生成的对话模型
- 挑战：保持连贯性、一致性、吸引力

### 2. 意图识别（Intent Detection）

意图识别是将用户话语映射到预定义意图类别的分类任务。

**常用方法**：

- 基于词袋 + 分类器（LR、SVM）
- 基于 RNN/CNN 的序列分类
- 基于预训练语言模型（BERT、RoBERTa）

**示例意图**：

```
查询天气 = {"意图": "weather_query", "槽位": {"城市": "北京", "时间": "明天"}}
播放音乐 = {"意图": "play_music", "槽位": {"歌手": "周杰伦", "歌名": "晴天"}}
```

### 3. 槽位填充（Slot Filling）

槽位填充是从用户话语中提取结构化信息（槽位值）的序列标注任务。

**标注BIO格式**：

```
北 O
京 B-city
明 B-date
天 I-date
会 O
下 B-weather
雨 I-weather
```

### 4. 对话管理（Dialogue Management）

对话管理包含三个子模块：

| 子模块 | 功能 |
|--------|------|
| 对话状态跟踪（DST） | 维护当前对话状态 |
| 对话策略（Policy） | 选择下一步动作 |
| 对话生成（NLG） | 生成自然语言回复 |

### 5. 评估指标

| 指标 | 说明 |
|------|------|
| 意图准确率 | 正确识别的意图比例 |
| 槽位 F1 | 槽位填充的 F1 分数 |
| 对话成功率 | 任务完成率 |
| BLEU | 生成回复质量 |
| AUC-ROC | 分类模型性能 |

### 意图识别（基于 BERT）

```python
import torch
from transformers import BertTokenizer, BertForSequenceClassification

model_name = "bert-base-chinese"
tokenizer = BertTokenizer.from_pretrained(model_name)
model = BertForSequenceClassification.from_pretrained(
    model_name,
    num_labels=len(intent_labels)
)

def predict_intent(utterance):
    inputs = tokenizer(
        utterance,
        return_tensors="pt",
        padding=True,
        truncation=True,
        max_length=128
    )
    with torch.no_grad():
        outputs = model(**inputs)
        probs = torch.softmax(outputs.logits, dim=-1)
        pred = torch.argmax(probs, dim=-1).item()
    return intent_labels[pred], probs[0][pred].item()
```

### 槽位填充（基于 CRF）

```python
import torch
from torchcrf import CRF

class SlotFillingModel(torch.nn.Module):
    def __init__(self, vocab_size, embed_dim, hidden_dim, num_tags):
        super().__init__()
        self.embedding = torch.nn.Embedding(vocab_size, embed_dim)
        self.lstm = torch.nn.LSTM(embed_dim, hidden_dim, batch_first=True, bidirectional=True)
        self.classifier = torch.nn.Linear(hidden_dim * 2, num_tags)
        self.crf = CRF(num_tags, batch_first=True)

    def forward(self, tokens, tags=None):
        mask = (tokens != 0)
        emissions = self.classifier(self.lstm(self.embedding(tokens))[0])

        if tags is not None:
            loss = -self.crf(emissions, tags, mask=mask)
            return loss
        else:
            return self.crf.decode(emissions, mask=mask)
```

### 对话状态跟踪

```python
class DialogueStateTracker:
    def __init__(self, slots):
        self.slots = slots
        self.state = {slot: None for slot in slots}

    def update(self, intent, slots_filled):
        """更新对话状态"""
        for slot, value in slots_filled.items():
            if slot in self.state:
                self.state[slot] = value
        return self.state

    def is_complete(self):
        """检查所有必需槽位是否已填充"""
        return all(v is not None for v in self.state.values())
```

## 应用场景

1. **智能客服**：自动回答用户咨询，减少人工成本
2. **语音助手**：Siri、小爱同学、天猫精灵等
3. **智能导诊**：医疗领域的症状询问和分诊
4. **在线教育**：AI 口语陪练和答疑
5. **智能家居控制**：通过对话控制设备

## 相关概念

- **检索式对话**：从预定义回复库中匹配最合适的回复
- **生成式对话**：使用序列到序列模型生成回复
- **意图识别**：判断用户想要完成的操作类型
- **槽位填充**：从用户话语中提取关键信息
- **对话管理**：维护对话状态并决策下一步动作
