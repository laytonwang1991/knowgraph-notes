# 自然语言处理基础

## 一句话定义

自然语言处理（Natural Language Processing, NLP）是计算机科学、人工智能与语言学的交叉领域，旨在让计算机理解、生成和处理人类自然语言。

## 核心公式

### 1. 序列标注特征函数

$$h(x, y) = \sum_{i=1}^{n} f(x, y, i)$$

其中 $f$ 为特征函数，$x$ 为输入序列，$y$ 为输出标签序列。

### 2. 条件概率公式

$$P(y|x) = \frac{P(x|y)P(y)}{P(x)}$$

用于基于概率的序列标注和分类任务。

## 详细说明

### 1. NLP的定义

自然语言处理是让计算机与人类语言进行交互的技术。它涵盖了从文本预处理到语义理解的完整技术栈。

**核心目标：**
- 让计算机理解人类语言的含义
- 让计算机生成符合人类习惯的文本
- 实现人机之间流畅的语言交互

### 2. 任务分类

#### 2.1 文本预处理任务
- **分词（Tokenization）**：将文本分割成词或子词单元
- **词性标注（POS Tagging）**：为每个词标注语法类别
- **命名实体识别（NER）**：识别人名、地名、机构名等实体
- **句法分析（Parsing）**：分析句子的语法结构

#### 2.2 文本理解任务
- **情感分析（Sentiment Analysis）**：判断文本的情感倾向
- **文本分类（Text Classification）**：将文本归类到预定义类别
- **语义相似度（Semantic Similarity）**：计算文本间的语义距离
- **问答系统（QA）**：从文本中提取问题答案

#### 2.3 文本生成任务
- **机器翻译（Machine Translation）**：将一种语言转换为另一种语言
- **文本摘要（Summarization）**：生成文本的简短概括
- **对话系统（Dialogue System）**：进行多轮自然语言对话
- **文本生成（Text Generation）**：根据条件生成连贯文本

### 3. 发展历程

#### 3.1 萌芽期（1950s-1980s）
- 基于规则的方法主导
- 符号主义人工智能
- 代表系统：ELIZA、SHRDLU

#### 3.2 统计学习期（1990s-2012）
- 统计机器学习方法兴起
- HMM、CRF等概率模型广泛应用
- 语料库语言学快速发展

#### 3.3 深度学习期（2013-2017）
- Word2Vec等词嵌入技术出现
- RNN/LSTM用于序列建模
- 注意力机制开始应用

#### 3.4 大模型时代（2018-至今）
- BERT、GPT等预训练模型
- Transformer架构统一建模
- 通用人工智能成为可能

### 4. 核心技术详解

#### 4.1 分词（Tokenization）

**定义：** 将连续文本分割成离散的语言单元。

**主要算法：**
```
# 字典匹配分词
def dictionary_match(text, dictionary):
    result = []
    i = 0
    while i < len(text):
        matched = False
        for j in range(len(text), i, -1):
            if text[i:j] in dictionary:
                result.append(text[i:j])
                i = j
                matched = True
                break
        if not matched:
            result.append(text[i])
            i += 1
    return result

# 正向最大匹配
def forward_max_match(text, dictionary, max_len=5):
    result = []
    i = 0
    while i < len(text):
        matched = False
        for j in range(min(i + max_len, len(text) + 1), i, -1):
            if text[i:j] in dictionary:
                result.append(text[i:j])
                i = j
                matched = True
                break
        if not matched:
            result.append(text[i])
            i += 1
    return result
```

**分词工具：** jieba、HanLP、LTP、SnowNLP

#### 4.2 词性标注（POS Tagging）

**定义：** 为句子中每个词标注其语法类别。

**常用词性标签集（宾州树库）：**
| 标签 | 含义 | 示例 |
|------|------|------|
| NN | 名词单数 | cat |
| NNS | 名词复数 | cats |
| VB | 动词原形 | run |
| VBD | 过去式 | ran |
| JJ | 形容词 | big |

**代码示例：**
```python
import jieba.posseg as pseg

text = "我爱自然语言处理技术"
words = pseg.cut(text)

for word, flag in words:
    print(f"{word}: {flag}")

# 输出:
# 我: r
# 爱: v
# 自然语言处理: nz
# 技术: n
```

#### 4.3 句法分析（Parsing）

**定义：** 分析句子的语法结构，生成语法树。

**两种主要方法：**

1. **短语结构分析**
   - 递归神经网络语法
   - Context-Free Grammar (CFG)

2. **依存结构分析**
   - 揭示词与词之间的依存关系
   - 适合中文等语言

```python
# 依存句法分析示例（使用spaCy）
import spacy

nlp = spacy.load("en_core_web_sm")
doc = nlp("The cat sits on the mat")

for token in doc:
    print(f"{token.text}: {token.dep_} <- {token.head.text}")
```

#### 4.4 语义分析（Semantic Analysis）

**定义：** 理解文本的深层含义。

**主要技术：**
- 词义消歧（WSD）
- 语义角色标注（SRL）
- 命名实体语义
- 分布式语义表示

```python
# 词义消歧示例
from nltk.corpus import wordnet as wn

# 获取"bank"的所有语义
bank_synsets = wn.synsets('bank')
for synset in bank_synsets:
    print(f"{synset.name()}: {synset.definition()}")
```
