# 语言模型

## 一句话定义

语言模型（Language Model, LM）是计算文本序列概率分布的模型，用于预测下一个词或评估文本的流畅度与合理性。

## 核心公式

### 1. 链式法则

$$P(w_1, w_2, ..., w_n) = P(w_1) \cdot P(w_2|w_1) \cdot P(w_3|w_1, w_2) \cdot ... \cdot P(w_n|w_1, ..., w_{n-1})$$

### 2. N-gram 模型概率

$$P(w_i|w_{i-n+1}, ..., w_{i-1}) = \frac{C(w_{i-n+1}, ..., w_i)}{C(w_{i-n+1}, ..., w_{i-1})}$$

### 3. 困惑度（Perplexity）

$$\text{PP}(W) = \sqrt[n]{\frac{1}{P(w_1, w_2, ..., w_n)}} = \left( \prod_{i=1}^{n} \frac{1}{P(w_i|w_1, ..., w_{i-1})} \right)^{\frac{1}{n}}$$

### 4. 交叉熵损失

$$H(T) = -\frac{1}{N} \sum_{i=1}^{N} \log_2 P(w_i|w_1, ..., w_{i-1})$$

## 详细说明

### 1. 统计语言模型

#### 1.1 什么是统计语言模型

统计语言模型使用概率统计方法来描述语言规律，通过计算词序列的联合概率来评估文本的可能性。

**核心思想：** 使用前 n-1 个词预测第 n 个词的概率。

#### 1.2 N-gram 模型

**定义：** 假设当前词只与前 n-1 个词相关。

**常见N-gram：**
- **Unigram (n=1)**：每个词独立，不考虑上下文
- **Bigram (n=2)**：只考虑前一个词
- **Trigram (n=3)**：考虑前两个词

```python
# N-gram 语言模型实现
from collections import defaultdict
import re

class NGramLM:
    def __init__(self, n):
        self.n = n
        self.ngrams = defaultdict(int)  # n-gram计数
        self.contexts = defaultdict(int)  # 上下文计数

    def tokenize(self, text):
        """简单分词"""
        return re.findall(r'\w+', text.lower())

    def train(self, corpus):
        """训练模型"""
        for sentence in corpus:
            tokens = ['<BOS>'] * (self.n - 1) + self.tokenize(sentence) + ['<EOS>']

            for i in range(len(tokens) - self.n + 1):
                context = tuple(tokens[i:i + self.n - 1])
                word = tokens[i + self.n - 1]

                self.ngrams[(context, word)] += 1
                self.contexts[context] += 1

    def probability(self, context, word):
        """计算条件概率 P(word|context)"""
        return self.ngrams.get((context, word), 0) / max(self.contexts[context], 1)

    def perplexity(self, sentence):
        """计算困惑度"""
        tokens = ['<BOS>'] * (self.n - 1) + self.tokenize(sentence) + ['<EOS>']
        log_prob = 0

        for i in range(len(tokens) - self.n + 1):
            context = tuple(tokens[i:i + self.n - 1])
            word = tokens[i + self.n - 1]
            prob = self.probability(context, word)
            log_prob += math.log(prob + 1e-10)

        return math.exp(-log_prob / (len(tokens) - self.n + 1))
```

#### 1.3 平滑技术

**问题：** 零概率问题，未登录词的计数为0。

**解决方案：**
- 加法平滑（Add-k Smoothing）
- Good-Turing 估计
- Kneser-Ney 平滑
- 插值平滑（Interpolated Smoothing）

```python
# 加法平滑实现
def add_k_smoothed_prob(ngram_count, context_count, vocab_size, k=1):
    """加法平滑"""
    return (ngram_count + k) / (context_count + k * vocab_size)
```

### 2. 神经语言模型

#### 2.1 词嵌入 + 神经网络

**核心思想：** 用分布式表示（词向量）表示词语，用神经网络学习上下文关系。

**架构：**
```
输入层 -> 词嵌入层 -> 隐藏层 -> 输出层
```

```python
# 简单神经语言模型（使用PyTorch）
import torch
import torch.nn as nn

class NeuralLM(nn.Module):
    def __init__(self, vocab_size, embed_dim, hidden_dim):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim)
        self.rnn = nn.LSTM(embed_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, vocab_size)

    def forward(self, x):
        # x: (batch_size, seq_len)
        embedded = self.embedding(x)  # (batch_size, seq_len, embed_dim)
        output, _ = self.rnn(embedded)  # (batch_size, seq_len, hidden_dim)
        logits = self.fc(output)  # (batch_size, seq_len, vocab_size)
        return logits
```

#### 2.2 RNN/LSTM 语言模型

**优势：** 能够处理任意长度的序列，捕捉长距离依赖。

```python
# 训练循环示例
def train_epoch(model, dataloader, optimizer, criterion):
    model.train()
    total_loss = 0

    for batch in dataloader:
        inputs, targets = batch  # inputs: (batch, seq_len), targets: (batch, seq_len)

        optimizer.zero_grad()
        outputs = model(inputs)  # (batch, seq_len, vocab_size)

        # 计算损失
        loss = criterion(outputs.view(-1, vocab_size), targets.view(-1))
        loss.backward()
        optimizer.step()

        total_loss += loss.item()

    return total_loss / len(dataloader)
```

#### 2.3 Transformer 语言模型

**核心组件：**
- 自注意力机制（Self-Attention）
- 多头注意力（Multi-Head Attention）
- 位置编码（Positional Encoding）
- 前馈神经网络（FFN）

```python
# Transformer语言模型核心实现
import torch
import torch.nn as nn
import math

class SelfAttention(nn.Module):
    def __init__(self, embed_dim, num_heads):
        super().__init__()
        self.embed_dim = embed_dim
        self.num_heads = num_heads
        self.head_dim = embed_dim // num_heads

        self.qkv = nn.Linear(embed_dim, embed_dim * 3)
        self.proj = nn.Linear(embed_dim, embed_dim)

    def forward(self, x, mask=None):
        B, T, C = x.shape

        # 生成 Q, K, V
        qkv = self.qkv(x).reshape(B, T, 3, self.num_heads, self.head_dim)
        q, k, v = qkv.unbind(2)

        # 缩放点积注意力
        scale = math.sqrt(self.head_dim)
        attn = (q @ k.transpose(-2, -1)) / scale

        if mask is not None:
            attn = attn.masked_fill(mask == 0, float('-inf'))

        attn = torch.softmax(attn, dim=-1)
        out = attn @ v

        return self.proj(out.reshape(B, T, C))
```

### 3. BPE 算法

#### 3.1 什么是 BPE

Byte Pair Encoding (BPE) 是一种基于字节对编码的分词算法，常用于大语言模型。

**核心思想：** 迭代合并最高频的相邻字符对，构建子词词表。

```python
# BPE 算法实现
from collections import Counter

def get_stats(vocab):
    """统计字符对频率"""
    pairs = Counter()
    for word, freq in vocab.items():
        symbols = word.split()
        for i in range(len(symbols) - 1):
            pairs[(symbols[i], symbols[i+1])] += freq
    return pairs

def merge_vocab(best_pair, vocab):
    """合并最高频字符对"""
    new_vocab = {}
    bigram = ' '.join(best_pair)
    replacement = ''.join(best_pair)

    for word, freq in vocab.items():
        new_word = word.replace(bigram, replacement)
        new_vocab[new_word] = freq

    return new_vocab

def learn_bpe(corpus, num_merges):
    """学习BPE"""
    # 初始化词表（每个字符为一个token）
    vocab = Counter()
    for word in corpus:
        vocab[' '.join(list(word)) + ' </w>'] = 1

    for i in range(num_merges):
        pairs = get_stats(vocab)
        if not pairs:
            break
        best_pair = max(pairs, key=pairs.get)
        vocab = merge_vocab(best_pair, vocab)
        print(f"Merge {i+1}: {best_pair}")

    return vocab

# 示例
corpus = ["low", "lowest", "newer", "wider"]
vocab = learn_bpe(corpus, 10)
```

#### 3.2 BPE 的优势

1. **OOV处理：** 将未登录词分解为已知子词
2. **词表大小可控：** 通过合并次数限制词表大小
3. **平衡粒度：** 介于字符和词之间

### 4. 困惑度评估

#### 4.1 定义

困惑度（Perplexity）是语言模型的标准评估指标，表示模型对测试数据的平均分支因子。

**解释：**
- 低困惑度 = 模型预测更确定 = 更好的模型
- 困惑度 = e^熵

#### 4.2 计算示例

```python
import math

def calculate_perplexity(model, test_data, tokenizer):
    """计算语言模型的困惑度"""
    model.eval()
    total_log_prob = 0
    total_tokens = 0

    with torch.no_grad():
        for text in test_data:
            tokens = tokenizer.encode(text)
            input_ids = torch.tensor(tokens[:-1]).unsqueeze(0)
            targets = torch.tensor(tokens[1:])

            outputs = model(input_ids)
            logits = outputs.logits[0]

            loss = nn.functional.cross_entropy(logits, targets)
            total_log_prob += loss.item() * len(tokens)
            total_tokens += len(tokens)

    perplexity = math.exp(total_log_prob / total_tokens)
    return perplexity
```
