# 分词器（Tokenizer）

## 一句话定义

分词器是将原始文本字符串转换为模型可处理的整数Token序列的核心组件，是NLP管道的第一个环节。

## 核心公式

$$T = \text{Tokenizer}(text)$$

$$T = [t_1, t_2, t_3, ..., t_n]$$

其中每个 $t_i$ 为词表中的整数索引。

## 详细说明

### 1. 文本到Token的转换流程

```
原始文本 → 归一化 → 分词 → Token映射 → ID序列
```

1. **归一化（Normalization）**：小写化、去除重音符号、Unicode规范化（NFKC）
2. **预分词（Pre-tokenization）**：按空格、标点进行初步切分
3. **分词算法**：应用BPE/WordPiece/Unigram等算法进行子词切分
4. **ID映射**：将子词映射为词表中的整数ID

### 2. BPE（Byte Pair Encoding）

BPE是一种基于频率的子词分词算法，源自数据压缩领域。

**训练过程：**
- 统计所有相邻字节对的出现频率
- 迭代合并最高频的字节对，生成新的符号
- 重复直到达到预设词表大小

**优点：**
- 简单高效，易于实现
- 能处理任意未知词
- 词表大小可精确控制

**缺点：**
- 贪心算法可能不是全局最优
- 无法捕捉语言的全部语义信息

### 3. WordPiece

Google在BERT中使用的分词算法，基于语言模型似然度进行合并。

**训练过程：**
- 初始化词表为所有字符
- 训练一个语言模型
- 选择使语言模型似然增益最大的相邻符号对进行合并
- 重复直到达到预设大小

**特点：**
- 合并依据是语言模型得分，而非纯频率
- 更倾向于保留有意义的语义单元

### 4. SentencePiece

Google提出的端到端分词框架，支持无监督训练。

**与BPE/WordPiece的区别：**
- 直接在原始文本上训练，不需要预分词
- 将空格也当作一种字符处理（空格编码为 ▁）
- 支持多种分词算法（BPE、Unigram、字符、词）

**优势：**
- 语言无关，适用于任何Unicode文本
- 训练和推理完全一致

### 5. Unigram Language Model

SentencePiece默认使用的算法，基于语言模型进行概率分词。

**训练过程：**
- 初始化大词表（可以用所有子词）
- 迭代优化词表，每次移除对似然贡献最小的子词
- 最终保留使似然最大化的词表

### 6. 特殊Token

| Token | 含义 | 用途 |
|-------|------|------|
| `[PAD]` | 填充 | 批量处理时补齐序列长度 |
| `[UNK]` | 未知词 | 未在词表中的字符 |
| `[CLS]` | 分类 | 分类任务的序列起始标记 |
| `[SEP]` | 分隔 | 序列分隔或多句子任务 |
| `[MASK]` | 掩码 | 掩码语言模型的掩码标记 |
| `<s>` | 起始 | GPT系列模型使用 |
| `</s>` | 结束 | 序列结束标记 |

## 代码示例

### Python - BPE分词器实现

```python
from collections import Counter

class BPETokenizer:
    def __init__(self, vocab_size=1000):
        self.vocab_size = vocab_size
        self.word_freq = Counter()
        self.merges = []
        self.vocab = {}

    def get_stats(self, words):
        """统计所有字节对的频率"""
        pairs = Counter()
        for word, freq in words.items():
            symbols = word.split()
            for i in range(len(symbols) - 1):
                pairs[(symbols[i], symbols[i + 1])] += freq
        return pairs

    def merge_vocab(self, best_pair, words):
        """合并最高频的字节对"""
        pattern = re.compile(re.escape(best_pair[0] + ' ' + best_pair[1]))
        new_words = {}
        for word, freq in words.items():
            new_word = pattern.sub(best_pair[0] + best_pair[1], word)
            new_words[new_word] = freq
        return new_words

    def train(self, corpus):
        """训练BPE"""
        # 初始化：将单词拆分为字符
        words = {}
        for word in corpus:
            words[' '.join(list(word)) + ' </w>'] = words.get(' '.join(list(word)) + ' </w>', 0) + 1

        self.vocab = {chr(i): i for i in range(256)}
        self.vocab['</w>'] = 256

        for i in range(self.vocab_size - 257):
            pairs = self.get_stats(words)
            if not pairs:
                break
            best_pair = max(pairs, key=pieces.get)
            words = self.merge_vocab(best_pair, words)
            self.merges.append(best_pair)
            self.vocab[best_pair[0] + best_pair[1]] = len(self.vocab)

    def encode(self, text):
        """编码文本为Token序列"""
        tokens = list(text) + ['</w>']
        for merge in self.merges:
            for i in range(len(tokens)):
                if i < len(tokens) - 1:
                    if tokens[i] == merge[0] and tokens[i + 1] == merge[1]:
                        tokens = tokens[:i] + [merge[0] + merge[1]] + tokens[i + 2:]
        return [self.vocab.get(t, self.vocab['[UNK]']) for t in tokens]
```

### Hugging Face Transformers 使用示例

```python
from transformers import AutoTokenizer

# 加载预训练分词器
tokenizer = AutoTokenizer.from_pretrained("bert-base-chinese")

# 编码文本
text = "今天天气真好"
tokens = tokenizer.encode(text, add_special_tokens=True)

# 解码
decoded = tokenizer.decode(tokens)

print(f"原始文本: {text}")
print(f"Token IDs: {tokens}")
print(f"分词结果: {tokenizer.convert_ids_to_tokens(tokens)}")
print(f"解码文本: {decoded}")
```

## 应用场景

| 场景 | 适用分词器 | 原因 |
|------|-----------|------|
| 中文NLP | Jieba / BERT分词器 | 中文字词边界不明显 |
| 多语言模型 | SentencePiece | 语言无关，支持Unicode |
| 机器翻译 | BPE / WordPiece | 跨语言子词对齐 |
| 代码生成 | Byte-level BPE | 处理任意未知字符 |
| 对话系统 | FastBPE / HuggingFace | 高效批量处理 |

## 相关概念

- **词表（Vocabulary）**：分词器能够识别的所有Token的集合
- **OOV（Out-of-Vocabulary）**：超出词表的词，通常映射为 `[UNK]`
- **子词分词（Subword Tokenization）**：将词进一步拆分为子词单元
- **序列填充（Padding）**：将不同长度序列补齐到相同长度
- **截断（Truncation）**：将超长序列截断到最大长度
