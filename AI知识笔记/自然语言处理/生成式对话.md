# 生成式对话系统

## 一句话定义

生成式对话系统使用序列到序列模型根据用户输入直接生成回复文本，而非从预定义库中检索。

## 核心公式

**Seq2Seq 概率模型**：

$$
P(y_1, y_2, \ldots, y_m | x_1, x_2, \ldots, x_n) = \prod_{t=1}^{m} P(y_t | y_{1:t-1}, x_{1:n})
$$

**注意力机制**：

$$
a_{ij} = \frac{\exp(e_{ij})}{\sum_{k=1}^{n} \exp(e_{ik})}, \quad e_{ij} = v^\top \tanh(W_h h_i + W_s s_j)
$$

$$
c_j = \sum_{i=1}^{n} a_{ij} h_i
$$

**人格嵌入**：

$$
\mathbf{p} = \text{Emb}(\text{persona\_id}), \quad P(y_t) = \text{Decoder}(s_{t-1}, y_{t-1}, c_t, \mathbf{p})
$$

**PPL（困惑度）**：

$$
\text{PPL} = \exp\left(-\frac{1}{T} \sum_{t=1}^{T} \log P(y_t | y_{<t})\right)
$$

## 详细说明

### 1. Seq2Seq 对话模型

**经典架构**（Encoder-Decoder）：

```
用户输入 → Encoder → Context Vector → Decoder → 回复
```

- **Encoder**：RNN/LSTM/GRU/Transformer 将输入编码为上下文向量
- **Decoder**：基于上下文向量和已生成词逐步生成回复
- **注意力机制**：允许 Decoder 关注输入的不同部分

**问题**：

- 通用回复（"我不知道"、"好的"）高频出现
- 缺乏一致性（人格、知识、情感）
- 曝光偏差（Exposure Bias）

### 2. 人格建模（Personality Modeling）

让对话系统具有一致的人设，提升真实感：

| 方法 | 说明 |
|------|------|
| 人格嵌入 | 为每个 persona 学习独立嵌入向量 |
| 人格描述编码 | 将 persona 描述文本编码后融入 decoder |
| 多任务学习 | 同时学习人格分类和对话生成 |
| 对话风格迁移 | 根据目标人格调整输出风格 |

**Persona-Chat 示例**：

```
Persona: 我是一名35岁的软件工程师。
         我喜欢科幻电影和户外运动。
         我有一只名叫Max的狗。

用户: 你周末一般做什么?
回复: 我通常会带Max去郊外徒步，偶尔看看科幻电影放松一下。
```

### 3. 一致性控制

**问题类型**：

- **人格一致**：前后回复的人设矛盾
- **知识一致**：与事实不符的错误
- **情感一致**：情绪波动不合理
- **逻辑一致**：前后文推理矛盾

**解决方法**：

| 技术 | 说明 |
|------|------|
| Knowledge-Grounded | 引入外部知识库 |
| Memory Network | 使用记忆网络存储上下文 |
| Consistency Loss | 引入一致性正则化 |
| Self-Check | 生成后用判别器检查一致性 |
| LLM Prompt | 使用大模型 + Prompt 控制 |

### 4. 生成策略

| 策略 | 特点 |
|------|------|
| Greedy Search | 每次选最高概率词，效率高但质量低 |
| Beam Search | 维护多个候选序列，平衡质量与效率 |
| Sampling | 根据概率分布随机采样，多样性高 |
| Temperature | 控制采样分布的平滑度 |
| Top-K Sampling | 只从最高K个词中采样 |
| Nucleus Sampling | 从累积概率达P的词中采样 |

### 5. 评估指标

| 指标 | 说明 |
|------|------|
| BLEU | n-gram 重叠度 |
| METEOR | 多维度相似度 |
| ROUGE-L | 最长公共子序列 |
| perplexity (PPL) | 语言模型困惑度 |
| DISTINCT | 生成多样性 |
| F1 (Intent/Slot) | 任务型对话专用 |
| 人工评估 | 流畅性、相关性、一致性 |

## 代码示例

### Seq2Seq 对话模型

```python
import torch
import torch.nn as nn
from transformers import BertModel, BertTokenizer

class Seq2SeqDialogue(nn.Module):
    def __init__(self, vocab_size, embed_dim=768, hidden_dim=768):
        super().__init__()
        self.bert = BertModel.from_pretrained("bert-base-chinese")
        self.embed_dim = embed_dim
        # Decoder 使用与 Encoder 相同的嵌入层，但独立处理
        self.decoder_embedding = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, batch_first=True, bidirectional=False)
        self.classifier = nn.Linear(hidden_dim, vocab_size)

    def encode(self, input_ids, attention_mask):
        outputs = self.bert(input_ids=input_ids, attention_mask=attention_mask)
        return outputs.last_hidden_state  # [B, T, H]

    def decode(self, encoder_outputs, target_ids, target_mask):
        embedded = self.decoder_embedding(target_ids)  # [B, T, H]
        outputs, _ = self.lstm(embedded)  # [B, T, H]
        logits = self.classifier(outputs)  # [B, T, V]
        return logits

    def forward(self, input_ids, attention_mask, target_ids, target_mask):
        encoder_outputs = self.encode(input_ids, attention_mask)
        logits = self.decode(encoder_outputs, target_ids, target_mask)
        return logits
```

### 人格嵌入模型

```python
class PersonaEmbedding(nn.Module):
    def __init__(self, num_personas, embed_dim=256):
        super().__init__()
        self.persona_embedding = nn.Embedding(num_personas, embed_dim)
        self.vocab_embed_dim = 768  # BERT hidden size
        # 投影层将 persona embedding 映射到 decoder 空间
        self.projection = nn.Linear(embed_dim, self.vocab_embed_dim)

    def forward(self, persona_ids):
        """获取人格嵌入并投影"""
        persona_embed = self.persona_embedding(persona_ids)  # [B, E]
        persona_embed = self.projection(persona_embed)  # [B, H]
        return persona_embed

class PersonaSeq2Seq(nn.Module):
    def __init__(self, num_personas, vocab_size):
        super().__init__()
        self.persona_emb = PersonaEmbedding(num_personas, embed_dim=256)
        self.seq2seq = Seq2SeqDialogue(vocab_size)

    def forward(self, input_ids, attention_mask, target_ids, target_mask, persona_ids):
        # 获取人格嵌入
        persona_embed = self.persona_emb(persona_ids)  # [B, H]
        # 在 decoder 中融入人格信息（简化处理：加到嵌入上）
        # 实际中可能更复杂地融合
        encoder_outputs = self.seq2seq.encode(input_ids, attention_mask)
        logits = self.seq2seq.decode(encoder_outputs, target_ids, target_mask)
        return logits
```

### 生成策略

```python
def generate_with_beam_search(model, encoder_outputs, tokenizer,
                                beam_size=5, max_len=50, start_token_id=101, end_token_id=102):
    """Beam Search 生成"""
    vocab_size = tokenizer.vocab_size
    device = encoder_outputs.device

    # 初始化：beam列表，每个元素(序列, 分数, 隐藏状态)
    beams = [(torch.tensor([[start_token_id]], device=device),
              0.0,
              None)]

    completed = []

    for step in range(max_len):
        all_candidates = []

        for beam in beams:
            seq, score, _ = beam
            if seq[0, -1].item() == end_token_id:
                completed.append(beam)
                continue

            # 解码一步
            target = seq
            logits = model.decode(encoder_outputs, target, None)  # [1, T, V]
            log_probs = torch.log_softmax(logits[:, -1, :], dim=-1)  # [1, V]

            # 取 top-k
            top_log_probs, top_ids = log_probs.topk(beam_size)

            for k in range(beam_size):
                cand_seq = torch.cat([seq, top_ids[:, k:k+1]], dim=-1)
                cand_score = score + top_log_probs[0, k].item()
                all_candidates.append((cand_seq, cand_score, None))

        # 排序并保留 top beams
        ordered = sorted(all_candidates, key=lambda x: x[1], reverse=True)
        beams = ordered[:beam_size]

        if not beams:
            break

    # 合并完成和未完成的序列
    all_seqs = completed + beams
    best_seq = max(all_seqs, key=lambda x: x[1] / len(x[0][0]))[0]
    return tokenizer.decode(best_seq[0].tolist(), skip_special_tokens=True)


def generate_with_nucleus_sampling(model, encoder_outputs, tokenizer,
                                     temperature=0.8, top_p=0.9, max_len=50):
    """Nucleus Sampling (Top-P Sampling)"""
    device = encoder_outputs.device
    generated = torch.tensor([[tokenizer.cls_token_id]], device=device)

    for _ in range(max_len):
        logits = model.decode(encoder_outputs, generated, None)
        log_probs = torch.log_softmax(logits[:, -1, :] / temperature, dim=-1)

        # Nucleus Sampling
        sorted_probs, sorted_indices = torch.sort(log_probs, descending=True)
        cumulative_probs = torch.cumsum(sorted_probs, dim=-1)

        # 保留累积概率达到 top_p 的词
        nucleus_indices = sorted_indices[:, cumulative_probs[0] <= top_p]
        if nucleus_indices.shape[1] == 0:
            nucleus_indices = sorted_indices[:, :1]

        # 从 nucleus 中采样
        probs = torch.gather(log_probs, 1, nucleus_indices)
        probs = torch.softmax(probs, dim=-1)
        next_token_idx = torch.multinomial(probs, 1)
        next_token = nucleus_indices[:, next_token_idx.item()]

        generated = torch.cat([generated, next_token], dim=-1)

        if next_token.item() == tokenizer.sep_token_id:
            break

    return tokenizer.decode(generated[0].tolist(), skip_special_tokens=True)
```

## 应用场景

1. **开放域聊天**：情感陪伴、无目的性交流
2. **创意写作助手**：故事续写、角色扮演
3. **虚拟人物**：游戏NPC、虚拟偶像
4. **教育辅助**：口语陪练、作文批改
5. **心理辅导**：倾听和情绪疏导

## 相关概念

- **检索式对话**：从预定义库检索回复
- **Seq2Seq**：编码器-解码器架构
- **人格建模**：让对话具有一致人设
- **注意力机制**：动态关注输入的不同部分
- **大语言模型**：如 GPT 系列，用于零样本对话生成
