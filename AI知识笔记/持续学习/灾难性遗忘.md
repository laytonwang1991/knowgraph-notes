# 灾难性遗忘（Catastrophic Forgetting）

## 一句话定义

灾难性遗忘是指神经网络在学习新任务时，由于权重被大规模更新而导致先前习得的知识被覆盖或丢失的现象。

## 核心公式

神经网络的权重更新可表示为：

$$
\theta_{t+1} = \theta_t - \eta \nabla L_{new}(\theta_t)
$$

当使用随机梯度下降（SGD）优化新任务损失 $L_{new}$ 时，权重 $\theta_t$ 会朝着最小化新任务的方向大幅调整，导致旧任务性能急剧下降。

## 详细说明

### 1. 现象成因

- **权重覆盖**：新任务的梯度更新直接覆盖了旧任务所需的权重
- **Representation Drift**：共享表示层在训练中逐渐偏向新任务
- **Context Interference**：不同任务的样本在训练过程中相互干扰
- **Capacity Saturation**：网络容量被新任务占满，旧知识无处存储

### 2. 解决方案分类

#### 2.1 弹性权重固定（Elastic Weight Consolidation, EWC）

EWC 通过对重要权重施加正则化约束来保护旧知识：

$$
L_{total} = L_{new}(\theta) + \frac{\lambda}{2} \sum_i \Omega_i (\theta_i - \theta_i^*)^2
$$

其中 $\Omega_i$ 是权重重要性度量，$\theta_i^*$ 是旧任务的最优权重，$\lambda$ 是正则化强度。

**代码示例（PyTorch）**：
```python
import torch
import torch.nn as nn

class EWC:
    def __init__(self, model, dataloader, device):
        self.model = model
        self.device = device
        self.params = {n: p.clone().detach() for n, p in model.named_parameters() if p.requires_grad}
        self.fisher = {n: torch.zeros_like(p) for n, p in model.named_parameters() if p.requires_grad}
        self._compute_fisher(dataloader)

    def _compute_fisher(self, dataloader):
        """计算 Fisher Information Matrix"""
        self.model.eval()
        for batch in dataloader:
            inputs, _ = batch
            inputs = inputs.to(self.device)
            self.model.zero_grad()
            output = self.model(inputs)
            loss = output.mean()
            loss.backward()
            for n, p in self.model.named_parameters():
                if p.requires_grad:
                    self.fisher[n] += p.grad.data ** 2 / len(dataloader)

    def penalty(self):
        """计算 EWC 惩罚项"""
        loss = 0
        for n, p in self.model.named_parameters():
            if p.requires_grad:
                loss += (self.fisher[n] * (p - self.params[n]) ** 2).sum()
        return loss * 0.5

def ewc_loss(loss, ewc, lambda_ewc):
    return loss + lambda_ewc * ewc.penalty()
```

#### 2.2 经验回放（Experience Replay）

保存一小部分旧任务样本，在训练新任务时混合回放：

```python
class ReplayBuffer:
    def __init__(self, buffer_size=1000):
        self.buffer_size = buffer_size
        self.buffer = []

    def add(self, task_id, state, action, reward, next_state, done):
        self.buffer.append((task_id, state, action, reward, next_state, done))
        if len(self.buffer) > self.buffer_size:
            self.buffer.pop(0)

    def sample(self, batch_size):
        batch = random.sample(self.buffer, min(batch_size, len(self.buffer)))
        task_ids, states, actions, rewards, next_states, dones = zip(*batch)
        return task_ids, states, actions, rewards, next_states, dones

    def sample_by_task(self, task_id, batch_size):
        task_samples = [s for s in self.buffer if s[0] == task_id]
        return random.sample(task_samples, min(batch_size, len(task_samples)))
```

#### 2.3 渐进式神经网络（Progressive Neural Networks）

为每个新任务添加独立的网络块，同时保留旧网络：

```python
class ProgressiveNetwork(nn.Module):
    def __init__(self, input_dim, task_configs):
        super().__init__()
        self.task_columns = nn.ModuleList()
        # 第一个任务：标准网络
        self.task_columns.append(self._build_column(input_dim, task_configs[0]))
        # 后续任务：连接所有先前列的输出
        for i, config in enumerate(task_configs[1:], 1):
            prev_dims = [task_configs[j]['hidden_dims'][-1] for j in range(i)]
            self.task_columns.append(self._build_column(input_dim + sum(prev_dims), config))

    def _build_column(self, input_dim, config):
        layers = []
        dims = [input_dim] + config['hidden_dims'] + [config['output_dim']]
        for i in range(len(dims) - 1):
            layers.extend([
                nn.Linear(dims[i], dims[i+1]),
                nn.ReLU() if i < len(dims) - 2 else nn.Identity()
            ])
        return nn.Sequential(*layers)

    def forward(self, x, task_id):
        outputs = []
        for i in range(task_id + 1):
            if i == 0:
                outputs.append(self.task_columns[i](x))
            else:
                outputs.append(self.task_columns[i](torch.cat([x] + outputs[-i:], dim=-1)))
        return outputs[task_id]
```

## 应用场景

| 场景 | 说明 |
|------|------|
| 机器人持续技能学习 | 机器人逐个学习新动作同时保留已学技能 |
| 推荐系统 | 模型持续学习用户新行为而不遗忘历史偏好 |
| 自动驾驶 | 车辆学习新路况同时保留已学驾驶模式 |
| 对话系统 | AI助手持续学习新知识而不遗忘基础对话能力 |
| 医疗诊断 | 模型学习新疾病特征同时保留对已有疾病的诊断能力 |

## 相关概念

- **增量学习（Incremental Learning）**：在不访问旧数据的情况下持续学习新任务
- **任务迁移（Task Transfer）**：知识在不同任务间的迁移与泛化
- **记忆巩固（Memory Consolidation）**：生物神经系统中长期记忆的形成机制
- **表示学习（Representation Learning）**：学习数据的有效表示
