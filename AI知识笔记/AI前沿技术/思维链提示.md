# 思维链提示（Chain-of-Thought Prompting）

## 一句话定义

> 思维链提示（Chain-of-Thought, CoT）是一种通过引导大语言模型逐步展示推理过程来提升复杂任务准确率的技术，其核心洞见是"思考过程本身可以引导出更好的答案"。

## 核心公式

### 标准CoT公式

$$P(\text{答案}) = P(\text{推理过程} \rightarrow \text{答案})$$

模型不仅生成答案，还生成中间的推理步骤：

$$\text{输出} = \underbrace{\text{推理链}}_{\text{Chain}} \rightarrow \underbrace{\text{最终答案}}_{\text{Thought}}$$

### 零样本CoT公式

$$\text{答案} = \text{LLM}(\text{问题} + \text{"Let's think step by step."})$$

### 自洽性CoT（Self-Consistency）

$$S = \arg\max_s \sum_{i=1}^{N} \mathbb{1}[s = \text{argmax}_{\text{answer}} \text{LLM}_i(\text{question})]$$

选择最一致的推理路径作为最终答案。

## 详细说明

### 1. 什么是思维链提示？

思维链提示是一种让大语言模型在给出最终答案之前，先展示其推理过程的提示技术。就像人类在解决复杂问题时会在纸上写下计算步骤一样，CoT引导模型"思考出声"。

### 2. CoT的类型

#### 2.1 标准思维链（Standard CoT）

需要人工设计带有推理示例的提示：

```
问题：小明有5个苹果，小红给了他3个，他又买了2个，现在有多少个？
推理：小明开始有5个苹果。小红给了他3个，所以是5+3=8个。然后他又买了2个，所以是8+2=10个。
答案：10个
```

#### 2.2 零样本思维链（Zero-shot CoT）

无需示例，只需一句触发语：

```
问题：小明有5个苹果，小红给了他3个，他又买了2个，现在有多少个？
"Let's think step by step."
```

#### 2.3 自动思维链（Auto-CoT）

自动生成带有推理的示例：

1. 从问题库中采样多样性问题
2. 使用零样本CoT生成候选推理
3. 验证推理正确性
4. 选择高质量示例构建提示

### 3. CoT与标准提示的对比

| 维度 | 标准提示 | 思维链提示 |
|------|----------|------------|
| **输出** | 仅答案 | 推理过程 + 答案 |
| **准确率** | 低 | 显著提升 |
| **适用场景** | 简单任务 | 复杂推理 |
| **计算成本** | 低 | 增加token输出 |
| **可解释性** | 低 | 高 |

### 4. CoT生效的原因

1. **注意力重分配**：将计算资源分配给推理过程而非直接拟合答案
2. **工作记忆扩展**：将中间结果外化为文本，扩展有效工作记忆
3. **错误定位**：便于发现推理中的错误步骤
4. **知识调用**：推理过程触发相关知识节点

## 代码示例

### 标准CoT实现

```python
"""
标准思维链提示实现
"""

from transformers import pipeline

def chain_of_thought_prompt(question, examples=None):
    """
    构建思维链提示

    Args:
        question: 待解决的问题
        examples: 可选的示例列表 [(问题, 推理, 答案), ...]
    """
    if examples is None:
        examples = []

    prompt_parts = []

    # 添加示例
    for q, reasoning, answer in examples:
        prompt_parts.append(f"问题：{q}")
        prompt_parts.append(f"推理：{reasoning}")
        prompt_parts.append(f"答案：{answer}\n")

    # 添加目标问题
    prompt_parts.append(f"问题：{question}")
    prompt_parts.append("推理：")

    return "\n".join(prompt_parts)

# 示例：数学问题
examples = [
    (
        "小明有10元钱，买了一本书花了3元，又买了一支笔花了2元，还剩多少？",
        "小明开始有10元。买书花了3元，10-3=7元。买笔花了2元，7-2=5元。",
        "5元"
    ),
    (
        "一箱苹果有24个，3个小朋友平分，每个人分到几个？还剩几个？",
        "总共24个苹果。24除以3等于8，所以每个人分到8个，没有剩余。",
        "8个，无剩余"
    )
]

question = "小红有15朵花，她把花平均分给3个朋友，每个朋友分到几朵？还剩几朵？"

prompt = chain_of_thought_prompt(question, examples)
print(prompt)

# 使用模型生成
generator = pipeline("text-generation", model="gpt2")
result = generator(prompt, max_new_tokens=100, do_sample=True, temperature=0.7)
print(result[0]['generated_text'])
```

### 零样本CoT实现

```python
"""
零样本思维链提示
无需示例，自动触发推理
"""

def zero_shot_cot(question, trigger_phrase="Let"s think step by step."):
    """
    零样本CoT提示

    Args:
        question: 问题
        trigger_phrase: 触发词
    """
    return f"{question}\n{trigger_phrase}"

def extract_answer(text):
    """
    从推理文本中提取最终答案
    """
    # 简单策略：找最后一个"答案是"或"所以"
    lines = text.split("\n")

    for line in reversed(lines):
        if "答案是" in line:
            return line.split("答案是")[-1].strip()
        if "所以" in line and "答案" in line:
            parts = line.split("所以")
            if len(parts) > 1:
                return parts[-1].strip()

    return text.split("\n")[-1].strip()

# 使用示例
question = "一个长方形的长是8厘米，宽是5厘米，面积是多少？"

prompt = zero_shot_cot(question)
print(f"提示：\n{prompt}\n")

# 生成推理
generator = pipeline("text-generation", model="gpt2-medium")
output = generator(prompt, max_new_tokens=150, do_sample=True)[0]["generated_text"]

print(f"推理过程：\n{output}")
print(f"\n提取的答案：{extract_answer(output)}")
```

### 自洽性CoT实现

```python
"""
自洽性思维链（Self-Consistency with CoT）
多次采样，选择最一致的答案
"""

from collections import Counter
from transformers import pipeline
import random

def self_consistency_cot(question, examples, n_samples=20):
    """
    自洽性CoT

    Args:
        question: 问题
        examples: CoT示例
        n_samples: 采样次数
    """
    prompt_template = chain_of_thought_prompt(question, examples)

    generator = pipeline(
        "text-generation",
        model="gpt2",
        do_sample=True,
        temperature=0.8,
        top_p=0.95
    )

    answers = []

    for _ in range(n_samples):
        output = generator(prompt_template, max_new_tokens=100)[0]["generated_text"]
        answer = extract_answer(output)
        answers.append(answer)

    # 统计最一致的答案
    answer_counts = Counter(answers)
    most_consistent = answer_counts.most_common(1)[0]

    return {
        "question": question,
        "most_common_answer": most_consistent[0],
        "count": most_consistent[1],
        "all_answers": dict(answer_counts),
        "consistency_rate": most_consistent[1] / n_samples
    }

# 测试
math_question = "一个水池有进水管和出水管。进水管每分钟进水10升，出水管每分钟出水7升。如果水池本来有100升水，10分钟后水池有多少升水？"

result = self_consistency_cot(math_question, examples)
print(f"问题：{result['question']}")
print(f"最一致答案：{result['most_common_answer']}")
print(f"一致性：{result['consistency_rate']:.1%}")
print(f"答案分布：{result['all_answers']}")
```
