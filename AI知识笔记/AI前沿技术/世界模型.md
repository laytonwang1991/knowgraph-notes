# 世界模型（World Models）

## 一句话定义

> 世界模型（World Models）是AI系统对物理世界环境进行内部表征与动态预测的机制，使智能体能够在心理层面对未来事件进行模拟和推理，从而以更高效、更安全的方式进行决策。

## 核心公式

### 世界模型基本架构

世界模型由三个核心组件构成：

$$M = (S, T, R)$$

其中：
- $S$: 表征函数（Perception/Encoder）
- $T$: 转移函数（Transition/Dynamics）
- $R$: 奖励函数（Reward/Predictor）

### 潜空间表示

将高维观察压缩到低维潜空间：

$$z_t = E(s_t)$$

其中 $z_t$ 是时刻 $t$ 的潜变量，$E$ 是编码器。

### 动态预测

在潜空间中进行时间序列预测：

$$\hat{z}_{t+k} = T(z_t, a_t, ..., a_{t+k-1})$$

或递归形式：

$$\hat{z}_{t+1} = T(z_t, a_t)$$

### Video Prediction Loss

$$\mathcal{L}_{video} = \mathbb{E}\left[\sum_{t=1}^{T} \|s_{t+1} - D(z_t, a_t)\|^2\right]$$

其中 $D$ 是解码器，$s_{t+1}$ 是真实下一帧。

## 详细说明

### 1. 什么是世界模型？

世界模型是智能体对环境的心智表征（Mental Model）。人类日常活动中，脑内会持续构建和更新关于世界的模型：

- **物理直觉**：知道物体受重力影响会下落
- **因果推断**：推动物体，物体按力的大小和方向移动
- **长期预测**：松手后物体将自由落体到地面

这种能力使人类无需穷举所有可能性就能做出合理决策。

### 2. 世界模型的核心组成

#### 2.1 表征模块（Perception）

负责将高维感知输入（图像、视频、传感器数据）压缩到低维潜空间：

$$z = E(s; \theta_e)$$

- 变分自编码器（VAE）
- 自编码器（Autoencoder）
- Diffusion-based 编码器

#### 2.2 动态模块（Dynamics/Transition）

学习环境在潜空间中的演化规律：

$$p(z_{t+1} | z_t, a_t)$$

关键能力：
- 时间一致性保持
- 长程依赖建模
- 不确定性量化

#### 2.3 奖励预测模块（Reward Prediction）

预测采取行动后的即时奖励：

$$\hat{r}_t = R(z_t, a_t; \theta_r)$$

import torch
import torch.nn as nn
from torch.distributions import Normal

class WorldModel(nn.Module):
    """
    简化世界模型
    """
    def __init__(self, obs_dim, act_dim, latent_dim=32, hidden_dim=256):
        super().__init__()

        self.latent_dim = latent_dim

        # 表征编码器：将观察压缩到潜空间
        self.encoder = nn.Sequential(
            nn.Linear(obs_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, latent_dim * 2)  # 均值 + 对数方差
        )

        # 解码器：从潜空间重建观察
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, obs_dim)
        )

        # 动态模型：预测下一潜状态
        self.dynamics = nn.Sequential(
            nn.Linear(latent_dim + act_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, latent_dim * 2)  # 预测分布参数
        )

        # 奖励预测器
        self.reward_predictor = nn.Sequential(
            nn.Linear(latent_dim + act_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )

    def encode(self, obs):
        """编码观察到潜空间"""
        params = self.encoder(obs)
        mean, log_std = params.chunk(2, dim=-1)
        std = torch.exp(log_std)
        dist = Normal(mean, std)
        z = dist.rsample()
        return z, dist

    def decode(self, z):
        """从潜空间解码"""
        return self.decoder(z)

    def predict_next(self, z, action):
        """预测下一潜状态"""
        z_action = torch.cat([z, action], dim=-1)
        params = self.dynamics(z_action)
        mean, log_std = params.chunk(2, dim=-1)
        std = torch.exp(log_std)
        dist = Normal(mean, std)
        next_z = dist.rsample()
        return next_z, dist

    def predict_reward(self, z, action):
        """预测奖励"""
        z_action = torch.cat([z, action], dim=-1)
        return self.reward_predictor(z_action)

    def forward(self, obs, action):
        """
        前向传播：编码 -> 动态预测 -> 奖励预测

        Args:
            obs: 当前观察 (batch, obs_dim)
            action: 当前动作 (batch, act_dim)
        """
        # 编码
        z, enc_dist = self.encode(obs)

        # 预测下一状态
        next_z, dyn_dist = self.predict_next(z, action)

        # 预测奖励
        reward = self.predict_reward(z, action)

        return {
            'z': z,
            'next_z': next_z,
            'reward': reward,
            'enc_dist': enc_dist,
            'dyn_dist': dyn_dist
        }


class WorldModelTrainer:
    """世界模型训练器"""

    def __init__(self, world_model, lr=1e-3):
        self.model = world_model
        self.optimizer = torch.optim.Adam(world_model.parameters(), lr=lr)

    def compute_loss(self, batch):
        """计算世界模型损失"""
        obs, actions, rewards, next_obs = batch

        # 编码当前观察
        z, enc_dist = self.model.encode(obs)

        # 预测下一状态
        next_z_pred, dyn_dist = self.model.predict_next(z, actions)

        # 预测奖励
        reward_pred = self.model.predict_reward(z, actions)

        # 重构损失
        obs_recon = self.model.decode(z)
        recon_loss = nn.functional.mse_loss(obs_recon, obs)

        # KL散度损失
        kl_loss = torch.distributions.kl_divergence(enc_dist, dyn_dist).sum(-1).mean()

        # 奖励预测损失
        reward_loss = nn.functional.mse_loss(reward_pred, rewards)

        # 总损失
        total_loss = recon_loss + 0.1 * kl_loss + reward_loss

        return {
            'loss': total_loss,
            'recon_loss': recon_loss,
            'kl_loss': kl_loss,
            'reward_loss': reward_loss
        }

    def train_step(self, batch):
        """单步训练"""
        self.optimizer.zero_grad()
        losses = self.compute_loss(batch)
        losses['loss'].backward()
        self.optimizer.step()
        return losses

import torch
import torch.nn as nn

class SimpleVideoPredictor(nn.Module):
    """
    简化视频预测器
    基于ConvLSTM架构
    """
    def __init__(self, in_channels=3, latent_dim=256):
        super().__init__()

        # 编码器：将视频帧序列编码
        self.encoder = nn.Sequential(
            nn.Conv3d(in_channels, 32, kernel_size=3, stride=2, padding=1),
            nn.ReLU(),
            nn.Conv3d(32, 64, kernel_size=3, stride=2, padding=1),
            nn.ReLU(),
            nn.Conv3d(64, 128, kernel_size=3, stride=2, padding=1),
            nn.ReLU(),
            nn.Conv3d(128, latent_dim, kernel_size=3, stride=2, padding=1),
        )

        # 解码器：从潜空间重建视频
        self.decoder = nn.Sequential(
            nn.ConvTranspose3d(latent_dim, 128, kernel_size=3, stride=2, padding=1, output_padding=1),
            nn.ReLU(),
            nn.ConvTranspose3d(128, 64, kernel_size=3, stride=2, padding=1, output_padding=1),
            nn.ReLU(),
            nn.ConvTranspose3d(64, 32, kernel_size=3, stride=2, padding=1, output_padding=1),
            nn.ReLU(),
            nn.ConvTranspose3d(32, in_channels, kernel_size=3, stride=2, padding=1, output_padding=1),
            nn.Tanh()  # 输出范围 [-1, 1]
        )

    def forward(self, video_sequence):
        """
        Args:
            video_sequence: (B, C, T, H, W) 批量视频序列
        Returns:
            predicted_sequence: (B, C, T, H, W) 预测的未来序列
        """
        batch_size, channels, time_steps, height, width = video_sequence.shape

        # 编码
        latent = self.encoder(video_sequence)

        # 解码（生成预测）
        prediction = self.decoder(latent)

        return prediction


class ConvLSTMCell(nn.Module):
    """
    ConvLSTM单元
    用于时空序列建模
    """
    def __init__(self, input_dim, hidden_dim, kernel_size):
        super().__init__()

        self.hidden_dim = hidden_dim
        self.kernel_size = kernel_size
        padding = kernel_size // 2

        # 门控机制
        self gates = nn.Conv2d(
            input_dim + hidden_dim,
            4 * hidden_dim,  # 输入门、遗忘门、输出门 + 细胞候选态
            kernel_size=kernel_size,
            padding=padding
        )

    def forward(self, x, prev_state):
        """前向传播"""
        h, c = prev_state

        # 拼接输入和隐藏状态
        combined = torch.cat([x, h], dim=1)

        # 计算门控
        gates = self.gates(combined)

        # 分割为四个部分
        i, f, o, g = gates.chunk(4, dim=1)

        # 门控操作
        i = torch.sigmoid(i)  # 输入门
        f = torch.sigmoid(f)  # 遗忘门
        o = torch.sigmoid(o)  # 输出门
        g = torch.tanh(g)     # 候选细胞态

        # 更新细胞状态和隐藏状态
        c_next = f * c + i * g
        h_next = o * torch.tanh(c_next)

        return h_next, c_next

    def init_hidden(self, batch_size, spatial_size):
        """初始化隐藏状态"""
        height, width = spatial_size
        device = next(self.parameters()).device
        return (
            torch.zeros(batch_size, self.hidden_dim, height, width, device=device),
            torch.zeros(batch_size, self.hidden_dim, height, width, device=device)
        )
