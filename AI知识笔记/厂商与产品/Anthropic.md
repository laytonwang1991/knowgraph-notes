# Anthropic

## 一句话定义

> Anthropic是一家由Dario Amodei等人创立的AI安全公司，以开发Claude AI助手和提出Constitutional AI等安全对齐技术著称。

## 基本信息

| 项目 | 内容 |
|------|------|
| 成立时间 | 2021年 |
| 总部 | 美国旧金山 |
| 创始人 | Dario Amodei、Daniela Amodei等 |
| 核心产品 | Claude AI助手系列 |
| 核心技术 | Constitutional AI、RLHF |
| 主要投资者 | Google、Salesforce、Spark Capital |
| 估值 | ~180亿美元（2024年） |
| 官方网站 | https://anthropic.com |

## 详细说明

### 1. 公司历史

**创立背景（2021）：**
- 由前OpenAI研究副总裁Dario Amodei创立
- 团队包括多名OpenAI安全和政策研究员
- 使命：构建安全、可控、受益于社会的AI系统

**发展历程：**
| 时间 | 事件 |
|------|------|
| 2021年 | Anthropic成立，获1.24亿美元融资 |
| 2022年 | 发布Claude（Slack集成版） |
| 2023年 | Claude 2发布，性能大幅提升 |
| 2024年 | Claude 3系列发布，包括Opus/Sonnet/Haiku |
| 2025年 | Claude 3.5系列，超越GPT-4o |

**投资背景：**
- Google投资约3亿美元
- Salesforce Ventures投资
- 总融资超过10亿美元

### 2. 核心产品线

**Claude系列：**
| 模型 | 发布时间 | 定位 | 主要特点 |
|------|----------|------|----------|
| Claude 1 | 2022 | 早期版本 | 长上下文 |
| Claude 2 | 2023 | 增强版 | 更长上下文、代码能力 |
| Claude 3 Opus | 2024 | 旗舰 | 最强推理、复杂任务 |
| Claude 3 Sonnet | 2024 | 均衡 | 性价比优 |
| Claude 3 Haiku | 2024 | 轻量 | 快速响应 |
| Claude 3.5 Sonnet | 2024 | 升级 | 性能超越前代 |
| Claude 3.5 Opus | 2025 | 新旗舰 | 增强推理 |

**产品形态：**
- **Claude.ai**：网页对话界面
- **Claude Pro**：付费订阅服务
- **Claude Team**：团队版
- **Claude Enterprise**：企业版
- **Claude API**：开发者接口
- **Claude for Work**：企业解决方案

### 3. 技术特点

**Constitutional AI（宪法AI）：**
- 提出AI行为应遵循的"宪法"原则
- 通过自我批评和RLHF结合
- 减少有害输出的同时保持有用性

**安全对齐技术：**
- 专注于AI安全和对齐研究
- 可解释性研究（Interpretability）
- Anthropic并重安全性和有用性

**模型特点：**
- 长上下文窗口（20万tokens）
- 强大的代码能力
- 优秀的推理能力
- 较少的幻觉问题
