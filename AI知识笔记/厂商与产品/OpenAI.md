# OpenAI

## 一句话定义

> OpenAI是一家由Sam Altman创立的人工智能研究机构，通过开发GPT系列、DALL-E、ChatGPT等产品，引领了生成式AI革命。

## 基本信息

| 项目 | 内容 |
|------|------|
| 成立时间 | 2015年12月 |
| 总部 | 美国旧金山 |
| 创始人 | Sam Altman、Elon Musk等 |
| 类型 | 人工智能研究实验室 |
| 核心产品 | GPT-4、ChatGPT、DALL-E、Sora |
| 估值 | ~2900亿美元（2024年） |
| 官方网站 | https://openai.com |

## 详细说明

### 1. 公司历史

**创立初期（2015-2018）：**
- 2015年12月成立，定位为非营利AI研究组织
- 使命：确保通用人工智能（AGI）造福全人类
- 早期投资者：Elon Musk、Peter Thiel、Sam Altman等
- 2018年Elon Musk离开董事会

**转型与融资（2019-2022）：**
- 2019年成立营利性子公司OpenAI LP
- 获得微软100亿美元投资
- 发布GPT-2（2019）、GPT-3（2020）
- 推出API服务和ChatGPT（2022年11月）

**快速发展期（2023-至今）：**
- GPT-4发布（2023年3月）
- ChatGPT用户破亿，成为史上增长最快的应用
- 推出GPT-4V、DALL-E 3、Sora等
- 估值达2900亿美元

### 2. 核心产品线

**GPT系列（大语言模型）：**
| 模型 | 发布时间 | 参数量 | 主要特点 |
|------|----------|--------|----------|
| GPT-1 | 2018 | 1.17亿 | Transformer解码器 |
| GPT-2 | 2019 | 15亿 | 零样本学习 |
| GPT-3 | 2020 | 1750亿 | 小样本学习 |
| GPT-3.5 | 2022 | - | ChatGPT底层模型 |
| GPT-4 | 2023 | 未公开 | 多模态、推理能力 |
| GPT-4o | 2024 | - | 实时语音/视频交互 |
| GPT-4.5 | 2025 | - | 增强推理 |

**其他产品：**
- **ChatGPT**：基于GPT的对话AI助手
- **DALL-E**：文本生成图像模型
- **Sora**：文本生成视频模型
- **Whisper**：语音识别模型
- **OpenAI API**：开发者接口服务

### 3. 技术特点

- **Scaling Law**：重视模型规模扩展规律
- **RLHF**：人类反馈强化学习对齐
- **安全研究**：AI安全与对齐研究领先
- **生态建设**：完善的API和开发者生态

### 4. 商业模式

- ChatGPT Plus订阅（20美元/月）
- API调用按token计费
- 企业版ChatGPT Enterprise
- Azure OpenAI服务合作
