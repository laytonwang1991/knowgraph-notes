# 状态空间模型（SSM）

## 一句话定义

状态空间模型（State Space Model, SSM）是一种通过潜在状态变量描述序列演化的概率模型，通过**选择性状态空间机制**在 O(1) 推理复杂度下实现长距离依赖建模，是Mamba等新型架构的核心。

## 核心公式

### 连续时间状态空间表示

$$
h(t) = A h(t) + B x(t) \quad \text{（状态更新）}
$$

$$
y(t) = C h(t) + D x(t) \quad \text{（输出）}
$$

其中：
- $h(t)$：隐藏状态向量（维度 N）
- $x(t)$：输入信号
- $y(t)$：输出信号
- $A, B, C, D$：系统矩阵（学习参数）

### 离散化形式（用于实际计算）

$$
h_k = \bar{A} h_{k-1} + \bar{B} x_k
$$

$$
y_k = C h_k
$$

其中 $\bar{A} = e^{\Delta A}$，$\bar{B} = (\Delta A)^{-1}(e^{\Delta A} - I) \Delta B$

### Mamba选择性状态空间核心公式

$$
h_k = \bar{A}_k h_{k-1} + \bar{B}_k x_k
$$

$$
y_k = C_k h_k
$$

关键区别：矩阵 $A_k, B_k, C_k$ 由输入 $x_k$ **选择性**生成

## 详细说明

### 1. SSM与RNN的对比

| 特性 | 传统RNN | SSM |
|------|---------|-----|
| 状态维度 | 固定 | 可选择性压缩 |
| 参数更新 | 门控机制 | 连续时间系统 |
| 梯度流 | 易消失 | 通过A矩阵稳定 |
| 并行化 | 困难 | 完全可并行 |
| 推理效率 | O(1) | O(1) |

### 2. 结构化状态空间序列模型（S4）

- **HiPPO矩阵**：用于高效逼近长时间历史

$$
A_{ij} = \begin{cases}
1 & i = j \\
0.5 & i < j \\
1 & i > j
\end{cases}
$$

- **对角化状态矩阵**：将 N×N 矩阵转化为 N 个标量，降低计算复杂度

### 3. Mamba选择性机制

Mamba的核心创新在于**输入依赖的参数生成**：

```python
class MambaBlock(nn.Module):
    def __init__(self, d_model, d_state=16, d_conv=4, expand=2):
        super().__init__()
        self.d_model = d_model
        self.d_state = d_state
        self.expand = expand

        # 输入投影
        self.in_proj = nn.Linear(d_model, self.expand * d_model * 2, bias=False)

        # 卷积
        self.conv = nn.Conv1d(expand*d_model, expand*d_model,
                              kernel_size=d_conv, padding=d_conv-1)

        # SSM参数投影
        self.x_proj = nn.Linear(expand*d_model, d_state * 2 + 1, bias=False)
        self.dt_proj = nn.Linear(1, expand*d_model, bias=True)

        # 输出投影
        self.out_proj = nn.Linear(expand*d_model, d_model, bias=False)

        # 可学习的A矩阵（负值确保稳定）
        self.A = nn.Parameter(torch.randn(expand*d_model, d_state))

    def forward(self, x):
        B, L, D = x.shape

        # 投影并分割为 halves 和 gate
        xz = self.in_proj(x)
        x_inner, gate = xz.chunk(2, dim=-1)

        # 因果卷积
        x_conv = self.conv(x_inner.transpose(1,2)).transpose(1,2)[:,:L,:]
        x_ssm = F.silu(x_conv) * x_inner

        # 生成SSM参数（选择性机制）
        x_dbl = self.x_proj(x_ssm)
        dt, B_param, C_param = x_dbl.split([1, self.d_state, self.d_state], dim=-1)

        # 软plus确保正值
        dt = F.softplus(self.dt_proj(dt))

        # 负A矩阵（稳定特征值）
        A = -torch.exp(self.A.float())

        # 离散化并扫描
        y = self.ssm_selective_scan(x_ssm, dt, A, B_param, C_param)

        # 门控输出
        y = y * F.silu(gate)
        return self.out_proj(y)

    def ssm_selective_scan(self, x, dt, A, B, C):
        """选择性扫描实现"""
        B_dim = B.shape[-1]

        # 扩展维度用于批量计算
        dt = dt.unsqueeze(-1)
        B = B.unsqueeze(-1)
        C = C.unsqueeze(-2)

        # 计算离散状态转移
        dA = torch.exp(dt * A)
        dB = dt * B

        # 扫描递归
        h = torch.zeros_like(x)
        ys = []
        for i in range(x.shape[1]):
            h = dA[:,i] * h + dB[:,i] * x[:,i:i+1]
            y = (h * C[:,i:i+1]).sum(dim=-1, keepdim=True)
            ys.append(y)

        return torch.cat(ys, dim=1)
```

### 4. 选择性 vs 非选择性

```
非选择性SSM（如S4）：
  A, B, C = 固定参数（与输入无关）
  → 高效但缺乏输入适应性

选择性SSM（如Mamba）：
  A, B, C = 输入依赖生成
  → 灵活选择记住/遗忘什么信息
```

### 5. 计算复杂度分析

| 操作 | Transformer | Mamba |
|------|-------------|-------|
| 推理 | O(L²) 注意力 | O(L) 线性 |
| 训练 | O(L²) | O(L log L) |
| 内存 | O(L²) KV缓存 | O(L) 无缓存 |
| 长序列 | 受限于显存 | 天然支持 |

## 代码示例：简化SSM实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class MinimalSSM(nn.Module):
    """简化版SSM：展示核心原理"""

    def __init__(self, d_model, d_state=8):
        super().__init__()
        self.d_model = d_model
        self.d_state = d_state

        # 输入、输出投影
        self.input_proj = nn.Linear(d_model, d_state * 2, bias=False)
        self.output_proj = nn.Linear(d_state, d_model, bias=False)

        # 状态矩阵（HiPPO初始化）
        self.A = nn.Parameter(self._init_A())

    def _init_A(self):
        """HiPPO-LegS矩阵初始化"""
        A = torch.zeros(self.d_state, self.d_state)
        for i in range(self.d_state):
            for j in range(self.d_state):
                if i >= j:
                    A[i, j] = 1.0 / (i + 1)
        return -A  # 负值确保稳定

    def forward(self, x):
        """
        x: (batch, seq_len, d_model)
        """
        B, L, D = x.shape

        # 生成动态B和C
        x_flat = x.view(B*L, -1)
        BC = self.input_proj(x_flat)  # (B*L, d_state*2)
        B_vec = BC[:, :self.d_state]
        C_vec = BC[:, self.d_state:]

        # 重塑
        B_vec = B_vec.view(B, L, self.d_state)
        C_vec = C_vec.view(B, L, self.d_state)

        # SSM扫描
        h = torch.zeros(B, self.d_state, device=x.device)
        outputs = []

        for t in range(L):
            # 状态更新
            h = h + torch.einsum('bn,bn->b', x[:,t,:], B_vec[:,t,:]) * self.A.unsqueeze(0)

            # 输出
            y_t = torch.einsum('bn,bn->b', h, C_vec[:,t,:])
            outputs.append(y_t)

        y = torch.stack(outputs, dim=1)  # (B, L, d_state)
        return self.output_proj(y)

# 使用示例
model = MinimalSSM(d_model=512, d_state=16)
x = torch.randn(2, 100, 512)  # (batch, seq_len, d_model)
y = model(x)
print(f"输入形状: {x.shape}")
print(f"输出形状: {y.shape}")
```
