# 第一次AI寒冬

## 一句话定义

> 第一次AI寒冬是1970年代由于AI研究未能达到早期期望而导致资金枯竭、发展停滞的时期。

## 详细说明

### 1. 背景

**AI的早期乐观预期（1956-1970）：**
- 1956年：达特茅斯会议，AI诞生
- 1960年代：专家系统兴起
- 普遍预期：10年内AI将达到人类智能水平

### 2. 寒冬成因

| 原因 | 说明 |
|------|------|
| 计算能力限制 | 计算机无法支持复杂AI |
| 复杂度爆炸 | 现实世界问题比预期复杂得多 |
| 知识获取瓶颈 | 专家系统需要大量人工知识工程 |
| 批评声音 | Lighthill报告（1973）批评AI缺乏实用性 |

### 3. Lighthill Report

1973年，英国科学家 James Lighthill 发布报告批评AI：
- AI的成就被夸大
- 通用问题求解器（GPPS）没有实用价值
- 机器人研究进展有限

**结果：** 英国政府大幅削减AI研究经费。

### 4. 复苏

**专家系统的崛起（1980年代）：**
- DENRAL、MYCIN等医疗专家系统
- 日本第五代计算机计划
- 专家系统商业化

## 教训与意义

| 教训 | 启示 |
|------|------|
| 过度承诺 | 避免夸大技术能力 |
| 计算瓶颈 | 硬件发展至关重要 |
| 垂直场景 | 先解决具体问题而非通用AI |
