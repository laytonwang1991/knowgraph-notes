# 感知机时代

## 一句话定义

> 感知机时代是1957年Rosenblatt提出感知机模型后，AI领域经历的第一波研究热潮，其基于神经网络思想，试图让机器模拟生物神经元的学习能力。

## 核心公式

**感知机的输出函数：**

$$
f(x) = \text{sign}\left(\sum_{i=1}^{n} w_i x_i + b\right) = \text{sign}(w \cdot x + b)
$$

**感知机的学习规则：**

$$
w_i^{(new)} = w_i^{(old)} + \eta (y - \hat{y}) x_i
$$

其中 $\eta$ 为学习率，$y$ 为真实标签，$\hat{y}$ 为预测标签。

## 详细说明

### 1. 历史背景

**时代背景：**
- 1956年达特茅斯会议正式宣告人工智能诞生
- 神经科学发现生物神经元的工作原理
- 计算机技术初步发展，为神经网络研究提供可能

**关键人物：**
| 人物 | 贡献 |
|------|------|
| Frank Rosenblatt | 感知机模型创始人，1957年提出 |
| Marvin Minsky | 神经网络先驱，后转向批判 |
| Warren McCulloch | 与Pitts合作建立神经元数学模型 |
| Walter Pitts | 首个神经网络数学模型 |

### 2. 感知机模型

**基本原理：**
- 单层神经网络，仅包含输入层和输出层
- 通过加权求和与激活函数进行分类
- 采用监督学习，通过迭代调整权重

**核心特点：**
1. **二分类器**：只能处理线性可分问题
2. **收敛性**：对于线性可分数据，算法保证收敛
3. **简单性**：结构简单，易于理解和实现

### 3. 代表性研究

**Rosenblatt的感知机（1957）：**
- 为知机（Mark 1）：第一个物理实现的感知机
- 能够学会识别简单的图像模式
- 引发学术界对神经网络的广泛兴趣

**Widrow的Adaline（1960）：**
- 自适应线性单元（Adaptive Linear Neuron）
- 最小均方误差（LMS）学习规则
- 成功应用于信号处理和控制系统

### 4. 第一次AI寒冬的起因

**感知机的局限性被发现：**
| 问题 | 说明 |
|------|------|
| 线性不可分 | 无法解决XOR问题 |
| 单层结构限制 | 无法表达复杂的非线性关系 |
| 过拟合风险 | 泛化能力有限 |

**Minsky与Papert的批判（1969）：**
- 出版《感知机：计算几何简介》
- 严格证明单层感知机的局限性
- 指出多层网络或许可行但缺乏有效学习算法

**深远影响：**
- 政府和企业大幅削减AI研究经费
- 神经网络研究进入长达十余年的低潮期
- 符号主义AI主导学术界
