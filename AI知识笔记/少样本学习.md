# 少样本学习

## 一句话定义

少样本学习是一类让模型能够从极少量（通常1-5个）标注样本中快速学习和泛化的技术，旨在模拟人类从少量示例中抽象新概念的能力。

## 核心公式

### 1. N-way K-shot问题定义

给定：
- **Support Set**: $S = \{(x_i, y_i)\}_{i=1}^{N \times K}$，包含N个类别，每类K个样本
- **Query Set**: $Q = \{(x_j, y_j)\}_{j=1}^{M}$，用于评估

目标是：根据Support Set中的少量样本，对Query Set中的样本进行分类。

### 2. MAML (Model-Agnostic Meta-Learning)

MAML寻求最优参数$\theta$使其能快速适应新任务：

$$\theta^* = \arg\min_\theta \sum_{\mathcal{T}_i} \mathcal{L}_{\mathcal{T}_i}(f_{\theta - \alpha \nabla_\theta \mathcal{L}_{\mathcal{T}_i}(f_\theta)})$$

内层更新：
$$\theta'_i = \theta - \alpha \nabla_\theta \mathcal{L}_{\mathcal{T}_i}(f_\theta)$$

外层更新：
$$\theta \leftarrow \theta - \beta \nabla_\theta \sum_{\mathcal{T}_i} \mathcal{L}_{\mathcal{T}_i}(f_{\theta'_i})$$

### 3. Prototypical Networks

每个类别的原型表示：
$$c_k = \frac{1}{|S_k|} \sum_{(x_i, y_i) \in S_k} f_\phi(x_i)$$

其中$f_\phi$是嵌入函数。分类通过计算query到各类原型的距离：

$$p(y=k|x) = \frac{\exp(-d(f_\phi(x), c_k))}{\sum_{k'} \exp(-d(f_\phi(x), c_{k'}))}$$

### 4. 对比学习InfoNCE损失

$$L = -\log \frac{\exp(sim(z_i, z_j)/\tau)}{\sum_{k=1}^{2N} \mathbb{1}_{[k \neq i]} \exp(sim(z_i, z_k)/\tau)}$$

其中$\tau$是温度参数，$sim$是余弦相似度。

### 5. Matching Networks

给定支持集S，定义注意力核$a$：

$$a(\hat{x}, S) = \sum_{k=1}^{|S|} a_{att}(\hat{x}, x_k) y_k$$

其中 $a_{att}(\hat{x}, x_k) = \text{softmax}(cosine(f(\hat{x}), g(x_k)))$

预测：
$$\hat{y} = \sum_{i=1}^{|S|} a(\hat{x}, x_i) y_i$$

## 详细说明

### 一、少样本学习问题设定

#### 1. 标准设定
- **N-way K-shot**: N个类别，每类K个样本（K通常为1-5）
- **Support Set**: 提供少量样本给模型"记忆"
- **Query Set**: 模型需要分类的新样本
- **Episode**: 一次训练/测试的单元，包含Support和Query

#### 2. 任务变体
- **Zero-shot Learning**: 完全无标注样本，通过属性描述识别
- **One-shot Learning**: 每类仅1个样本
- **Few-shot Learning**: 每类少量样本（通常1-5）
- **Domain shift**: 训练域与测试域分布不同

### 二、MAML (Model-Agnostic Meta-Learning)

#### 1. 核心思想
学习一个模型的初始化参数，使其能通过少量梯度步快速适应新任务。

#### 2. 算法流程
```
for each epoch:
    sample batch of tasks {T_i}
    for each task T_i:
        compute adapted parameters theta'_i using K support samples
    update theta by minimizing sum of loss on query samples
    across all tasks using theta'_i
```

#### 3. 优点
- 模型无关，可应用于任意可微分模型
- 学习到的初始化使模型高度适应新任务
- 对比Reptile等方法通常泛化更好

#### 4. 缺点
- 内外循环计算开销大
- 对新任务需要多步梯度更新
- 难以处理离散、无梯度任务

### 三、Prototypical Networks

#### 1. 核心思想
将样本映射到嵌入空间，每个类别的原型是其在支持集中的均值表示。

#### 2. 距离度量
- 欧氏距离：$d(x, y) = \|x - y\|^2$
- 余弦距离：$d(x, y) = 1 - \cos(x, y)$
- 可学习度量层

#### 3. 优缺点
- **优点**: 简单高效，训练稳定
- **缺点**: 假设类内分布单一，难以处理复杂决策边界

#### 4. 改进版本
- **Semi-prototypical Networks**: 结合原型和样本级别特征
- **Transductive Prototypical**: 利用query集统计信息

### 四、匹配网络 (Matching Networks)

#### 1. 独特设计
- 注意力机制定义样本间相似性
- 支持集样本作为"记忆"参与预测
- 完全可微的最近邻分类器

#### 2. 训练策略
- 使用Episode训练模拟测试场景
- 训练/测试任务同分布

### 五、对比学习

#### 1. 核心机制
通过拉近相似样本（正样本对）、推远不相似样本（负样本对）学习特征表示。

#### 2. 典型方法对比

| 方法 | 正样本构建 | 负样本 | 特点 |
|------|-----------|--------|------|
| **SimCLR** | 同一图像不同增强 | Batch内其他样本 | 简单有效 |
| **MoCo** | 同一图像不同增强 | 动态负样本队列 | 更大负样本库 |
| **BYOL** | 同一图像不同增强 | 无需负样本 | Bootstrap方式 |
| **SwAV** | 同一图像不同增强 | 无需显式负样本 | 聚类约束 |

#### 3. 在少样本学习中的应用
- 预训练阶段学习通用特征表示
- 微调阶段结合分类器适配新类别

### 六、数据增强在少样本学习中的角色

#### 1. 弱增强
- 随机翻转、裁剪、旋转
- 颜色抖动
- 适用于样本极少的场景

#### 2. 强增强
- **Cutout**: 随机遮挡图像区域
- **MixUp**: 样本混合
- **CutMix**: 剪切粘贴混合
- **AutoAugment/RandAugment**: 学习/随机增强策略

#### 3. 类级别增强
- 特征空间插值
- 合成新样本（GAN/扩散模型）

## 代码示例

### Python - MAML实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class MAML(nn.Module):
    def __init__(self, model, inner_lr=0.01, outer_lr=0.001):
        super().__init__()
        self.model = model
        self.inner_lr = inner_lr
        self.outer_lr = outer_lr

    def forward(self, x_spt, y_spt, x_qry, y_qry, num_inner_steps=5):
        """
        x_spt: [task_size, support_size, features...]
        y_spt: [task_size, support_size]
        x_qry: [task_size, query_size, features...]
        y_qry: [task_size, query_size]
        """
        task_size = x_spt.size(0)
        query_size = x_qry.size(1)

        # 存储每个任务的元梯度
        qry_losses = []

        for task_idx in range(task_size):
            # 克隆模型用于内层更新
            model_copy = self.clone_model()

            # 内层循环：在支持集上多步梯度下降
            for _ in range(num_inner_steps):
                spt_pred = model_copy(x_spt[task_idx])
                inner_loss = F.cross_entropy(spt_pred, y_spt[task_idx])
                model_copy.update(inner_loss, self.inner_lr)

            # 外层：计算查询集损失
            qry_pred = model_copy(x_qry[task_idx])
            qry_loss = F.cross_entropy(qry_pred, y_qry[task_idx])
            qry_losses.append(qry_loss)

        # 元更新
        meta_loss = torch.stack(qry_losses).mean()
        self.model.zero_grad()
        meta_loss.backward()
        self.meta_update(self.outer_lr)

        return meta_loss.item()

    def clone_model(self):
        """创建模型副本用于内层更新"""
        clone = type(self.model)()
        clone.load_state_dict(self.model.state_dict())
        clone.train()
        return clone

    def meta_update(self, lr):
        """执行元更新"""
        with torch.no_grad():
            for param, meta_param in zip(self.model.parameters(), self.model.parameters()):
                if param.grad is not None:
                    param -= lr * param.grad
```

### Python - Prototypical Networks实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from typing import Tuple

class PrototypicalNetwork(nn.Module):
    def __init__(self, encoder: nn.Module, hidden_dim: int = 64):
        super().__init__()
        self.encoder = encoder
        self.hidden_dim = hidden_dim

    def forward(self, support_x, support_y, query_x, query_y,
                n_support: int, n_query: int) -> Tuple[float, float]:
        """
        support_x: [batch_size * n_support, *input_shape]
        query_x: [batch_size * n_query, *input_shape]
        """
        # 计算嵌入
        support_emb = self.encoder(support_x)  # [B * N*K, embedding_dim]
        query_emb = self.encoder(query_x)      # [B * N*K, embedding_dim]

        # 重塑为 [batch_size, n_classes * k, embedding_dim]
        # 然后计算每类的原型
        classes = torch.unique(support_y)
        n_classes = len(classes)

        # 计算每类的原型：类内样本嵌入的均值
        prototypes = []
        for c in classes:
            mask = (support_y == c)
            class_emb = support_emb[mask].mean(dim=0)
            prototypes.append(class_emb)

        prototypes = torch.stack(prototypes)  # [n_classes, embedding_dim]

        # 计算query到各原型的距离
        query_emb = query_emb.unsqueeze(1)  # [B*N*Q, 1, emb_dim]
        prototypes = prototypes.unsqueeze(0)  # [1, n_classes, emb_dim]
        dists = torch.norm(query_emb - prototypes, dim=2)

        # 计算损失和准确率
        log_probs = F.log_softmax(-dists, dim=1)  # 负距离作为相似度
        loss = F.nll_loss(log_probs, query_y)

        preds = torch.argmax(log_probs, dim=1)
        accuracy = (preds == query_y).float().mean()

        return loss, accuracy

    def get_prototypes(self, support_x, support_y):
        """提取类别原型（用于推理）"""
        with torch.no_grad():
            support_emb = self.encoder(support_x)
            classes = torch.unique(support_y)
            prototypes = []
            for c in classes:
                mask = (support_y == c)
                class_emb = support_emb[mask].mean(dim=0)
                prototypes.append(class_emb)
            return torch.stack(prototypes), classes
```

### Python - 对比学习预训练 (SimCLR风格)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torchvision import transforms

class SimCLR(nn.Module):
    def __init__(self, base_encoder, projection_dim=128, hidden_dim=512):
        super().__init__()
        self.encoder = base_encoder
        self.projection_head = nn.Sequential(
            nn.Linear(self.encoder.output_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, projection_dim)
        )

    def info_nce_loss(self, z_i, z_j, temperature=0.5):
        """
        z_i, z_j: [batch_size, projection_dim]
        """
        batch_size = z_i.size(0)

        # 拼接正样本对
        z = torch.cat([z_i, z_j], dim=0)  # [2*B, proj_dim]

        # 计算相似度矩阵
        sim = torch.mm(z, z.T) / temperature

        # 创建掩码：正样本对位置
        sim_i_j = torch.diag(sim, batch_size)  # 正样本对 i-j
        sim_j_i = torch.diag(sim, -batch_size)  # 正样本对 j-i
        positives = torch.cat([sim_i_j, sim_j_i], dim=0)

        # 排除自相似度
        logits = positives / (sim.sum(dim=1) - 1 + 1e-8)

        # 交叉熵损失
        labels = torch.zeros(2 * batch_size).to(z.device).long()
        loss = F.cross_entropy(logits.unsqueeze(0), labels)

        return loss

    def forward(self, x_i, x_j):
        # 编码
        h_i = self.encoder(x_i)
        h_j = self.encoder(x_j)

        # 投影
        z_i = self.projection_head(h_i)
        z_j = self.projection_head(h_j)

        # 归一化
        z_i = F.normalize(z_i, dim=1)
        z_j = F.normalize(z_j, dim=1)

        loss = self.info_nce_loss(z_i, z_j)
        return loss

# 数据增强
class ContrastiveTransform:
    def __init__(self, size=32):
        self.train_transform = transforms.Compose([
            transforms.RandomResizedCrop(size),
            transforms.RandomHorizontalFlip(),
            transforms.ColorJitter(0.4, 0.4, 0.4, 0.1),
            transforms.RandomGrayscale(p=0.2),
            transforms.ToTensor(),
            transforms.Normalize(mean=[0.485, 0.456, 0.406],
                               std=[0.229, 0.224, 0.225])
        ])

    def __call__(self, x):
        return self.train_transform(x), self.train_transform(x)
```
